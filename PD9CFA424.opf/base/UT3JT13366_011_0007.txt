སེམས་གཏད་དོ། །དེ་ལས་སེམས་གཡེངས་པར་གྱུར་ན་ཐིག་ལེའི་འོད་ཟེར་དེ་འོག་ནས་འོག་ཏུ་འཕྲོས་པས་བདག་གི་སྤྱི་གཙུག་ནས་མར་ཞུགས་ཤིང་ལུས་ཐམས་ཅད་ལ་ཁྱབ་པས་ལུས་ཐམས་ཅད་ལ་བདེ་བ་སྐྱེས་པར་བསམ་ཞིང༌། དེས་སེམས་མི་གཡེངས་ན་ཡང་སྔ་མའི་ཐིག་ལེ་ལ་སེམས་གཏད་དོ། །དེ་ལ་བརྟན་པ་ཐོབ་པ་ལ་བདེ་བ་དང་མི་རྟོག་པ་གཉིས་སུ་མེད་པ་རྟོགས་པར་འགྱུར་ཞིང་སངས་རྒྱས་ཉིད་དུའང་འགྱུར་རོ། །དེ་ནི་བརྟན་པར་བྱ་བའོ། །དེ་ནས་སྤྲོ་བ་ནི། གཞན་ཡང་དེ་ལྟ་བུའི་བརྟན་པ་ཐོབ་པའི་ཐིག་ལེ་སྣ་ཚོགས་རྡོ་རྗེ་ཅན་དེ་ཉིད་མིག་དང༌། རྣ་བ་དང༌། སྣ་དང༌། ལྕེ་དང༌། མཚན་མའི་རྩེ་མོའི་མཐར་ཐུག་པར་སྦྱར་ཞིང་འཁོར་ལོ་བཞིའི་ཡི་གེ་རྣམས་ཀྱི་ཐིག་ལེ་ལ་སྦྱར་ཏེ། ཐོག་མར་སྙིང་ག་ཆོས་ཀྱི་འཁོར་ལོ་ལ་བསླབ། དེ་ནས་སྤྱི་བོ་བདེ་བ་ཆེན་པོའི་འཁོར་ལོ་ལ་བསླབ། དེ་ནས་མགྲིན་པ་ལོངས་སྤྱོད་ཀྱི་འཁོར་ལོ་ལ་བསླབ། དེ་ནས་ལྟེ་བ་སྤྲུལ་པའི་འཁོར་ལོ་ལ་བསླབས་ན་གཞུང་ནས་འབྱུང་བ་བཞིན་ཡོན་ཏན་གྱི་མཆོག་ཐམས་ཅད་སྐྱེ་བར་འགྱུར་རོ། །རྣམ་པ་སྣ་ཚོགས་པའི་སྣ་ཚོགས་རྡོ་རྗེའི་རིམ་པའོ། །ཡང་ས་བོན་མེད་པའི་སྣ་ཚོགས་རྡོ་རྗེའི་རིམ་པ་ནི་སྔོན་དུ་འགྲོ་བ་ཐམས་ཅད་སྔ་མ་བཞིན་བྱས་ལ། སྣ་གཉིས་ཀྱི་རླུང་གང་ནས་རྒྱུ་བ་བརྟགས་ཏེ་གཡས་ནས་རྒྱུ་ན་ཕྱིར་རྒྱུ་བའི་དུས་སུ་མཛོད་སྤུར་བསྟིམ་ལ་ཉི་མའི་དཀྱིལ་འཁོར་ཞིག་བསམ། ཡང་གཡོན་ནས་རྒྱུ་བར་བསམ་ལ། དེའི་སྟེང་དུ་ཟླ་བའི་དཀྱིལ་འཁོར་ཞིག་བསམ། སྤྱི་རླུང་གཡོན་ནས་རྒྱུ་ན་སྔོན་ལ་ཟླ་བའི་དཀྱིལ་འཁོར་བཀོད་ལ། དེ་ནས་ཉི་མའི་དཀྱིལ་འཁོར་བསམ

༑ གང་ལྟར་ཡང་རླུང་གསུམ་པ་ཕྱིར་འགྲོ་བའི་ཚེ་དེ་གཉིས་ཀྱི་སྟེང་དུ་སྐར་མའི་གཟུགས་ལྔ་བསམས་ཏེ། དབུས་སུ་སྔོན་པོ། ཤར་དུ་དཀར་པོ། ལྷོར་སེར་པོ ༑ ནུབ་ཏུ་དམར་པོ། བྱང་དུ་ལྗང་བའི་ཐིག་ལེ་དེ་བཞིན་གཤེགས་པ་ལྔའི་རང་བཞིན་དུ་བསམ་ལ། དེ་ལ་ཡང་ནས་ཡང་དུ་སེམས་གཏད་ནས་བསྒོམས་པས་ཇི་ཞིག་ལྟར་ན་མི་དམིགས་པའི་རང་བཞིན་གྱི་བསམ་གཏན་ཐོབ་པར་འགྱུར་ཏེ། དེའང་དཔེར་ན་ལག་པའི་རི་མོ་ལྟ་བུ་ཞིག་ལ་མིག་གཏད་ཅིང་མི་འཛུམས་པར་བརྟགས་པས། ཇི་ཞིག་ལྟར་ན་མིག་སྦྲིད་ནས་རི་མོ་དེ་རྣམས་མི་དམིགས་པར་གཏད་ཡལ་དུ་འགྲོ་བ་བཞིན་ནོ། །དེ་བཞིན་དུ་ཆོས་ཐམས་ཅད་སྣང་ལ་རང་བཞིན་མེད་པར་རྟོགས་པར་འགྱུར་ཏེ། ( ¡¶ ཞ ¡· + དེ་ ) བརྟན་པས་སངས་རྒྱས་ཉིད་དུ་འགྲུབ་པར་འགྱུར་རོ། །ས་བོན་མེད་པའི་སྣ་ཚོགས་རྡོ་རྗེའི་རིམ་པ་བསྟན་ཟིན་ཏེ། རིམ་པ་གཉིས་པ་སྣ་ཚོགས་རྡོ་རྗེའི་རིམ་པ་བསྟན་ཟིན་ཏོ།། །།ད་ནི་དབང་པོ་འབྲིང་པོ་བྱིན་རླབས་ཀྱི་དུས་སུ་བདེ་བ་ཆེན་པོའི་ཏིང་ངེ་འཛིན་སྐྱེས་པ་རྣམས་ལ་རིམ་པ་གསུམ་པ་ནོར་བུ་འགེངས་པའི་རིམ་པ་བརྗོད་པར་ ¡Ã བྱའོ། ། ( ¡¶ ཞ ¡· བྱ་སྟེ། ) དེ་ལ་བཞི་སྟེ། དམ་ཚིག་གི་ཕྱག་རྒྱ་དང༌། ལས་ཀྱི་ཕྱག་རྒྱ་དང༌། ཆོས་ཀྱི་ཕྱག་རྒྱ་དང༌། ཕྱག་རྒྱ་ཆེན་པོའོ། །དང་པོ་དམ་ཚིག་གི་ཕྱག་རྒྱ་ནི། སྔོན་དུ་འགྲོ་བ་ཐམས་ཅད་སྔ་མ་བཞིན་བྱས་ནས། འོད་ཀྱྀ་ཕྱག་རྒྱ་ལ་བརྟེན་ནས་དེ་ཁོ་ན་ཉིད་ཀྱི་མཆོད་པ་བྱ་བ་བཞིན་དུ་དེ་ལ་གནས་ལྔར་བྱིན་གྱིས་བརླབ་པར་བྱ་སྟེ། དེ་ལ་གནས་ལྔ་ནི་གསང་གནས་དང༌། ལྟེ་བ་དང༌། སྙིང་ག་དང་ ༑ མགྲིན་པ་དང༌། དཔྲལ་བ་རྣམས་ཡིན་ཏེ། དེ་དག་བྱིན་གྱིས་བརླབ་པ་ནི། ཧཱུྃ་ལས་རྡོ་རྗེ། བྷྱྃ་ལས་རྡོ་རྗེའི་ནོར་བུ་དམར་པོ། བྷྱྃ་ལས་རྡོ་རྗེའི་རྩེ་མོ་སེར་པོ་བུ་གར་བྷྱྃ་གིས་མཚན་པ་བསྒོམ། ཡབ་ཀྱི་ལྟེ་བར་ཡུམ་གྱི་སྙིང་པོ་དང༌། ( ¡¶ ལུ ¡· + སྙིང་གར་ཡུམ་གྱི་ཉེ་སྙིང༌། ) མགྲིན་པར་ཡུམ་ ( ¡¶ ལུ ¡· ཡབ་ ) གྱི་སྙིང་པོ་དང༌།

¡Ã ལྐོག་མར་ཡབ་ཀྱི་སྙིང་པོ་དང༌། ( ¡¶ ལུ ¡· - ལྐོག་མར་་་་་་་པོ་དང༌། ) དཔྲལ་བར་ཡབ་ཀྱི་ཉེ་སྙིང་བཀོད་དེ་ཡབ་ཀྱི་གནས་ལྔའོ། །ཡུམ་གྱི་མཁར་ཨ་ལས་པདྨ་དམར་པོ་འདབ་མ་གསུམ་པ། དྷྱྃ་ལས་ཟེའུ་འབྲུ་དཀར་པོ་ཟེའུ་འབྲུའི་རྩེ་མོ་སེར་པོའི་བུ་གར་དྷྱྃ་གིས་མཚན་པ། སྔ་མ་བཞིན་དུ་ཡུམ་གྱི་ལྟེ་བ་དང༌། སྙིང་གར་ཡབ་ཀྱི་སྙིང་པོ་དང་ཉེ་སྙིང༌། མགྲིན་པ་དང་དཔྲལ་བར་ཡུམ་ཉིད་ཀྱི་སྙིང་པོ་དང་ཉེ་སྙིང་དགོད་དོ། །སྔགས་རྣམས་ཀྱང་ཕན་ཚུན་སྙོམས་པར་འཇུག་པར་བསམ་མོ། །དེ་ལྟར་རང་གི་འོད་ཀྱི་རིག་མ་དང༌། རྡོ་རྗེ་དང་པདྨ་སྙོམས་པར་འཇུག་པ་དང༌། ལྟེ་བ་ལ་སོགས་པའི་གནས་བཞིར་སྔགས་ཀྱི་སྙོམས་འཇུག་གི་བྱང་སེམས་ཀྱིས་ནོར་བུ་བཀང་བ་ན་དེ་ལ་བདེ་སྟོང་གི་ཏིང་ངེ་འཛིན་སྐྱེ་ལ། དེ་ཉིད་ཉེ་བར་བརྟག་ཅིང་བརྟན་པར་བྱའོ།། གལ་ཏེ་བདེ་བ་ཆེར་སྐྱེས་དྲགས་ན་འཛག་པའི་ཉེན་ཡོད་པས་ལག་པ་རྡོ་རྗེ་ཁུ་ཚུར་སྙིང་གར་བསྣོལ་ཏེ། མིག་དང་ཁ་སྟེང་དུ་བལྟས་ཏེ་བགྲད་ལ་ཧཱུྃ་ཞེས་དྲག་པོ་བཅུ་བདུན་ནམ་བཅུ་དགུ་ཙམ་རེ་བརྗོད་པར་བྱ་སྟེ། དེ་ལས་མང་དུ་མི་བྱའོ། །དེ་ལྟར་བསྒོམས་ན་དཔྱད་པ་མེད་པར་རྡོ་རྗེ་འཆང་བ་ཉིད་འགྲུབ་པར་འགྱུར་རོ། །དམ་ཚིག་གི་ཕྱག་རྒྱའོ། །གཉིས་པ་ལས་ཀྱི་ཕྱག་རྒྱ་ནི། དམ་ཚིག་གི་ཕྱག་རྒྱ་ལ་སེམས་བརྟན་པ་ཐོབ་པའི་རྣལ་འབྱོར་པས་རྟེན་མིའི་བུ་མོ་ལོ་བཅུ་གཉིས་མ་ནས་ཉི་ཤུ་རྩ་ལྔའི་བར་གཟུགས་མཛེས་ཤིང་རྩ་མཆོག་ཏུ་གྱུར་པ་ཡོད་པ། དབང་ཡོངས་སུ་རྫོགས་ཤིང་དམ་ཚིག་དང༌། ཐོས་བསམ་དང༌། ལམ་རིམ་པ་གཉིས་བསྒོམ་པའི་ཚུལ་གྱིས་རྒྱུད་ཡོངས་སུ་སྦྱངས་པའི་ཕྱག་རྒྱ་ཉེ་བར་བཟུང་ལ། བདག་དང་ཕྱག་རྒྱ་བྱིན་གྱིས་བརླབ་པའི་ཚུལ་དམ་ཚིག་གི་ཕྱག་རྒྱ་ལྟར་བྱས་ཏེ། ཕྱག་རྒྱ་དེའི་གཟུགས་ལ་བལྟ་བ་དང༌། སྒྲ་ལ་ཉན་པ་དང༌། སོ་དང་སེན་མོས་འདེབས་པ་དང༌། འཁྱུད་པ་དང་འོ་ལ་སོགས་པའི་ཚུལ་གྱིས་སྐད་ཅིག་དང་པོ་ལ་དགའ་བའི་ཡེ་ཤེས

༑ རྡོ་རྗེ་དང་པདྨ་མཉམ་པར་སྦྱོར་བ་སྐད་ཅིག་གཉིས་པ་ལ་མཆོག་ཏུ་དགའ་བའི་ཡེ་ཤེས། བསྲུབ་པ་དང་སྲུབ་པའི་སྦྱོར་བའི་ཚུལ་གྱིས་སྐད་ཅིག་གསུམ་པ་ལ་དགའ་བྲལ་གྱི་དགའ་བའི་ཡེ་ཤེས། དེ་ནས་ཀུན་རྫོབ་བྱང་ཆུབ་ཀྱི་སེམས་དབང་གི་གནས་སུ་མངོན་དུ་བྱ་བ་ལ་བརྟེན་ནས་སྐད་ཅིག་བཞི་པ་ལ་དོན་དམ་བྱང་ཆུབ་ཀྱི་སེམས་བདེ་སྟོང་བརྗོད་བྲལ་གྱི་ཏིང་ངེ་འཛིན་ལྷན་ཅིག་སྐྱེས་པའི་ཡེ་ཤེས་རྒྱུད་ལ་སྐྱེད་ལ། གལ་ཏེ་བྱང་ཆུབ་ཀྱི་སེམས་ཉམས་ཀྱིས་དོགས་ན་དམ་ཚིག་གི་ཕྱག་རྒྱ་བསྟེན་པའི་སྐབས་བཞིན་ཧྲིཾ་གི་སྒྲ་བརྗོད་པས་ལུས་ཐམས་ཅད་ལ་བདེ་སྟོང་གི་ཏིང་ངེ་འཛིན་གྱིས་ཁྱབ་པར་བྱས་ཤིང་བྱང་ཆུབ་ཀྱི་སེམས་མི་ཉམས་པར་བསྲུང་བ་ཤིན་ཏུ་གཅེས་སོ། །དེ་ལྟར་མཚོན་ཅིང་བསྒོམས་པ་ལ་བརྟེན་ནས་རྡོ་རྗེ་འཆང་བ་ཉིད་འགྲུབ་པར་འགྱུར་རོ། །ལས་ཀྱི་ཕྱག་རྒྱའོ། །གསུམ་པ་ཆོས་ཀྱི་ཕྱག་རྒྱ་ནི། སྔར་དབང་གསུམ་པ་ ( ¡¶ ལུ ¡· + ཡིད་ཀྱིའམ་ ) བརྟེན་ཅན་ནམ། ལས་ཀྱི་ཕྱག་རྒྱ་ལ་བརྟེན་པའི་བདེ་སྟོང་གི་ཉམས་སུ་མྱོང་བ་དྲན་པར་བྱས་ལ། དེ་ནས་ ( ¡¶ ལུ ¡· + ཡིད་ཀྱི ) བདག་ཉིད་ཀྱི་རྡོ་རྗེའི་ནོར་བུའི་དབུས་སུ་ཧཱུྃ་ཁ་དོག་དམར་སྐྱ་ཞིག་བསམས་ཏེ། ཡིད་ཀྱི་ཕྱག་རྒྱ་དང་སྙོམས་པར་ཞུགས་པས་བདེ་སྟོང་གི་ཏིང་ངེ་འཛིན་དྲན་པར་བྱས་ལ། དེ་ནས་ཡིད་ཀྱི་ཕྱག་རྒྱ་དེ་ལུས་ལ་བསྟིམ་ནས་རྡོ་རྗེ་ནོར་བུའི་ཧཱུྃ་ལ་དམིགས་པ་གཏད་ཅིང་བསྒོམ། དེར་བདེ་བ་མི་བཟོད་ན་ཧཱུྃ་དེ་ཉིད་གསང་གནས་ཀྱི་རྩ་བར་སྤཽ། དེར་ཡང་མི་བཟོད་ན་ལྟེ་བར་སྤོ། དེར་ཡང་མི་བཟོད་ན་སྙིང་གར། དེར་ཡང་མི་བཟོད་ན་མགྲིན་པར། དེར་ཡང་མི་བཟོད་ན་སྤྱི་བོར་སྤོ། དེ་ནས་ལུས་ཐམས་ཅད་ལ་ཁྱབ་པར་བསམ་པ་བརྟན་པར་བསྒོམས་ན་རྡོ་རྗེ་འཆང་བ་ཉིད་རྙེད་པར་འགྱུར་རོ། །ཆོས་ཀྱི་ཕྱག་རྒྱའོ། །བཞི་པ་ཕྱག་རྒྱ་ཆེན་པོ་བསྒོམ་པའི་བསམ་གཏན་ནི། སྔ་མ་ལྟ་བུར་འོད་ཀྱི་རིག་མ་དང༌། དངོས་ཀྱི་རིག་མ་དང༌།

ཡི་གེ་ཧཱུྃ་གི་བདེ་བ་ལ་དམིགས་པ་ལྟ་བུ་ནི་མ་ཡིན་ཏེ། དེ་རྣམས་ཐམས་ཅད་དང་བྲལ་བ་རྩ་ཐམས་ཅད་འདུས་པའི་བདེ་བ་ཆེན་པོའི་རང་བཞིན་རྩ་དབུ་མའི་གནས་སུ་འདུས་ལ། དེ་ཉིད་རྡོ་རྗེ་ནོར་བུར་བདེ་བ་ཆེན་པོའི་རང་བཞིན་དུ་བསམ་ལ། དེ་ལྟ་བུའི་བདེ་སྟོང་གི་ཏིང་ངེ་འཛིན་ལ་རེ་ཞིག་ངོས་བཟུང་སྟེ། དེ་ལུས་ཐམས་ཅད་ལ་ཁྱབ་པར་བྱས་ལ། དེ་ལ་སྔར་གྱི་འོད་ཀྱི་རིག་མ་ལ་སོགས་པ་གསུམ་གྱི་རྟེན་ཐམས་ཅད་དང་བྲལ་བ། བརྟན་པ་དང་གཡོ་བ་མཐའ་དག་གི་རང་བཞིན་དུ་གྱུར་པའི་སེམས། ཁམས་གསུམ་ཐམས་ཅད་ལ་ཁྱབ་པའི་སྟོང་པ་ཉིད་ཀྱི་བདག་ཉིད་ཕྱག་རྒྱ་ཆེན་པོ་དང་གཉིས་སུ་མེད་པར་འཁྱུད་པས་བསམ་གྱིས་མི་ཁྱབ་པའི་རང་བཞིན་དུ་ཡིད་བརྟན་པར་བྱས་པས་རྡོ་རྗེ་འཆང་བ་ཉིད་འགྲུབ་པར་བྱེད་དོ། །ཕྱག་རྒྱ་ཆེན་པོའི་རིམ་པ་བསྟན་ཟིན་ཏེ། ནོར་བུ་འགེངས་པ་ཞེས་བྱ་བའི་རིམ་པ་གསུམ་པ་བསྟན་ཟིན་ཏོ།། །།ད་ནི་ཛཱ་ལནྡྷ་རའི་རིམ་པ་བརྗོད་པར་བྱ་སྟེ། དེའང་བྱིན་རླབས་ཀྱི་དུས་སུ་སྨྲ་བ་དང་དགོད་པ་ལ་སོགས་པའི་རྟགས་བྱུང་བའི་གང་ཟག་གིས་འདི་བསྒོམ་ ¡Ã པར་བྱ་སྟེ། ( ¡¶ ཞ ¡· པ་ལ་འཇུག་པར་བྱའོ། ། ) དེའི་བསྒོམ་ཚུལ་ནི་བསྐྱེད་པའི་རིམ་པའི་ལུས་ཀྱི་ལྷ་རྣམས་རང་གི་གནས་སུ་ཐིམས་ནས། ཡུམ་འོད་ཀྱི་གོང་བུར་བྱས་ཏེ་སྣ་བུག་གཡོན་པ་ནས་ལུས་ལ་ཞུགས་པས། དེའི་འོད་ཀྱིས་ལྟེ་བའི་འོག་སུམ་མདོར་ཆུ་སོ་མན་ཆད་ཀྱི་མདོམ་གྲུ་གསུམ་གྱི་གནས་འདིར་ཆོས་འབྱུང་ཁ་དོག་དམར་སྐྱ། དབྱིབས་གྲུ་གསུམ་སྟེ་ཁ་རྒྱས་པ་ཞབས་ཞུམ་པ་གཏིང་རིང་བ་ཞིག་བསམ། དེའི་ནང་དུ་ཐིག་ལེ་ཁ་དོག་རྣམ་པ་ལྔ་སངས་རྒྱས་ལྔའི་རང་བཞིན། དབུས་སུ་དཀར་པོ ༑ ཤར་སྔོ། ལྷོ་སེར། ནུབ་དམར། བྱང་ལྗང་བ་རྣམས་གསལ་བར་བལྟས་ལ། དེ་ནས་རླུང་ནང་དུ་འཇུག་པ་དང་བསྟུན་ཏེ། རླུང་ཐིག་ལེ་དབུ་མ་ལ་བསྟིམ་ལ། རླུང་ཁ་སྦྱར་གྱི་བསམ་གཏན་བུམ་པ་ཅན་ཞེས་བྱ་བ་བསྒོམས་པས་གཏུམ་མོའི་མེ་གྲུབ་པར་འགྱུར་རོ།

།དེ་ནས་གཏུམ་མོ་མེའི་ནུས་པ་གྲུབ་ནས་ལུས་དྲོད་ཀྱི་རང་བཞིན་ཅན་དུ་བརྟན་པར་གྱུར་པ་དེའི་ཚེ་གཡས་རོ་མ་ནས་གཏུམ་མོའི་མེ་སྦར་ཏེ། སྣ་བུག་གཡས་པ་ནས་འཐོན་ཏེ་འཇིག་རྟེན་གྱི་ཁམས་ཐམས་ཅད་ཀྱི་རྡོ་རྗེ་འཛིན་པ་ཐམས་ཅད་མཆོད་ནས། དེ་བཞིན་གཤེགས་པ་ཐམས་ཅད་ཀྱི་ཤངས་ཀྱི་བུ་ག་གཡས་པར་ཞུགས་ཏེ་དེ་བཞིན་གཤེགས་པ་ཐམས་ཅད་ཀྱི་འཁོར་ལོ་བཞི་བསྲེགས་ནས། དེ་བཞིན་གཤེགས་པ་ཐམས་ཅད་ཀྱི་བྱང་ཆུབ་ཀྱི་སེམས་ཤངས་ཀྱི་བུ་ག་གཡོན་པ་ནས་དྲངས་ནས་བདག་གི་སྣ་བུག་གཡོན་པར་ཞུགས་ཏེ། སྤྱི་གཙུག་ཛཱ་ལནྡྷ་ར་ན་མར་ལ་རྩ་དབུ་མའི་ལམ་ནས་ཞུགས་ཏེ་ལྟེ་བའི་དཀྱིལ་འཁོར་དུ་བརྟན་པར་བྱས་པས་ལུས་བདེ་སྟོང་གི་ཏིང་ངེ་འཛིན་རྒྱས་པར་འགྱུར་ཞིང་ ༑ ལྟེ་བའི་དཀྱིལ་འཁོར་དཀར་ལ་ ¡Ã སྲབ་ཅིང་ ( ¡¶ ཞ ¡· - སྲབ་ཅིང་ ) སྲ་ཞིང་བརྟན་པར་གྱུར་པ་ནི། གང་ཚེ་བསྒོམ་པ་བརྟན་གྱུར་པ། །ཞེས་པའི་དོན་ཡིན་ནོ།། སྙིང་གར་བརྟན་ཙམ་ན་མངོན་པར་ཤེས་པ་ལྔ་ཐོབ་པ་ནི། སྣང་བ་དངོས་སུ་མཐོང་བར་འགྱུར། །ཞེས་པའི་དོན་ཡིན་ནོ། །མགྲིན་པར་བརྟན་ཙམ་ན་ཁམས་གསུམ་གྱི་རོ་ལ་ལོངས་སྤྱོད་ཅིང་ཚིག་གི་མཆོག་གྲུབ་པ་ནི་ལྕེས་གདོང་གི་དཀྱིལ་འཁོར་ཁེབས་པ་སྟེ། འབར་བ་བརྟན་པར་འགྱུར་བ་ནི། །ཞེས་པའི་ཚིག་རྐང་བཞིས་བསྟན་ཏོ། །དཔུང་པར་བརྟན་པས་དཔུང་པ་འཕེལ་བའི་སྟོབས་ཀྱིས་ལུས་རྒྱས་ཤིང་སྟོབས་ཚད་མེད་པར་གྱུར་པ་ནི། དཔུང་པའི་རྩ་བར་སོན་པའི་རླུང༌། ། ཞེས་པའི་རྐང་པ་གཉིས་ཀྱིས་བསྟན་ཏོ། །སྤྱི་གཙུག་ཏུ་བརྟན་པར་གྱུར་པས་ཆོས་ཀྱི་སྐུ་རྡོ་རྗེ་འཆང་བ་ཉིད་མངོན་དུ་བྱས་ནས་སེམས་ཅན་གྱི་དོན་བྱེད་པ་ནི། ཁམས་གསུམ་པོ་ནི་ཐམས་ཅད་དུ། །ཞེས་པའི་རྐང་པ་གཉིས་ཀྱིས་བསྟན་ཏོ། ། ཛཱ་ལནྡྷ་ར་ཞེས་བྱ་བའི་རིམ་པ་བཞི་པ་བསྟན་ཟིན་ཏོ།། །།ད་ནི་གང་ཟག་དབང་པོ་ཡང་རབ་བྱིན་རླབས་ཀྱི་དུས་སུ་སེམས་ལ་མི་རྟོག་པ་སྐྱེས་པ་ལ་བསམ་གྱིས་མི་ཁྱབ་པ་ཞེས་བྱ་བའི་རིམ་པ་ལྔ་པ་བརྗོད་པར་བྱ་སྟེ

༑ བསྐྱེད་པའི་རིམ་པའི་རྣལ་འབྱོར་གོ་ཆ་ཆེན་པོའི་ཚུལ་གྱིས་ལུས་ལ་བསྡུས་ལ ༑ བླ་མ་སྤྱི་བོར་བསྒོམས་ཏེ་མོས་གུས་བྱས་ནས། སྙིང་གར་ཟླ་བའི་དཀྱིལ་འཁོར་གྱི་སྟེང་དུ་ཧཱུྃ་དམར་སྐྱ་ཞིག་བསམ། དེ་ནས་འདི་ལྟར་ཕྱི་དང་ནང་གིས་བསྡུས་པའི་ཆོས་ཐམས་ཅད་རང་གི་སེམས་ཡིན་པའི་ཕྱིར་གཟུགས་ལ་སོགས་པ་རྣམས་ཀྱིས་དྲངས་པ་ཕྱི་རོལ་གྱི་དོན་ངོ་བོ་ཉིད་ཀྱིས་གང་གཡོ་བ་འདི་རྣམས་ནི་རྣམ་པར་རྟོག་པ་ཡིན་པའི་ཕྱིར་བདག་གི་འཛིན་པ་མེད་པའི་ཡེ་ཤེས་མངོན་དུ་བྱའོ་སྙམ་དུ་བསམས་ལ། ཡི་གེ་ཧཱུྃ་དེ་ལས་འོད་ཟེར་སྤྲོས་ཏེ་ཁམས་གསུམ་ཐམས་ཅད་ཁྱབ་པར་བྱས་ནས་འོད་ཟེར་གྱི་རང་བཞིན་དུ་གྱུར་པའི་ཁམས་གསུམ་རང་གི་ལུས་ལ་བསྡུ། དེ་བཞིན་དུ་ཡུམ་ཡབ་ལ། ཡབ་སྙིང་གའི་ཧཱུྃ་ལ། ཟླ་བ་ཧཱུྃ་གི་ཞབས་ཀྱུ་ལ། ཞབས་ཀྱུ་ཡང་ཁོག་པ་ལ། ཁོག་པ་མགོ་བོ་ལ། མགོ་བོ་ཟླ་ཚེས་ལ ༑ ཟླ་ཚེས་ཐིག་ལེ་ལ་བསྡུ། ཐིག་ལེ་ནཱ་དའི་རང་བཞིན་དུ་བྱ། ནཱ་ད་ཡང་མི་དམིགས་པར་བྱས་ལ། དེའི་ངོ་བོ་ནི་འདི་ཡིན་ཏེ། གཟུང་བ་དང་འཛིན་པ་ཐམས་ཅད་ཀྱིས་སྟོང་པ། དམིགས་པ་ཐམས་ཅད་དང་བྲལ་བ། བསམ་གྱིས་མི་ཁྱབ་པའི་རང་བཞིན་དེར་ཅི་ཡང་མི་བསམ་པའི་ཚུལ་གྱིས་བསྒོམ་པར་བྱ་སྟེ། དེའང་གཟུང་འཛིན་དང་ནི་བྲལ་བས་ན། །ཞེས་བྱ་བའི་ཚིགས་སུ་བཅད་པ་གཅིག་གིས་བསྟན་ཏོ། །དེའི་གནས་སྐབས་ཀྱི་སེམས་ནི་སྣང་བའི་རང་བཞིན་ཡིན་ལ། དེ་མི་དམིགས་པའི་ངོ་བོར་བྱེད་པས་ན། སེམས་ནི་སྟོང་ཉིད་ལ་གཞུག་ཅིང༌། །ཞེས་བྱ་བ་ཡིན་ནོ། །སྟོང་པ་ཉིད་ཀྱི་རང་བཞིན་དེ་ཉྀད་ངོ་བོ་ཉིད་དུ་འདོད་ན་གནས་ལུགས་དང་མི་མཐུན་པའི་ཕྱིར་ནཱ་ད་དང་ཡི་གེ་ཧཱུྃ་དང་ཧེ་རུ་ཀ་ཡབ་ཡུམ་ནས་ཁམས་གསུམ་གྱི་བར་དུ་སྣང་བར་ལྡང་བར་བྱ་བ་ནི། སྟོང་པ་སེམས་སུ་གཞུག་པར་བྱ། །ཞེས་བྱ་བའི་དོན་ཡིན་ནོ། ། ¡Ã དེ་ནས་ ( ¡¶ ཞ ¡· དེས་ན་ ) དབྱེར་མེད་དམ། ཟུང་འཇུག་གམ་བརྗོད་བྲལ་ཞེས་བྱ་བའི་རང་བཞིན་དུ་བསྟན་པ་ནི། སྟོང་དང་སྟོང་མིན་ལ་སོགས་པ།

།ཞེས་བྱ་བ་ཤླཽ་ཀ་ཕྱེད་དང་གཉིས་ཀྱིས་བསྟན་པ་ཡིན་ནོ ༑ ༑ དེ་ལྟར་ཆོས་ཐམས་ཅད་སྟོང་པ་ཉིད་བརྗོད་བྲལ་གྱི་རང་བཞིན་ཡིན་པའི་རྒྱུ་མཚན་ནི། ཤིང་དང་ཚེར་མ་ལྕུག་མ་སོགས། །ཞེས་པ་ལ་སོགས་པ་རྐང་པ་དྲུག་གིས་བསྟན་པ་ཡིན་ནོ། །དེ་ལྟར་རྣལ་འབྱོར་པས་མཉམ་གཞག་གི་དུས་སུ་སྣང་བ་ལ་ཞེན་ན་སྟོང་པར་ལྡང་ཞིང༌། སྟོང་པ་ལ་ཞེན་ན་སྣང་བར་ལྡང་བའི་ཚུལ་གྱིས་བསམ་གྱིས་མི་ཁྱབ་པའི་རང་བཞིན་ངེས་པར་བྱས་ལ། རྗེས་ལ་ཉམས་ཅི་སྐྱེས་པ་སྐྱོང་བའི་ཚུལ་གྱིས་སྤྱོད་ལམ་གྱི་རྣལའབྱོར་བྱའོ། །དེ་ལྟར་བསྒོམས་ན་འབྲས་བུ་ཐམས་ཅད་མཁྱེན་པའི་སངས་རྒྱས་འགྲུབ་པ་ནི། གལ་ཏེ་དེ་ནི་ཀུན་རྟོགས་ན། ། ཞེས་པའི་ཚིག་རྐང་པ་དྲུག་གིས་བསྟན་ཏོ། །དེའང་ཡོད་པ་དང་མེད་པ་གཉིས་ཀ་དང་ ༑ གཉིས་ཀ་མ་ཡིན་པའི་མཐའ་བཞི་ལས་གྲོལ་བའི་བསམ་གཏན་རང་བཞིན་ལྷུན་གྱིས་གྲུབ་པའི་གནས་ལུགས་དུས་གསུམ་སངས་རྒྱས་ཐམས་ཅད་ཀྱི་ལམ་འདི་བསྒོམས་ན་མྱུར་དུ་སངས་རྒྱས་འགྲུབ་པ་ནི། མཐའ་བཞི་ལས་ནི་རྣམ་གྲོལ་བ། །ཞེས་བྱ་བའི་ཚིགས་སུ་བཅད་པ་གཅིག་གིས་བསྟན་ཏོ། །བསམ་གྱིས་མི་ཁྱབ་པ་ཞེས་བྱ་བའི་རིམ་པ་ལྔ་པ་བསྟན་ཟིན་ཏོ།། །།ད་ནི་རིམ་པ་ལྔ་པོ་དེ་དག་གི་དོན་བསྡུས་ཏེ་བསྟན་ན། སྤྱིར་རིམ་པ་བཞི་ནི་བདེ་བ་སྔོན་དུ་འགྲོ་བའི་སྟོང་པ་བསྒོམ་པ་ཞེས་བྱ་བ་ཡིན་ལ། ཛཱ་ལནྡྷ་རའི་རིམ་པ་ནི་རླུང་འགོག་པའི་སྒོ་ནས་སྟོང་པ་སྔོན་དུ་སོང་བའི་བདེ་བ་བསྒོམ་པ་ཞེས་བྱ་བ་ཡིན་ཞིང༌། གང་ལྟར་ཡང་ཟུང་དུ་འཇུག་པའི་བསམ་གྱིས་མི་ཁྱབ་པ་ཞེས་བྱ་བའི་རིམ་པས་མཐར་ཕྱིན་པ་ཡིན་ནོ། །དེ་ཡང་ཐོག་མར་བྱིན་རླབས་ཀྱི་རིམ་པས་སློབ་མའི་རྒྱུད་བྱིན་གྱིས་བརླབས་ནས། སློབ་མའི་རྒྱུད་བརྟགས་ཏེ་རིམ་པ་ལྔ་གང་ཡང་རུང་བ་ནས་ཅིག་ཅར་དུ་སྟོན་པའམ། རིམ་པ་དང་པོ་ནས་རིམ་གྱིས་བསླབ་ཀྱང་རུང་ངོ༌། །དང་པོ་ཅིག་ཅར་དུ་སློབ་ན་བྱིན་རླབས་ཀྱི་དུས་སུ་ལུས་ལ་འཕར་གཡོ་བྱུང་ན་རིམ་པ་དང་པོ་ནས་

( ¡¶ ཞ ¡·¡¶ ལུ ¡· + རིམ་གྱིས་ ) སློབ། འདར་བ་དང་སྤྲུགས་པ་བྱུང་ན་རིམ་པ་གཉིས་པ་མན་ཆད་ལ་སློབ། བདེ་བ་བྱུང་ན་རིམ་པ་གསུམ་པ་མན་ཆད་ལ་སློབ། ངག་སྨྲ་བ་ལ་སོགས་པ་བྱུང་ན་རིམ་པ་བཞི་པ་མན་ཆད་ལ་སློབ། སེམས་ལ་མི་རྟོག་པ་ལ་སོགས་པ་སྐྱེས་ན་རིམ་པ་ལྔ་པ་འབའ་ཞིག་ལ་སླཽབ་བོ་ཞེས་གསུང་ངོ༌། ཁྲིད་ཀྱི་དུས་སུ་ནི་བྱིན་རླབས་ཀྱི་ ¡Ã དུས་སུ་ ( ¡¶ ཞ ¡· - དུས་སུ་ ) རྟགས་གང་བྱུང་ཡང་རུང་སྟེ། རིམ་པ་དང་པོ་ཉིད་ནས་བཟུང་སྟེ་རིམ་གྱིས་སློབ་པ་ཞེས་བྱ་བ་རིམ་པ་ལྔའི་བར་དུ་དམིགས་པ་ཐམས་ཅད་ཚང་བར་སྟོན་པ་ལ་ཕྱག་ལེན་མཛད་དོ། །གཞུང་གི་ཐད་སོ་ན་ནི་གང་ལ་མོས་པའི་རིམ་པ་རེ་རེས་ཀྱང་ལམ་མཐར་ཕྱིན་པར་ ¡Ã འགྱུར་བར་ ( ¡¶ ཞ ¡· - འགྱུར་བར་ ) མངོན་ཏེ། རིམ་པ་རེ་རེའི་གཤམ་ནའང་འདིས་རྡོ་རྗེ་འཆང་བ་ཉིད་མངོན་དུ་བྱེད་དོ་ཞེས་འབྱུང་བ་ཡིན་ནོ། །དེ་ལ་གཞུང་འདིར་རིམ་པ་དང་དམིགས་པའི་རྣམ་པ་དུ་ཞིག་འབྱུང་ཞེ་ན ༑ རིམ་པ་ནི་ལྔ་སྟེ། བདག་བྱིན་གྱིས་བརླབ་པའི་རིམ་པ་དང༌། སྣ་ཚོགས་རྡོ་རྗེའི་རིམ་པ་དང༌། ནོར་བུ་འགེངས་པའི་རིམ་པ་དང༌། ཛཱ་ལནྡྷ་རའི་རིམ་པ་དང༌། བསམ་གྱིས་མི་ཁྱབ་པའི་རིམ་པ་སྟེ་ལྔའོ། །རིམ་པ་དང་པོ་ལ་སེམས་གཟུང་བའི་རིམ་པ་བསྟན་པའི་ཡན་ལག་ནི། ས་བོན་དང་བཅས་པ་དང༌། ས་བོན་མེད་པ་གཉིས་སོ། །དེ་རེ་རེ་ལའང་སྤྲོ་བའི་ཡན་ལག་ཏུ་དྲུག་དྲུག་སྟེ། མིག་ལ་སོགས་པ་དབང་པོའི་སྒོ་རྣམས་ལ་སྤོས་ཤིང་བསྒོམ་པའོ། །རིམ་པ་གཉིས་པ་སྣ་ཚོགས་རྡོ་རྗེའི་རིམ་པའང་གཉིས་ཏེ། ས་བོན་དང་བཅས་པ་དང༌། ས་བོན་མེད་པའོ། །ས་བོན་དང་བཅས་པ་ལ་སྤྲོ་བའི་ཡན་ལག་ནི། དབང་པོ་ལྔ་དང༌། འཁོར་ལོ་བཞི་སྟེ་དམིགས་པ་དགུ་ཡིན་ནོ། །རིམ་པ་གསུམ་པ་ནོར་བུ་འགེངས་པའི་རིམ་པ་ནི། དམིགས་པ་བཞི་སྟེ། དམ་ཚིག་གི་ཕྱག་རྒྱ་སྟེ་འོད་ཀྱི་རིག་མ་དང༌། ལས་ཀྱི་ཕྱག་རྒྱ་སྟེ་དངོས་ཀྱི་རིག་མ་དང༌། ཆོས་ཀྱི་ཕྱག་རྒྱ་སྟེ་བདེ་བ་དྲན་པ་དང༌། ཕྱག་རྒྱ་ཆེན་པོ་སྟེ་བདེ་སྟོང་རྒྱུད་ལ་སྐྱེས་པའོ།

།དེ་ལྟར་ན་རིམ་པ་ལྔ། དམིགས་པའི་གཙོ་བོ་རིམ་པ་དང་པོ་ལ་གཉིས། གཉིས་པ་ལ་གཉིས། གསུམ་པ་ལ་བཞི། བཞི་པ་དང་ལྔ་པ་དང་གཉིས་པ་ལ་རེ་རེ་སྟེ་བཅུ་ཐམ་

¡Ã པའམ། ( ¡¶ ཞ ¡· པའོ། ། ) དམིགས་པ་སྤྲོ་བ་དང༌། རིམ་པ་དང་པོ་ལ་དྲུག་ཚན་གཉིས། རིམ་པ་གཉིས་པ་ལ་དགུ་སྟེ་ཉེར་གཅིག་ཡིན་ནོ། །དེ་ལྟར་ན་བྱིན་རླབས་ཀྱི་རིམ་པ་མ་ཡིན་པ་ལམ་གྱི་རིམ་པ་ནི་ལྔ ༑ དམིགས་པའི་གཙོ་བོ་ནི་བཅུ། ¡Ã སྤྲོ་བ་ན་ཉེར་གཅིག་སྟེ། ( ¡¶ ཞ ¡· - སྤྲོ་བ་ན་ཉེར་གཅིག་སྟེ། ) སེམས་གཟུང་བའི་ཐབས་ཕྲ་མོ་སུམ་ཅུ་སོ་གཅིག་ཡོད་ལ། ཛཱ་ལནྡྷཱ་རའི་གཤམ་ན་ལྕེ་འཕེལ་བ་ལ་སོགས་པ་ནི་རྟགས་ཀྱི་ཁྱད་པར་ཡིན་ནོ། །དོན་བསྡུས་པ་བསྟན་ཟིན་ཏོ། །གསོལ་བ་འདེབས་པ་དག་ནི་གཙོར་བྱས་ཏེ། །གང་ཟག་གཞན་ལའང་ཕན་པར་འགྱུར་བསམས་ནས། །རིམ་པ་ལྔ་ཡི་ཤིན་ཏུ་གསལ་བྱེད་འདི ༑ ༑ བགྱིས་པའི་དགེ་བས་འགྲོ་ཀུན་རྒྱལ་གྱུར་ཅིག །སློབ་དཔོན་རྡོ་རྗེ་དྲིལ་བུ་པའི་རིམ་པ་ལྔའི་གསལ་བྱེད་འདི་ ( ¡¶ ཞ ¡· - འདི་ ) ཤཱཀྱའི་དགེ་བསྙེན་གྲགས་པ་རྒྱལ་མཚན་གྱིས་ཉེ་བར་སྦྱར་བ་འདི་ཡོངས་སུ་རྫོགས་སོ།། །།༄༅། །ཚེས་བཅུའི་དུས་ཀྱི་མཆོད་པ་བཞུགས། ། ༄༅། །བླ་མ་དམ་པ་རྣམས་ལ་ཕྱག་འཚལ་ལོ། །ཚེས་བཅུའི་མཆོད་པ་ནི། བསོད་ནམས་བསོག་པར་འདོད་པས་ཡར་གྱི་ཚེས་བཅུ། སྡིག་པ་དག་པར་འདོད་པས་མར་གྱི་ཚེས་བཅུ་ལ་བྱ། ཡོ་བྱད་ནི་ཅི་འབྱོར་བ་ཡིན་ན་ཡང༌། མེད་དུ་མི་རུང་བ་ཤ་དང་ཆང༌། གནས་ནི་མདོར་བསྡུ་ན་དབེན་པར། གྲོགས་དམ་ཚིག་ཐོབ་ལ་མ་ཉམས་པའི་དཔའ་བོ་དང་རྣལ་འབྱོར་མ། ཆོ་ག་ལ་གསུམ། སྦྱོར་བ་དང༌། དངོས་གཞི། རྗེས་ཀྱི་བྱ་བའོ། །སྦྱོར་བ་ནི་གནས་ཁང་ཕྱག་དར་བྱ། རྟེན་དགྲམ། མཆོད་ཡོན་ཞབས་བསིལ་སྔོན་དུ་འགྲོ་བའི་མཆོད་པ་དང༌། གཏོར་མ་ཇི་ལྟར་འབྱོར་པ་དང༌། ནང་བདུད་རྩིའི་མཆོད་པ་བཤམ། ཕྱི་མཆོད་དང་འབྲེལ་བར་བྱེད་ན་ཕྱི་མཆོད་ཀྱི་སྐབས་ནས་འབྱུང་བ་བཞིན་བཤམ། གྲལ་ལ་རིམ་གྱིས་འཁོད་དེ། ལྷའི་རྣལ་འབྱོར་ལ་གནས་པར་གསོལ་བ་གདབ། ཡི་གེ་བརྒྱ་པས་རྒྱུད་སྦྱངས་ཏེ་སྦྱོར་བའོ། །དངོས་གཞི་ནི། མངོན་པར་རྟོགས་པ་རྒྱས་འབྲིང་བསྡུས་གསུམ་གང་རུང་བསྒོམ། བདག་ཉིད་སངས་རྒྱས་སུ་བསྐྱེད་པ་ལ་མཆོད་པ་བྱ

༑ དམ་ཚིག་ཅན་ལ་གཏོར་མ་གཏང༌། ཕྱི་མཆོད་དང་འབྲེལ་ན་གཏོར་མའི་གོང་དུ་ཕྱི་མཆོད་རྒྱས་པར་བྱས་ལ། དེ་ནས་གཏོར་མ་གཏང༌། ཚོགས་ཀྱི་ཡོ་བྱད་འོངས་པ་དང༌། གཏོར་མ་བཞིན་བྱིན་གྱིས་བརླབས་ལ། ཕུད་བླ་མ་དང་དཀྱིལ་འཁོར་གྱི་འཁོར་ལོ་དང་གཏོར་མ་ལ་དབུལ། དེ་ནས་ཚོགས་མཆོད་བྱེད་སྙིང་འདོད་ན་ཕུད་ལ་ལག་མཆོད་ཀྱི་སྐབས་བཞིན་རྒྱས་པར་བྱ། ཡང་འཇིག་རྟེན་ན་རྒྱལ་པོའི་ཕྱག་ཚང་བ་མཉེས་པར་བྱས་ནས་རྒྱལ་པོ་མཉེས་པར་བྱེད་པ་བཞིན་ཐོག་མར་བདག་ཉིད་སངས་རྒྱས་སུ་བསྐྱེད་པ་ལ་མཆོད་ནས། ལག་མཆོད་བྱས་ཏེ་ལག་པ་མཆོད་པ་དང་བཅས་པ་དེས་ཕྱི་མཆོད་བྱེད་པ་ཡིན་ནོ་ཞེས་ཀྱང་གསུང་ངོ་ ༑ ༑ གང་ལྟར་ཡང་ཚོགས་ལ་ལོངས་སྤྱོད་པའི་དུས་སུ་ཚིགས་སུ་བཅད་པ་བརྗོད་ཅིང་ཕྱག་རྒྱ་དང་བཅས་པས་མ་ད་ན་དང༌། ཚོགས་ཀྱི་ཁ་ཟས་དང༌། གླུ་གར་གྱིས་ཀྱང་ཚིམ་པར་བྱ་སྟེ་དངོས་གཞིའོ།། །།དེ་ནས་ལྷག་མ་བསྡུས་ཏེ་འབར་བའི་ཕྱག་རྒྱས་ཁ་ཕྲུ་བཏབ་སྟེ་ལྷག་མ་བཏང་ ༑ ཐུན་མོང་གི་བཤད་པའི་རྒྱུད་སཾ་པུ་ཊ་ལས་འབྱུང་བའི་རྡོ་རྗེའི་གླུ་བླངས། བཀྲ་ཤིས་ཀྱི་ཚིགས་བཅད་བརྗོད། སྨོན་ལམ་གདབ། དཔའ་བོ་དང༌། དཔའ་མོ་དང་ ༑ གོ་ཆ་ཆེན་པོས་བསྲུངས་ཏེ་གཏོར་མ་གཏང་བ་ལ་སོགས་པ་ལས་ཀྱྀ་རྗེས་མི་མངོན་པར་བྱའོ། །རྗེས་ཀྱི་བྱ་བའོ། །དབང་བསྐུར་དང་འབྲེལ་ན་བདག་ཉིད་སངས་རྒྱས་སུ་བསྐྱེད་པ་ལ་མཆོད་པ་ཡན་ཆད་སྔ་མ་བཞིན་བྱས་ལ། སྟ་གོན་གྱི་ནུབ་མོ་སའི་ལྷ་མོ་སྟ་གོན་དང༌། ལྷ་སྟ་གོན་བྱས་ལ་གཏོར་མ་གཏང༌། དངོས་གཞིའི་དུས་སུ་དཀྱིལ་འཁོར་སྒྲུབ་མཆོད་དང༌། བྱིན་གྱིས་བརླབ་པ་དང༌། མཆོད་པ་ཡན་ཆད་བྱས་ལ་གཏོར་མ་གཏང༌། དེ་ནས་ཚོགས་བྱིན་གྱིས་བརླབ་པ་མན་ཆད་ལ་ལས་ཀྱི་རྗེས་མི་མངོན་པར་བྱ་བ་མ་གཏོགས་པ་ཐམས་ཅད་སྔ་མ་བཞིན་བྱས་ལ། དེ་ནས་སློབ་མའི་བྱ་བ་རྣམས་བྱས ( ¡¶ ཞ ¡· བྱ ) ། སློབ་མའི་བྱ་བ་རྣམས་ཚར་བ་དང༌། ནམ་ལངས་ཁར་འཇིག་རྟེན་ལས་འདས་པའི་གཏོར་མ་མདང་བཞག་པ་དེ་བཏང་སྟེ་ལས་ཀྱི་རྗེས་མི་མངོན་པར་བྱའོ།

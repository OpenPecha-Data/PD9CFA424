ལུ ¡· མམ་ ) མཚན་མའང་བསྐུའོ། །དེ་ནས་སྐད་ཅིག་གིས་སློབ་མ་རྣམས་ལྷར་དམིགས་ཤིང༌། མེ་ཏོག་ནི་མགོ་ལའོ། །ཞེས་བྱ་བ་ལ་སོགས་པའི་ཚུལ་གྱིས་མཆོད། དེ་ནས་སྲུང་སྐུད་ནི་ཐ་མལ་པ་ལས་དམ་ཚིག་སྲུང་སྐུད་དུ་བསྐྱེད་ལ། སྐུད་པ་དེ་ཉིད་དང་སྙིང་གའི་འོད་ཟེར་གྱིས་མཁའ་འགྲོ་མའི་ཐུགས་ཀ་ནས། ཡེ་ཤེས་ཀྱི་སྲུང་སྐུད་བཀུག་ལ་བསྟིམ་མོ། །སློབ་མའི་ལུས་ཚད་ལ་སུམ་སྒྲིལ་བྱས་ལ། མདུད་པ་གསུམ་མམ་བདུན་བོར་ལ། སྔགས་བཟླས་ཏེ་དཔུང་པའམ་འགུལ་དུ་བཏགས་ལ། དེ་ནས་ཡི་གེ་བརྒྱ་པས་བརྟན་པར་བྱའོ། ¡Ã ཁ་ནས་བླུད་པ་དང་གཏོར་བ་ཡང་བྱའོ། ( ¡¶ ཞ ¡· - ཁ་ནས་བླུད་པ་དང་གཏོར་བ་ཡང་བྱའོ ) ༑ ༑ དེ་ནས་ཚེམས་ཤིང་མ་ཉམས་པར་གསེར་རམ་དངུལ་དུ་བསམ་སྟེ། སོ་ ¡Ã རྩ་དྲུད་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· རྩེ་སྐྲུག་ ) ཅིང་སྡིག་པ་དྲང་བར་བསམས་ལ། དེ་ནས་མཎྜལ་ ¡Ã བའི་ཀོ་བ་ཙམ་གྱི་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· བ་བྱུང་གིས་བ་ཀོ་ཙམ་བྱུགས་པའི་ ) སྟེང་དུ་ཁ་ཤར་དུ་བལྟས་ཏེ་དོར་ལ་ལྟས་བརྟག་གོ །དེ་ནས་ཀུ་ཤའི་ཆུན་པོ་གཉིས་ལ་གཅིག་ལ་ངྷཱིཿ བཟླས་པས་སྔས་སུ་བཅུག་གོ །གཅིག་ལ་ཧ་བཟླས་ལ་སྟན་དུ་ཐིངས་ཤིག་པར་བསྒོའོ ༑ ༑ འཇིག་རྟེན་ན་ནི་ཀུན་མཁྱེན་དག །ཅེས་བྱ་བ་ནས། དེ་བཞིན་གཤེགས་པ་ཉིད་དུ་འགྱུར། །ཞེས་བྱ་བའི་བར་གྱིས་ཆོས་ཟབ་མོ་བསྟན་པས་སྤྲོ་བ་བསྐྱེད་པར་བྱའོ། །དེ་ནས་སེང་གེའི་ཉལ་སྟངས་ཀྱིས་ཉལ་དུ་གཞུག་གོ །སློབ་དཔོན་དུ་དབང་བསྐུར་བར་འོས་པ་ནི། སློབ་དཔོན་དུ་ལྷག་པར་གནས་པར་གསོལ་བ་འདེབས་སུ་གཞུག་ཅིང༌། དེ་ཉིད་ཀྱིས་ལྷག་པར་གནས་པར་འགྱུར་རོ།། སློབ་མ་ལྷག་པར་གནས་པའོ། །བུམ་པ་ནི་སྐྱོན་མེད་པ་བཅུ་ཇི་ལྟ་བ་བཞིན་དུ་བྱས་ལ། གཅིག་རྣམ་པར་རྒྱལ་བ་སྟེ་རིགས་ལྔའི་མཚན་མ་བྲིའོ། །བརྒྱད་ནི་སྒོ་མཚམས་མའི་སྟེ་གྲི་གུག་བྲིའོ། །གཅིག་ལས་ཐམས་ཅད་པ་སྟེ་སྣ་ཚོགས་རྡོ་རྗེའམ་གྲི་གུག་བྲིའོ། །དྲི་མའི་ཁྲུས་རྒྱས་པར་བྱེད་ན། དྲི་ཆུ་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· - ཆུ་ ) སྣ་ཚོགས་པའི་བུམ་པ་བརྒྱད་དེ།

དེ་ལྟར་བཅོ་བརྒྱད་དོ། །བསྡུས་པར་བྱེད་ན། ལྷའི་བུམ་པ་གཉིས་སུ་བསྡུ་བར་བྱ་སྟེ་དེ་ལྟར་བཅུའོ། །ཡང་ན་རྫས་ཀྱི་བུམ་པ་གཅིག་དང་ ( ¡¶ ཞ ¡· ཏུ་བྱས་ཏེ ) ། ལྷའི་བུམ་པ་བཅུ་དང་བཅུ་གཅིག་གོ །དེ་ལྟར་བྱས་པས་བུམ་པའི་གདན་རས་ཀྱི་སྟེང་དུ་བཞག་ལ། ཐ་མལ་པའི་བུམ་པ་སྟོང་པར་བྱས་ལ། བྷྲཱུྃ་མམ་ཁྃ་ལས་བདུད་རྩིའི་བུམ་པ་བསྐྱེད་དེ། རྣམ་རྒྱལ་དུ་ལྔའམ་གསུམ་མམ་ཡི་གེར་གྱུར་པ་ལས། དཀྱིལ་འཁོར་གྱི་ལྷ་ཚང་བར་བསྐྱེད། བརྒྱད་ ¡Ã པོར་དེ་བཞིན་དུ་ ( ¡¶ ཞ ¡· པར་ ) སྒོ་མཚམས་མ་བརྒྱད་བསྐྱེད། ལས་ཐམས་ཅད་པར་བདེ་བ་ཆེན་པོའི་འཁོར་ལོ་འམ་ཁཎྜ་རོ་ཧཱ་བསྐྱེད་པའོ། །དེ་ནས་ཕྱོགས་ཕྱོགས་ནས་ཡེ་ཤེས་པ་སྤྱན་དྲངས་ལ་བསྟིམ། རང་གི་སྔགས་བརྒྱ་རྩ་བརྒྱད་བཟླས་པ་འམ། ཡང་ན་ཕག་མོའི་ཉེ་སྙིང་བརྒྱ་རྩ་བརྒྱད་དམ། ཡང་ན་དེ་ཉིད་ཉི་ཤུ་རྩ་གཅིག་བཟླའོ ༑ ༑ དེ་ནས་ཕྱི་ནང་མཆོད་པ་ལ་སོགས་བྱའོ། །རྣམ་པར་རྒྱལ་བའི་བུམ་པའི་ཐོག་ཏུ་ ¡Ã ཐོད་པ་ ( ¡¶ ཞ ¡· - ཐོད་པ་ ) བདུད་རྩི་ལྔ་དང་ལྡན་པ་བཞག་ལ། འབར་བ་ལ་སོགས་པའི་ཚུལ་གྱིས་ཡེ་ཤེས་ཀྱི་བདུད་རྩིའི་ངོ་བོར་བསམས་ལ། བུམ་པ་དེ་ཉིད་ནས་བཟུང་ནས་ལྷའི་བུམ་པ་རྣམས་སུ་ཧུབ་རེ་བླུགས་པས་དེའི་རྐྱེན་གྱིས་ལྷ་རྣམས་ཀྱང་ཞུ་ནས། ཡེ་ཤེས་ཀྱི་བདུད་རྩིའི་ངོ་བོར་བསམས་ལ་གནས་དབེན་པར་བཞག་པ་ནི་བུམ་པ་ལྷག་པར་གནས་པའོ། །ཐིག་དང་ཚོན་གྱི་ཆོ་ག་ནི། བསྐྱེད་རིམ་སྔོན་དུ་འགྲོ་བས་ཡིག་འབྲུ་ལྔའམ་མཁའ་འགྲོ་ལྔར་བསྐྱེད་ཅིང་དེ་བཞིན་དུ་ཡེ་ཤེས་པ་བསྟིམ་ལ་མཆོད། དཀྱིལ་འཁོར་བྲི་བ་དང་ལྷ་དགོད་པའི་ཆོ་ག་ནི་དཀྱུས་ན་གསལ། རྡུལ་ཚོན་སྦྱང་ཞིང་བྱིན་གྱིས་བརླབ་པ་ནི། ཚོགས་བསགས་པ་ནས་དབུ་རྒྱན་གྱི་ལྷ་དྲན་པའི་བར་དུ་བསྒོམས་ལ། སྔ་མ་བཞིན་དུ་མཆོད་ཅིང་བསྟོད་དེ་བསྡུའོ། །དེ་བཞིན་དུ་མདུན་གྱི་ཡང་དབུ་རྒྱན་གྱི་ ¡Ã བར་དུ་དྲན་པར་ ( ¡¶ ཞ ¡· ལྷ་དྲན་གྱི་བར་དུ་ ) བསྒོམས་ལ། ལྷའི་གྲངས་ཀྱི་གཏོར་མའམ།

ཡང་ན་གཏོར་མ་གཅིག་གིས་སརྦ་བིགྷྣཱན་ན་མསྶརྦ་ཏ་ཐཱ་ག་ཏེ་བྷྱོ་བི་ཤཱ་མུ་ཁེ་བྷྱཿསརྦ་ཐཱ་ཁཾ་ཨུངྒ་ཏེ་སྥ་ར་ཎ་ཨི་ ( ¡¶ ཞ ¡· ཧི་ ) མཾ་ག་ག་ན་ཁཾ། གྲིཧྣེ་དཾ་བ་ལིཾ་ཏེ། ཞེས་པས་ལན་གསུམ་དུ་བསྐོར་ཞིང་གཏོར་མ་བཏང་ངོ༌། །ཏིང་ལོ་དང་ཆང་བུ་ཡང་མ་འབྱོར་ན་ཟུང་གཅིག་བཏང་སྟེ་དཀྱིལ་འཁོར་སོ་སོའི་ལས་ཐམས་ཅད་པའི་སྔགས་ཀྱིས་བསྐོར་ལ་གཏང་ངོ༌། །དེ་བཞིན་དུ་གུ་གུལ་ལ་སོགས་པའི་བདུགས་པ་རྣམས་ཀྱང་སྔགས་དེ་ཉིད་བཟླས་ཤིང་བདུག་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· བརྡུངས་ ) ལ་བསྐྲད་དོ།། དྲི་མའི་ཁྲུས་ནི་ལྷ་ལ་སོགས་པའི་སྒོ་ནས་མེ་ལོང་ལ་གཟུགས་བརྙན་འཆར་བར་བྱས་ལ། དྲི་ལ་སོགས་པའི་བུམ་པ་བརྒྱད། ཇི་ལྟར་བལྟམས་པ་དང༌། འདི་ནི་ཁྲུས་མཆོག་དང༌། ཕ་རོལ་ཏུ་ཕྱིན་པ་དྲུག་སྟེ། དེ་ལྟར་བརྒྱད་ཀྱིས་ཤེས་པ་བརྗོད་ཅིང་ ༑ མེ་ལོང་ལ་ཏིལ་མར་གྱིས་བྱུག སྲན་དཀར་གྱི་འདག་ཆལ་གྱིས་གདབ། དཱུར་བའི་པིར་གྱིས་བྱུག་ཅིང་རྫས་སོ་སོའི་སྔགས་ཀྱིས་བཀྲུའོ། །དྲི་སྣ་ཚོགས་ཀྱིས་ནི་རྫས་གཞན་གྱིས་བུམ་པ་རེ་རེའི་གཞུག་ལ་བཤལ་ལོ། །བུམ་པ་རེ་རེའི་ཁྲུས་ཟིན་པ་དང༌། སྐུ་ཕྱིས་ཀྱིས་རྡུལ་དེ་ཕྱིའོ། །དེ་ལྟར་ཁྲུས་ཟིན་པ་དང༌། མཁའ་འགྲོ་ལྔའི་ས་བོན་བཟླས་པའི་དྲིའི་ཐིག་ལེ་ལྔ་ ( ¡¶ ཞ ¡· - ལྔ་ ) བྱས་ལ། ན་བཟའ་མ་འབྱོར་ན་སྤྱིར་ཟུང་གཅིག་དབུལ་ལོ། །དེ་ནས་གཉིས་སུ་མེད་པའི་སྔགས་ཀྱིས་བྱིན་གྱིས་རླབས་བརླབ་བོ། །དེ་ནས་ཁྱད་པར་གྱི་སྔགས་ཀྱིས་སོ། །དེ་ནས་སྒོ་ ( ¡¶ ལུ ¡· + འཚམས་ ) མ་བཞིའི་སྔགས་ཀྱིས་སོ། །དེ་ནས་ལྷའི་གྲངས་བཞིན་དུ་ཨཱ་ལི་ཀཱ་ལིས་སོ། །དེ་ནས་ལྷ་ཐམས་ཅད་རང་གི་མིང་དང་ལྡན་པའི་ཡི་གེ་བརྒྱ་པས་སོ།། དེ་ནས་སྔགས་ལ་སོགས་པའི་བར་གྱི་མཆོད་ཅིང་བསྟོད་ལ། ཚོགས་བྱིན་གྱིས་བརླབ་ཅིང་ཕུད་གཏོར་མ་ལ་དབུལ་བ་ལ་སོགས་པ་སྔ་མ་བཞིན་དཀྱིལ་འཁོར་བྱིན་གྱིས་བརླབ་བོ། །དེ་ནས་སློབ་མ་དཀྱིལ་འཁོར་དུ་གཞུག་པ་ནི། འཇིག་རྟེན་པའི་གཏོར་མ་བཏང་ལ་སློབ་དཔོན་གྱིས་ལས་ཐམས་ཅད་པའི་བུམ་པ་དང༌།

གདོང་གཡོགས་དང༌། རྡོ་རྗེ་ཁྱེར་ལ་སློབ་མའི་དྲུང་དུ་ཕྱིན་པ་དང༌། རང་གི་ལྷའི་རྣལ་འབྱོར་གསལ་བར་དྲན་ལ་བདེན་པ་བརྗོད་པར་བྱ་སྟེ། ཆོས་རྣམས་གཟུགས་བརྙན་ལྟ་བུ་སྟེ། །གསལ་ཞིང་དག་ལ་རྙོག་པ་མེད། །གཟུང་དུ་མེད་ཅིང་བརྗོད་དུ་མེད ༑ ༑ རྒྱུ་དང་ལས་ལས་ཡང་དག་འབྱུང༌། །དེ་བཞིན་དེ་ཉིད་ངེས་པར་འབྱུང༌།། བདེན་པ་དེ་ཡིས་དཀྱིལ་འཁོར་དེ། །གཟུགས་བརྙན་གསལ་བར་སློབ་མ་རྣམས།། དྲི་མེད་ཐམས་ཅད་མཐོང་བར་ཤོག །ཅེས་ལན་གསུམ་བརྗོད་ལ། ཡོལ་བའི་ཕྱི་རོལ་དུ་འཇུག་པའི་ཡོན་ཕུལ་ཏེ། སློབ་མ་རྣམ་པར་སྣང་མཛད་དུ་དམིགས་ལ། ཨ་ཁཾ་བཱི་ར་ཧཱུྃ། ཞེས་བརྗོད་ལ། མེ་ཏོག་གི་ཕྲེང་བ་དང་བཅས་པའི་གདོང་གཡོགས་བཅིངས་ལ། བུམ་པའི་ཆུས་བྲན་ཞིང་འཇུག་པའི་ཕྱིར་བྲི་བར་བྱ་སྟེ། ཁྱོད་སུ་ཅི་ལ་དགའ། ཞེས་དྲི་བར་བྱའོ། །དེས་ཀྱང་བརྗོད་པར་བྱ་སྟེ། སྐལ་བཟང་བདག་བདེ་བ་ཆེན་པོ་ལའོ། །སློབ་མའི་སྙིང་གར་རྡོ་རྗེ་བཞག་ལ། ཨོཾ་བཛྲ་ཡོ་ག་ཙིཏྟ་ཨུཏྤཱ་ད་ཡཱ་མེ། སུ་ར་ཏི་ས་མ་ཡ་སྟཱཾ། བོ་ངྷི་ཙིཏྟ་བཛྲ་ཡ་ཐཱ་སུ་ཁཾ། ཞེས་བརྗོད་པས་སེམས་བསྐྱེད། འདི་སྐད་ཅེས་བརྗོད་པར་བྱ་སྟེ། དེང་ཁྱོད་དེ་བཞིན་གཤེགས་པ་ཐམས་ཅད་ཀྱི་བྱིན་གྱིས་རློབ་པར་འགྱུར་གྱིས། དེ་བཞིན་གཤེགས་པ་ཐམས་ཅད་ཀྱི་གསང་བ་མཆོག་འདི་ཁྱོད་ཀྱིས་དཀྱིལ་འཁོར་དུ་མ་ཞུགས་པ་རྣམས་ཀྱི་མདུན་དུ་སྨྲ་བར་མི་བྱ་ཞིང༌། མ་དད་པ་རྣམས་ལ་ཡང་མ་ཡིན་ནོ། །དེ་ནས་ཧཱུྃ་྄ ¡Ã ཛཿཧཱུྃ་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· ཛ་ཧུཾ་ ) ཞེས་བྱ་བས་བཀུག་སྟེ། ཡོལ་བའི་ནང་དུ་བཅུགདཀྱིལ་འཁོར་གྱི་ལྷོ་སྒོ་ལ་སོགས་པར་བཞག་སྟེ། སློབ་དཔོན་གྱིས་གོང་ནས་གོང་མའི་ཞེས་བྱ་བ་ལ་སོགས་པའི་ཚིག་ཀྱང་བརྗོད་དོ། །དེ་ནས་ཡང་བདུད་རྩིས་བསངས་ཤིང་བསྲུང་བར་བྱའོ། །དེ་ནས་ལྷར་མོས་ཤིང་གོ་ཆས་བརྒྱན་ལ། སློབ་མའི་གཙོ་བོའི་ལག་པ་བཟུང་སྟེ། ཨོཾ་ཤྲཱི་ཧེ་རུ་ཀ་བི་ཏ་ཅེས་བྱ་བ་ལ་སོགས་པའི་སྔགས་བརྗོད་ལ། དཔའ་བོ་བསྐུལ་ལོ། །དེ་ནས།

གང་གི་དྲིན་གྱིས་བདེ་ཆེན་ཉིད ༑ ༑ ཅེས་བྱ་བ་བཅུག་ལ། རྣལ་འབྱོར་གནས་ལས་ཞེས་པའི་བར་གྱིས་ཕྱག་འཚལ་ཞིང་སྐྱབས་སུ་འགྲོ་བའོ། །གསོལ་བ་གདབ། སློབ་དཔོན་གྱིས་ཀྱང་ལྷ་ལ་གསོལ་བ་གདབ། དེ་ནས་སློབ་མ་སྦྱངས་ཏེ། གཏོར་མ་ནས་འདུག་པས་བགེགས་བསྐྲད་པའི་བར་དུ་རྡུལ་ཚོན་དང་མཐུན། སྡིག་པ་སྦྱང་བ་ཇི་ལྟ་བ་བཞིན་བྱས་ལ་སློབ་མའི་ཐ་མལ་པའི་ལུས་མི་དམིགས་པར་བྱས་ལ། མངོན་པར་བྱང་ཆུབ་པ་ལས་ཡབ་ཡུམ་ཙམ་བསྐྱེད་ལ། དེའི་སྙིང་གར་རྫོགས་པར་བསྐྱེད་ལ་ཡེ་ཤེས་ཀྱང་རྫོགས་པར་བསྟིམ་པའམ། ཡང་ན་ཨཱ་ལི་ཀཱ་ལི་ཕྲག་ ¡Ã ལྔའི་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· ལྷའི་ ) གྲངས་བཞིན་བསྟིམ་ལ། ཕྱི་ནང་གི་མཆོད་པས་མཆོད་ལ། དེ་ཙམ་གྱིས་ནི་དཀྱིལ་འཁོར་དུ་བཅུག་པའི་ངོ་བོའོ། །དེ་ནས་བྱིན་གྱིས་བརླབ་པའི་ཕྱག་གོ ། དེ་ནས་སྡོམ་པ་དབོག་གོ །དེ་ནས་དམ་ལ་བཞག་པ་བཞིའོ། །དེ་ནས་དབབ་པའི་གསོལ་བ་འདེབས་སུ་གཞུག་གོ །དེ་ནས་སློབ་མའི་སྙིང་གར་ཧེ་རུ་ཀ ལྟེ་བར་ཕག་མོ་བསམས་ལ། ཡེ་ཤེས་ཀྱི་ཡབ་ཡུམ་སྤྱན་དྲངས་ལ་དམ་ཚིག་པ་ལ་བསྟིམ། ཡང་སྙིང་གའི་ས་བོན་ལས་འོད་ཟེར་འཕྲོས་པས། ཡེ་ཤེས་ཀྱི་འཁོར་ལོའི་ལྷ་ཚང་བར་སྤྱན་དྲངས་ལ་བསྟིམ་ཞིང་གནས་བཞིར་ཡང་འབྱུང་བ་བཞི་ལ་ཡི་གེ་བཞི་དགོད་ཅིང༌། རྐང་མཐིལ་གྱི་ཡི་གེས་ཡར་ཕུལ་ཞིང་ལྷ་རྣམས་མར་འཇུག་པར་བསམས་ལ། སྔགས་ཐམས་ཅད་བཟླས་པའམ། ཡང་ན་ཡབ་ཡུམ་གྱི་སྔགས་བཞི་པོ་བཟླས་ཤིང༌། ཨོཾ་ཤྲཱི་ཧེ་རུ་ཀ་ཏིཥྛ་མ་ཧཱ་ཀྲོ་ངྷ། ཞེས་བྱ་བ་ལ་སོགས་པའང་བཟླས་ཤིང༌། ཇི་སྐད་གསུངས་པའི་བདུག་པས་ཀྱང་བདུག་པར་བྱའོ། །དེ་ནས་ཇི་ལྟར་ཕེབས་པའི་རྟགས་འབྱུང་ངོ༌། །དེ་ནས་དཀྱིལ་འཁོར་གྱི་འཇུག་གི་སྒོར་སོང་ལ་སྣང་བ་བརྟག་ཅིང༌། བདེན་པ་བརྗོད་ལ་མེ་ཏོག་དོར་རོ། །དེ་ནས་སློབ་མའི་མིག་གཉིས་སུ་མ་དང་ཊ་ལས་ཉི་མ་ཟླ་བར་བསམས་ལ། གདོང་གཡོགས་དགྲོལ་ཞིང་དཀྱིལ་འཁོར་བསྟན་ཏོ། །དེ་ཙམ་གྱིས་དཀྱིལ་འཁོར་དུ་འཇུག་པའོ།། འདི་ནི་སློབ་མ་འཇུག་པའི་དབང་དུ་བྱས་པ་སྟེ།

སློབ་དཔོན་ནམ་སྒྲུབ་པ་པོ་བདག་ཉིད་འཇུག་པར་འདོད་ན་ལོའམ་ཟླ་བར་བསྒྲུབས་ཤིང་སློབ་མ་འཇུག་པའི་ཆོ་གས་བདག་ཉིད་ཞུགས་ཤིང་དབང་རྣམས་ཀྱང་རྫོགས་པར་བླངས་ན་དམ་ཚིག་ཉམས་པ་སྐོང་བར་འགྱུར་ཞིང༌། ཇི་ལྟ་བ་བཞིན་དུ་འགྲུབ་པར་འགྱུར་རོ། །དེ་ནས་སེང་གེའི་ཁྲི་དང༌། རས་སམ་སྣམ་བུ་ལ་པདྨ་འདབ་བརྒྱད་ཀྱི་མཎྜལ་བྱེད་དུ་བཅུག་ལ། དབང་གི་ཡོན་ཕུལ་ནས་ཧེ་རུ་ཀ་དཔལ་ཞེས་བྱ་བའི་ཚིག་གིས་གསོལ་བ་གདབ་བོ། །དེ་ནས་ཡུངས་དཀར་བརྡེག་པ་དང༌། ཏིལ་བསྲེག་པར་བྱའོ། །དེ་ནས་ད་ཅིའི་མེ་ཏོག་དེ་ཡང་དཀྱིལ་འཁོར་དུ་ཕུལ་ལ་དེ་ནས་བླངས་ཏེ་སྤྱི་གཙུག་ཏུ་བཏགས་ལ། སྤྱི་གཙུག་ནས་མར་ཐིམ་ནས། དེའི་རིགས་མ་འབྲེལ་པ་དང༌། ཡང་ན་སྐྲ་ལ་བཏགས་ནས་ཅུང་ཟད་ཅིག་བཞག་པས་དེའི་རིགས་སུ་འབྲེལ་བར་བསམ་མོ། །མེ་ཏོག་དེ་དང་འབྲེལ་བའི་ལྷའི་སྔགས་སྦྱིན་པར་བྱས་ཏེ། དང་པོ་སློབ་དཔོན་གྱིས་སློབ་མའི་བགྲང་ཕྲེང་ཐོགས་ཏེ། བཅོམ་ལྡན་འདི་ལ་སྩལ་ལགས་ཀྱིས། །བཅོམ་ལྡན་འདི་ལ་བྱིན་གྱིས་རློབས། །ཞེས་གསོལ་བ་བཏབ་ལ། སློབ་མ་ལ་སྦྱིན་ནོ། །སློབ་མས་ཀྱང༌། བཅོམ་ལྡན་བདག་ལ་སྩལ་ལགས་ཀྱིས། ། བཅོམ་ལྡན་བདག་ལ་བྱིན་གྱིས་རློབས། །ཞེས་བརྗོད་ལ། སྤྱི་གཙུག་ཚངས་པའི་བུ་ག་ནས་ཞུགས་པས་ལུས་ཐམས་ཅད་བཀྲུས་ཤིང་དག་པར་བསམ་མོ། །ཕྱིའི་དབང་ངོ༌། །སློབ་དཔོན་གྱི་སྙིང་གའི་ས་བོན་གྱི་འོད་ཀྱིས་མི་བསྐྱོད་པ་བཀུག་ཅིང་ ༑ ཚངས་པའི་བུ་ག་ནས་བསྟིམ་པས་སློབ་མའི་རྣམ་པར་ཤེས་པ་གནས་གྱུར་ནས ༑ མེ་ལོང་ལྟ་བུའི་ཡེ་ཤེས་སུ་བྱས་པ་ནི་ནང་གི་དབང་ངོ༌། །དབུ་རྒྱན་གྱི་དབང་ནི་རིན་ཆེན་འབྱུང་ལྡན་གྱི་རང་བཞིན་འཛིན་དུ་བཅུག་ལ། གང་ཡང་རུང་བའི་ཅོད་པན་ཆོ་ག་ཡན་ལག་གསུམ་ལས་རིན་ཆེན་འབྱུང་ལྡན་དུ་བསྐྱེད་ལ། དེ་སློབ་མའི་སྤྱི་བོར་བཞག་ཅིང༌། ཨོཾ་པྲ་ཏཱིཙྪ་མ་ཧཱ་མུ་ཀུ་ཊ་ཧོཿཞེས་བརྗོད་པ་ནི་ཕྱིའིའོ ༑ ༑ སློབ་མའི་སྤྱི་བོར་སྣ་ཚོགས་རྡོ་རྗེ་བསམས་ལ། སློབ་དཔོན་གྱིས་སྙིང་གའི་འོད་ཀྱིས་དཀྱིལ་འཁོར་གྱི་ལྷ་བཀུག་ལ།

སྣ་ཚོགས་རྡོ་རྗེའི་ལྟེ་བ་ནས་མར་བསྟིམ་པས། རང་རང་གི་སྔགས་དྲན་ཞིང་གཞུག་པར་བསམ། དེ་ནས་གཙོ་བོ་དང་མཁའ་འགྲོ་མ་བཞིའི་སྔགས་ཟློས་ཤིང༌། དབུ་རྒྱན་དཔྲལ་བར་འཛིན་ཅིང་དེ་བཞིན་གཤེགས་པ་ལྔ་དབུ་ལ་གནས་པར་བསམ་པ་ནི། རིན་ཆེན་འབྱུང་ལྡན་གྱི་ས་བོན་བཞག་པར་རིགས་པར་བྱའོ། །ནང་སྟེ་ཅོད་པན་གྱི་དབང་ངོ༌། །དེ་ནས་ ¡Ã རྡོ་རྗེ་ ( ¡¶ ཞ ¡· - རྡོ་རྗེ་ ) ཆོ་ག་ཡན་ལག་གསུམ་གྱིས་སྣང་བ་མཐའ་ཡས་སུ་བསྐྱེད་ལ ༑ ¡Ã སངས་རྒྱས་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· རྡོ་རྗེ་ ) ཀུན་གྱི་རྡོ་རྗེའི་དབང༌། །ཞེས་བ་བྱ་བ་ལ་སོགས་པའི་ཚིག་གིས་བསྙད་པ་ནི་ཕྱིའི་དབང་ངོ༌། །མགྲིན་པར་པདྨའི་ལྟེ་བ་ནས་སྣང་བ་མཐའ་ཡས་བཅུག་ཅིང་སློབ་མ་དང་གཉིས་སུ་མེད་པར་བྱས་པ་ནི་ནང་སྟེ་རྡོ་རྗེའི་དབང་ངོ༌། །དེ་ནས་དྲིལ་བུ་ཆོ་ག་ཡན་ལག་གསུམ་ལས་དོན་ཡོད་གྲུབ་པ་བསྐྱེད་དེ། འདི་ནི་རྣལ་འབྱོར་མ་ཀུན་གྱི་བྱ་བའི་ཚིག་གིས་སྦྱིན་པ་ནི་ཕྱིའི་དབང་ངོ་ ༑ ༑ ཡང་སྤྱི་བོར་སྣ་ཚོགས་རྡོ་རྗེ། ལྟེ་བ་ནས་སློབ་དཔོན་གྱིས་དོན་ཡོད་གྲུབ་པ་བཅུག་ཅིང༌། སློབ་མ་དེའི་རང་བཞིན་དུ་བྱས་པར་བསམ་པ་ནི་ནང་གིའོ། །དེ་ནས་སློབ་མ་རྣམ་པར་སྣང་མཛད་ཀྱི་རྣལ་འབྱོར་འཛིན་དུ་བཅུག་སྟེ། མཁའ་འགྲོ་མ་ལ་སོགས་པ་བཞིའི་སྔགས་ཟློས་ཤིང༌། དེ་དག་གིས་མིང་འདོགས་པར་བསམས་ལ། མེ་ཏོག་བབས་པ་དང་མཐུན་པའི་མིང་མདོགས་སོ། །ཕྱིའི་འོ། ། ཡང་སྤྱི་བོར་ཟླ་བ་ལས་ཨོཾ་ཐུར་བལྟས། ཨོཾ་གྱི་ཐིག་ལེ་ནས་རྣམ་པར་སྣང་མཛད་བཅུག་པས་དེའི་རང་བཞིན་དུ་བྱས་པ་ནི་ནང་སྟེ། དེས་ནི་སློབ་མའི་རྒྱུད་ལ་རྣམ་པར་སྣང་མཛད་ཀྱི་ས་བོན་བཞག་པའོ། །དེ་ཡན་ཆད་སོ་སོའི་རྣལ་འབྱོར་ནི་སློབ་དཔོན་དང་སློབ་མ་གཉིས་ཀས་འཛིན་ནོ། །དེ་ནས་སློབ་མ་ཧེ་རུ་ཀར་མོས་པ་བྱས་ནས

༑ འདི་དེ་སངས་རྒྱས་ཐམས་ཅད་དེ། །ཞེས་བྱ་བ་ལ་སོགས་པའི་ཚིག་གིས་ཐུན་མོང་གི་བརྟུལ་ཞུགས་དང༌། ཁ་ཊྭཱཾ་ག་དང་ ( ¡¶ ཞ ¡· - དང་ ) ཌ་མ་རུ། །ཞེས་བྱ་བ་ལ་སོགས་པའི་ཚིག་གིས་ཐུན་མོང་མ་ཡིན་པའི་བརྟུལ་ཞུགས་ཀྱི་བར་དུ་སློབ་དཔོན་དང་སྒྲུབ་པ་པོ་ཐམས་ཅད་ལ་ཐུན་མོང་དུ་བྱིན་ཏེ། དེ་ལྟར་ན་ཟུང་མ་ཐེམ་པའི་སློབ་མ་ཉི་ཤུ་རྩ་ལྔའི་བར་དུ་གཟུང་བར་བྱ་སྟེ། དེ་ཡང༌། མཁས་པས་སློབ་མ་གཅིག་གམ་གསུམ། །ལྔའམ་ཡང་ན་བདུན་དག་གམ། །ཉི་ཤུ་རྩ་ནི་ལྔ་ཡི་བར།། ཟུང་དུ་མ་གྱུར་ཡོངས་སུ་གཟུང༌། །དེ་ལས་ལྷག་པའི་སློབ་མ་དག །ཡོངས་སུ་གཟུང་བར་མི་ཤེས་སོ། །ཞེས་གསང་བ་སྤྱི་རྒྱུད་ལས་གསུངས་པས་སོ། །དེ་རྣམས་ཀྱི་ནང་ནས་སློབ་དཔོན་དུ་གཟུང་བ་ནི་གཅིག་ཡིན་ཏེ། དེ་ཡང་གསང་བ་སྤྱི་རྒྱུད་ལས། སློབ་དཔོན་བློ་གྲོས་ཆེན་པོ་ཡིས། །གསང་སྔགས་རྒྱུད་ནི་འཛིན་པ་པོར ༑ ༑ སློབ་མ་དམ་པ་སྙིང་རྗེ་ཅན། །གཅིག་ནི་ནན་ཏན་བསྐྱེད་དེ་བསྒྲུབ།། ཅེས་གསུངས་པས་སོ། །དེ་ལྟ་བུ་སློབ་དཔོན་དུ་འོས་པ་ཞིག་སེང་གེའི་ཁྲི་ལ་སོགས་པ་ཚོགས་པར་བྱས་ལ། དཔའ་ཆེན་པོ་མི་ལྡོག་པའི། །ཞེས་བྱ་བ་ལ་སོགས་པའི་ཚིག་གིས་སློབ་དཔོན་དུ་འོས་པ་དེ་ལ་གསོལ་བ་འདེབས་སུ་བཅུག་ལ་སེང་གེའི་ཁྲིའི་སྟེང་དུ་བཞག་སྟེ། ཐོག་མཐའ་མེད་པའི་སེམས་དཔའ་ནི།། ཞེས་བྱ་བ་ལ་སོགས་པའི་ཚིག་གིས་ཆོས་ཀྱི་ཕུང་པོ་བརྒྱད་ཁྲི་བཞི་སྟོང་གི་བརྗོད་བྱའི་རང་བཞིན་ཅན་གཉིས་སུ་མེད་པའི་རྡོ་རྗེ་ལག་པ་གཡས་པར་སྦྱིན། འདི་ནི་རྣལ་འབྱོར་མ་ཀུན་གྱི། །བྱ་བ་ནས། སྲིད་པ་ངོ་བོ་ཉིད་ཀྱིས་བདག །ཅེས་བྱ་བ་ལ་སོགས་པ་ཆོས་ཀྱི་ཕུང་པོ་རྗོད་བྱེད་ཀྱི་རང་བཞིན་གྱི་དྲིལ་བུ་ལག་པ་གཡོན་པར་སྦྱིན་ལ། དེ་གཉིས་འཁྱུད་པའི་ཚུལ་དུ་བསྣོལ་ལ། རང་གི་འོད་ཀྱི་རིག་མ་ཕག་མོ་ལ་སོགས་པའི་ཤེས་རབ་བཅུ་དྲུག་ལོན་པ་ལ་འཁྱུད་པར་བསམ་དུ་གཞུག དེ་ཙམ་ནི་སློབ་དཔོན་གྱི་དབང་གི་ངོ་བོ་ཡིན་ལ། དེ་ཡང་ཀྱེ་རྡོ་རྗེ་ལས།

ཤེས་རབ་བཅུ་དྲུག་ལོན་པ་དང༌། །ལག་པ་གཉིས་ཀྱིས་འཁྱུད་པ་དང༌། །རྡོ་རྗེ་དྲིལ་བུ་མཉམ་ལྡན་ལ། །སློབ་དཔོན་དབང་དུ་ཤེས་པར་བྱ། །ཞེས་གསུངས་པས་སོ། །དེ་ནས་དེ་ལ་དཀྱིལ་འཁོར་དང་ལྷའི་དེ་ཁོ་ན་ཉིད་བསྟན་ཅིང་བསྙེན་པ་ལ་སོགས་པ་སློབ་དཔོན་གྱི་ལས་རྣམས་ཀྱང་བསླབས་ལ། སློབ་དཔོན་གྱི་རྗེས་སུ་གནང་བ་སྦྱིན་པར་བྱ་སྟེ། དེ་རིང་ ¡Ã དཀྱིལ་འཁོར་ ( ¡¶ ལུ ¡· རྡོ་རྗེ་ ) སློབ་དཔོན་འགྱུར། ། སྔགས་དང་རྒྱུད་རྣམས་འཛིན་གྱུར་བར། །སངས་རྒྱས་བྱང་ཆུབ་སེམས་དཔའ་དང་ ༑ ༑ ལྷ་རྣམས་ཀྱིས་ཀྱང་བཞེད་པ་ཡིན། །སེམས་ཅན་རྣམས་ནི་རྗེས་གཟུང་ཕྱིར ༑ ༑ དཀྱིལ་འཁོར་ཆོ་ག་བཞིན་དུ་ནི། ཁྱོད་ཀྱིས་ནན་ཏན་བསྐྱེད་ནས་བྲིས། ། སྒྲུབ་པ་པོ་རྣམས་རྒྱུད་ལ་སྦྱོར། །ཞེས་བྱ་བ་རྗེས་སུ་གནང་བ་བྱིན་ལ། དེ་ནས་བསྒྲུབ་པ་དང་ཐུན་མོང་དུ་སྦྱིན་པ་ནི། དེང་ནས་བརྩམས་ཏེ་རྫོགས་སངས་རྒྱས།། ཞེས་བྱ་བ་ལ་སོགས་པས་རྗེས་སུ་གནང་བའི་དབང་སྦྱིན་པར་བྱ་བ་ནི་གསུམ་སྟེ། མཆོད་པའི་རྗེས་གནང་དང༌། ཆོས་ཀྱི་དེ་ཁོ་ན་ཉིད་དང༌། ཞི་དང་རྒྱས་དང༌། ཞེས་བྱ་བ་ལ་སོགས་པས་དངོས་གྲུབ་བསྒྲུབ་པའི་རྗེས་སུ་གནང་བའོ། །ནངས་པར་སྔར་ལངས་སེམས་དག་ཉིད། །ཅེས་བྱ་བ་ནས། འདོད་ཆགས་བྲལ་བ་ལྟ་བུ་ཡི།། ཞེས་བྱ་བའི་བར་ནི། དམ་ཚིག་བཀའི་དབང་སྟེ། དཀྱུས་ན་གསལ། ལུང་བསྟན་པ་དང་དབུགས་དབྱུང་བ་ཡང་དཀྱུས་ན་གསལ། དེ་ནས་དྲིན་ལ་ལན་གྱིས་བླན་པའི་ཚུལ་དུ་སློབ་མས་ཡོ་བྱད་ཅི་འབྱོར་བ་དབུལ་ཞིང༌། མཐར་བདག་ཉིད་ཀྱང་དབུལ་བར་བྱའོ། །དེ་ལྟར་ན་གོང་གི་དབང་དེ་རྣམས་རེ་རེ་ཞིང་ཡང་ཆུའི་དབང་འཇུག་པས་ཐམས་ཅད་དུ་བུམ་པའི་བྱ་བ་འཇུག་པའི་ཕྱིར། བུམ་པའི་དབང་ཞེས་བྱའོ ༑ ༑ གསང་བའི་དབང་ནི། སློབ་དཔོན་ཞི་བ་ཆེན་པོ་ཀྱེ། །ཞེས་བྱ་བ་ལ་སོགས་པས་གསོལ་བ་གདབ་ཅིང༌། རྒན་པོ་དང་བསླབ་པས་བཅིངས་པ་རྣམས་ཀྱིས་ནི། ཆོ་ག་ཇི་ལྟ་བ་བཞིན་དུ་བདུད་རྩི་ལ་སོགས་པ་དང་བསྲེས་ལ་སྦྱིན་ནོ། ། སློབ་དཔོན་གཞན་གྱིས་ནི་གསང་བའི་ཡུམ་ཇི་ལྟ་བ་བཞིན་དུ་བསྟེན་ལ་སྦྱིན་ནོ།།

ཤེས་རབ་ཡེ་ཤེས་ནི་རྒན་པོ་དང་བསླབ་ ( ¡¶ ཞ ¡· བསླབས་ ) པས་བཅིངས་བའི་སློབ་མ་རྣམས་ལ་ནི་དགྱེས་པ་རྡོ་རྗེ་ལས་གསུངས་པའི་རང་གི་གཡས་དང་ལག་གཞན་གྱིས ༑ ༑ ཞེས་བྱ་བས་རྩ་གཉིས་པོ་བཙིར་ཏེ་ཉམས་མྱོང་སྐྱེས་པའམ་ཡེ་ཤེས་ཀྱི་ཕྱག་རྒྱ་ལས་སྦྱིན་པའོ། །དེ་ལས་གཞན་པའི་སློབ་མ་རྣམས་ལ་ནི་ཇི་ལྟ་བ་བཞིན་དུ་དངོས་ཀྱི་ཕྱག་རྒྱ་བསྟེན་དུ་གཞུག་གོ །དབང་དེ་གཉིས་ཀ་ཡང་གསོལ་བ་འདེབས་པའི་ཚིག་ནི་གཅིག་གོ །དེ་ནས་བཞི་པའི་གསོལ་བ་གདབ་པར་བྱ་སྟེ། ཁྱོད་ཞབས་པདྨའི་བཀའ་དྲིན་གྱིས། །གཙོ་བོ་དབང་བསྐུར་རྣམས་གསུམ་ཐོབ།། དེ་རིང་རིན་ཆེན་བཞི་པ་ཡང༌། །གཙོ་བོས་བདག་ལ་བཀའ་དྲིན་སྩོལ། །ཞེས་བྱ་བ་ལན་གསུམ་གྱི་བར་དུ་བཀའ་དྲིན་གསོལ་བ་བཏབ་ལ། སློབ་དཔོན་གྱིས་ཀྱང༌། ཁྱོད་ཀྱིས་སྲོག་ཆགས་བསད་པར་བྱ། །ཞེས་བྱ་བ་ལ་སོགས་པའི་དམ་ཚིག་བཞི་དབོག་ཅིང༌། ཡེ་ཤེས་ཆེན་པོ་ཕྲ་བ་ནི། །ཞེས་བྱ་བ་ལ་སོགས་པའི་ཚིག་གི་སྒོ་ནས ༑ གསུམ་པ་དང་བརྡ་སྦྱར་བ་ནི་བཞི་པའོ། །ནག་པོ་པའི་དཀྱིལ་འཁོར་གྱི་ཐོ་ཡིག་ཙམ་དུ་སྦྱར་བའོ། །རྫོགས་སོ། །ས་མཱཔྟ་མི་ཐི། །དགེའོ།། །། ས་ཆེན་ཀུན་དགའ་སྙིང་པོའི་གསུང་རབ་དཔེ་བསྡུར་མ་པོད་གསུམ་པ། མེས་པོའི་ཤུལ་བཞག་ལས། ས་སྐྱ་བཀའ་འབུམ། PAGE 540PAGE 541

འདི་དག་ནི་སྔར་ཐུན་མཚམས་ཀྱི་སྐབས་སུ་བསྟན་པ་ལྟར་བྱའོ། །སྤྲོས་པ་ལ་དགའ་བ་རྣམས་ཀྱིས་ནི། མཆོད་པ་དང་བསྟོད་པ་རྣམས་ལ་དབྱངས་དང་སྦྱར་ཏེ་དབུལ་ལོ། །དེ་ནས་ལག་པ་གཡས་པས་རྡོ་རྗེ་དང་བཅས་པས་སྤོས་བཟུང་སྟེ། ། པུས་མོ་གཡས་པའི་ལྷ་ང་ས་ལ་བཙུགས་ཏེ། དྲིལ་བུ་གསིལ་ཞིང་སྙན་གསན་ཕབ་པ་བརྗོད་པར་བྱའོ། །ཧཱུྃ་ཧེ་རུ་ཀ་དཔལ་དཔའ་བོ་ཆེ། །རིག་པའི་རྒྱལ་པོར་ཕྱག་འཚལ་བསྟོད། །དཀྱིལ་འཁོར་ཐུགས་རྗེའི་བདག་ཉིད་ཅན། །མགོན་པོ་བདག་ནི་འདྲི་བར་འཚལ། །སློབ་མ་རྣམས་ལ་བརྩེ་བ་དང༌། ཁྱོད་ལ་མཆོད་པ་བྱ་བའི་ཕྱིར

༑༑ བཅོམ་ལྡན་བདག་ནི་གུས་པ་ལ། །དེ་བས་བཀའ་དྲིན་མཛད་པར་འོས། ། འགྲོ་བ་འཁོར་ལོའི་དོན་མཛད་པའི། །སངས་རྒྱས་བདག་ལ་དགོངས་སུ་གསོལ།། འབྲས་གནས་བྱང་ཆུབ་སེམས་དཔའ་དང༌། །སྔགས་ཀྱི་ལྷ་གཞན་གང་ལགས་དང་ ༑༑ འཇིག་རྟེན་སྐྱོང་བའི་ལྷ་རྣམས་དང༌། །གང་ཡང་འབྱུང་པོ་མཐུ་ཆེན་དང༌།། སེམས་ཅན་བསྟན་ལ་དགའ་བ་དང༌། །གང་དུ་རྡོ་རྗེ་འཛིན་པ་རྣམས། །བདག་ལ་དགོངས་པར་མཛད་དུ་གསོལ། །བདག་མིང་འདི་ཞེས་བགྱི་བ་ཡིས། །ཧེ་ཞེས་བྱ་བའི་དཀྱིལ་འཁོར་ནི། །ཡོ་བྱད་ཅི་འབྱོར་དག་གིས་ནི། །འགྲོ་ལ་ཕན་ ( ¡¶ ཞ ¡· དག་ ) ཕྱིར་བདག་འདྲི་བས། །བདག་དང་སློབ་མར་བཅས་པ་ལ། །ཐུགས་རྗེས་ཐུགས་དཔག་མཛད་ནས་སུ། །དཀྱིལ་འཁོར་བར་བཅས་ཐམས་ཅད་ཀྱིས། ། ཐུགས་རྗེས་དགོངས་ནས་གནང་བར་མཛོད། །ཅེས་དེ་ལྟར་ལན་གསུམ་གསོལ་བ་གདབ་སྟེ་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· + ཁྱེད་ཀྱིས་གཞན་དོན་རྒྱ་ཆེན་པོ་གྱིས་ཤིག་ཅེས་ ) ལྷས་གནང་བ་མཛད་པར་བསམ། དེ་རྗེས་ལ་གཏོར་མ་གཏོང་བའམ་ཡང་ན་བུམ་པ་སྟ་གོན་གྱི་རྗེས་ལ་གཏོར་མ་བཏང་ངོ༌། ། ¡Ã གཏོར་མ་དེའི་རྗེས་ལ་བཏང་ན་དེ་ཉིད་ཀྱི་དུས་སུ་དཀྱིལ་འཁོར་ནམ་མཁའ་ལ་གཤེགས་སུ་གསོལ། ཡང་ན་ནངས་པར་ཐོ་རངས་གཤེགས་སུ་གསོལ་བར་བྱ་སྟེ། ( ¡¶ ལུ ¡· - གཏོར་མ་་་་་་་བྱ་སྟེ། ) དེ་དག་ནི་ལྷ་སྟ་གོན་ལ་གནས་པའི་ཆོ་གའོ། །དེ་ནས་བུམ་པ་སྟ་གོན་ལ་གནས་པར་བྱ་བའི་ཕྱིར ༑ ( ¡¶ ཞ ¡· + རང་ ) ལྷའི་ང་རྒྱལ་གསལ་བས་བདུད་རྩི་འཁྱིལ་པའི་སྔགས་བརྗོད་ཅིང་ ༑ ལས་ཀྱི་རྡོ་རྗེའམ་ཀུ་ཤའི་ཆུན་པོས་བདུད་རྩིའི་བསངས་ཆུས་བུམ་པ་རྣམས་ལ་གཏོར་ཞིང་བགེགས་བསལ། ཨོཾ་སྭ་བྷཱ་ཝའི་སྔགས་བརྗོད་སྟོང་པར་བསམས་ལ། སྟོང་པའི་ངང་ལས་པཾ་བཅུ་ལས་པདྨ་བཅུའི་སྟེང་དུ་ཨ་ལས་ཟླ་བའི་དཀྱིལ་འཁོར་བཅུ། དེ་རྣམས་ཀྱི་སྟེང་དུ་བྷྲཱུཾ་ལས་རིན་པོ་ཆེའི་བུམ་པ་ལྟོ་བ་ཡངས་ཤིང་མགྲིན་པ་ཕྲ་བ་བཅུ་བསྐྱེད། དེ་ནས་རྣམ་པར་རྒྱལ་བའི་བུམ་པའི་ནང་དུ་མངོན་པར་བྱང་ཆུབ་པ་ལྔ་ལས་གཙོ་བོ་ཡབ་ཡུམ་དུ་བསྐྱེད།

དེའི་མཁའ་གསང་སྒྲུབ་ཐབས་བཞིན་བྱིན་གྱིས་བརླབས་ལ། སྙོམས་པར་ཞུགས་པའི་སྒྲས་དེ་བཞིན་གཤེགས་པ་ཐམས་ཅད་སྤྱན་དྲངས་ཏེ། ཞལ་ནས་ཞུགས་ཏེ་ཐུགས་ཀར་འོད་དུ་ཞུ་སྟེ་ཡུམ་གྱི་བྷ་གར་ས་བོན་དུ་གྱུར། དེ་ལས་ལྷ་མོ་བརྒྱད་བདུད་རྩི་འཁྱིལ་པ་ ( ¡¶ ཞ ¡· + སྐུ་མདོགས་ལྗོང་ཁུ་ ) གཡས་རྡོ་རྗེ་རྒྱ་གྲམ། གཡོན་དེས་ཡུ་བ་བྱས་པའི་དྲིལ་བུ་འཛིན་པ་ ¡Ã སྐུ་མདོག་ལྗང་གུ་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· - སྐུ་མདོག་ལྗང་གུ་ ) དང་དགུ་བསྐྱེད་ལ། བུམ་པ་དགུར་སྤྲོས་ཏེ་དམ་ཚིག་པ་བསྐྱེད་པ་དང༌། སྐྱེ་མཆེད་དྲུག་སྐུ་གསུང་ཐུགས་བྱིན་གྱིས་བརླབས་པ་དང༌། དེ་ལ་ཡེ་ཤེས་པ་གཞུག་པ་དང༌། དབང་བསྐུར་བའི་བར་དུ་ཡན་ལག་བཞི་རྫོགས་པར་བྱས་ལ། ཕྱི་ནང་གི་མཆོད་པས་མཆོད་དེ། གཟུངས་ཐག་ནས་བཟུང་སྟེ་བཟླས་པ་བྱ། དེའི་དམིགས་པ་ནི། བདག་གི་ཞལ་ནས་སྔགས་ཀྱི་འོད་ཟེར་དམར་པོ་བྱུང་ནས་གཟུངས་ཐག་ལ་ཞོན་སྟེ་སོང་བས། བུམ་པའི་ནང་གི་ལྷ་རྣམས་ལ་ཕོག་པས་དེ་རྣམས་དམར་ལམ་ལམ་སོང་ནས་བདག་གི་དབང་དུ་འདུ་ཞིང༌། དེ་དག་གི་བ་སྤུའི་བུ་ག་ནས་བདུད་རྩིའི་ཆུ་རྒྱུན་འཛག་པར་བསམ་ཞིང་སྔགས་བརྒྱ་རྩ་བརྒྱད་བཟླས་སོ། །དེ་ལ་རྣམ་པར་རྒྱལ་བའི་བུམ་པ་ལ་ནི། ཨོཾ་དེ་ཝ་པི་ཙུ་བཛྲ་ཧཱུྃ་ཧཱུྃ་ཧཱུྃ་ཕཊ་སྭཱ་ཧཱ། ཞེས་པ་དང༌། ཨོཾ་ཨཱཿཨྃ་ཧཱུྃ་ཕཊ་སྭཱ་ཧཱ། ཞེས་པ་བརྒྱ་རྩ་བརྒྱད་བཟླས་སོ། །ལྷ་མོ་བརྒྱད་ཀྱི་བུམ་པ་ལ ༑ ཨོཾ་ཨཱཿགྃ་ ( ¡¶ ལུ ¡· + གྷཽ་རཱི་ ) ཧཱུྃ་ཕཊ་སྭཱ་ཧཱ། ཞེས་བྱ་བ་ལ་སོགས་པ་རང་རང་གི་སྔགས་བརྒྱ་རྩ་བརྒྱད་བཟླས་ལ་ལས་ཐམས་ཅད་པའི་བུམ་པ་ལ། ཨོཾ་བཛྲ་ཨ་མྲྀ་ཏ་ཀུཎྜ་ལི་ཧ་ན་ཧ་ན་ཧཱུྃ་ཧཱུྃ་ཧཱུྃ་ཕཊ་སྭཱ་ཧཱ། ཞེས་བརྒྱ་རྩ་བརྒྱད་བཟླས་ལ། མཐར་ཅི་འབྱོར་པས་མཆོད་ལ་བསྟོད་པ་བྱས་ལ། རྣམ་རྒྱལ་བུམ་པའི་ནང་ ( ¡¶ ཞ ¡· སྟེང་ ) གི་ཆུ་དེ་ཐིགས་རེ་བླུགས་པས་ཞུ་བར་བསམ་མོ། །བུམ་པ་ལྷག་པར་གནས་པའི་ཆོ་གའོ ༑༑ སྔར་གཏོར་མ་མ་གཏང་ན་སྐབས་འདིར་གཏོར་མ་གཉིས་པོ་གཏང༌། རྣལ་འབྱོར་པ་རྣམས་མཉེས་པར་བྱ་བའི་ཕྱིར་ཚོགས་ཀྱི་བཟའ་བ་དང་བཏུང་བ་བླངས་ལ་བྱིན་གྱིས་བརླབས་ལ།

ཕུད་དབུལ་བ་ ( ¡¶ ཞ ¡· + ལ་སོགས་པ་ ) ནས་ལྷག་མ་གཏོང་བའི་བར་དུ་ཚོགས་ཀྱི་འཁོར་ལོའི་ཆོ་ག་བཞིན་བྱ། སློབ་མ་སྟ་གོན་དཀྱིལ་འཁོར་གྱི་ཁང་པ་དེ་ཉིད་དུ་བྱེད་ན་བྲིས་སྐུ་ལ་སོགས་པ་ཞལ་དགབ། ལྷ་སྟ་གོན་གྱི་མེ་ཏོག་ཚོམ་བུ་ལ་སོགས་པ་ལ་མེ་ཏོག་ ( ¡¶ ཞ ¡· + སིལ་མ་ ) གཏོར་ཏེ་མི་མཐོང་བར་བྱ། ཡང་ན་དར་ལ་སོགས་པས་དགབ། བུམ་པ་རྣམས་ཀྱང་ཞལ་བཀབ་ལ་མི་མཐོང་བར་བྱའོ། །འཇིག་རྟེན་པའི་གཏོར་མ་གཏང༌། འདས་པའི་གཏོར་མ་བར་ཆད་ཞི་བར་བྱ་བའི་ཕྱིར་བཞག་གོ །དེ་ནས་སློབ་མ་སྟ་གོན་ལ་གནས་པ་ལ་དོན་བཅུ་བཞི་སྟེ། ལུས་ཀྱི་བྱ་བ་དང༌། མྱ་ངན་ལས་འདས་པའི་བསམ་པ་བརྟན་པོ་བསྐྱེད་དུ་གཞུག་པ་དང༌། ནང་གི་དབང་བསྐུར་བ་དང༌། ཁྱད་པར་ཅན་གྱི་དོན་ཞུ་བ་དང་ ༑ གསང་སྔགས་ལ་སྤྲོ་བ་བསྐྱེད་པ་དང༌། རྒྱུན་བཤགས་སྔོན་དུ་སོང་བས་སེམས་བསྐྱེད་པ་དང༌། བསྲུང་བ་བྱ་བ་དང༌། མཆོད་པ་རྒྱས་པས་མཆོད་པ་དང༌། སོ་ཤིང་དོར་བ་དང༌། སྐྱོར་ཆུ་བླུད་པ་དང༌། ཀུ་ཤ་སྦྱིན་པ་དང༌། སྲུང་སྐུད་གདགས་པ་དང༌། ཆོས་ཟབ་མོ་བཤད་པའི་སྒོ་ནས་སྤྲོ་བ་བསྐྱེད་པ་དང༌། རྨི་ལམ་བརྟག་ཅིང་ངན་ན་བསྲུང་བར་བྱའོ། །དེ་ལ་དང་པོ་ནི། སློབ་མ་ཁྲུས་ལེགས་པར་བྱས་ལ་གྲལ་རིམ་པ་བཞིན་འཁོད་དོ། །གླིང་བཞི་རི་རབ་དང་བཅས་པ་ཉི་མ་དང་ཟླ་བ་དང་བཅས་པ་རིན་པོ་ཆེ་སྣ་བདུན་གྱིས་བཀང་བ་བླ་མ་ལ་སེམས་བསྐྱེད་པའི་ཡོན་དུ་དབུལ་བར་བསམ་ཞིང༌། མཎྜལ་འབུལ། མེ་ཏོག་ཕུལ་ཏེ་མེ་ཏོག་གིས་བཀང་བའི་ཐལ་མོ་སྦྱར་ལ་མ་ཡེངས་པར་འདུག་ཏུ་གཞུག་གོ །གཉིས་པ་མྱ་ངན་ལས་འདས་པའི་ ¡Ã སེམས་ ( ¡¶ ཞ ¡· བསམ་པ་བརྟན་པོ་ ) བསྐྱེད་པ་ནི། གང་ཞིག་ཚེ་འདིའམ་ཚེ་གཞན་ལ། སྔགས་ཀྱི་དངོས་གྲུབ་དོན་གཉེར་གང༌། །དཀྱིལ་འཁོར་འདིར་ནི་གཞུག་པར་བྱ། །བསོད་ནམས་འདོད་པས་དེ་ལས་གཞན། །བདག་གཞན་འཇིག་རྟེན་ཕ་རོལ་གཉེར། །དད་པ་ཆེན་པོ་ལྡན་པ་དག །འཇིག་རྟེན་ཕ་རོལ་ལ་དམིགས་ནས། །བློ་ལྡན་དཀྱིལ་འཁོར་གཞུག་བྱ་སྟེ། །འདི་ཡི་འབྲས་བུ་གཉེར་མི་བྱ།

།གལ་ཏེ་འདི་འབྲས་དོན་གཉེར་ན། །འཇིག་རྟེན་ཕ་རོལ་འགྲུབ་མི་འགྱུར ༑ ༑ འཇིག་རྟེན་ཕ་རོལ་ལ་དམིགས་ན། །འབད་པ་མེད་པར་འདིར་འབྲས་འགྲུབ ༑ ༑ ཅེས་ལན་ཅིག་བརྗོད་ལ་དེའི་དོན་རྒྱས་པར་བཤད། མདོར་བསྡུ་ན། ཚེ་འདི་ཞེས་བྱ་བ་ནི་ཚེ་འདིར་ཚེ་རིང་བ་དང༌། ནད་མེད་པ་ལ་སོགས་པ་དང༌། སྐྱེ་བ་ཕྱིར་མར་ལྷ་དང་མི་ལ་སོགས་པའི་བདེ་བ་ཐོབ་པར་འདོད་པ་ཡིན་ཏེ་དེ་ལ་སྙིང་པོ་མེད་པ་ཡིན། འཁོར་བ་ནི་ཉེས་དམིགས་དུ་མ་དང་ལྡན་པ་ཡིན་པར་བཤད་པར་བྱ། གལ་ཏེ་དོན་དུ་གཉེར་ན་དཀྱིལ་འཁོར་ཆེན་པོར་འཇུག་མི་དགོས ༑ དེ་བས་ཁྱེད་དཀྱིལ་འཁོར་ཆེན་པོར་འཇུག་པ་དོན་དུ་གཉེར་བས་ན། སྔགས་ཀྱི་དངོས་གྲུབ་ཅེས་བྱ་བ་བླ་ན་མེད་པ་ཡང་དག་པར་རྫོགས་པའི་སངས་རྒྱས་ཕྱག་རྒྱ་ཆེན་པོ་མཆོག་གི་དངོས་གྲུབ་ཅེས་བྱ་བ་དེ་འདོད་པར་བྱ་དགོས་ཏེ། དེ་དོན་དུ་གཉེར་བ་སྐྱེ་བ་འདི་དང་སྐྱེ་བ་གཞན་གྱི་ཕུན་སུམ་ཚོགས་པ་དང༌། ཉན་ཐོས་དང་རང་སངས་རྒྱས་ཀྱི་རྣམ་པར་གྲོལ་བའམ་འབད་པ་མེད་པར་འཐོབ་སྟེ། དཔེར་ན་འབྲས་ ¡Ã མཁར་མ་ ( ¡¶ ཞ ¡· ཀྱི་ ) ལོ་ལེགས་ན་སོག་ཕུབ་པ་ལ་སོགས་པ་ཞར་ལ་འབྱུང་བ་དང་འདྲའོ། །ཞེས་བྱ་བ་རྒྱས་པར་བཤད་པར་བྱའོ། །གསུམ་པ་ནང་གི་དབང་བསྐུར་བ་ནི། རང་དཔལ་ཀྱེའི་རྡོ་རྗེ་ཡབ་ཡུམ་དུ་གསལ་གདབ་ལ། སློབ་མ་རྣམས་རིམ་པ་བཞིན་དུ་རང་གི་ཞལ་ནས་བཅུག་སྟེ། སྙིང་གའི་ཧཱུྃ་གི་གཟི་བརྗིད་ཀྱིས་ཚེ་ཐོག་མ་མེད་པ་ནས་བསགས་པའི་སྡིག་སྒྲིབ་ཐམས་ཅད་བསྲེགས་ནས་སྐུའི་དབྱིབས་བརྒྱུད། རྡོ་རྗེའི་ལམ་ནས་ཡུམ་གྱི་བྷ་གར་ཧེ་རུ་ཀ་ཕྱག་གཉིས་པའི་རྣམ་པས་བཞག་ལ། རང་གི་སྙིང་གའི་ས་བོན་གྱི་འོད་ཀྱིས་དེ་བཞིན་གཤེགས་པ་ཐམས་ཅད་སྤྱན་དྲངས་ལ། ཡབ་ཀྱི་ཞལ་ལམ་སྣ་གཡོན་ནས་ཞུགས་སྙིང་གར་འོད་དུ་ཞུ། སྐུའི་དབྱིབས་བརྒྱུད་རྡོ་རྗེའི་ལམ་ནས་ཡུམ་གྱི་བྷ་གར་ཕབ་སྟེ ༑ སློབ་མ་ཐམས་ཅད་ཀྱི་སྤྱྀ་བོ་ནས་བྱང་ཆུབ་སེམས་ཀྱི་ཆུ་རྒྱུན་གྱིས་དབང་བསྐུར་བས། ལུས་ཐམས་ཅད་གང་བས་ལུས་ངག་ཡིད་གསུམ་གྱི་དྲི་མ་དག་པར་བསམས་ལ།

ཡུམ་གྱི་དཀུ་ ( ¡¶ ཞ ¡· སྐུ་ ) གཡོན་པ་ནས་རིམ་གྱིས་ཕྱུང་ལ་རང་རང་གི་གྲལ་ལ་རིམ་གྱིས་བཞག་གོ །བཞི་པ་ཁྱད་པར་ཅན་གྱི་དོན་ཞུ་བ་ནི། བདག་མེད་མ་ཡིས་མགྲིན་འཁྱུདཔའི། །དགའ་ཆེན་ཁྱོད་བདག་སྟོན་པ་སྟེ། །སློབ་དཔོན་ཁྱོད་ནི་དགོངས་སུ་གསོལ། །བྱང་ཆུབ་ཆེན་པོའི་ཚུལ་བསྟན་པ། །མགོན་པོ་ཆེན་པོ་བདག་འཚལ་ལོ། །དམ་ཚིག་དེ་ཉིད་བདག་ལ་སྩོལ། །བྱང་ཆུབ་སེམས་ཀྱང་བདག་ལ་སྩོལ། །སངས་རྒྱས་ཆོས་དང་དགེ་འདུན་ཏེ། །སྐྱབས་གསུམ་དག་ཀྱང་བདག་ལ་སྩོལ། །ཐར་པ་ཆེན་པོའི་གྲོང་མཆོག་ཏུ། །མགོན་པོས་བདག་ནི་གཞུག་ཏུ་གསོལ། །ཞེས་པ་སློབ་དཔོན་གྱི་རྗེས་སུ་བཟླས་པ་ལན་གསུམ་གྱིས་གསོལ་བ་གདབ་པར་བྱའོ། །ལྔ་པ་གསང་སྔགས་ཀྱི་ཐེག་པ་ལ་སྤྲོ་བ་བསྐྱེད་པ་ནི། བུ་ཚུར་ཐེག་པ་ཆེན་པོ་ཡི། ཁྱོད་ནི་ཚུལ་ཆེན་སྣོད་ཡིན་ཏེ། །གསང་སྔགས་སྤྱཽད་ཚུལ་ཆོ་ག་ནི། ཁྱོད་ལ་ཡང་དག་བསྟན་པར་བྱ། །རྫོགས་པའི་སངས་རྒྱས་གང་འདས་དང་ ༑ ༑ དེ་བཞིན་གང་དག་མ་བྱོན་དང༌། །ད་ལྟར་བྱུང་བའི་ ¡Ã མགོན་པོ་རྣམས ( ¡¶ ཞ ¡· རྫོགས་སངས་རྒྱས ) ། །འགྲོ་ལ་ཕན་ཕྱིར་བཞུགས་པ་ཀུན། །དེ་དག་ཀུན་གྱི་གསང་སྔགས་ཀྱི། །ཆོ་ག་བཟང་པོ་འདི་མཁྱེན་ནས། །དཔའ་བོས་བྱང་ཆུབ་ཤིང་དྲུང་དུ། །ཐམས་ཅད་མཁྱེན་པའི་མཚན་མེད་བརྙེས། །གསང་སྔགས་སྦྱོར་བ་མཉམ་མེད་དེ། །ཤཱཀྱ་སེང་གེ་སྐྱོབ་པ་ཡིས། །དྲག་པོ་ཆེན་པོ་བདུད་ཀྱི་སྡེ། ། སྟོབས་ཆེན་དག་ཀྱང་དེས་བཅོམ་སྟེ། །འཇིག་རྟེན་མཐུན་པར་གཤེགས་ནས་ནི།། འཁོར་ལོ་བསྐོར་ནས་མྱ་ངན་འདས། །དེ་བས་ཀུན་མཁྱེན་ཐོབ་བྱའི་ཕྱིར། །བུ་ཡི་བློ་གྲོས་འདི་གྱིས་ཤིག །ཅེས་ལན་ཅིག་བརྗོད་ལ་དེའི་དོན་རྒྱས་པར་བཤད་དེ། མདོར་ན་ཁྱོད་ནི་ཚུལ་ཆེན་སྣོད་ཡིན། ཞེས་པ་ནི། ཁྱོད་སྙིང་རྗེ་དང་ཤེས་རབ་དང་ལྡན་པས། ཐེག་པ་ཆེན་པོའི་ཆོས་ཀྱི་སྣོད་ཡིན་ལ། ཐེག་པ་ཆེན་པོ་ལའང་གཉིས་ཏེ ༑ ཕ་རོལ་ཏུ་ཕྱིན་པའི་ཐེག་པ་ཆེན་པོ་དང༌། གསང་སྔགས་ཀྱི་ཐེག་པ་ཆེན་པོ་གཉིས་ལས། ཁྱོད་ལ་གསང་སྔགས་ཀྱི་ཐེག་པ་ཆེན་པོ་སྟོན་པར་བྱེད་དེ།

གསང་སྔགས་ཀྱི་ཐེག་པ་ཆེན་པོ་འདི་ནི་དུས་གསུམ་གྱི་སངས་རྒྱས་ཐམས་ཅད་ཀྱང་གཤེགས་པའི་ལམ་ཡིན་པས། ཁྱོད་ཀྱང་བློ་འདི་ལས་མི་འགྱུར་བར་གྱིས་ཤིག་ཅེས་བཤད་པ་རྒྱས་པར་བྱའོ། །དྲུག་པ་རྒྱུན་བཤགས་སྔོན་དུ་སོང་ནས། བྱང་ཆུབ་ཀྱི་མཆོག་ཏུ་སེམས་བསྐྱེད་པ་ནི། ཐོག་མ་མེད་པ་ནས་བསགས་པའི་སྡིག་པ་ལ་འགྱོད་པའི་སེམས་བསྐྱེད་དུ་གཞུག སླར་མི་བྱེད་པར་བསྡམ་དུ་གཞུག སོ་སོ་སྐྱེ་བོའི་དང་འཕགས་པ་ཉན་ཐོས་དང་རང་སངས་རྒྱས་དང་བྱང་ཆུབ་སེམས་དཔའ་སྟེ། སེམས་ཅན་ཐམས་ཅད་ཀྱི་དགེ་བའི་རྩ་བ་ལ་ཕྲག་དོག་མི་བྱེད་པར་ཡི་རང་དུ་གཞུག་སྟེ། དེ་ལྟ་བུའི་དགེ་བའི་རྩ་བ་ཐམས་ཅད་བླ་ན་མེད་པའི་བྱང་ཆུབ་ཏུ་ཡོངས་སུ་བསྔོ་རུ་གཞུག དུས་ད་ནས་བཟུང་ནས་སྙིང་པོ་བྱང་ཆུབ་ལ་མཆིས་ཀྱི་བར་དུ་སངས་རྒྱས་ལ་སྟོན་པ། ཆོས་ལ་ལམ། དགེ་འདུན་ལ་གྲོགས་སུ་འགྲོར་གཞུག སེམས་ཅན་ཐམས་ཅད་ཀྱི་དོན་དུ་བླ་ན་མེད་པ་ཡང་དག་པར་རྫོགས་པའི་སངས་རྒྱས་ཐོབ་པར་བྱ། དེའི་དོན་དུ་བྱང་ཆུབ་སེམས་དཔའི་བསླབ་པ་ཐམས་ཅད་ལ་བསླབ་བར་བྱའོ། །ཞེས་བྱང་ཆུབ་ཀྱི་མཆོག་ཏུ་སེམས་བསྐྱེད་དུ་བཅུག་ལ། འདི་སྐད་ཅེས་བརྗོད་དུ་གཞུག་པར་བྱ་སྟེ། དཀོན་མཆོག་གསུམ་ལ་སྐྱབས་སུ་མཆི ༑༑ སྡིག་པ་ཐམས་ཅད་སོ་སོར་བཤགས། །འགྲོ་བའི་དགེ་ལ་རྗེས་ཡི་རངས།། སངས་རྒྱས་བྱང་ཆུབ་ཡིད་ཀྱིས་གཟུང༌། །སངས་རྒྱས་ཆོས་དང་ཚོགས་མཆོག་ལ།། བྱང་ཆུབ་བར་དུ་སྐྱབས་སུ་མཆི། །རང་བཞན་དོན་ནི་རབ་བསྒྲུབ་ཕྱིར། །བྱང་ཆུབ་སེམས་ནི་བསྐྱེད་པར་བགྱི། །བྱང་ཆུབ་མཆོག་གི་སེམས་ནི་བསྐྱེད་བགྱིས་ནས ༑ ༑ སེམས་ཅན་ཐམས་ཅད་བདག་གིས་མགྲོན་དུ་གཉེར། །བྱང་ཆུབ་སྤྱོད་མཆོག་ཡིད་འོང་སྤྱད་པར་བགྱི། །འགྲོ་ལ་ཕན་ཕྱིར་སངས་རྒྱས་འགྲུབ་པར་ཤོག། ཅེས་ལན་གསུམ་བྱའོ། །དེའི་རྗེས་ལ་རིག་ལྔའི་སྡོམ་པ་གཟུང་བ་ ¡Ã བྱས་ཀྱང་ ( ¡¶ ཞ ¡· རྒྱས་པ་བ་འདིར་བྱས་ན་ཡང་ ) འགལ་བ་ཆེན་པོ་མེད་ནའང༌། འོན་ཀྱང་འོག་ཏུ་སློབ་མ་འཇུག་པའི་སྟབས་སུ་བྱའོ།

།བདུན་པ་དེ་ལྟར་བྱང་ཆུབ་ཏུ་སེམས་བསྐྱེད་པ་ལ་བར་ཆད་མི་འབྱུང་བར་བྱ་བའི་ཕྱིར་རྩ་བའི་སྔགས་ཀྱིས་བསྲུང་ཞིང་ལྷར་བསྐྱེད་ཅིང་བྱིན་གྱིས་བརླབ་པ་ནི། སློབ་མ་རྣམས་ལ་དམིགས་ལ་རྩ་བའི་སྔགས་བཟླས་པ་དང༌། སྙིང་གའི་ས་བོན་གྱི་འོད་ཀྱིས་སྡིག་པ་རྣམས་བསྲེགས་པས་ཕྱི་ནང་གི་བར་ཆད་བསྲུངས་ལ། སློབ་མ་རྣམས་སྐད་ཅིག་གིས་ཕྱག་གཉིས་པར་བསམས་ལ། སློབ་དཔོན་གྱི་ཁུ་ཚུར་གཡས་པར་རྡོ་རྗེ་བཟུང༌། གཡོན་པར་དྲི་ཞིམ་པོ་དང་བཅས་པའི་ཆུ་བཟུང་ནས་ལངས་ཏེ། སློབ་མའི་སྤྱྀ་བོར་དྲི་ཞིམ་པོའི་ཆུས་ཐིག་ལེ་བྱ། ཨོཾ་བརྗོད་པས་འཁོར་ལོའི་ལྟེ་བར་ཨོཾ་གྱིས་མཚན་པའམ་སྐུ་རྡོ་རྗེ་མར་བསམ། དེ་བཞིན་དུ་མགྲིན་པར་རེག་ཅིང་ཨཱཿབརྗོད་པས་པདྨ་ཨཱཿས་མཚན་པའམ་གསུང་རྡོ་རྗེ་མར་བསམ། ཐུགས་ཀར་རེག་ཅིང་ཧཱུྃ་བརྗོད་པས་རྡོ་རྗེ་ཧཱུྃ་གིས་མཚན་པའམ་ ( ¡¶ ལུ ¡· + བདག་མེད་མ་ ) ཐུགས་རྡོ་རྗེ་མར་བསམ། དེ་ནས་སྙིང་ག་ནས་མས་རིམ་གྱིས་ཧཱུྃ་ཨཱ་ཨོཾ་བརྗོད་ཅིང་རྡོ་རྗེ་དང་བཅས་པའི་ལག་པས་རེག་ཅིང་གསལ་གདབ་བོ། །ཡང་ཡས་མར་རེག་ཅིང་ཨོཾ་ཨཱཿཧཱུྃ་ཞེས་བརྗོད་པས་གསལ་གདབ་བོ། །བརྒྱད་པ་དེ་ལྟར་སེམས་བསྐྱེད་ཅིང་བྱིན་གྱིས་བརླབས་པ་ལ་མཆོད་པ་ནི། ཨོཾ་བཛྲ་པུཥྤེ་ཨཱཿཧཱུྃ། ཞེས་བྱ་བ་ལ་སོགས་པའི་སྔགས་དང་སྤྱྀ་བོར་མེ་ཏོག་མདུན་དུ་སྤོས། ཐུགས་ཀར་བྱུག་པ། སྤྱན་སྔར་མར་མེ། ཞལ་ཟས་ཞལ་དུ་སྦྱིན་ནོ། །དགུ་པ་དངོས་གྲུབ་ཀྱི་ལྟས་བརྟག་པའི་ཕྱིར། སོ་ཤིང་སྦྱིན་པ་ནི། དཀྱིལ་འཁོར་གྱི་ཤར་སྒ྄ོའམ་ལྷོ་རུ་མཎྜལ་ཁྲུ་གང་བ་བྱུང་རྣམ་པ་ལྔས་བ་ལང་གི་ཀོ་བ་ཙམ་དུ་བྱུགས་ལ། དེའི་སྟེང་དུ་མེ་ཏོག་ཚོམ་བུ་དགུ་དགོད། དེའི་དྲུང་དུ་སློབ་མ་ཁ་ཤར་རམ་བྱང་དུ་བལྟས་ཏེ་བཞག དེ་ནས་སློབ་དཔོན་གྱིས་སྔར་བཤམས་པའི་སོ་ཤིང་ལ་རྩ་བའི་སྔགས་ཀྱི་མངོན་པར་བཟླས། ཨོཾ་ཨཱཿཛི་ཧ་བཛྲ་སཾ་ཝརྡྷ་ནི་ཧཱུྃ་སྭཱ་ཧཱ། ཞེས་བརྗོད་དེ་སློབ་མའི་ཐལ་མོ་སྦྱར་བའི་བར་དུ་འཛིན་དུ་བཅུག་ལ།

འདི་རིན་པོ་ཆེ་གསེར་རམ་དངུལ་ལས་བྱས་པར་མོས་ཤིག དེ་ ¡Ã སོས་ ( ¡¶ ལུ ¡· སོ་ཤིང་ ) ནེམ་ནེམ་བཅའ་བ་དང་སོ་བྲུད་པ་བཞིན་གྱིས་ལ། སྨན་པ་མཁས་པས་སྐྱུག་སྨན་བཏང་བ་ལྟར། བདག་ཉིད་ཀྱི་སྡིག་སྒྲིབ་ཐམས་ཅད་ཕྱིར་ཕྱུང་བར་བསྒོམས་ལ་དབུས་སུ་ཐལ་མོ་ཁ་ཕྱེ་ལ་དོར་ཅིག་ཅེས་བརྗོད་དོ། །དེ་ལྟར་སོ་བྲུད་པ་ལ་སོགས་པ་བྱས་ནས་དོར་བས། བདག་ཉིད་ལ་རྩེ་མོ་བསྟན་ན་མཆོག་འགྲུབ་བོ། །ཐུར་དུ་བལྟས་ན་ས་འོག་འགྲུབ་བོ། །སྟེང་དུ་བལྟས་ན་མཁའ་སྤྱོད་ལ་སོགས་པ་འབྲིང་མཆོག་འགྲུབ་བོ། །གཞན་དག་གིས་ནི་དངོས་གྲུབ་ཕྲན་ཚེགས་གང་ཡང་རུང་བ་འགྲུབ་སྟེ། ཤར་དུ་ཞི་བ། ལྷོར་མངོན་སྤྱོད། ནུབ་ཏུ་དབང་ ༑ བྱང་དུ་རྒྱས་པ། མཚམས་རྣམས་སུ་དངོས་གྲུབ་ཕྲན་ཚེགས་གང་ཡང་རུང་བ་འགྲུབ་ཅེས་བསྟན་ཏོ། །དེ་ནས་ཀུ་ཤ་སྲུང་སྐུད་སྐྱོར་ཆུ་སྔོན་ལ་གང་བྱས་ཀྱང་འདྲ་སྟེ། བརྗོད་པའི་གོ་རིམ་ཀྱི་དབང་དུ་བྱས་ན། བཅུ་པ་ཁ་དག་པར་བྱ་བ་ནི ༑

 རྨི་ལམ་གསལ་པོ་འབྱུང་བར་བྱ་བའི་ཕྱིར་སྐྱོར་ཆུ་བླུད་པ་ནི། དྲི་ཞིམ་པོའི་ཆུ་ལ ༑ ཨོཾ་ཧྲིཿབི་ཤུདྡྷཿསརྦ་དྷརྨཱཿསརྦ་པཱ་པཾ་ནི་ཙ་སྱ་ ( ¡¶ ཞ ¡· སཾ་ ) བི་ཤོ་དྷ་ཡ་སརྦ་བི་ཀལྤ་ཨ་པ་ན་ཡ་ཧཱུྃ། ཞེས་བྱ་བས་མངོན་པར་བསྔགས་ཏེ་གཡས་པའི་སྐྱོར་མོར་ཆུ་བླུགས་ལ་འཐུང་དུ་གཞུག དེ་ནས་བཅུ་གཅིག་པ་རྨི་ལམ་མི་འཁྲུག་པར་བྱ་བའི་ཕྱིར་ཀུ་ཤ་སྦྱིན་པ་ནི། ཡི་གེ་དྷིཿབཟླས་པའི་ཀུ་ཤ་ཆུང་བ་ ( ¡¶ ཞ ¡· + ལག་པ་གཡས་དང་ ) སྔས་སུ་སྦྱིན་ཞིང༌། ཡི་གེ་ཧ་བཟླས་ ¡Ã པའི་ཀུ་ཤ་ཆེ་བ་སྟན་ ( ¡¶ ཞ ¡· པ་ཆེ་བ་གཡོན་དང་གདན་ ) དུ་བཏིང་ངོ༌། །དེ་ནས་བཅུ་གཉིས་པ་རྨི་ལམ་ལ་བར་ཆད་མི་འབྱུང་བར་བྱ་བའི་ཕྱིར་སྲུང་སྐུད་བཅིང་བ་ནི། སྐུད་པ་རྐྱང་པ་ལུས་ཀྱི་ཚད་དང་མཉམ་པ་སུམ་སྒྲིལ་བྱས་ཏེ། དེ་ལ་ཧཱུྃ་བཟླས་ཤིང་རྡོ་རྗེ་མདུད་པ་ཞེས་བྱ་བ་མདུད་པ་སུམ་བརྩེགས་བྱས་ཏེ། དེ་ཐལ་མོའི་བར་དུ་བཅུག་ལ་མི་དམིགས་ཏེ་སྟོང་པའི་ངང་ལས་ཧཱུྃ་ལས་བྱུང་བའི་སྐུད་པར་བསྐྱེད་ལ། རང་གི་སྙིང་གའི་ས་བོན་གྱི་འོད་ཀྱིས་ ( ¡¶ ཞ ¡· + སྟ་གོན་གྱི་ལྷ་ ) ཏ་ཐཱ་ག་ཏ་ཐམས་ཅད་ཀྱི་ཐུགས་ཀ་ནས། ཡེ་ཤེས་ཀྱི་སྐུད་པ་བཀུག་སྟེ་གཉིས་སུ་མེད་པར་བསྟིམ་ལ་རྩ་སྔགས་བཟླས་ཤིང་སྐྱེས་པའི་དཔུང་པ་གཡོན ༑ བུད་མེད་ཀྱི་གཡས་པ་དེ་ལ་ལག་པ་གཉིས་འཇུས་ཏེ། ཨོཾ་བཛྲ་ས་ཏྭ་ས་མ་ཡ་མ་ནུ་པཱ་ལ་ཡ། བཛྲ་ས་ཏྭ་ཏྭེ་ནོ་པ་ཏིཥྛ་དྲྀ་ཌྷོ་མེ་བྷ་ཝ། སུ་ཏོ་ཥྱོ་མེ་བྷ་ཝ། ཨ་ནུ་རཀྟོ་མེ་བྷ་ཝ། ཤ་ཤྭ་ཏོ་མེ་བྷ་ཝ། ཧྲྀ་ད་ཡ་མེ། ཨ་དྷི་ཏིཥྛ། སརྦ་སིདྡྷིམྨེ་པྲ་ཡཙྪ། སརྦ་ཀརྨ་སུ་ཙ་མེ། ཙིཏྟཾ་ཤྲི་ཡཿཀུ་རུ་ཧཱུྃ་ཧ་ཧ་ཧ་ཧ་ཧོཿ བྷ་ག་ཝཱཾ་སརྦ་ཏ་ཐཱ་ག་ཏ་བཛྲ་མ་མཱེ་མུཉྩ། བཛྲཱི་བྷ་ཝ་མ་ཧཱ་ས་མ་ཡ་ས་ཏྭ་ཨ། ཞེས་བརྗོད་པས་བརྟན་པར་བྱའོ ༑༑ ཕྱག་ལེན་ལ་ནི་གཙོར་སོ་ཤིང་རྗེས་ལ་སྲུང་སྐུད། དེ་ནས་སྐྱོར་ཆུ། དེ་ནས་ཀུ་ཤ་སྦྱིན་པ་མཛད་དོ། །དེ་ནས་བཅུ་གསུམ་པ་ཆོས་ཟབ་མོ་བཤད་པའི་སྒོ་ནས་སྤྲཽ་བ་བསྐྱེད་པ་ནི།

སློབ་དཔོན་གྱིས་འདི་སྐད་ཅེས་བརྗོད་དེ། འཇིག་རྟེན་དུ་ནྀ་ཀུན་མཁྱེན་དག །ཨུ་དུམྺ་རའི་མེ་ཏོག་བཞིན། །བརྒྱ་ལམ་ན་ནི་རེས་འགའ་ཞིག། འབྱུང་འགྱུར་ཡང་ནི་མི་འབྱུང་ངོ༌། །གསང་སྔགས་སྤྱོད་ཚུལ་ཆོ་ག་ནི། །དེ་བས་ཀྱང་ནི་དཀོན་པ་སྟེ། །གང་གིས་མཉམ་མེད་སེམས་ཅན་དོན། །བླ་མེད་བྱེད་པར་ནུས་པ་ཡིན། །བསྐལ་པ་བྱེ་བ་དུ་མར་ནི། །སྔོན་དུ་སྡིག་པ་བྱས་པ་གང༌། ། དཀྱིལ་འཁོར་འདི་འདྲ་མཐོང་མ་ཐག །དེ་དག་ཟད་པར་འགྱུར་ཡིན་ན། །གྲགས་པ་མཐའ་ཡས་སྔགས་སྤྱོད་པའི། །ཚུལ་ལ་གནས་ན་སྨོས་ཅི་དགོས། །སྐྱོབ་པའི་གསང་སྔགས་བཟླས་ན་ནི། །གོ་འཕང་བླ་ན་མེད་པར་འགྱུར། །གང་དག་སྤྱོད་པ་མཆོག་འདི་ལ། །བློ་གྲོས་ཤིན་ཏུ་མི་གཡོ་ན། །སྡུག་བསྔལ་ཐམས་ཅད་འབྱུང་བ་ཡི ༑༑ ངན་འགྲོ་དེ་ཡིས་གཅོད་པར་འགྱུར། །བདག་ཉིད་ཆེན་པོ་ཁྱོད་རྣམས་ཀྱིས།། རྙེད་དཀའ་བླ་མེད་དེ་རིང་རྙེད། །གང་ཕྱིར་རྒྱལ་བ་སྲས་བཅས་པ། །བདག་ཉིད་ཆེན་པོ་ཀུན་གྱིས་ནི། །བསྟན་པ་འདི་ལ་ཁྱེད་རྣམས་ཀུན། །ཡོངས་སུ་བཟུང་སྟེ་སྐྱེ་བཞིན་པ། །དེ་ཕྱིར་ཁྱེད་རྣམས་ཐེག་ཆེན་དུ། །སང་ནི་རབ་ཏུ་སྐྱེ་བར་འགྱུར། ། ཐེག་པ་ཆེན་པོ་ཆེར་འབྱུང་བའི། །ལམ་མཆོག་འདི་ནི་དཔལ་དང་ལྡན། ཁྱེད་རྣམས་དེར་ནི་དོང་བས་ན། །རང་བྱུང་སྐལ་པ་ཆེན་པོ་པ། །འཇིག་རྟེན་ཐམས་ཅད་མཁྱེན་པ་ཡི། །དེ་བཞིན་གཤེགས་པ་ཉིད་དུ་འགྱུར། །ཞེས་བརྗོད་ལ་དེ་དག་གི་དོན་རྒྱས་པར་བཤད་པར་བྱའོ། །དེའི་དོན་འདི་ཡིན་ཏེ། འཇིག་རྟེན་ན་གང་དཀོན་ན་ ( ¡¶ ཞ ¡· + སངས་རྒྱས་འབྱུང་བ་དཀོན་ཏེ། དཔེར་ན། ) ཨུ་དུམྺཱ་རའི་མེ་ཏོག་ ¡Ã དཀོན་ཏེ ( ¡¶ ཞ ¡· བཞིན ) ། དེ་ལ་སངས་རྒྱས་གཅིག་འཇིག་རྟེན་དུ་བྱོན་ན། ཨུ་དུམྺཱ་རའི་མེ་ཏོག་གཅིག་འབྱུང་སྟེ། དེའང་རྒྱལ་རིགས་ལ་དཀར་པོ། བྲམ་ཟེའི་རིགས་འཚང་རྒྱ་ན། སེར་པོ་འབྱུང༌། དེ་ཡུམ་གྱི་ལྷུམས་སུ་འཇུག་པའི་དུས་ན་ཁ་འབུ། བལྟམས་པའི་དུས་ན་ཁ་འབྱེ། གཞོན་ནུ་རོལ་རྩེད་མཛད་པ་ན་རྒྱས། དཀའ་ཐུབ་མཛད་པ་ན་རྙིང༌།

ཆོས་ཀྱི་འཁོར་ལོ་བསྐོར་བ་ཤིན་ཏུ་རྒྱས། མྱ་ངན་ལས་འདས་པ་ན་དེ་བསྐམས་ནས་འགྲོའོ། །སངས་རྒྱས་པས་ཀྱང་གསང་སྔགས་དཀོན་ཏེ། གསང་བ་འདུས་པ་ལས། ཐུབ་པ་ཆེན་པོ་མར་མེ་མཛད་ནས་ཐུབ་པ་ཆེན་པོ་འོད་སྲུངས་ཀྱི་བར་གྱིས་ཀྱང་གསང་སྔགས་མ་གསུངས། མ་བྱོན་པའི་སངས་རྒྱས་བྱམས་པ་ནས་ ¡Ã སངས་རྒྱས་ ( ¡¶ ཞ ¡· - སངས་རྒྱས་ ) མོས་པའི་བར་གྱིས་ཀྱང་གསང་སྔགས་མི་གསུང་བས། ཤཱཀྱ་ཐུབ་པའི་འདི་ཁོ་ནས་སྨོན་ལམ་གྱིས་གསུངས་པ་ཡིན་པས་ན་ཤིན་ཏུ་དཀོན་ནོ། །དཀོན་པས་ཅི་སྟེ། དཀོན་པ་དག་རི་བོང་གི་རྭ་ཡང་དཀོན། ནམ་མཁའི་མེ་ཏོག་ཀྱང་དཀོན་ཟེར་ན། བསྐལ་པ་དཔག་ཏུ་མེད་པར་སྡིག་པ་བྱས་པ་ཐམས་ཅད་ཟད་པར་བྱེད་དེ། དཔེར་ན་བྲག་ཕུག་སྟོང་པ་ཁ་བྱང་དུ་བལྟས་པ་ལ་སྣང་བ་གཏན་འོང་མ་མྱོང་བའི་ལོ་བརྒྱ་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· སྟོང་ ) ཕྲག་མང་པོར་ལོན་པ་དེར་མར་མེ་གཅིག་བཏེག་ན། མུན་པ་དེ་ན་རེ། ང་བུང་ཆེ་བ་གྲངས་མང་བ་ལོ་མང་པོ་བསྡད་པ་ཡིན་གྱིས། ཁྱོད་ཀྱིས་བསལ་བར་མི་ནུས་མི་ཟེར་བར་མར་མེ་ཆུང་ངུ་སྐྱེས་མ་ཐག་མུན་པ་ཐམས་ཅད་འཇོམས་པར་བྱེད་པ་ཆོས་ཉིད་ཡིན་ནོ། །དཔེ་དེ་བཞིན་དུ་བསྐལ་པ་དཔག་ཏུ་མེད་པར་སྡིག་པ་བྱས་པ་ཐམས་ཅད་སང་ནུབ་ ( ¡¶ ལུ ¡· + ཀྱི་ ) དཀྱིལ་འཁོར་མཐོང་མ་ཐག་ཟད་པར་བྱེད་པ་ཡིན་ནོ། །དེ་ལྟར་དཀྱིལ་འཁོར་ཆེན་པོ་མཐོང་བ་ལ་ཕན་ཡོན་དེ་ཙམ་ཡོད་ན། གསང་སྔགས་ཀྱི་ལམ་ལ་ཞུགས་པ་ལ་སྨོས་ཅི་དགོས། དེ་ལྟ་བུའི་རྙེད་དཀའ་ལ་ཕན་ཡོན་ཆེ་བའི་གསང་སྔགས་ཁྱེད་ཀྱིས་ཐོབ་ ( ¡¶ ཞ ¡· ཐོས་ ) ཀྱིས་ཁྱེད་དགའ་བར་གྱིས་ཤིག ཅེས་བྱ་བ་ལ་སོགས་པའི་ཆོས་ཟབ་མོ་བཤད་དོ། །དོན་བཅུ་བཞི་པ་རྨི་ལམ་བརྟག་ཅིང་ངན་ན་བསྲུང་བ་ནི། སློབ་མ་རྣམས་ལ། ཁྱེད་དཀྱིལ་འཁོར་གྱི་དྲུང་ལ་སོགས་པ་ཇི་ལྟར་བདེ་བའི་གནས་སུ་སྔར་བསྟན་པའི་ཀུ་ཤ་སྔས་དང་སྟན་དུ་ཐིང་ལ། ཕན་ཚུན་གཅིག་ལ་གཅིག་མ་རེག་པར་གྱིས་ལ། སེང་གེའི་ཉལ་སྟབས་ཞེས་བྱ་བ་གློ་གཡས་པ་ས་ལ་ཕོབ་ལ་ཉོལ་ཅིག

བསམ་པའང་སང་ནུབ་དཀྱིལ་འཁོར་ཆེན་པོ་ཇི་ལྟ་བུ་ཞིག་མཐོང་སྙམ་པ་ཡིད་ལ་གྱིས་ལ། གཞན་འདོད་ཆགས་དང་ཞེ་སྡང་ལ་སོགས་པ་ཡིད་ལ་མ་སྐྱེད་ཅིག རྨི་ལམ་ཅི་མཐོང་བ་ཡང་ང་ལ་སྨྲོས་ཤིག ¡Ã བུ་ཀྱེ་ནངས་པར་དཀྱིལ་འཁོར་ནི། །མཐོ་བར་འགྱུར་རོ་ཉོལ་ཅིག་དང་ ༑ ༑ རྨི་ལམ་མཐོང་བ་གང་ཡིན་པ། །དེ་ཀུན་དྲང་པོར་ང་ལ་སྨྲོས། ། ( ¡¶ ལུ ¡· - བུ་ཀྱེ་་་་་་་ལ་སྨྲོས། ། ) ཞེས་བསྒོས་ལ་བཏང་ངོ༌། །སློབ་མས་ཀྱང་སློབ་དཔོན་ལ་ཕྱག་འཚལ་ཏེ་ཇི་ལྟར་བདེ་བའི་གནས་སུ་སོང་ལ། སློབ་དཔོན་གྱིས་ཇི་ལྟར་བསྒོ་བ་བཞིན་བྱས་ལ་ཉལ་བར་བྱའོ། །དེ་ནས་སློབ་དཔོན་བདག་ཉིད་ནངས་པར་ཐོ་རངས་ལྡང་བའི་རྣལ་འབྱོར་གྱིས་ཕྱག་གཉིས་པའི་ཧེ་རུ་ཀར་ལངས་ལ། ཆོས་ཐམས་ཅད་རང་བཞིན་གྱིས་རྣམ་པར་དག་པར་མོས་ཏེ། ཨོཾ་སྭ་བྷཱ་ཝ་ཤུདྡྷཿསརྦ་དྷརྨཱཿསྭ་བྷཱ་ཝ་ཤུདྡྷོ྅་ཧཾ། ཨོཾ་བཛྲ་ཤུདྡྷཿསརྦ་དྷརྨཱ་བཛྲ་ཤུདྡྷོ྅་ཧཾ། ཨོཾ་ཡོ་ག་ཤུདྡྷཿསརྦ་དྷརྨཱ་ཡོ་ག་ཤུདྡྷོ྅་ཧཾ། ཞེས་བརྗོད་དེ་དག་པ་རྣམ་གསུམ་དྲན་པར་བྱའོ། །དེ་ནས་དྲིའི་དཀྱིལ་འཁོར་ལ་གནས་པའི་སྟ་གོན་ཅི་འབྱོར་པས་མཆོད་པ་དང་བསྟོད་པ་བྱས་ཏེ་ཡིད་ཀྱིས་ནམ་མཁའ་ལ་འདེགས་སོ། །དེ་ནས་མེ་ཏོག་གི་ཚོམ་བུ་རྣམས་བསྡུ་བ་དང༌། བུམ་པ་རྣམས་གནས་དབེན་པར་བཞུགས་ཏེ་དགབ་པ་དང༌། མཆོད་པ་དམན་པ་རྣམས་གཤེགས་སུ་གསོལ་བ་དང༌། དཀྱིལ་འཁོར་ཀུན་ཏུ་བྱུགས་ཏེ་ལས་ཀྱི་རྗེས་མི་མངོན་པར་བྱའོ། །དེ་ནས་སློབ་མ་དེ་རྣམས་དེར་བོས་ལ་ཁྱེད་ཀྱིས་རྨི་ལམ་དུ་ཅི་ལྟ་བུ་མཐོང༌། ཞེས་དྲི་བར་བྱའོ། །སློབ་མས་ཅི་ལྟ་བ་བཞིན་དུ་སྨྲ་བར་བྱ་བ་དང༌། ཐོས་ནས་བཟང་དང་མི་བཟང་བ། །ཐེ་ཚོམ་མེད་པར་བཤད་པར་བྱ། །རྨི་ལམ་ཐམས་ཅད་བདེན་མིན་ཡང༌། །དེ་ལ་མ་དད་རིགས་མ་ཡིན། །དེ་ལ་རྨི་ལམ་གྱི་ལྟས་བཟང་བ་རྣམས་གསང་བ་སྤྱྀ་རྒྱུད་ལས་འབྱུང་བ། རྨི་ལམ་སངས་རྒྱས་མཐོང་བ་ནི། །ཡོན་ཏན་རྒྱ་མཚོ་མངོན་སུམ་བཞིན ༑ ༑ མཆོད་རྟེན་དང་ནི་སྐུ་གཟུགས་མཐོང༌།

།དེ་ལ་མཆོད་པར་བརྩམ་པ་དང་ ༑ ༑ དགེ་སློང་དག་ལ་ཆོས་ཉན་ཏམ། །ཡང་ན་མ་ཉན་ན་ཡང་རུང༌། །ཆོས་ནི་རྣམ་པར་སྤྱོད་པ་དང༌། །དེ་བཞིན་མདོ་སྡེ་ཀློག་པ་དང༌། །རྨི་ལམ་དགེ་འདུན་མཐོང་བ་དང༌། །ཡང་ན་དགེ་སློང་གཅིག་པུ་མཐོང༌། །དེ་དང་འགོགས་ཤིང་གཏམ་ཟེར་དང༌། །ཡང་ན་དགེ་སློང་དངོས་སུ་གྱུར། །གཙུག་ལག་ཁང་དག་མཐོང་བ་དང༌། །དེ་བཞིན་དགེ་སློང་མ་དག་མཐོང༌། །རྒྱལ་པོའི་སྲས་དག་མཐོང་བ་དང༌། །དེ་བཞིན་ཕ་མ་བླ་མ་རྣམས། །མཛའ་བོ་རྣམ་པ་སྣ་ཚོགས་དང༌། །རྨི་ལམ་དག་ཏུ་མཐོང་ན་ནི ( ¡¶ ཞ ¡· ཤིས ) ། །རིག་སྔགས་དང་ནི་དངོས་གྲུབ་ཐོབ། །དེ་བཞིན་སློབ་མ་འཛིན་པ་དང༌། །ཀུན་དགའ་རྣམ་པ་སྣ་ཚོགས་དང༌། །འབབ་ཆུ་རྣམས་དང་རྒྱ་མཚོ་དང༌། །རི་དང་གླང་པོ་ཆེ་དག་ཀྱང༌། །མཐོང་ན་ཡང་ནི་བཟང་པོར་བསྒྲགས། །ཞེས་བྱ་བ་ལ་སོགས་པ་རྒྱལ་པོར་དབང་བསྐུར་བ་དང༌། བཞོན་པ་ཞོན་པ་དང༌། ནོར་བུ་དང་འབྲུ་དང་ཟས་མཆོག་ཏུ་གྱུར་པ་ཐོབ་པ་ལ་སོགས་པ་རྣམས་བཟང་ ( ¡¶ ལུ ¡· མང་ ) པོར་ཤེས་པར་བྱའོ། །མི་དགེ་བའི་ལྟས་ནི། རྒྱ་སྐྱེགས་ཀྱི་རི་ལ་འཛེག་པ་དང༌། ཀ་ར་བཱི་རའི་མེ་ཏོག་རྒྱས་པ་ཐུ་བ་དང༌། མཆོད་སྡོང་དང་བྱེ་མའམ་ཐལ་ཕུང་ལ་འཛེག་པ་འམ། བོང་བུ་ལ་ཞོན་ནས་ཁ་ལྷོ་ཕྱོགས་སུ་བལྟས་ཏེ་སོང་ནས་མ་ལོག་པ་དང༌། སྤྲེའུ་ལ་ཞོན་ནས་ལྷོ་ཕྱོགས་སུ་འགྲོ་བའམ། བུད་མེད་ ( ¡¶ ཞ ¡· + ནག་མོ་ ) རབ་ཏུ་ཁྲོས་པའམ། བུད་མེད་ནག་མོ་གོས་ནག་ཅན་ལ་འདོད་པ་སྤྱཽད་ཅིང༌། ( ¡¶ ལུ ¡· + ཀ་ར་བི་རའི་ཕྲེང་བ་བཅིངས་བ་དང༌། ) གཡང་ས་དང་གཅོང་རོང་དུ་འདུག་ ( ¡¶ ལུ ¡· འཇུག་ ) པ་དང༌། ཤིང་ལས་ལྷུང་བ་དང༌། ལྗོན་པའི་འབྲས་བུ་མེད་པའི་སྐམ་པོ་མཐོང་བ་དང༌། ཁྱི་དང་ཁྭ་དང་སྲིན་པོ་ལ་སོགས་པས་ཟོས་པ་དང་ ༑ ཕག་དང་རྔ་མོང་དང་མ་ཧེ་ལ་ཞོན་པ་དང༌། མི་ནག་པོ་ལྕགས་ཀྱི་དབྱུག་པ་ཐོགས་པས་བསྡིགས་པ་དང༌། ལུས་ཏིལ་མར་གྱིས་བྱུགས་ནས་ཕྲེང་བ་དམར་པོ་བཅིངས་པ་དང༌། སྐྲ་བྲེགས་ནས་ཕྲེང་བ་དམར་པོ་བཅིངས་པ་ལ་སོགས་པ་རྣམས་མི་དགེ་བའི་ལྟས་སོ།

།དེ་ལྟར་དགེ་བའི་ལྟས་རྣམས་ལ་སྔགས་ཀྱི་དངོས་གྲུབ་རབ་འབྲིང་གསུམ་དུ་འགྲུབ་པར་བསྟན་ཏོ། །མི་དགེ་བའི་ལྟས་རྣམས་ལ་ཆོས་ཐམས་ཅད་སྒྱུ་མ་ལྟ་བུའི་རང་བཞིན་དུ་བསྟན་ལ། རྩ་བའི་སྔགས་བརྒྱ་རྩ་བརྒྱད་ཀྱིས་བསྲུངས་ལ། སངས་རྒྱས་སྤྱན་གྱིས་མན་ངག་ཇི་ལྟ་བ་བཞིན་དུ་བྱིན་ལ ༑ སྲེག་བླུགས་བརྒྱ་རྩ་བརྒྱད་དུ་སྦྱིན། ཞི་བའི་སྦྱིན་སྲེག་བྱའོ། །སློབ་མ་ལྷག་པར་གནས་པའི་ཆོ་གའོ། །དེ་དག་གིས་ནི་སྟ་གོན་གྱི་ཆོ་གའོ། །དེ་ནས་དཀྱིལ་འཁོར་ཆེན་པོ་བྲི་ཞིང་རྒྱན་དགྲམ་པ་ནི། སྤྱྀར་དཀྱིལ་འཁོར་ལ་བྲིས་སྐུ་ལ་བརྟེན་པའི་ལུགས་དང༌། རྡུལ་ཚོན་གྱི་དཀྱིལ་འཁོར་བྲིས་པ་ལ་བརྟེན་པའི་ལུགས་གཉིས་ལས། བྲིས་སྐུ་ལ་བརྟེན་པའི་ལུགས་ལ་ནི། དེའི་ཉིན་མོ་ཇི་ལྟར་བདེ་བར་གནས་ཏེ་ལྷའི་དཀྱིལ་འཁོར་བསྒོམ་པ་ལ་སོགས་པས་དུས་འདའ་བར་བྱའོ། ། རས་རིས་ལ་བརྟེན་པའི་ལུགས་འདི་ནི་བདེ་མཆོག་ཨ་བྷི་དྷ་ནར་གསུངས་པ་ལ་བརྟེན་ནས་སློབ་དཔོན་རྡོ་རྗེ་དྲིལ་བུ་པ་དང༌། སློབ་དཔོན་དགའ་རབ་རྡོ་རྗེ་ལ་སོགས་པས་གསུངས་ལ། དེང་སང་རྒྱ་གར་གྱི་ཡུལ་ནའང་ཆོ་ག་ཕལ་ཆེར་འདི་ལ་བྱེད ༑ བླ་མ་གོང་མ་རྣམས་ཀྱང་ཕལ་ཆེར་འདི་ལ་ཕྱག་ལེན་མཛད་པས་ལུགས་འདི་ནི་ལེགས་པ་ཡིན་ནོ། །དེ་ལ་རྡུལ་ཚོན་གྱི་དཀྱིལ་འཁོར་བྲི་བར་འདོད་ན། དཀྱིལ་འཁོར་ཐིག་གིས་བྲི་བ་དང༌། ཚོན་གྱིས་བྲི་བའོ། །ཐིག་གིས་བྲི་བ་ལ། ཡེ་ཤེས་ཀྱི་ཐིག་གིས་ས་གཞི་བྱིན་གྱིས་བརླབ་པ་དང༌། ལས་ཀྱི་ཐིག་གིས་ཆ་རྣམ་པར་བགོ་བའོ། །ཐིག་གཉིས་པོ་འདི་ལ་གོ་རིམ་ངེས་པ་མེད་དེ། དང་པོར་ཡེ་ཤེས་ཀྱི་ཐིག་བཏབ་ནས་ལས་ཀྱི་ཐིག་བཏབ་ཀྱང་ཆོག དང་པོར་ལས་ཐིག་གདབ་ནས། དེ་ནས་ཡེ་ཤེས་ཀྱི་ཐིག་གི་བྱིན་གྱིས་བརླབས་ཀྱང་ཆོག་ཏེ་གང་བདེ་བར་བྱའོ། །དེ་ནས་ལས་ཀྱི་ཐིག་འདེབས་པར་འདོད་ན། དེའི་ཚུལ་ནི་འདི་ཡིན་ཏེ ༑ དཀྱིལ་འཁོར་ལེགས་པར་བྱུགས་ལ་ ¡Ã ནུབ་མོ་ ( ¡¶ ཞ ¡· - ནུབ་མོ་ ) ལྷ་སྟ་གོན་གྱི་གནས་མི་འཁྲུག་པར་ངོ་ཤེས་པར་བྱས་ལ།

དང་པོ་ཚངས་ཐིག་གདབ་ལ་དེའི་དབུས་ངོ་ཤེས་པར་བྱ། དེ་ནས་ཤར་དང་ནུབ་ཏུ་པཉྩ་རེ་ཁའི་མཚམས་སུ་རི་མོ་རེ་བྱ ༑ དེ་ནས་ཕྱིའི་མཚམས་གཉིས་ཀྱང་དེ་དང་མཉམ་པོར་བྱ་སྟེ། མཉམ་པོ་བཞིར་འགྱུར ༑ དེ་ནས་པཉྩ་རེ་ཁའི་ནང་གཉིས་པོ་དཀྱིལ་འཁོར་གྱི་ཚད་དེ། དེ་བལྟབ་པ་བསྡམས་པའི་ཐིག་ ( ¡¶ ཞ ¡· ཐ་ ) གུ་པཉྩ་རེ་ཁའི་མཚམས་དེར་བཟུང་སྟེ་ལྷོ་བྱང་གཉིས་སུ་བྱ་རོག་རྗེས་བྱ། དེ་ལ་ཚངས་ཐིག་གཉིས་པ་གདབ། ཚད་དེ་ཉིད་ཀྱིས་ཚངས་ཐིག་གི་ཕྱོགས་བཞི་གར་མཉམ། མཉམ་པའི་ས་ནས་ཚད་དེ་ཉིད་བཟུང་སྟེ་མཚམས་བཞིར་བྱ་རོག་རྗེས་རེ་བྱ། དེ་ཉིད་ལ་ཟུར་གྱི་ཐིག་དང་ཕྱིའི་གྲུ་བཞི་བྱ། གྲུ་བཞི་གྲུ་ཆོད་དུ་བྱས་ལ་ཕྱི་ཕྱེད་ནང་ཕྱེད་དུ་བྱས་པ་ནི་རྩ་བའི་ཐིག་གོ །དེ་བཞིན་དུ་གྲུ་ཆོད་བྱ་ཞིང་ཕྱེད་ཕྱེད་གཤགས་ལ་ཆ་ཆེན་བཅུ་དྲུག་ཏུ་བཏང་ངོ༌། །དེ་ལྟར་ན་ཆ་ཆེན་བཅུ་དྲུག་ཏུ་ཐོང་བ་དང་དཀྱིལ་འཁོར་གྱི་ཆ་ཕྲ་མོ་བགོ་བར་བྱ་སྟེ། ཚངས་པའི་ཐིག་གི་དྲུང་ནས་ཆ་ཆེན་དང་པོ་ཞེས་བྱ་བ་བཟུང་སྟེ་གཡས་ན་བརྒྱད་གཡོན་དུ་བརྒྱད་ཡོད་པས་བཅུ་དྲུག་ཡིན་ལ། དེའང་ཕྱོགས་གཅིག་གི་ཤེས་ན་ཕྱོགས་བཞི་ག་དེས་འགྲེའོ། །དེ་ལ་ཕྱོགས་གཅིག་གི་དབང་དུ་བྱས་ན། ཆ་ཆེན་དང་པོ་ཕྱེད་དུ་གདབ། །དེ་ལ་སྒོ་དང་ཉི་དཀྱིལ་འཁོར། ། ¡Ã དེ་ཡི་ཕྱི་ཡང་ཕྱེད་དུ་གདབ ༑ ༑ ( ¡¶ ལུ ¡· - དེ་ཡི་་་་་་་དུ་གདབ། ། ) དེ་ལ་སྒོ་ཁྱུད་པདྨའི་ལྟེ། །ཆ་ཆེན་གཉིས་པ་ཕྱེད་དུ་གདབ། །དེ་ཡི་ནང་མ་ཕྱེད་དུ་གདབ། །དེ་ལ་སྒོ་ལོགས་ནམ་མཁའ་དང༌།། དཀྱིལ་འཁོར་ནང་དུ་པད་ཟེའུ་འབྲུ། །གཉིས་པའི་ཕྱི་མ་ཕྱེད་དུ་གདབ། །དེ་ལ་ཀ་བ་ཀ་ཞུ་འབྱུང༌། །ཆ་ཆེན་གསུམ་པ་ཚུན་ཆོད་ནི། །པདྨ་དང་ནི་རྡོ་རྗེའོ། ། ¡Ã གཉིས་པ་མེ་རི་བཅས་པ་ཡིན། ། ( ¡¶ ཞ ¡·¡¶ ལུ ¡· ར་བ་མེ་རི་བཅས་པ་འབྱུང༌། ། ) ཆ་ཆེན་བཞི་པ་ལྷ་སྣམ་འདོད། །ནང་གི་ཆ་ཡི་དབྱེ་བ་འདི། །འཕང་མ་གཉིས་པའི་ལུགས་ཀྱིས་ནི། །ཡང་ན་ཆ་གཉིས་ལྟེ་བ་སྟེ། །ཆ་གཉིས་པདྨའི་འདབ་མ་ཡིན།

༑ ༑ དེའི་ལག་ལེན་ནི་ཚོན་རྩི་ཁ་དོག་ལྔ་ལེགས་པར་བྱས་ལ། དུར་ཁྲོད་ཀྱི་ ¡Ã སོལ་བ་ལ་སོགས་པས་སྐྱེས་པའི་རུས་པའི་ཕྱེ་མ་ ( ¡¶ ཞ ¡· རུས་པས་ ) སྦགས་པར་བྱས་ལ ༑ དབུས་སུ་ཚོན་རྩི་སྔོན་པོ་ལ་ཧཱུྃ་བྲིས་ལ་དེ་ལ་སྐུད་པ་སྔོན་པོ་བཞག ཤར་དུ་དཀར་པོ་ལ་བྷྲུཾ། ལྷོ་རུ་སེར་པོ་ལ་ཨོཾ། ནུབ་ཏུ་དམར་པོ་ལ་ཛྲཱྀཾ། བྱང་དུ་ལྗང་གུ་ལ་ཁྃ་བྲི། དེ་དག་གི་ཁ་དོག་དང་མཐུན་པར་བསྒྱུར་པའི་སྐུད་པ་རྣམས་ཀྱང་བཞག དེ་དག་གི་མདུན་དུ་མཆོད་པ་ལྔ་དང་གཏོར་མ་གཅིག་བཤམ། རང་རྡོ་རྗེ་དྲིལ་བུ་བཟུང་ལ་ཤར་ཕྱོགས་ནས་འདུག་ལ། ལྷའི་རྣལ་འབྱོར་དྲན་པ་ཙམ་གྱིས་གསལ་གདབ ༑ ཐིག་ཚོན་བྱིན་གྱིས་བརླབས་ལ། དང་པོར་སྭ་བྷཱ་ཝའི་སྔགས་བརྗོད་པས་ཐིག་ཚོན་སྟོང་པར་བསྒོམ། སྟོང་པའི་ངང་ལས་པྃ་ལས་པདྨ་ལྔ། རྃ་ལས་ཉི་མ་ ( ¡¶ ཞ ¡· + དཀྱིལ་འཁོར་ ) ལྔའི་སྟེང་དུ། སྔོན་པོའི་ཐིག་ཚོན་ཧཱུྃ་ལས་མི་བསྐྱོད་པར་བསྐྱེད ༑ དཀར་པོ་བྷྲཱུཾ་ལས་རྣམ་སྣང༌། སེར་པོ་ཨཱཾ་ལས་རིན་འབྱུང༌། དམར་པོ་ཛྲཱྀཾ་ལས་འོད་དཔག་མེད། ལྗང་གུ་ཁཾ་ལས་དོན་ཡོད་གྲུབ་པར་བསྐྱེད། དེའང་ནུས་ན་ཞལ་བརྒྱད་ཕྱག་བཅུ་དྲུག་པ་ཡུམ་དང་བཅས་པར་བསྐྱེད། མི་ནུས་པས་ ( ¡¶ ཞ ¡· + ཀྱང་ ) ཕྱག་གཉིས་པ་ ¡Ã ཡབ་ཡུམ་དུ་ ( ¡¶ ཞ ¡· - ཡབ་ཡུམ་དུ་ ) བསྐྱེད་པས་ཆོག་ ¡Ã སྐྱེ་མཆེད་དང་ ( ¡¶ ཞ ¡· - སྐྱེ་མཆེད་དང་ ) སྐུ་གསུང་ཐུགས་བྱིན་གྱིས་བརླབ་ ¡Ã པ། གཽ་རཱི་ལ་སོགས་པ་ལྷ་མོ་བཞིས་ཡེ་ཤེས་པ་དགུག་པ་ནས། ( ¡¶ ཞ ¡· དེ་ནས་ ) རང་གི་མིག་གཉིས་མ་དང་ཊ་ལས་ཟླ་བ་དང་ཉི་མར་བསྐྱེད། མིག་འབྲས་ ( ¡¶ ཞ ¡· + གཉིས་ ) ཧཱུྃ་ལས་ ( ¡¶ ཞ ¡· + རྡོ་རྗེ་ ) བྱིན་གྱིས་བརླབས་ཏེ། མིག་གི་རྫི་མ་མྱུར་དུ་བརྡབ་ཅིང་གྱེན་དུ་བལྟ་བ་ནི། འབར་བའི་ལྟ་བ་ཞེས་བྱ་བ་སྟེ་སྟ་གོན་གྱི་ལྷ་དང༌། དེ་བཞིན་གཤེགས་པ་ཐམས་ཅད་ཀྱི་ཐུགས་ཀ་ནས་ཡེ་ཤེས་ཀྱི་ཚོན་དང་སྐུད་པ་ཚོན་སྐུད་ཀྱི་རྣམ་པར་བྱོན་པའམ། ཡང་ན་རིགས་ལྔའི་རྣམ་པར་སྤྱན་དྲངས་ཏེ་བསྐྱེད་པ་ལ་བསྟིམ་ལ།

དབང་བསྐུར་ཏེ་རིགས་ ¡Ã ཀྱི་བདག་པོའི་རྒྱས་གདབ། ཆུ་བཞི་སྔོན་དུ་འགྲོ་བའི་ ( ¡¶ ཞ ¡· རིགས་བདག་གི་དབུ་བརྒྱན་པ་ལ་ ) མཆོད་པ་ ( ¡¶ ཞ ¡· + རྣམ་པ་ ) ལྔ། ཨོཾ་བཛྲ་པུཥྤེ་ཨཱཿཧཱུྃ་ ¡Ã སྭཱ་ཧཱ། ( ¡¶ ཞ ¡· - སྭཱ་ཧཱ། ) ཞེས་བྱ་བ་ལ་སོགས་པའི་སྔགས་ ¡Ã ཀྱིས་འབུལ ( ¡¶ ཞ ¡· ཙམ་གྱིས་དབུལ ) ། ¡Ã ཁྱོད་ནི་རྣལ་འབྱོར་མ་ཚོགས་ཡོངས་སུ་བསྐོར་གྱིས་བསྟོད། ( ¡¶ ཞ ¡· ཁྱོད་ནི་་་་་་་གྱིས་བསྟོད། ) མི་བསྐྱོད་རྡོ་རྗེ་ཡེ་ཤེས་ཆེ། །ཞེས་བྱ་ ¡Ã བའི་ ( ¡¶ ཞ ¡· ལ་སོགས་པའི་ ) བསྟོད་པས་ཀྱང་བཏུབ ( ¡¶ ཞ ¡· བསྟོད ) ། ( ¡¶ ཞ ¡· + ཨོཾ་ ) ཨ་ཀཱ་རོ་མུ་ཁཾ་གིས་གཏོར་མ་བཏང་ལ། ཨོཾ་བཛྲ་ཙི་ཏྲ་ས་མ་ཡ་མ་ཏི་ཀྲ་མ་ཧཱུྃ། ཞེས་བྱ་བ་ལན་བདུན་ནམ་ཉེར་གཅིག་བཟླས། མཐེ་བོ་དང་སྲིན་ལག་ ¡Ã གིས་ ( ¡¶ ཞ ¡· གཉིས་ཀྱིས་ ) ཚོན་རྣམས་ལ་རེག་སྟེ་མཉེས་པས་ལྷ་རྣམས་འོད་དུ་ཞུ་སྟེ། ཏ་ཐཱ་ག་ཏའི་རྡུལ་ཕྲ་རབ་ལས་གྲུབ་པའི་ཚོན་རྩི་དང་ཐིག་སྐུད་དུ་བསམ། ཚོན་རྩི་བྱིན་གྱིས་བརླབས་པ་དེ་ཙམ་ཉིད་ཀྱིས་ཆོག་གོ །དེ་ནས་ཐིག་སྐུད་རྣམས་སློབ་དཔོན་གྱིས་བཟུང་ལ། རང་དཔལ་ཀྱེའི་རྡོ་རྗེའི་རྣལ་འབྱོར་ལ་གནས་ཏེ། སློབ་མ་གཽ་རཱིར་སྤྲུལ་ལ་ཐིག་སྐུད་ལྔ་པོ་ནག་པོ་དང་དཀར་པོ་དང་སེར་པོ་དང་དམར་པོ་དང་ལྗང་གུ་རྣམས་རིམ་གྱིས་བརྐྱང་ལ། ཏ་ཐཱ་ག་ཏ་སོ་སོར་མོས་ཏེ་ཕན་ཚུན་རྗེས་སུ་རྟོགས་པ་སྟེ། ཆོས་ཀུན་ཕན་ཚུན་རྗེས་སུ་འགྲོ །ཆོས་རྣམས་ཐམས་ཅད་ ¡Ã སེམས་སུ་ཡང་ ( ¡¶ ཞ ¡· གཅིག་ཏུ་ཡང་ ) ། །ཀུན་ཏུ་འགྲོ་བ་ཆོས་ཀུན་བརྗོད། །ཅེས་བརྗོད་ཅིང་བསྒྲིལ། ཏ་ཐཱ་ག་ཏ་ཐམས་ཅད་གཉིས་སུ་མེད་པར་བསམས་ཏེ་རྡོ་རྗེའི་ལྟེ་བ་ལ་དཀྲིས་ཏེ། བདུད་རྩིས་ཅུང་ཟད་གཏོར་ཏེ་བཞག་གོ །དེ་དག་ནི་ཐིག་དང་ཚོན་བྱིན་གྱིས་བརླབས་པའོ ༑ ༑ དེ་ནས་ཚོན་དང་མཆོད་པ་ལ་སོགས་པ་གཞན་དུ་གསལ་ལ། ཡེ་ཤེས་ཀྱི་ཐིག་གིས་ས་གཞི་བྱིན་གྱིས་བརླབ་པའི་ཕྱིར་གདབ་པར་བྱ་སྟེ། རང་ནུབ་ཕྱོགས་ནས་ཤར་ཕྱོགས་སུ་ཁ་བལྟས་ཏེ་དཔལ་ཀྱེའི་རྡོ་རྗེར་བསྐྱེད།

སློབ་མ་ཤར་ནས་ནུབ་ཏུ་བལྟས་ལ་གཽ་རཱིར་སྤྲུལ་ལ། རང་གི་ལག་པ་གཡོན་པའི་རྡོ་རྗེ་ལ་དཀྲིས་པའི་ཐིག་སྐུད་ ( ¡¶ ཞ ¡· + ལྟེའི་ ) ཐད་ཀར་བཟུང་ སྟེ་ཐིག་སྐ྄ུད་ཀྱི་སྣ་ནས་བཟུང་ལ། ཛ་ཛ་ཛ། ཞེས་བརྗོད་ཅིང་སློབ་མ་ལ་གཏད། སློབ་མས་ཀྱང༌། ཛ་ཛ་ཛ། ཞེས་བརྗོད་ཅིང་རྣ་ནས་དྲངས་ལ་ལྟེ་བའི་ཐད་ཀར་བཟུང་སྟེ་བརྐྱང་ལ། ལག་པ་གཡས་པས་སྟེང་ནས་ཐིག་སྐུད་བཟུང་ལ་གཅུས་ཏེ་གྱེན་ལ་བཏང༌། ངག་ཏུ། ཨོཾ་བཛྲ་ས་མ་ཡ་སཱུ་ཏྲ་མ་ཏི་ཀྲ་མ་ཧཱུྃ། ཞེས་བརྗོད། ཡང་ཐུར་ལ་དེ་བཞིན་དུ་གདབ། དེའི་ཚེ་སེམས་ཀྱིས་ཐིག་སྒྲ་དེ་ལས། ནམ་མཁའ་གང་བའི་དེ་བཞིན་གཤེགས། །འགྲོ་དོན་མཛད་དུས་འདི་ལགས་པས། ཁྱེད་རྣམས་འདིར་ནི་གཤེགས་གསོལ་བར། ། ཐིག་སྐུད་སྒྲས་ནི་སྐུལ་མ་འདེབས། །ཞེས་སྒྲ་དེ་གྲགས་པས་ཏ་ཐཱ་ག་ཏ་ཐམས་ཅད་ནམ་མཁའི་ཁམས་འདིར་གང་བར་བྱོན་པར་བསམ་ཞིང༌། ཚངས་ཐིག་དང་པོ་གདབ་བོ། །དེ་བཞིན་དུ་ཐིག་འོག་མ་རྣམས་འདེབས་པའི་ངག་ཏུ་བརྗོད་པ་དང་ ༑ ཏིང་ངེ་འཛིན་དེ་ཉིད་ཀྱིས་ཤེས་པར་བྱའོ། །དེ་ནས་གཡས་སྐོར་གྱི་རིམ་པས་སློབ་དཔོན་ལྷོ་ན་འདུག་སྟེ་བྱང་དུ་བལྟ། སློབ་མ་བྱང་དུ་འདུག་པ་དང་ཚངས་ཐིག་གཉིས་པ་གདབ། དེ་བཞིན་དུ་གཡས་སྐོར་དུ་སོང་སྟེ། ཤར་ལྷོར་སློབ་དཔོན་འདུག་སྟེ། བྱང་ཤར་དུ་སློབ་མ་འདུག་པ་དང་ཤར་གྱི་ལོགས་ཐིག་གདབ། ཡང་གཡས་སྐོར་གྱིས་ནུབ་བྱང་དུ་སློབ་དཔོན་འདུག་ལ། ལྷོ་ནུབ་མཚམས་སུ་འདུག་པའི་སློབ་མ་དང་ནུབ་ཀྱི་ལོགས་ཐིག་གདབ། སློབ་དཔོན་དེ་ཉིད་དུ་འདུག་ལ་སློབ་མ་བྱང་ཤར་དུ་སོང་ལ་བྱང་གི་ལོགས་ཐིག་གདབ། ཡང་གཡས་སྐོར་གྱིས་སློབ་དཔོན་ཤར་ལྷོར་སོང་སྟེ། སློབ་མ་ལྷོ་ནུབ་ཏུ་འདུག་པ་དང་ལྷོའི་ལོགས་ཐིག་གདབ ༑ སློབ་དཔོན་དེ་ཉིད་དུ་འདུག་ལ་སློབ་མ་བྱང་ནུབ་ཏུ་འདུག་པ་དང་མཚམས་ཐིག་དང་པོ་གདབ། སློབ་དཔོན་ལྷོ་ནུབ་ཏུ་སོང་ལ་སློབ་མ་བྱང་ཤར་དུ་སོང་སྟེ་མཚམས་ཐིག་གཉིས་པ་གདབ་བོ། དེ་ལྟར་སྟེང་དང་འོག་ཏུ་ཐིག་ཆེན་བརྒྱད་གདབ་པས་དཀྱིལ་འཁོར་དུ་ལམ་གྱིས་བྱུང་བར་བསམ།

དེ་ནས། ཨོཾ་བཛྲ་བཻ་ག ( ¡¶ ཞ ¡·¡¶ ལུ ¡· + ཀྲ་མ་ཧོ། ) ཞེས་བརྗོད་ཅིང་ཡིད་ཀྱིས་ཡེ་ཤེས་ཀྱི་ཐིག་བཏེག་ལ། སློབ་དཔོན་ནང་དུ་ཞུགས་ཏེ་ཕྱོགས་བཞིར་སྨིན་མ་གཅུས་ཏེ་ཁྲོ་བོའི་ལྟ་བས་བལྟ་ཞིང་ཧིཿཞེས་བརྗོད་པས་སྒོ་བཞི་པོ་དེ་བྱེ་བར་བསམ། སྔར་ཕྱི་ནང་གི་རྡོ་རྗེ་ར་བ་དང་མེ་རིས་མ་བསྐོར་ན། དེའི་དུས་སུ་དཀྱིལ་འཁོར་གྱི་དབུས་སུ་ཕུར་པ་བཏབ་ལ་བསྐོར། ཐིག་གདབ་པའི་ཆོ་གའོ། །དེ་ནས་ཚོན་གྱིས་བྲི་བ་ནི། དཀྱིལ་འཁོར་གྱི་རྡུལ་ལ་སོགས་པ་ཐམས་ཅད་ཕྱགས་ལ། སློབ་དཔོན་དབུས་སུ་འདུག་ལ་ཚོན་རྩི་རྣམས་བླངས། ལག་པ་གཡོན་པའི་ཁུ་ཚུར་གྱིས་བཟུང་ལ་འདི་སྐད་ཅེས་བརྗོད་དོ ༑ ༑ ཆོས་དབྱིངས་འདི་ནི་རྣམ་པར་དག སེམས་ཅན་ཁམས་ནི་སྒྲོལ་བར་བྱེད།། ཀྱེ་ཡི་རྡོ་རྗེ་རྒྱལ་པོ་ཆེ། །དཀྱིལ་འཁོར་སྙིང་པོ་བྲི་བར་བྱ། །ཨོཾ་བཛྲ་ཙི་ཏྲ་ས་མ་ཡ་མ་ཏི་ཀྲ་མ་ཧཱུྃ། ཞེས་བརྗོད་ཅིང་དབང་ལྡན་གྱི་མཚམས་ནས་བརྩམས་ཏེ། གཡས་སྐོར་གྱིས་ཆ་ཆེན་གཅིག་གི་བར་དུ་བྲི་བར་བྱའོ། །དེའི་རིམ་པ་ནི་དང་པོ་ནང་དུ་ནག་པོ་བྲི། དེའི་ཕྱི་རོལ་དུ་ལྗང་གུ། དེའི་ཕྱི་རོལ་དུ་དམར་པོ། དེའི་ཕྱི་རོལ་དུ་སེར་པོ། དེའི་ཕྱི་རོལ་དུ་དཀར་པོ། ཡང་ན་ནག་པོའི་རྩར་དཀར་པོ། དེའི་ ¡Ã རྩར་ ( ¡¶ ཞ ¡· དེའི་ཕྱི་རོལ་དུ་ ) སེར་པོ། དེའི་ ¡Ã རྩར་ ( ¡¶ ཞ ¡· ཕྱི་རོལ་དུ་ ) དམར་པོ། དེའི་ ¡Ã རྩར་ ( ¡¶ ཞ ¡· ཕྱི་རོལ་དུ་ ) ལྗང་གུ་སྟེ། གོང་མ་ནི་རྡོ་རྗེ་རྩེ་མོའི་དགོངས་པ་ཕྱོགས་ཀྱི་རིམ་པ་ཡིན་ལ། འོག་མ་ནི་གདན་བཞིའི་དགོངས་པ་ཡེ་ཤེས་ཀྱི་རིམ་པའོ། །དེ་ལྟར་པཉྩ་རེ་ཁ་ཅུང་ཟད་ཅིག་བྲིས་ལ་ཐོད་པས་ཁ་བཅད་དེ། དེའི་སྟེང་དུ་རྡོ་རྗེ་བཞག་ལ་མཆོད་པ་རྣམ་པ་ལྔས་མཆོད་ལ་བཞག་གོ །འདི་ནི། སློབ་དཔོན་གྱིས་ནི་དང་པོར་དགྱེ། །ཞེས་བྱ་བའི་དོན་ཡིན་ཏེ། སློབ་དཔོན་གྱི་ཁྱད་པར་གྱི་ལས་ཡིན་ནོ། །དེ་ནས་དབུས་སུ་ཕུར་བའི་བུ་ག་འོག་ཏུ་དཀར་པོ་ནས་བརྩམས་ཏེ་ཚོན་རྩི་སྔ་མ་རྣམས་ཀྱིས་རིམ་པ་བཞིན་བཀང་ལ།

དེ་ནས་སློབ་མ་རྣམས་ཀྱིས་ཇི་ལྟར་བདེ་བར་དགྱེ་བར་བྱའོ། །དཀྱིལ་འཁོར་གྱི་སྟེང་དུ་འགྲོ་བ་དང༌། ཕྱིར་འགྲོ་བའི་དུས་ཐམས་ཅད་དུ། ( ¡¶ ཞ ¡· + ཨོཾ་ ) བཛྲ་བི་ག ( ¡¶ ཞ ¡·¡¶ ལུ ¡· + ཀྲ་མ་ཧོ། ) ཞེས་བརྗོད་ཅིང་ཡིད་ཀྱིས་ཡེ་ཤེས་ཀྱི་ཐིག་དང་ཚོན་རྩི་ལ་སོགས་པ་བཏེག་ཅིང་འགྲོ་བར་བྱའོ ༑ ༑ དེ་ལ་སྙིང་པོའི་དཀྱིལ་འཁོར་གྱི་ས་གཞི་ཕྱི་ཐམས་ཅད་སྔོན་པོར་བྱ། དེའི་དབུས་ཀྱི་པདྨའི་ལྟེ་བ་ལ་ཉི་མ་དམར་པོར་བྱ། དེའི་ཕྱི་རོལ་དུ་ཆ་ཆུང་གཅིག་སྔོན་པོར་བྱ། དེའི་ཕྱི་རོལ་དུ་པདྨའི་ཟེའུ་འབྲུ་སྣ་ཚོགས་སུ་བྱ། དེའི་ཕྱི་རོལ་དུ་པདྨ་འདབ་མ་བརྒྱད་པོ་རྣམས་ལྷ་མོའི་ཁ་དོག་དང་མཐུན་པར་བྱ། ཡང་ན་ཕྱོགས་ཚོན་དུ་བྱ། འཕར་མ་གཉིས་པ་བྱེད་ན་དེའི་ཕྱི་རོལ་རྡོ་རྗེ་ར་བ་སྔོན་པོ་མེ་རིས་དམར་སེར་དུ་བྱ། དེ་ནས་ཤར་ཕྱོགས་ཀྱི་ལྷ་སྣམ་དང་སྒོ་དཀར་པོ་བྱ། ལྷོའི་སེར་པོ ༑ ནུབ་ཀྱི་དམར་པོ། བྱང་གི་ལྗང་གུ། འཕུར་མ་གཅིག་པར་བྱེད་ན། པདྨ་ཚན་པ་གཅིག་གིས་སྒོའི་བར་དུ་ཁེབས་པར་བྱས་ལ། དེ་ལ་རྡོ་རྗེ་ར་བ་དང་མེ་རིས་བྱས་ཀྱང་བཏུབ། མེད་ཀྱང་བཏུབ་བོ། །དེའི་ཕྱི་རོལ་དུ་པཉྩ་རེ་ཁ་སྔ་མ་བཞིན་བྱ། སློབ་དཔོན་རྩེ་མོའི་གསུང་རབ་དཔེ་བསྡུར་མ་པོད་ གསུམ་ པ། མེས་པོའི་ཤུལ་བཞག་ལས། ས་སྐྱ་བཀའ་འབུམ། PAGE 310PAGE 309

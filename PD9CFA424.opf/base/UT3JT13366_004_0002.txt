བཟང་པོ་བཟུང་ལ། ཡང་སྟོན་པ་ངེས་དོན་དུ་སངས་རྒྱས་ནས་རྡོ་རྗེ་ཐེག་པ་ལ་སོགས་པ་ཟབ་པ་དང་རྒྱ་ཆེ་བའི་ཆོས་ཀྱི་འཁོར་ལོ་བསྐོར་བའི་རྗེས་ལ་གདོད་འཛམ་བུ་གླིང་དུ་ཤཱཀྱ་ཐུབ་པ་བྱུང་བ་ཡིན་ཏེ། འཕགས་པ་དེ་ཁོ་ན་ཉིད་འདུས་ པའི་རྒྱུད་དང༌། རྩ་བའི་རྒྱུད་དང༌། རྒྱུད་ཕྱི་མ་དང༌། ཕྱི་མའི་ཕྱི་མ་གསུངས་ནས། རྒྱུད་ཉེ་བར་བསྡུ་

¡Ã བ་ ( ¡¶ ཞ ¡· བའང་ ) གསུངས་པའི་སྐབས་སུ་གསུངས་ལ། མི་ཡི་ནང་དུ་བབས་ནས་ཀྱང༌། །དགོས་ཕྱིར་སེམས་ཅན་ཀུན་གྱི་དོན། །རྡོ་རྗེ་འཁོར་ལོ་མཆོག་བསྐོར་ཞིང༌། །མགོན་པོས་རབ་ཏུ་བསྒྲུབ་པར་མཛོད། །ཅེས་བྱ་བ་དང༌། དེ་ནས་རི་རབ་ཀྱི་རྩེ་མོ་ནས་སྙིང་པོ་བྱང་ཆུབ་ག་ལ་བ་དེར་གཤེགས་ནས། ཆེད་དུ་བཅོམ་ལྡན་འདས་ཀྱིས་བྱང་ཆུབ་ཀྱི་ཤིང་དྲུང་དུ་འཇིག་རྟེན་དང་མཐུན་པར་བྱ་བའི་ཕྱིར་རྩྭ་བླངས་ནས་ཞེས་བྱ་བ་ལ་སོགས་པ་གསུངས་པ་ལྟ་བུའོ། །ཕྱག་ན་རྡོ་རྗེ་དབང་བསྐུར་བའི་རྒྱུད་ལས་ཀྱང༌། བྱང་ཆུབ་སེམས་དཔའ་ཆེན་པོ་ཀུན་ཏུ་བཟང་པོས་ཕྱག་ན་རྡོ་རྗེ་འཇིག་རྟེན་གྱི་ཁམས་གཞི་དང་སྙིང་པོ་མེ་ཏོག་གི་རྒྱན་གྱིས་བརྒྱན་པར་དབང་བསྐུར་བའི་རྗེས་ལ། དེ་ནས་འཇིག་རྟེན་གྱི་ཁམས་མི་མཇེད་འདིར་བཅོམ་ལྡན་འདས་ཤཱཀྱ་ཐུབ་པའི་མཚན་གྱིས་གླིང་བཞི་པ་ ( ¡¶ ཞ ¡· - པ་ ) བར་མའི་འཇིག་རྟེན་གྱི་ཁམས་ཀྱི་འཛམ་བུའི་གླིང་དུ་བདུད་ཕམ་པར་མཛད་ནས་བྱང་ཆུབ་ཏུ་མངོན་པར་རྫོགས་པར་སངས་རྒྱས་ཏེ་ཞེས་གསུངས་པ་ལྟ་བུའོ།། གསུམ་པ་སྟོན་པ་ཇི་ལྟར་སངས་རྒྱས་པའི་ཚུལ་ནི། སྟོན་པ་དེ་བཞིན་གཤེགས་པ་སྤྲུལ་པའི་སྐུ་ནི་འཇིག་རྟེན་དང་མཐུན་པར་མཛད་པའི་ཕྱིར་རང་གི་ངོ་བོ་སངས་རྒྱས་ཟིན་ཀྱང་འཚང་རྒྱ་བའི་ཚུལ་སླར་སྟོན་པ་ནི་སངས་རྒྱས་རྣམས་ཀྱི་ཆོས་ཉིད་ དེ ༑ དེ་ལྟར་ཡང་དེ་ཁོ་ན་ཉིད་འདུས་པ་ལས་གསུངས་པ། ལྷ་རྣམས་ཀྱིས་སྨྲས་པ། གྲོགས་པོ་རྣམས་འདི་ ( ¡¶ ཞ ¡· - འདི་ ) ནི་སྔོན་དུ་མངོན་པར་རྫོགས་པར་སངས་རྒྱས་ཟིན་གྱིས་འོན་ཀྱང་འཇིག་རྟེན་དང་མཐུན་པར་མཛད་ཅིང་སླར་ཡང་བླ་ན་མེད་པའི་བྱང་ཆུབ་སྟོན་གྱིས་ཞེས་བྱ་བ་ནས། དེ་ནས་དེའི་མཚན་མོ་བདུད་ཕམ་པར་མཛད་ནས། ཐོ་རངས་བླ་ན་མེད་པ་ཡང་དག་པར་རྫོགས་པའི་བྱང་ཆུབ་ཏུ་མངོན་པར་རྫོགས་པར་སངས་རྒྱས་ནས། སེམས་ཅན་གྱི་ཁམས་མ་ལུས་པ་ལ་ཕན་པར་མཛད་པ་ཞེས་བྱ་བའི་བར་དུ་གསུངས་པ་ལྟ་བུའོ། །གལ་ཏེ་དེ་ལྟར་ན་སློབ་དཔོན་ཤཱཀྱ་བཤེས་གཉེན་གྱི་ལུགས་ལེགས་ཏེ།

མཚམས་འདིར་རྩྭ་བླངས་ནས་ཞེས་བྱ་ བའམ ༑ དེའི་མཚན་མོ་ཞེས་བྱ་བ་ནས་མཚམས་བཟུང་བའི་ཕྱིར་རོ་སྙམ་ན། མ་ ཡིན་ཏེ་དབང་བསྐུར་བའི་རྒྱུད་ལས། དབང་བསྐུར་བའི་རྗེས་ལ་དགའ་ལྡན་གྱི་གནས་ན་གནས་པ་ནས་བཟུང་སྟེ། མཐར་གྱིས་སྤྱོད་པ་སྤྱད་ཅིང་ཡོངས་སུ་མྱ་ངན་ལས་འདས་པ་ཆེན་པོའི་མཐར་ཐུག་པར་བྱིན་གྱིས་བརླབས་ནས་ཆོས་སྟོན་ཏོ་ཞེས་གསུངས་པ་ལྟ་བུའོ། །བཞི་པ་དེའི་མཛད་པ་ནི་སྟོན་པ་དེ་བཞིན་གཤེགས་པ་དགའ་ལྡན་ན་གནས་པ་ནས་བརྩམས་ཏེ་མྱ་ངན་ལས་འདས་པའི་བར་དུ་མཛད་པ་བརྒྱད་དམ་བཅུ་གཉིས་སམ་བཅུ་དྲུག་གམ་བསམ་གྱིས་མི་ཁྱབ་པ་དེ་དག་མཛད་དེ་ཐུན་མོང་བ་འདི་ནི་རྒྱ་ཆེ་རོལ་པའི་མདོ་ལ་སོགས་པའི་རྗེས་སུ་འབྲངས་ཏེ་ཤེས་པར་བྱའོ། །སྤྱི་དོན་གཉིས་པ་འཁོར་ཇི་ལྟར་བྱུང་བའི་ཚུལ་ལ་གཉིས་ཏེ། སྤྱིར་སྐལ་བ་དང་ལྡན་པའི་འཁོར་ཇི་ལྟ་བུ་ཞིག་ཡིན་པ་དང༌། བྱེ་བྲག་ཏུ་སྡུད་པ་པོ་ཇི་ལྟར་བྱུང་བའི་ཚུལ་བསྟན་ པའོ། །དང་པོ་ནི་སྤྱིར་བཅོམ་ལྡན་འདས་རྣམ་པར་སྣང་མཛད་ཀྱིས་ཡོངས་སུ་བཟུང་ཞིང་སྨོན་ལམ་བཏབ་པ་དེ་བྱང་ཆུབ་སེམས་དཔའ་རྣམས་ཀྱིས་ཡོངས་སུ་སྨིན་པར་བྱས་པས། ཚོགས་ལམ་ཆེན་པོ་ཡན་ཆད་ དུ་གྱུར་པ་སྟེ། ཉན་ཐོས་ཀྱི་རིགས་ཅན་ནམ་བྱང་ཆུབ་སེམས་དཔའི་རིགས་ཅན་གཉིས་ཀའི་དབང་དུ་བྱས་ནས། ཁ་ཅིག་ཚོགས་ལམ་ཆེན་པོ། ཁ་ཅིག་དྲོད། ཁ་ ཅིག་རྩེ་མོ། ཁ་ཅིག་བཟོད་པ། ཁ་ཅིག་ཆོས་མཆོག ཁ་ཅིག་མཐོང་ལམ་ནས་བརྩམས་ནས་བྱང་ཆུབ་སེམས་དཔའི་ས་ཆེན་པོར་ཆུད་པའི་བར་ཏེ། དེ་དག་ཀྱང་སྐལ་བ་མཐུན་པ་ཕྱོགས་གཅིག་ཏུ་སྐྱེ་བ་བཟུང་བ་རྣམས་སོ། །གལ་ཏེ་ཚོགས་ལམ་ཡན་ཆད་ལ་མི་རིགས་ཏེ། སངས་རྒྱས་མཐོང་ཡང་ལོག་པར་ལྟ་བ་དང་ལྡན་པ་དང༌། དམྱལ་བར་སྐྱེས་པ་ལ་སོགས་པ་དུ་མ་ཡོད་པའི་ཕྱིར་རོ་སྙམ་ན། མ་ཡིན་ཏེ་མི་དགེ་བའི་བཤེས་གཉེན་གྱིས་ཟིན་པའི་དབང་དང་ལས་ཀྱི་སྒྲིབ་པས་བསྒྲིབས་པས། བདེན་པ་མ་མཐོང་གི་བར་དུ་ལོག་པར་ལྟ་བ་འཛིན་པའང་ཡོད་ལ ༑

བཟོད་པ་མ་ཐོབ་ཀྱི་བར་དུ་ངན་སོང་དུ་འགྲོ་བ་ཡང་ཡོད་དོ། །འོ་ན་དེ་དེ་དག་གིས་སངས་རྒྱས་མཐོང་བས་ཅི་ཞིག་བྱ་སྙམ་ན། ཡང་དེ་དག་གི་ལས་ཀྱི་སྒྲིབ་པ་སྲབ་མོར་བྱེད་ཅིང་ཡོངས་སུ་སྨིན་པར་བྱེད་དོ། །གཉིས་པ་བྱེ་བྲག་ཏུ་སྡུད་པ་པོ་ཇི་ལྟར་བྱུང་བ་ཡིན་ཞེ་ན། དེ་ལ་སྡུད་པ་པོ་ནི་བཅོམ་ལྡན་འདས་ཀྱིས་ཆོས་ཐམས་ཅད་ཐུགས་སུ་ཆུད་པས་སྡུད་པར་འོས་ཀྱིས་གཞན་ ( ¡¶ ཞ ¡· + གྱི་ ) མ་ཡིན་ནོ། །ཆོས་ཐུགས་སུ་ཆུད་པ་དེའང་ཡེ་ཤེས་བསམ་གྱིས་མི་ཁྱབ་པ་ལ་དབང་བ་ཡིན་གྱི་གཞན་མ་ཡིན་ནོ། །དེ་ལ་སྤྱིར་ཉན་ཐོས་རྣམས་ནི་སངས་རྒྱས་ཀྱི་མཐུ་མ་ཡིན་པར་ཆོས་སྨྲ་བའི་སྤོབས་པ་མེད་ལ་བྱང་ཆུབ་སེམས་དཔའ་རྣམས་ནི་རང་གི་སྤོབས་པས་དེ་བཞིན་གཤེགས་པ་རྣམས་ཀྱི་དྲུང་དུ་ཆོས་སྟོན་པ་ཡིན་མོད་ཀྱང༌། དེ་ཐམས་ཅད་ཀྱིས་ཀྱང་སྡུད་པའི་འོས་མ་ཡིན་ནོ། །འོ་ན་གང་ཡིན་ན། བཅོམ་ལྡན་འདས་ཉིད་ཀྱིས་བྱང་ཆུབ་ཀྱི་སེམས་ལས་སྤྲུལ་པའི་བྱང་ཆུབ་སེམས་དཔའ་ཡིན་ནོ ༑ ༑ དེའང་སྟུག་པོ་བཀོད་པ་ནི་རྡོ་རྗེ་སེམས་དཔའ་ཞེས་བྱ་ལ་དེའང་ཐོག་མཐའ་མེད་པའི་བྱང་ཆུབ་ཀྱི་སེམས་དེ་བཞིན་གཤེགས་པ་ཐམས་ཅད་ཀྱི་བདག་པོ་དེའང་གཟུགས་ཀྱི་སྐུའི་རྣམ་པར་བཞུགས་ཏེ། ཨེ་མ་བདག་ནི་རང་བྱུང་བའི། ། ཞེས་གོང་དུ་བསྟན་ཏོ། །དེས་མེ་ཏོག་གི་སྙིང་པོའི་རྒྱན་གྱི་འཇིག་རྟེན་དུ་བྱང་ཆུབ་སེམས་དཔའ་ཀུན་ཏུ་བཟང་པོ་ཞེས་བྱ་བ་བསྟན་པའི་བདག་པོར་གཏད་ལ། དེ་འཇིག་རྟེན་ཆགས་འཇིག་ལ་སོགས་པའི་གནས་སྐབས་ཀྱི་སྐྱེ་བ་ཐ་དད་པ་ལེན་ པར་སྨོན་ལམ་བཏབ་སྟེ། མི་མཇེད་ཀྱི་འཇིག་རྟེན་གྱི་ཁམས་འདིར་བསྐལ་པ་བཟང་པོ་ལ་གནོད་སྦྱིན་གྱི་སྡེ་དཔོན་ལག་ན་རྡོ་རྗེ་ཞེས་བྱ་བ་ ¡Ã གསང་བ་ཐམས་ ཅད་དང་ ( ¡¶ ཞ ¡· གསོལ་བ་ནི་ཐམས་ཅད་དུ་ ) བསྟན་པ་ཐམས་ཅད་ཀྱི་བདག་པོ་ཡིན་ཏེ། ཕྱག་ན་རྡོ་རྗེ་དབང་བསྐུར་བའི་རྒྱུད་ལས་གསུངས་པ། བྱང་ཆུབ་སེམས་དཔའ་ ཀུན་ཏུ་བཟང་པོ་ ( ¡¶ ཞ ¡· པོས་ ) བྱིན་གྱིས་ བརླབས་ནས།

སངས་རྒྱས་ཀྱི་མཛད་པ་གཅིག་ནས་གཅིག་ཏུ་བརྒྱུད་པ་རྒྱུན་མི་འཆད་པར་བྱ་བའི་ཕྱིར་དུས་ཀྱི་དབང་གིས་སྐྱེ་བ་ཡོངས་སུ་བཟུང་བར་བྱིན་གྱིས་བརླབས་ཏེ་ཞེས་བྱ་བ་དང༌། བྱང་ཆུབ་ སེམས་དཔའ་ཀུན་ཏུ་བཟང་པོ་དབང་བསྐུར་ནས་དེ་ཉིད་ལ་བསྟན་པ་གཏད་དོ། ། ཞེས་གསུངས་པ་དང༌། ཡང་དེ་ཉིད་ལས། བྱང་ཆུབ་སེམས་དཔའ་ཀུན་ཏུ་བཟང་པོ་ཞེས་བྱ་བ་འཇིག་རྟེན་གྱི་ཁམས་མེ་ཏོག་གི་སྙིང་པོའི་རྒྱན་བཀོད་པ་ཐམས་ཅད་དུ་བཅོམ་ལྡན་འདས་རྣམ་པར་སྣང་མཛད་ཀྱི་སྔོན་གྱི་སྤྱོད་པ་བསྟན་པར་སྨོན་ལམ་བཏབ་ནས་གནོད་སྦྱིན་འདུལ་བའི་དབང་ཉེ་བར་བཟུང་སྟེ། དབུས་ཀྱི་འཛམ་བུའི་གླིང་མི་མཇེད་འདིར་ལྕང་ལོ་ཅན་དུ་གནོད་སྦྱིན་གྱི་སྡེ་དཔོན་ཆེན་པོ་འབྲོག་གནས་ཀྱི་གནས་སུ་གནས་སོ། །ཞེས་གསུངས་པ་ལྟ་བུའོ། །མི་རྣམས་ཀྱི་ཕྱིར་ནི་ གནས་བརྟན་ཀུན་དགའ་བོ་ཡིན་ནོ་ཞེས་ཐུན་མོང་མ་ཡིན་པའི་གསང་བ་ཞེས་བྱ་ བ་ལས་གསུངས་པ ( ¡¶ ཞ ¡· ལ ) ། དེ་ནས་ཡང་དག་པར་རྫོགས་པའི་སངས་རྒྱས་ཀྱི་འཁོར་ཕུན་སུམ་ཚོགས་པ་བཤད་པར་བྱ་སྟེ། འདི་ལྟ་སྟེ་འཇམ་དཔལ་ནི་རྒྱལ་པོ་ཟས་གཙང་མར་གྱུར་ཏོ། །འཇིག་རྟེན་དབང་ཕྱུག་ནི་ལྷ་མོ་སྒྱུ་འཕྲུལ་དུ་གྱུར་ཏོ། ( ¡¶ ཞ ¡·¡¶ ལུ ¡· + དཔལ་ལྷ་མོ་ནི་གྲགས་འཛིན་ནོ། །རྡོ་རྗེ་སེམས་དཔའ་ནི་སྒྲ་གཅན་འཛིན་ཏོ། །སྒྲིབ་པ་ཐམས་ཅད་རྣམ་པར་སེལ་བ་ནི་ཤཱ་རིའི་བུའོ། །ལྷའི་དབང་པོ་ནི་ལྷ་སྦྱིན་ནོ། ) །ཀུན་ཏུ་བཟང་པོ་ནི་གནས་བརྟན་ཀུན་དགའ་བོར་གྱུར་ཏོ། །ཞེས་བྱ་བ་ལ་སོགས་པ་གསུངས་པ་ལྟ་ བུའོ ༑༑ དེ་ལྟར་སྡུད་པ་པོ་ ¡Ã ཡོན་ཏན་ ( ¡¶ ཞ ¡· དང་སྟོན་པ་ ) དང་མཛད་པ་དང་དབང་བསྐུར་བ་ལ་སོགས་པ་རྣམས་དབང་བསྐུར་གྱི་རྒྱུད་དང་གསང་བ་བསམ་གྱིས་མི་ཁྱབ་པའི་མདོ་ལ་སོགས་པ་ལས་ཤེས་པར་བྱའོ། ། ( ¡¶ ཞ ¡· + དོན་ ) གསུམ་པ་བཅོམ་ལྡན་འདས་དེས་ཆོས་ཀྱི་འཁོར་ལོ་ཇི་ལྟར་བསྐོར་ཞེ་ན། དེ་ལ་གཉིས་ཏེ། ཐུན་མོང་གི་ཆོས་ཀྱི་འཁོར་ལོ་བསྐོར་ཚུལ་དང༌། ཐུན་མོང་མ་ཡིན་པའི་རྡོ་རྗེ་ཐེག་པ་ཇི་ལྟར་བསྐོར་ཚུལ་ལོ།

།དང་པོ་ལ་གཉིས་ཏེ། གཞན་གྱི་ལོག་རྟོག་དགག་པ་དང༌། རང་གི་འདོད་པ་བརྗོད་པའོ། །དེ་ལ་དང་པོ་ནི་མི་མཁས་པ་ཁ་ཅིག་ན་རེ་བཅོམ་ལྡན་འདས་ཀྱིས་ཆོས་ཀྱི་འཁོར་ལོ་རིམ་པ་གསུམ་དུ་བསྐོར་ཏེ། དང་པོ་ཝཱའ་རཱ་ཎ་སཱིར་ཉན་ཐོས་ཀྱི་རིགས་ཅན་དྲང་བའི་དོན་དུ་བདེན་པ་བཞིའི་ཆོས་ཀྱི་འཁོར་ལོ་བསྐོར་ལ། དེས་ཡོད་པར་འཛིན་པ་ཤས་ཆེ་བར་གྱུར་པས། བར་དུ་བྱ་རྒོད་ཕུང་པོ་ལ་སོགས་པར་ངོ་བོ་ཉིད་མེད་པ་སྟོང་པར་སྟོན་པའི་ཆོས་ཀྱི་འཁོར་ལོ་བསྐོར་ལ ༑ དེས་མེད་པར་འཛིན་པ་ཤས་ཆེ་བར་གྱུར་པས་ཐ་མར་མཉན་ཡོད་ལ་སོགས་པར་ངོ་བོ་ཉིད་མེད་པ་ལས་བརྩམས་ནས། ལེགས་པར་ཕྱེ་བའི་འཁོར་ལོ་གསུམ་ པ་ངེས་པའི་དོན་ ( ¡¶ ལུ ¡· + དུ་ ) བསྐོར་རོ། །དེ་ལྟར་འཕགས་པ་དགོངས་པ་ངེས་པར་འགྲེལ་པ་ལས་གསུངས་པ། བཅོམ་ལྡན་འདས་ཀྱིས་དང་པོར་ཡུལ་ཝཱ་རཱ་ཎ་སཱིར་ཉན་ཐོས་ཀྱི་ཐེག་པ་ལ་ཡང་དག་པར་ཞུགས་པ་རྣམས་ལ་འཕགས་པའི་བདེན་པ་བཞིའི་རྣམ་པ་བསྟན་པས། སུས་ཀྱང་མ་བསྐོར་བའི་འཁོར་ལོ་བསྐོར་ཏེ། འཁོར་ལོ་དེའང་བླ་ན་མཆིས་པ་དྲང་བའི་དོན་རྩོད་པའི་གནས་སུ་གྱུར་པ་ལགས་ལ། བཅོམ་ལྡན་འདས་ཀྱིས་ཆོས་རྣམས་ཀྱི་ངོ་བོ་ཉིད་མ་མཆིས་པ་ཉིད་ལས་བརྩམས་ ཏེ་ཐེག་པ་ཆེན་པོ་ལ་ཡང་དག་པར་ཞུགས་པ་རྣམས་ལ་སྟོང་པ་ཉིད་སྨོས་པའི་རྣམ་པས་ཆོས་ཀྱི་འཁོར་ལོ་གཉིས་པ་བསྐོར་ཏེ། འཁོར་ལོ་དེ་ཡང་བླ་ན་མཆིས་པ་དྲང་བའི་དོན་རྩོད་པའི་གནས་སུ་གྱུར་པ་ལགས་ལ། བཅོམ་ལྡན་འདས་ཀྱིས་ཆོས་ རྣམས་ཀྱི་ངོ་བོ་ཉིད་མ་མཆིས་པ་ཉིད་ལས་བརྩམས་ཏེ་ཐེག་པ་ཐམས་ཅད་ལ་ཡང་དག་པར་ཞུགས་པ་རྣམས་ལ་ལེགས་པར་ཕྱེ་བ་དང་ལྡན་པ་ཤིན་ཏུ་ངོ་མཚར་དང་ལྡན་པའི་ཆོས་ཀྱི་འཁོར་ལོ་གསུམ་པ་བསྐོར་ཏེ། ཆོས་ཀྱི་འཁོར་ལོ་ ( ¡¶ ཞ ¡· + འདི་ ) བླ་ན་མ་མཆིས་པ་ངེས་པའི་དོན་རྩོད་པའི་གནས་སུ་གྱུར་པ་མ་ལགས་སོ་ཞེས་བཤད་དོ ༑ ༑ ཤེས་རབ་ཀྱི་ཕ་རོལ་ཏུ་ཕྱིན་པ་ལས་ཀྱང༌། ཀྱེ་མ་འཇིག་རྟེན་དུ་ཆོས་ཀྱི་འཁོར་ལོ་གཉིས་པ་བསྐོར་རོ་ཞེས་བཤད་པའི་ཕྱིར་རོ།

།ཞེས་ཟེར་རོ། །དེ་ནི་ཡང་དག་པ་མ་ཡིན་ཏེ། ཆོས་ཀྱི་འཁོར་ལོ་རིགས་མི་འདྲ་བ་གསུམ་དུ་བསྐོར་བ་དེ་ནི་བདེན་ན། ཡུལ་ཐ་དད་པར་དུས་རིམ་པ་ཅན་དུ་བསྐོར་རོ་ཞེས་ཟེར་བ་དེ་ནི་མི་རིགས་སོ། །ཇི་ལྟར་མི་རིགས་ཞེ་ན། ལུང་རིགས་དང་འགལ་བ་དང༌། ཉན་ཐོས་ཀྱི་རྩོད་པ་མི་སྤོང་བ་དང༌། བདེ་བར་གཤེགས་པ་ལ་སྐུར་པ་བཏབ་པར་འགྱུར་བ་ལ་སོགས་པ་སྐྱོན་དུ་མར་འགྱུར་བས་མི་རིགས་པས་སོ། །གཉིས་པ་རང་གི་འདོད་པ་བརྗོད་པ་ནི། ཁོ་བོ་ཅག་ནི་འདི་ལྟར་འདོད་དེ་དེའང་མདོ་སྡེ་རྒྱན་ལས་ཀྱང་ཐེག་པ་ཆེན་པོ་བཀར་བསྒྲུབ་པའི་དུས་སུ། སྔར་ལུང་མ་བསྟན་ལྷན་ཅིག་བྱུང༌། ། ཞེས་གསུངས་ལ་དགོངས་པ་ལུང་བསྟན་པ་ལས་ཀྱང༌། རྟོག་པ་མེད་ཅིང་འཁྲུག་མེད་པ། །ཡིད་འོང་རྡོ་རྗེ་ཚིག་གཅིག་པུ། །གདུལ་བྱའི་བསམ་པའི་དབང་གིས་ ན ༑ ༑ བྱེ་བྲག་མང་པོར་འགྱུར་བ་ཡིན། །ཞེས་གསུངས་པས། དུས་གཅིག་ཉིད་ལ་ བཅོམ་ལྡན་འདས་ཀྱི་གསུང་གཅིག་ཉིད་གདུལ་བྱ་ཐ་དད་པ་རྣམས་ལ་ཆོས་ཐ་དད་པ་སྣང་བ་ཡིན་ནོ། །དེ་ནི་གོང་དུ་བཤད་པའི་དགོངས་པ་ངེས་པར་འགྲེལ་པའི་ལུང་དང་ཡང་མི་འགལ་ཏེ། ལུང་དེ་ལས་ཀྱང་ཆོས་ཐ་དད་པ་གསུངས་ཀྱི་གནས་ཐ་དད་པ་དང་དུས་ཐ་དད་པ་མ་གསུངས་སོ། །དེས་ན་གདུལ་བྱ་རྣམས་ལ་དུས་གཅིག་ཏུ་གསུང་གཅིག་གིས་ཆོས་ཐ་དད་པ་གསུངས་པ་ནི་གསུང་གི་བཀོད་པ་ཞེས་བྱ་བ་ཡིན་ལ། དུས་གཅིག་ལ་གནས་ཐ་དད་པར་སྐུ་ཐ་དད་པ་དུ་མ་སྟོན་པ་ནི་སྐུའི་བཀོད་པས་དོན་མཛད་པའོ།།དོན་གཉིས་པ་ཐུན་མོང་མ་ཡིན་པ་རྡོ་རྗེ་ ཐེག་པ་ཇི་ལྟར་བསྐོར་ཚུལ་ལ་གཉིས།ཇི་ལྟར་བསྐོར་ཚུལ་དངོས་དང་དེ་ལ་འགལ་བ་སྤང་བའོ། །དང་པོ་ནི་སྤྱིར་ཆོས་ཀྱི་འཁོར་ལོ་བསྐོར་བའི་ཚུལ་དེ་ཉིད་ཀྱིས་ གསང་སྔགས་ཀྱང་གསུངས་པར་རིག་པར་བྱ་མོད། འོན་ཀྱང་བྱེ་བྲག་ཏུ་བཤད་ན། སྤྱིར་གསང་སྔགས་མཐའ་དག་སྔར་བཤད་པའི་ཚུལ་གྱིས་ཤཱཀྱ་ཐུབ་པའི་སྔ་རོལ་ དུ་སྤྲུལ་པའི་སྐུ་མཆོག་གིས་གསུངས་ལ། ཕྱིས་ཤཱཀྱ་ཐུབ་པས་བཟླས་ཏེ་གསུངས་

པའང་ཡོད། ཕྱིས་བཟླས་ཏེ་མ་གསུངས་པའང་ཡོད། དེ་ཉིད་ཀྱི་རྣམ་འཕྲུལ་ གཞན་ ( ¡¶ ཞ ¡· - གཞན་ ) གྱིས་གསུངས་པའང་ཡོད་དོ། །དེ་ལ་བཟླས་ཏེ་གསུངས་པ་ ནི་དཔེར་ན་ཕྱག་ན་རྡོ་རྗེ་དབང་བསྐུར་བའི་རྒྱུད་ལྟ་བུ་སྟེ། དང་པོར་འཇིག་རྟེན་གྱི་ཁམས་གཞི་དང་སྙིང་པོ་མེ་ཏོག་གི་རྒྱན་གྱིས་བརྒྱན་པར་ཀུན་ཏུ་བཟང་པོ་དབང་ བསྐུར་བའི་ ¡Ã གླེང་གཞི་ ( ¡¶ ཞ ¡· རྒྱུད་ ) གསུངས་ནས། དེ་ནས་མི་མཇེད་ཀྱི་ གླེང་གཞི་གྲོང་ཁྱེར་དཔལ་ཡོན་ཅན་དུ་ཕྱག་ན་རྡོ་རྗེ་དབང་བསྐུར་བ་ག སུངས་ པ་ལྟ་ བུའོ ༑ ༑ དེ་བཞིན་དུ། འཛམ་བུའི་གླིང་སོགས་ད་ལྟའི་གནས། །ཆོས་འཁོར་ གནས་ དང་ཡངས་པ་ཅན། །ས་དཀར་ཅན་དང་ལྷ་ཡི་གནས། །ཞེས་བྱ་བ་ལ་སོགས་པ་དང་ ༑ འཁོར་ལྔ་སྡེ་ལ་སོགས་པ་དང་བཅས་པ་ཐུན་མོང་བ་སྨོས་པའི་རྒྱུད་རྣམས་ནི་གསུང་གི་བཀོད་པས་ཆོས་གསུངས་པ་ཡིན་ལ། གནས་ཐུན་མོང་མ་ཡིན་པ་ དཔལ་ལྡན་འབྲས་སྤུང་གི་མཆོད་རྟེན་ལ་སོགས་པ་རྣམས་དང༌། འཁོར་ཐུན་མོང་ མ་ཡིན་པ་སྨོས་པ་རྣམས་ནི་སྐུའི་བཀོད་པ་མཛད་དེ་ཆོས་གསུངས་པའོ། །དེ་ལྟར་ ན་ཕྱིས་བཟླས་ཏེ་གསུངས་པའོ། །ཕྱིས་བཟླས་ཏེ་མ་གསུངས་པའང་ཡོད་དེ། དེ་ ཁོ་ན་ཉིད་འདུས་པའི་རྒྱུད་ཐོག་མར་ གསུངས་ནས་དེའི་རྒྱུད་རྫོགས་པའི་རྗེས་ལ། མི་ཡི་ནང་དུ་བབས་ནས་ཀྱང༌། །ཞེས་བྱ་བ་སཽགས་པ་འཛམ་བུའི་གླིང་དུ་གཤེགས་པ་ལ་སོགས་པ་མཛད་པ་བསྟན་པ་ལྟ་བུའོ། །དཔལ་བདེ་མཆོག་གི་རྒྱུད་ཀྱང་ཕྱིས་ བཟླས་ཏེ་མ་གསུངས་པ་ཡིན་ནོ་ཞེས་སློབ་དཔོན་སྐལ་ལྡན་གྲགས་པ་བཞེད་དོ། ( ¡¶ ཞ ¡· དེ ) ༑ དཔལ་བདེ་མཆོག་རྩ་རྒྱུད་ཀྱི་འགྲེལ་པ་ལས། ཆོས་ཀྱི་རྣམ་གྲངས་འདི་ ནི་ཐོག་མ་མེད་པའི་རྒྱུད་ ( ¡¶ ཞ ¡· རྒྱུ་ ) ཅན་ཡིན་ཏེ། བཅོམ་ལྡན་ཤཱཀྱ་ཐུབ་པ་ལས་སྔ་བ་ཉིད་དུ་གནས་སོ། །དུས་ཀྱི་དབང་གིས་སྲེག་པའི་བསྐལ་པ་ལ་སོགས་པས་ཤེས་རབ་ཀྱི་ཕ་རོལ་ཏུ་ཕྱིན་པ་ལ་སོགས་པ་ནུབ་པ་ན། བཅོམ་ལྡན་འདས་ཤཱཀྱ་ཐུབ་པས་སླར་ཡང་གསུངས་པ་ཡིན་ནོ། །དཔལ་བདེ་མཆོག་ནི་དེ་ལྟ་མ་ཡིན་ཏེ།

སངས་རྒྱས་ཀྱི་ཞིང་བརྗོད་དུ་མེད་པ་ན་དཔའ་བོ་དཔའ་མོ་ལ་སོགས་པས་ཉམས་སུ་བླངས་ཏེ་གནས་པའི་ཕྱིར་མ་ནུབ་བོ། །ཞེས་བཤད་པ་ལྟ་བུའོ། །འདི་ཉིད་ཀྱི་ རྣམ་འཕྲུལ་གཞན་གྱིས་གསུངས་པ་ ¡Ã ནི་ ( ¡¶ ཞ ¡· གསང་བ་ ) འདུས་པ་ལ་སོགས་ པའི་རྒྱུད་ཕལ་ཆེ་བ་ཡིན་ནོ་ཞེས་གསུངས་ཏེ། དཔལ་གསང་བ་འདུས་པ་ལས་དེ་ ནས་ཕྱག་ན་རྡོ་རྗེ་དེ་བཞིན་གཤེགས་པ་ཐམས་ཅད་ཀྱི་བདག་པོས། དེ་བཞིན་ གཤེགས་པ་ཐམས་ཅད་དང་བྱང་ཆུབ་སེམས་དཔའ་ཐམས་ཅད་ལ་བཀའ་སྩལ་པ། བཅོམ་ལྡན་འདས་དེ་བཞིན་གཤེགས་པ་ཐམས་ཅད་སྔོན་བྱུང་བ་བརྗོད་ཀྱིས་མི་ལང་བའི་ཡང་སངས་རྒྱས་ཀྱི་ཞིང་གི་རྡུལ་ཕྲ་རབ་དང་མཉམ་པའི་བསྐལ་པའང་འདས ༑ བཅོམ་ལྡན་འདས་དེ་བཞིན་གཤེགས་པ་དགྲ་བཅོམ་པ་ཡང་དག་པར་རྫོགས་པའི་སངས་རྒྱས་མར་མེ་མཛད་འདས་པ་ནས་ཐུབ་པ་ཆེན་པོ་འོད་སྲུངས་ སངས་རྒྱས་པའི་བར་གྱིས་ཀྱང་མ་གསུངས་ཏེ། དེ་ཅིའི་ཕྱིར་ཞེ་ན། བཅོམ་ལྡན་ འདས་གསང་བ་ཆེན་པོའི་གནས་ཀྱི་དོན་བཤད་པ་དེའི་དུས་སུ་སེམས་ཅན་རྣམས་ལ་ སྐལ་ བ་མེད་པའི་ཕྱིར་ཞེས་གསུངས་པ་ལ། སློབ་དཔོན་ཁ་ཅིག་འོད་སྲུངས་ཀྱི་ བར་མ་གསུངས་ཞེས་འབྱུང་བས་ཤཱཀྱ་ཐུབ་པས་གསུངས་ཞེས་ཤུགས་ལས་འབྱུང་བས་ཤཱཀྱ་ཐུབ་པས་གསུངས་པར་

¡Ã ཤེས་སོ ( ¡¶ ཞ ¡· ཞེས་སོ ) ། །འོ་ན་དེ་ལྟ་བུའི་རྡོ་རྗེ་བཙུན་མོའི་བྷ་ག་དེ་གནས་གང་དུ་བྱུང་ཞེ་ན། འདི་ལ་ཁ་ཅིག་ན་རེ། བཅོམ་ ལྡན་འདས་ཡུམ་གྱི་ལྷུམས་ན་བཞུགས་པ་དེའི་ཚེ་གསུངས་པ་ཡིན་པས་དེ་ན་རྡོ་རྗེ་བཙུན་མོའི་བྷ་གའོ་ཞེས་ཟེར་ཏེ་འདི་ནི་གྱི་ནའོ། །གྱི་ན་ཇི་ལྟར་ཡིན་ཞེ་ན། རྒྱུད་དང་རྟོག་པ་དང་བསྟན་བཅོས་ལས་ཀྱང་མ་བཤད་ལ་གཏམ་དུ་ཡོངས་སུ་གྲགས་པའང་མ་ཡིན་པས། རང་བཟོ་མཁན་རྣམས་ཀྱིས་རང་བཟོར་བྱས་པ་ཡིན་པར་རིག་པར་བྱའོ། །ཡང་སློབ་དཔོན་གཞན་དག་ན་རེ་གཏམ་དུ་གྲགས་པ་འདི་ཡོད་དེ ༑ བཅོམ་ལྡན་འདས་གྲོང་ཁྱེར་ཆེན་པོ་མཉན་ཡོད་ན་བཞུགས་པ་དེའི་ཚེ། དེ་ནས་དཔག་ཚད་སུམ་བརྒྱ་ན་ཨཽ་ཌྱན་ཞེས་བྱ་བའི་ཡུལ་ན་རྒྱལ་པོ་ཨིནྡྲ་བྷཱུ་ཏི་ཞེས་བྱ་བ་ཡོད་དེ། དེས་སྔ་དྲོའི་དུས་གཅིག་གི་ཚེ་དགེ་སློང་གི་གཟུགས་དུ་མ་ནམ་མཁའ་ལས་འགྲོ་བ་མཐོང་སྟེ། དེས་དེ་དག་ཅི་ཡིན་མ་ཤེས་ཏེ། བློན་པོ་རྣམས་ལ་དྲིས་པས་དེ་དག་གིས་ཀྱང་མ་ཤེས་སོ། །དེ་ནས་གྲོང་ཁྱེར་བ་ལ་དྲིས་པས་དེ་དག་གི་ནང་ན་གཅིག་ན་རེ་འདི་ནས་དཔག་ཚད་སུམ་བརྒྱ་ན་མཉན་ཡོད་ཅེས་བྱ་བའི་གྲོང་ཁྱེར་ན་རྒྱལ་པོ་ཟས་གཙང་མ་ཞེས་བྱ་བའི་སྲས་སངས་རྒྱས་པ་བཞུགས་པས། འདི་དག་དེའི་འཁོར་ཡིན་ནོ་ཞེས་སྨྲས་པ་དེ་རྒྱལ་པོས་ཐོས་པས་སངས་རྒྱས་ཞེས་བྱ་བའི་སྒྲ་དེས་རྒྱལ་པོ་མངོན་པར་དད་དེ་དེའི་ཕྱོགས་སུ་ཕྱག་བཙལ་ཏེ། མེ་ཏོག་དང་བཅས་པས་ཐལ་མོ་སྦྱར་ཏེ་འདུག་པས་མཉན་ཡོད་ན་བཅོམ་ལྡན་འདས་འཁོར་དང་བཅས་པ་བཞུགས་པ་མཐོང་ངོ༌། །དེར་དད་པ་སྐྱེས་ཏེ། བཅོམ་ལྡན་འདས་ཐམས་ཅད་མཁྱེན་པ་ཐུགས་རྗེ་ཅན། རྫུའཕྲུལ་གྱི་ཞབས་མངའ་བས་ སང་གི་གདུགས་ཚོད་ལ་འདིར་གཤེགས་པར་གསོལ་ཞེས་གསོལ་བ་བཏབ་པས། སང་གི་གདུགས་ཚོད་ལ་བཅོམ་ལྡན་འདས་འཁོར་དང་བཅས་པ་བྱོན་ཏེ། རྒྱལ་པོ་ འཁོར་དང་བཅས་པས་མཆོད་ནས་གསོལ་བ། བདག་ཅག་སྲིད་པའི་སྡུག་བསྔལ་ལས་ཐར་བར་མཛད་དུ་གསོལ་ཞེས་ཞུས་སོ།

།བཀའ་སྩལ་པ་ཁྱིམ་ནས་ཁྱིམ་མེད་པར་རབ་ཏུ་བྱུང་ཤིག་དང་ཐབས་ཡོད་དོ། །གསོལ་བ། བདག་ཅག་ནི་འདོད་པའི་ཡོན་ཏན་གྱི་ ( ¡¶ ཞ ¡· གྱིས་ ) གདུང་བ་ཆེ་ན་འདི་མ་སྤངས་པ་ཉིད་དུ་ཐར་པའི་ཐབས་ཞུའོ་ཞེས་གསོལ་བ་དང༌། དེ་ཉིད་ཀྱི་ཚེ་བཅོམ་ལྡན་འདས་ཀྱིས་སྐྱེ་བོ་འདོད་ཆགས་ཅན་དེ་དག་རྗེས་སུ་གཟུང་བའི་ཕྱིར། རྡོ་རྗེ་བཙུན་མོའི་བྷ་ག་ལ་བཞུགས་ནས་བཙུན་མོའི་འཁོར་གྱི་དབུས་སུ་རྒྱལ་པོ་ཨིནྡྲ་བྷཱུ་ཏི་ ( ¡¶ ཞ ¡· + ལ་ ) དབང་བསྐུར་བས་དེ་ཉིད་ཀྱི་ཚེ་གྲོལ་བར་མཛད་དོ། །དེ་བཞིན་དུ་རྡོ་རྗེ་ཐེག་པའི་ཆོས་གསུངས་པས་རྒྱལ་པོ་འཁོར་དང་བཅས་པ་གྲོལ་བར་མཛད་དེ་གསང་བའི་བདག་པོ་ལ་བསྟན་པ་གཏད་དེ་བསྡུས་སོ་ཞེས་འདོད་དོ། །གཞན་དག་ན་རེ། གཏམ་དུ་གྲགས་པ་དེའང་ཆོས་བསྡུས་པའི་དུས་ཀྱི་གཏམ་ཡིན་ལ་ཆོས་ཀྱི་རྣམ་གྲངས་འདིའང་ཐོག་མ་མེད་པའི་རྒྱུ་ཅན་དུ་རིག་པར་བྱའོ། །ཇི་ལྟར་ཞེ་ན། དཔལ་ཡེ་ཤེས་ཐིག་ལེའི་རྒྱུད་ལས་གསུངས་པ། མ་འོངས་པའི་དུས་སུ་ཡང་དག་པའི་གསང་བའི་རྒྱུད་སུས་སྐལ་བ་དང་ལྡན་པའི་སེམས་ཅན་ལ་བཤད་པ་བྱེད་པར་འགྱུར། བཀའ་སྩལ་པ་ཕྱག་ན་རྡོ་རྗེ་མགོན་པོས་བཤད་པར་བྱེད་དོ། །མཐའ་ཡས་པ་ཡི་འཇིག་རྟེན་ཁམས ༑ ༑ སྐྱེ་དང་འཇིག་པའི་རྒྱུ་ཡིན་ནོ། །ཕྱག་ན་རྡོ་རྗེ་མགོན་ཆེན་པོ། །རྡོ་རྗེ་འཛིན་པ་ཀུན་གྱི་བདག །དེ་ལ་སྐྱེ་དང་འགག་པ་མེད། །རང་བྱུང་མཆོག་གི་དབང་ཕྱུག་གོ །ལ་ལར་སངས་རྒྱས་ཀྱི་ནི་གཟུགས། །ལ་ལར་ཧེ་རུ་ཀ་ཡི་གཟུགས། །ལ་ལར་རྒྱུད་དང་བསྟན་བཅོས་ནི། །མཛད་དང་ལ་ལ་རུ་ནི་འཛིན། །ཞེས་གསུངས་པས་འཇིག་རྟེན་ཆགས་ ( ¡¶ ཞ ¡· - ཆགས་ ) འཇིག་ལ་སོགས་པ་བྱུང་བའི་སྔ་རོལ་ཉིད་དུ་ཆགས་ཅན་གཟུང་བའི་ཕྱིར་བཅོམ་ལྡན་འདས་ཀྱི་བཙུན་མོའི་བྷ་ག་ལ་སོགས་པའི་རྣམ་འཕྲུལ་མཛད་ནས་གསུངས་པ། ཕྱིས་ཕྱག་ན་རྡོ་རྗེས་བསྡུས་པ་ལྟ་བུར་གསལ་ལོ། །དཔལ་གསང་བ་གྲུབ་པ་ལས་སྔ་མའི་གཏམ་དེ་སྡུད་པ་ཡིན་པར་འཆད་དེ་འོག་ཏུ་བཤད་པར་བྱའོ།

།ཡང་སློབ་དཔོན་ཁ་ཅིག་ན་རེ། མཚན་ཡང་དག་པར་བརྗོད་པ་ལས། འདས་པའི་སངས་རྒྱས་རྣམས་ཀྱིས་གསུངས། །མ་འོངས་རྣམས་ཀྱང་གསུངས་འགྱུར་ལ། །ད་ལྟར་བྱུང་བའི་རྫོགས་སངས་རྒྱས། །ཡང་དང་ཡང་དུ་གསུངས་པ་གང༌། །ཞེས་འབྱུང་བས། ཐོག་མར་ཡང་གསུངས་ལ། ད་ལྟར་གྱི་སངས་རྒྱས་ཀྱིས་ཀྱང་གསུངས་པས་རྒྱུན་ཏུ་གསུངས་པ་ཡིན་ནོ། །ཞེས་ འདོད་དོ། །དོན་གཉིས་པ་ ¡Ã གསང་སྔགས་ ( ¡¶ ཞ ¡· - གསང་སྔགས་ ) དེ་ལྟར་གསུངས་ པ་ལ་འགལ་བ་སྤང་བ་ལ་གཉིས་ཏེ། གསུངས་པའི་དུས་ལ་འགལ་བ་སྤང་བ་ དང་ ༑ སྟོན་པ་ལ་འགལ་བ་སྤང་བའོ། །དང་པོ་ནི་གལ་ཏེ་མཚན་བརྗོད་ལས་དུས་གསུམ་གྱི་སངས་རྒྱས་ཐམས་ཅད་ཀྱིས་གསུངས་པར་བཤད་པ་དེ་དང་གསང་བ་འདུས་པ་ལས་མར་མེ་མཛད་ནས་འོད་སྲུངས་ ¡Ã ཀྱི་བར་གྱིས་ ( ¡¶ ཞ ¡· ཀྱིས་ ) མ་གསུངས་པར་བཤད་པ་དེ་དག་ ( ¡¶ ཞ ¡· དང་ ) འགལ་ལོ་ཞེ་ན། དེ་ལ་སློབ་དཔོན་ཡེ་ཤེས་ཞབས་ཀྱི་རྗེས་སུ་འབྲང་བ་ཁ་ཅིག་གསུང་བར་བཤད་པ་ནི་ལོངས་སྤྱོད་རྫོགས་པའི་སངས་རྒྱས་ཀྱི་དབང་དུ་མཛད་པ་ཡིན་ལ། མི་གསུངས་བར་བཤད་པ་ནི་སྤྲུལ་པའི་སྐུའི་དབང་དུ་བྱས་པ་ཡིན་ཏེ་སྤྲུལ་པའི་སྐུས་མ་ ( ¡¶ ཞ ¡· - མ་ ) བཤད་པའམ ༑ ཕྱག་ན་རྡོ་རྗེ་བདག་གི་དེའི་བར་དུ་མ་བཤད་དོ། །བར་དུ་ཞེས་བྱ་བ་འདིས་གོང་དུ་བྱུང་བ་དང་ཕྱིས་འབྱུང་བ་མ་བཀག་གོ །ཞེས་འཆད་དོ། །ཡང་ཁ་ཅིག་གསུང་བར་བཤད་པ་ནི། སྤྱིར་གསང་སྔགས་གཞན་ཡིན་ལ། མི་གསུང་བར་བཤད་པ་ནི་གསང་བ་འདུས་པ་འབའ་ཞིག་ཡིན་ནོ། །ཞེས་ཟེར་རོ། །ཡང་སློབ་དཔོན་ཟླ་བ་གྲགས་པ་ལ་སོགས་པ་མང་པོའི་བཞེད་ཀྱིས། གསུངས་པར་བཤད་པ་ནི་ངེས་པའི་དོན་ཡིན་ལ་མི་གསུང་བར་བཤད་པ་ནི་དཔལ་གསང་བ་འདུས་པའི་ཆེ་བ་བརྗོད་པ་ཙམ་ཡིན་ཏེ། དཔེར་ན་སངས་རྒྱས་ཐམས་ཅད་ཀྱིས་གསུངས་ཀྱང་འཇིག་རྟེན་དུ་མ་བྱུང་བའི་ཆོས་ཀྱི་འཁོར་ལོ་བསྐོར་རོ་ཞེས་གསུང་པ་ལྟ་ ¡Ã བུའམ ( ¡¶ ཞ ¡· བུའོ། ) ། སངས་རྒྱས་དང་བྱང་ཆུབ་སེམས་དཔའ་ཐམས་ཅད་ཀྱང་འཇིགས་པར་གྱུར་ཏོ་ཞེས་ལ་དོར་བ་ལྟ་བུའམ།

སངས་རྒྱས་ཐམས་ཅད་ཀྱི་གསུང་གཅིག་ཡིན་ཞིང་མཐུན་ཡང་དུས་གཅིག་གི་ཚེ་འདི་ཐོས་ཀྱི་གཞན་དུ་ ( ¡¶ ཞ ¡· + ནི་ ) འདི་མ་ཐོས་ཞེས་གསུངས་པ་ལྟ་བུའོ་ཞེས་བཞེད་དོ། །འདིས་ནི་ཡང་དང་ཡང་དུ་གསུངས་པ་དང༌། དུས་གཅིག་གི་ཚེ་འདི་ཐོས་བྱ་བའི་འགལ་བའང་བསལ་བ་ཡིན་ནོ། །སློབ་དཔོན་ཨཱརྱ་དེ་བའང་འདི་བཞེད་དེ། སྤྱོད་པ་བསྡུས་པའི་སྒྲོན་མ་ལས ༑ རྒྱུད་ཡང་དག་པ་རྣམས་ལས་བཏུས་ཏེ་སྒྲུབ་ཐབས་ཀྱི་ཡན་ལག་མར་མེ་ལྟ་བུ་དག་སྔོན་བྱེད་པ་འབྱུང་བ་མི་རིགས་ཏེ། རྫོགས་ལྡན་དང་སུམ་ལྡན་དང་ཉིས་ལྡན་གྱི་དུས་ན་ཡོན་ཏན་ཕུན་སུམ་ཚོགས་པ་དང་ལྡན་པས་རྒྱུད་ཀྱི་ཚིག་རྒྱ་ཆེ་བ་དང་སྤས་པའི་སྒྲའི་སྒོ་ནས་དེ་ཁོ་ན་ཉིད་སྟོན་ལ་དེ་རྣམས་ཀྱིས་ཀྱང་ཁོང་དུ་ཆུད་པ་ཡིན་ནོ། །ད་ལྟ་ནི་རྩོད་པའི་དུས་ཏེ་ཞེས་འབྱུང་བས། ཤཱཀ་ཐུབ་ནི་རྩོད་དུས་སུ་བྱུང་ལ་རྫོགས་ལྡན་ལ་སོགས་པར་བསྟན་བཅོས་བཀག་ལ་རྒྱུད་ཡོད་པར་བཤད་པས་ཤཱཀ་ཐུབ་པས་སྔ་བར་ཡོད་པར་བཞེད་པ་ཡིན་ནོ། །གཉིས་པ་སྟོན་པ་ལ་འགལ་བ་སྤང་བ་ནི། ཤཱཀ་ཐུབ་ལས་སྔ་བར་ཡོད་ན། ཤཱཀྱ་སེང་གེ་མཆོག་གིས་ ནི ༑ ༑ དྲག་པོ་ཆེན་པོ་བདུད་ཀྱི་སྡེ། །སྟོབས་ཆེན་དག་ཀྱང་དེས་བཅོམ་ནས། ། འཁོར་ལོ་བསྐོར་ནས་མྱ་ངན་འདས། །ཞེས་བྱ་བ་ལ་སོགས་པ་ཤཱཀྱ་ཐུབ་པའི་མཚན་ཁོ་ན་འབྱུང་བ་མ་ཡིན་ནམ་ཞེ་ན། མཚན་བྱུང་མོད་དེ་ནི་སྔོན་ ¡Ã བྱུང་བའི་ ( ¡¶ ཞ ¡· གསུངས་པའི་ ) བཅོམ་ལྡན་འདས་རྣམ་པར་སྣང་མཛད་ཉིད་ ( ¡¶ ཞ ¡· + དང་ ) ད་ལྟའི་ཤཱཀྱ་ཐུབ་པ་སྟེ་ཐ་དད་པ་མེད་པས་ཐ་དད་དགག་པའམ་མཚོན་སླ་བའི་དབང་དུ་བྱས་ནས་སྡུད་པ་པོས་དེ་ལྟར་གསུངས་པ་ཡིན་ཏེ་དཔེར་ན་འཕགས་པ་མཽང་གལ་གྱི་བུའི་སྐྱེ་བ་དཔག་ཏུ་མེད་པའི་གོང་གི་ལས་བྱས་པ་དེའང༌། བཅོམ་ལྡན་འདས་ཀྱིས་དགེ་སློང་མཽང་གལ་གྱི་བུས་ལས་བྱས་པ་དེ་གཞན་ས་ལ་མ་ཡིན་ཆུ་ལ་མ་ཡིན་མེ་ལ་མ་ཡིན་རླུང་ལ་མ་ཡིན་མཽང་གལ་གྱི་བུ་ཉིད་ལ་སྨིན་ཅིང་ཉམས་སུ་མྱོང་ངོ་ཞེས་བཤད་པ་ལྟ་བུའོ།

།དེ་བས་ན་ཤཱཀྱ་ཐུབ་པའི་མཚན་ལ་འགལ་བ་མེད་དོ། །དོན་བཞི་པ་བཀའ་བསྡུ་བ་ཇི་ལྟར་བྱས་པ་ལ་གསུམ། ཞར་ལ་བྱུང་བ་ཉན་ཐོས་ཀྱི་བསྡུ་བ་དང༌། ཐེག་པ་ཆེན་པོའི་བསྡུ་བ་དང༌། གསང་སྔགས་ཇི་ལྟར་བསྡུས་ཚུལ་ལོ། །དང་པོ་ནི་ཉན་ཐོས་ཀྱི་སྡེ་སྣོད་མཐའ་དག་བི་མ་ལ་ཡ་ནའི་ཕུག་ཅེས་བྱ་བར་དགྲ་བཅོམ་པ་ལྔ་བརྒྱ་འདུས་ནས། མདོ་སྡེ་རྣམས་ནི་ཀུན་དགའ་བོས་ བསྡུས ༑ འདུལ་བ་ཉེ་བ་འཁོར་གྱིས་བསྡུས། མངོན་པ་འཕགས་པ་འོད་སྲུངས་ཀྱིས་བསྡུས་སོ། །གཉིས་པ་ཐེག་པ་ཆེན་པོའི་མདོ་རྣམས་ནི་སློབ་དཔོན་ཁ་ཅིག་ན་རེ་ལྷོ་ཕྱོགས་ཀྱི་རྒྱུད་བི་མ་ལ་སཾ་བྷ་བ་ཞེས་བྱ་བར་བྱང་ཆུབ་སེམས་དཔའ་བྱེ་བ་ཕྲག་དགུ་བཅུ་འདུས་ནས་བསྡུས་སོ་ཞེས་བཞེད་ན། དེ་ལའང་སྐྱོན་ཆེན་པོ་ནི་ མེད་ལ། འོན་ཀྱང་གསང་སྔགས་དང་ཐེག་པ་ཆེན་པོ་ཐུན་མོང་མ་ཡིན་པ་གཉིས་ཀ་ ཕྱག་ན་རྡོ་རྗེ་ཉིད་ཀྱིས་བསྡུས་ཏེ། སློབ་དཔོན་དབྱིག་གཉེན་གྱིས་ཕ་རོལ་ཏུ་ཕྱིན་པའི་གཞུང་འགྲེལ་ལས། བྱང་ཆུབ་སེམས་དཔའ་ཕྱག་ན་རྡོ་རྗེ་ལ་འཕགས་པ་བྱམས་པས་དྲིས་པ་ལས། ཤེས་རབ་ཀྱི་ཕ་རོལ་ཏུ་ཕྱིན་པ་འདི་བདག་གིས་ཐོས་ པ་ཡིན་པར་འཁོར་གྱི་ནང་དུ་ཁས་ ¡Ã འཆེས་སོ་ ( ¡¶ ཞ ¡· འཆེའོ་ ) ཞེས་བཤད་པ་ལྟ་ བུའོ ༑ ༑ གནས་ཐ་དད་པར་བསྡུས་པ་ནི། དགག་པར་ཡང་མི་ནུས་ བསྒྲུབ་ པར་ཡང་ མི་ནུས་སོ། །གསུམ་པ་གསང་སྔགས་ཇི་ལྟར་བསྡུས་པ་ལ་གཉིས། གཞན་གྱི་ ལོག་རྟོག་དགག་པ་དང༌། རང་གི་འདོད་པ་བརྗོད་པའོ། །དང་པོ་ནི་འདི་ན་ཁ་ ཅིག་སྨྲ་བ། དཔལ་སྡོམ་པའི་རྒྱུད་ཕྱི་མ་ཞེས་བྱ་བ་ལས། ང་ནི་འདི་ནས་མི་སྣང་ནས ༑ ༑ ལོ་ནི་བརྒྱ་དང་བཅུ་གཉིས་ན། །བསྟན་པའི་སྙིང་པོ་དམ་པ་ཞིག །ལྷ་ གནས་གསུམ་དུ་གྲགས་པ་དེར། །འཛམ་གླིང་ཤར་ལྷོའི་ཕྱོགས་མཚམས་ན། ། དཔལ་ཞེས་བྱ་བའི་རི་བོ་ལ། །ཐུགས་རྗེ་མཐུན་པའི་བྱིན་རླབས་ཀྱིས། །རྒྱལ་པོ་ཛ་ཞེས་བྱ་བ་ལ། །ལག་ན་རྡོ་རྗེས་གནང་བར་བྱེད། །མི་ལ་སྐལ་ལྡན་དམ་པ་རྣམས ༑ ༑ བྱང་ཆུབ་མཆོག་ཏུ་གྲུབ་འགྱུར་བས།

།དེ་ལ་རྟག་ཏུ་གུས་པར་བྱོས། ། ཞེས་འབྱུང་བ་ཡོད་དོ་ཞེས་ཟེར་རོ། །དེ་བས་ན་ཤཱཀྱ་ཐུབ་པ་མྱ་ངན་ལས་འདས་ནས་ལོ་ ¡Ã ཉི་ཤུ་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· བརྒྱ་ཉི་ཤུ་ ) ན་ཤར་ལྷོའི་ཕྱོགས་མཚམས་ཟ་ཧོར་གྱི་ཡུལ་ཨཽ་ཌྱན་ཞེས་བྱ་བར་རྒྱལ་པོ་ཛ་ཞེས་བྱ་བའམ། ཨིནྡྲ་བྷཱུ་ཏི་ཞེས་བྱ་བའི་ཁང་པའི་སྟེང་དུ་གསེར་གྱི་གླེགས་བམ་ལ་བཻ་ཌཱུརྻ་བཞུ་བས་བྲིས་པའི་པོ་ཏི་དཔག་ཏུ་མེད་པ་བབས་སོ། །དེ་རྒྱལ་པོས་བལྟས་པས་ཤེས་ཏེ། བསྒྲུབས་པས་གྲུབ་པ་ཐོབ་བོ་ཞེས་ཟེར་རོ། ཁ་ཅིག་ན་རེ་རྒྱལ་པོས་མ་ཤེས་ཏེ་དགེ་སློང་ཀུ་ཀུ་རི་ ( ¡¶ ཞ ¡· + པ་ ) ཞེས་བྱ་བ་སྤྱན་དྲངས་པས་དེས་ཀྱང་མ་ཤེས་ཏེ་རྡོ་རྗེ་སེམས་དཔའ་ཞལ་མཐོང་གི་ལེའུ་ཤེས་ནས་བསྒྲུབས་པས་རྡོ་རྗེ་སེམས་དཔའ་ཞལ་མཐོང་སྟེ་དེར་རྒྱུད་སྡེ་བཅོ་བརྒྱད་ ( ¡¶ ཞ ¡· + པོ་ ) ཐམས་ཅད་ཤེས་སོ། །དེ་ནས་རྒྱལ་པོ་ལ་སོགས་པ་འཁོར་དང་བཅས་པས་གྲུབ་བོ། །དེས་གཞན་ལ་འཕེལ་ལོ་ཞེས་ཟེར་བ་ལ་སོགས་པའི་གཏམ་མང་པོ་ཡོད་དེ་མ་བྲིས་སོ། །ཇི་ལྟར་མི་འབྲྀ་ཞེ་ན། འདི་ནི་རང་བཟོ་ ¡Ã མཁན་རྣམས་ཀྱིས་རང་བཟོར་སྦྱར་བ་ ( ¡¶ ཞ ¡· བྱས་པ་ ) ཅིའང་རུང་བ་སྨྲས་པའི་ཕྱིར་རོ། ། འདི་རང་བཟོ་ཇི་ལྟར་ཡིན་ཞེ་ན། རྒྱལ་པོ་ཛ་ལ་ཕྱག་ན་རྡོ་རྗེས་འཆད་པར་འགྱུར་རོ་ཞེས་བྱ་བ་དེ་དགག་པར་མི་ནུས་སོ། །གླེགས་བམ་བལྟས་ནས་བསྒྲུབས་པས་གྲུབ་ན། གསང་སྔགས་རང་ཉིད་ལེན་པ་ནི། །ལུས་ཅན་དམྱལ་བ་འབའ་ཞིག་རྒྱུ།། ཞེས་བཤད་པ་དང་འགལ་ལ། བླ་མ་མེད་པར་འགྲུབ་པ་མེད་པར་བཤད་པ་དང་ཡང་འགལ་ལོ། །གཞན་ཡང་ཤར་ལྷོའི་ཕྱོགས་མཚམས་ཞེས་ཟེར་བ་དེ་ཨཽ་ཌྱན་གྱི་ཡུལ་ཡིན་ཞེས་བྱ་བ་འདི་འཇིག་རྟེན་གྱི་གྲགས་པ་དང་མངོན་སུམ་དང་ཡང་འགལ་ལོ། །དེ་ལ་སོགས་པའི་བརྫུན་གྱི་གཏམ་དུ་མ་ཡོད་དེ་བྲི་བར་སུས་ནུས། དེ་བས་ན་གྱི་ན་དེ་བོར་བར་བྱའོ། །དེ་བཞིན་དུ་དེ་དག་གིས་བཤད་པའི་རྒྱུད་སྡེའི་དབྱེ་བ་ལ་སོགས་པ་ཕན་ཚུན་འགལ་བ་དུ་མ་ཡོད་དེ། དེ་དག་གི་སུན་དབྱུང་བ་གཞན་དུ་ཤེས་པར་བྱ་བས་འདིར་མ་བྲིས་སོ།

༑ ༑ ཚུལ་ཇི་ལྟར་བསྡུས་ན། བཅོམ་ལྡན་འདས་ཀྱིས་རྣམ་པར་འཕྲུལ་པ་མཛད་པས་འདོད་ཆགས་དང་ལྡན་པ་ཉིད་དུ་བསྟན་ནས། ཆགས་ཅན་ཀྱི་སེམས་འདུན་ པར་མཛད་དེ་སྐལ་བ་དང་ལྡན་པ་ཐམས་ཅད་དབང་བསྐུར་ནས་རྒྱུད་སྨིན་པར་མཛད་པ་ལ་ཕྱག་ན་རྡོ་རྗེ་དེ་བཞིན་གཤེགས་པ་ཐམས་ཅད་ཀྱི་བདག་པོས་ གསུངས་ པ་སྟེ། དཔལ་གསང་བ་གྲུབ་པ་ལས། ཡེ་ཤེས་སྒྱུ་མའི་རང་བཞིན་ཐུགས་རྡོ་རྗེ་སྐྱེས་བུ་ཆེན་པོའི་མཚན་སུམ་ཅུ་རྩ་གཉིས་ཀྱིས་བརྒྱན་པའི་སྐུར་གྱུར་ནས་འདི་སྐད་བདག་གིས་ཞེས་བྱ་བ་ལ་སོགས་པས་དོན་དམ་པའི་དེ་ཁོ་ན་ཉིད་དང་སངས་རྒྱས་དང་བྱང་ཆུབ་སེམས་དཔའ་རྣམས་ལ་བདག་ཉིད་སྟོན་ཏོ་ཞེས་འབྱུང་བ་དང༌། འདི་ཡི་སྡུད་པར་བྱེད་པ་པོ། །གཞན་དག་ཡོད་པ་མ་ཡིན་ཞེས། །བདག་ཉིད་འབའ་ཞིག་སྨྲ་བར་བྱེད། །རྒྱུད་མཛད་པ་ནི་ཐུགས་རྡོ་རྗེ། །ཞེས་གསུངས་སོ། ། དེ་བས་ན་ངེས་པའི་དོན་ལ་མོས་པ་རྣམས་ཀྱིས་འདི་སྐད་ལ་སོགས་པ་རྒྱུད་ཀྱི་ལུས་རྣམ་པར་བཞག་པ་སྟེ་ཉམས་སུ་བླང་བྱ་ཉིད་དུ་གོའོ། །བློ་དམན་པ་རྣམས་ཀྱིས་ ནི་བཅོམ་ལྡན་འདས་ཕྱག་ན་རྡོ་རྗེ་འདིས་བཅོམ་ལྡན་འདས་ལས་ཐོས་ནས་སྡུད་པ་ མཛད་དོ་ཞེས་གོ་བར་གྱུར་ཏོ། །དེས་ན་གཏམ་རྒྱུད་དང༌། འདི་སྐད་བདག་གིས་ ཐོས་པ་སངས་རྒྱས་དང་བྱང་ཆུབ་སེམས་དཔའ་རྣམས་ལ་སྟོན་ཏོ་ཞེས་བྱ་བ་འདིས་དུས་བཅོམ་ལྡན་འདས་བཞུགས་པའི་དུས་སུ་བསྡུས་པར་ཤེས་སོ། །དེ་ལྟར་ན་ཕྱག་ན་རྡོ་རྗེས་རྒྱུད་སྡེ་མཐའ་དག་བསྡུས་ནས་རྒྱལ་པོ་འཁོར་དང་བཅས་པ་ལ་བྱིན་ནོ། །དེ་དག་གིས་ཀྱང་དངོས་གྲུབ་ཐོབ་པར་གྱུར་པས་རང་བཞིན་གྱིས་གྲུབ་པའི་དཔའ་བོ་དང་རྣལ་འབྱོར་མ་ཞེས་བྱའོ། །དེས་ཨཽ་ཌྱན་གྱི་ཡུལ་དེ་སྟོངས་ཏེ་དེར་མཚོ་ཆེན་པོ་ཀླུས་གང་བ་ཅིག་བྱུང་བར་གྱུར་ཏོ། །དེ་ནས་ཡང་ཕྱག་ན་རྡོ་རྗེས་རྒྱུད་མཐའ་དག་གསེར་གྱི་གླེགས་བམ་ལ་བཻ་ཌཱུརྻ་ཞུན་མས་བྲིས་ཏེ། ཡུལ་དེ་ན་མཚོ་ཆེན་པོའི་ཀླུ་རྣམས་སྨིན་པར་མཛད་ནས་དེ་རྣམས་ལ་རྒྱུད་ཀྱི་གླེགས་བམ་རྣམས་ཕྱག་ན་རྡོ་རྗེས་གཏད་དོ།

༑ ༑ བུད་མེད་རྣམས་ཕལ་ཆེར་མཁའ་འགྲོ་མར་གྱུར་ཏོ། །དེ་སྐད་ཅེས་དེང་སང་ཡང་གཏམ་དུ་གྲགས་ཏེ། དཔལ་ཨཽ་ཌྱན་མཁའ་འགྲོ་མའི་གནས་ཞེས་བྱའོ། ། ¡Ã དེ་ནས་ཡུལ་དེའི་ ( ¡¶ ཞ ¡· དེས་ན་ཡུལ་དེ་ན་གནས་པའི་ ) རྒྱལ་པོ་རྣམས་ཀྱང་ཆོས་དང་ལྡན་པ་ཤ་སྟག་སྟེ། དེའང་རིམ་གྱིས་འགྱུར་ཏེ་འདིའི་གཏམ་ནི་ལྷན་ཅིག་སྐྱེས་པ་གྲུབ་པའི་འགྲེལ་པ་ལས་གསལ་ལོ། །དེང་སང་ནི་དང་པོའི་རྒྱལ་པོ་ཨིནྡྲ་བྷཱུ་ཏིའི་རིགས་ཡིན་ནོ་ཞེས་ཟེར་བ་དག་ཀྱང་ཡོད་དེ་ཕྱིས་ཀྱིས་བཏགས་པ་ཡིན་པར་བས་སོ ༑ ༑ དེ་ལྟར་ན་བྱང་ཕྱོགས་ཨཽ་ཌྱན་ཡོན་ཏན་གྱི་འབྱུང་གནས་ཞེས་བྱའོ། །དོན་ལྔ་པ་དེ་ནས་ཡུལ་གཞན་དུ་ཇི་ལྟར་འཕེལ་བ་ནི། ཡུལ་ལྷོ་ཕྱོགས་ཀྱི་རྒྱུད་ན་རྒྱལ་པོ་བི་སུ་ཀལྤ་ཞེས་བྱ་བ་ཉན་ཐོས་ཀྱི་ཐེག་པ་ལ་ཞུགས་པ་ཅིག་གི་རྨི་ལམ་དུ་ཁྱོད་ཨཽ་ཌྱན་དུ་སོང་ཤིག་སྐྱེ་བ་འདིར་རྡོ་རྗེ་འཛིན་པ་ཐོབ་པར་འགྱུར་རོ་ཞེས་ཐོས་ཏེ་སད་ནས་ཐེ་ཚོམ་མེད་པར་སོང་སྟེ། ཨཽ་ཌྱན་དུ་བུད་མེད་ཅིག་དང་ཕྲད་དེ་བསྙཻན་བཀུར་བྱས་པས་དེས་བྱིན་གྱིས་བརླབས་ཏེ་གསང་བ་འདུས་པ་དང་དེ་ཉིད་འདུས་པ་བྱིན་ཏེ ༑ དེས་དེ་ཁྱེར་ནས་འོངས་ཏེ་ཡུལ་ལྷོ་ཕྱོགས་སུ་འཕེལ་ལོ་ཞེས་ཟེར་རོ། །དེ་བཞིན་དུ་སློབ་དཔོན་དུ་མས་ཨཽ་ཌྱན་ནས་རྒྱུད་སྡེ་དུ་མ་སྤྱན་དྲངས་སོ། །ཀྱེའི་རྡོ་རྗེའི་རྒྱུད་ནི་སློབ་དཔོན་ཆེན་པོ་ཌོཾ་བི་ཧེ་རུ་ཀ་ཞེས་བྱ་བས་སྤྱན་དྲངས་སོ། ། ཞེས་ཟེར་རོ། ཁ་ཅིག་ན་རེ་སློབ་དཔོན་ཀླུ་སྒྲུབ་གླེན་པར་བརྫུས་ནས་བརྐུས་སོ་ཞེས་ཟེར་ཏེ། འདི་ནི་བཞད་གད་ཀྱི་གནས་སོ། །ཇི་ལྟར་བཞད་གད་ཡིན་ཞེ་ན། ¡Ã སློབ་དཔོན་ལ་སྐུར་པ་བཏབ་པར་འགྱུར་ཏེ་ས་དང་པོ་པ་ལ་བརྐུ་བ་ལས་རྫུ་འཕྲུལ་མེད་པའི་ཕྱིར་རོ། །མཁའ་འགྲོ་མ་ལ་སྐུར་པ་བཏབ་པར་འགྱུར་ཏེ་བརྐུ་བ་དང་ཀླུ་སྒྲུབ་ཏུ་ངོ་མི་ཤེས་པའི་དཔེ་གཉེར་བ་ག་ལ་འོང་ཞེས་བྱ་བ་ལ་སོགས་ པའོ ༑ ༑ རང་གི་ཆོས་ལའང་སྐུར་པ་བཏབ་པར་འགྱུར་ཏེ། བརྐུས་པའི་དཔེ་ལས་བརྒྱུད་པའི་ཕྱིར་རོ།

། ( ¡¶ ཞ ¡· - སློབ་དཔོན་་་་་་་ཕྱིར་རོ། ། ) དེ་བཞིན་དུ་སློབ་དཔོན་ཀུ་ཀུ་ རི་པས་མ་ཧཱ་མཱ་ཡའི་རྒྱུད་སྤྱན་དྲངས་སོ། །དེ་བཞིན་དུ་སློབ་དཔོན་ལ་ལི་ཏ་བཛྲས་ རྡོ་རྗེ་འཇིགས་བྱེད་ཀྱི་རྒྱུད་སྤྱན་དྲངས་ཏེ་ ( ¡¶ ཞ ¡· སོ། ) ། དེ་ལ་སོགས་པ་བསམ་ གྱིས་མི་ཁྱབ་སྟེ། སློབ་དཔོན་རྣམས་ཀྱི་གཏམ་རྒྱུད་དང་བོད་ཀྱི་ཡུལ་དུ་རྒྱུད་གང་དག་ཇི་ལྟར་བསྒྱུར་བའི་གཏམ་རྒྱུད་ནི་གཞན་དུ་ཤེས་པར་བྱའོ། །དེ་ལྟར་གསང་ སྔགས་བྱུང་ཚུལ་འདི་ནི་བླ་མའི་གསུང་ཉིད་ལ། །གཙོ་བོར་བགྱིས་ནས་སྣ་ཚོགས་རྒྱུད་མདོ་བསྟན་བཅོས་ལས། །ལེགས་པར་བསྡུས་ ( ¡¶ ཞ ¡· བཏུས་ ) ཏེ་རང་བཟོ་སྤངས་ནས་ཕྱོགས་གཅིག་ཏུ། །གསལ་བར་བགྱིས་པ་འདིས་ནི་འཕགས་རྣམས་ མཉེས་གྱུར་ཅིག རྒྱུད་སྡེ་སྤྱིའི་རྣམ་པར་བཞག་པ་ལས་ ( ¡¶ ཞ ¡· དང་ ) གསང་སྔགས་ཇི་ལྟར་བྱུང་བའི་ཚུལ་རྣམ་པར་བཞག་པའི་སྐབས་ཏེ་བདུན་པའོ། །དེ་ལྟར་སྐབས་རྣམ་པ་བདུན་གྱིས་རྒྱུད་སྡེ་སྤྱིའི་རྣམ་པར་བཞག་པ་བསྟན་ཏེ་ལེའུ་དང་པོའོ།། །། ད་ནི་སྤྱི་དོན་གཉིས་པ་རྒྱུད་ཀྱི་མཚན་དོན་བརྗོད་པར་བྱའོ། །དེ་ལ་གཉིས་ཏེ ༑ བརྗོད་བྱ་དོན་གྱི་རྒྱུད་དང༌། རྗོད་བྱེད་སྒྲའི་རྒྱུད་བསྟན་པའོ། །དང་པོ་ ¡Ã ལ་ ལྔ ༑ ( ¡¶ ཞ ¡· ལའང་ ) སྤྱིར་རྒྱུད་ཅེས་བྱ་བ་ཙམ་ ( ¡¶ ཞ ¡· - ཙམ་ ) གང་ཡིན་པ་བསྟན་པ་ དང་ ༑ དེ་ལ་གསུམ་དུ་དབྱེ་བ་དང༌། གསུམ་པོའི་ངེས་ཚིག་རྣམ་པར་བཞག་པའི་ཚུལ་དང༌། དེ་ལ་རྒྱུད་སོ་སོའི་མཚན་ཐབས་དང་ཤེས་རབ་ལ་སོགས་པར་བཞག་པ་དང༌། རྒྱལ་པོ་ལ་སོགས་པར་བཞག་པའོ། །དང་པོ་ནི་སྤྱིར་རྒྱུད་ཅེས་བྱ་བ་ནི་རང་གི་སེམས་ཉིད་གཉིས་སུ་མེད་པའི་ཡེ་ཤེས་འདི་ཉིད་ལ་རྒྱུད་ཅེས་བྱ་སྟེ་ཐོག་མ་མེད་པ་ནས་སངས་རྒྱས་ཀྱི་བར་དུ་ཟག་བཅས་ཀྱི་སེམས་རྒྱུན་མི་འཆད་པས་ ན་རྒྱུད་དོ། །དེ་ལྟར་འདུས་པའི་རྒྱུད་ཕྱི་མ་དང༌། རྡོ་རྗེ་རྩེ་མོ་ལས། རྒྱུད་ནི་རྒྱུན་ ཆགས་ཞེས་བྱ་སྟེ། །ཞེས་གསུངས་པ་དང༌། སྒྱུ་འཕྲུལ་དྲ་བ་ལས། རྒྱུད་ནི་རྒྱུན་ ཞེས་བྱ་བ་སྟེ།

།ཞེས་གསུངས་སོ། །གཉིས་པ་དེ་ལ་གསུམ་དུ་དབྱེ་བ་ནི། སེམས་ དེ་ཉིད་ལ་རྒྱུ་དང་ཐབས་དང་འབྲས་བུའི་རྒྱུད་དུ་འགྱུར་ཏེ། དང་པོ་སེམས་ཅན་འཁོར་བའི་གནས་སྐབས་ནི་རྒྱུ་རྒྱུད། བར་དུ་ལམ་ལ་ཞུགས་པ་ནི་ཐབས་རྒྱུད། འབྲས་བུ་ཡེ་ཤེས་ཀྱི་སྐུ་གྲུབ་པ་ནི་འབྲས་བུའི་རྒྱུད་དོ། །དེ་ལྟར་ཡང་གསང་བ་འདུས་པའི་རྒྱུད་ཕྱི་མ་ལས། རྒྱུད་ནི་རྒྱུན་ཆགས་ཞེས་བྱ་སྟེ། །རྒྱུན་དེ་རྣམ་པ་གསུམ་དུ་འགྱུར། །གཞི་དང་དེ་ཡི་རང་བཞིན་དང༌། །མི་འཕྲོགས་པ་ཡིས་རབ་ཕྱེ་བའོ ༑ ༑ རང་བཞིན་རྣམ་པ་རྒྱུ་ཡིན་ཏེ། །གཞི་ནི་ ཐབས ་ཞེས་བྱ་བ་ཡིན། །དེ་བཞིན་ མི་འཕྲོགས་འབྲས་བུ་སྟེ། །གསུམ་གྱིས་རྒྱུད་ཀྱི་དོན་བསྡུས་པའོ། །ཞེས་བཤད་པ་ལྟ་བུའོ། །གསུམ་པ་དེའི་ངེས་ཚིག་རྣམ་པར་བཞག་པ་ནི། དེ་ལ་རྒྱུ་ཞེས་བྱ་བ་ནི། ད་ལྟ་དངོས་སུ་མེད་ཀྱང་སྤྱིར་རྐྱེན་དང་ཕྲད་ན་འབྲས་བུ་སྐྱེར་རུང་གི་ནུས་པ་ལ་བྱ་སྟེ ༑ དཔེར་ན་སྣོད་ན་གནས་པའི་ས་བོན་དང་འདྲའོ། །དེ་བས་ན་རང་གི་སེམས་འདི་ད་ལྟ་སངས་རྒྱས་དངོས་སུ་མ་གྲུབ་ཀྱང་ཐབས་དང་ཕྲད་ན་སངས་རྒྱས་འབྱུང་ བས་སངས་རྒྱས་ཀྱི་རྒྱུ་ཞེས་བྱ་སྟེ། ཧེ་བཛྲ་ལས། སེམས་ཅན་རྣམས་ནི་སངས་ རྒྱས་ཉིད། །འོན་ཀྱང་གློ་བུར་དྲི་མས་བསྒྲིབས། །དེ་བསལ་ན་ནི་སངས་རྒྱས་ ཉིད ༑ ༑ ཅེས་གསུངས་པ་ལྟ་བུའོ། །དེ་རྒྱུད་དུ་ཇི་ལྟར་འགྲོ་ཞེ་ན། གནས་སྐབས་ཀྱིས་འགྲོ་བ་རིགས་དྲུག་ལ་སོགས་པ་དང༌། ལམ་སྨིན་གྲོལ་དུ་མ་དང༌། འབྲས་བུ་སྣ་ཚོགས་སུ་འགྱུར་ཡང༌། སེམས་ཉིད་ཀྱི་ངོ་བོ་ལ་འགྱུར་བ་མེད་པར་རྒྱུན་མ་ ཆད་པས་རྒྱུད་དེ། གུར་ལས། ནམ་མཁའ་ཇི་སྲིད་མི་འཇིག་པ། །ནམ་མཁའ་ ཇི་སྲིད་དེ་སྲིད་སེམས། །ཞེས་གསུངས་པ་ལྟ་བུའོ། །ཐབས་ཞེས་བྱ་བ་ནི་རྒྱུ་ལ་ གནས་པ་དེ་འབྲས་བུར་འབྱིན་པའི་ཐབས་སུ་གྱུར་པས་ཐབས་ཏེ་ཆུ་ལུད་ལྟ་བུའོ།། དེས་ན་སྨིན་གྲོལ་གྱི་ལམ་འཁོར་དང་བཅས་པ་ལ་ཐབས་ཞེས་བྱ་སྟེ། དཔལ་སཾ་ པུ་ཊ་ལས། སེམས་ཅན་ཡེ་ཤེས་སྤེལ་བའི་ཕྱིར།

།ཐབས་ནི་སངས་རྒྱས་རྣམས་ ཀྱིས་བསྟན། །ཞེས་སོ། །ཐབས་དེ་རྒྱུད་ཇི་ལྟར་ཡིན་ཞེ་ན། གདམས་ངག་གི་བརྒྱུད་པ་བར་མ་ཆད་པའི་ཕྱིར་དང༌། ཉམས་མྱོང་ཏིང་ངེ་འཛིན་བར་མ་ཆད་པའི་ ཕྱིར་རོ། །དེ་ལྟར་ཡང་སཾ་པུ་ཊ་ལས། བསྟན་བཅོས་ཚད་མ་སློབ་དཔོན་དང༌། ། ལུང་གི་རྗེས་འབྲངས་དེ་ཉིད་རིག །གསང་དོན་དེ་ཉིད་དངོས་པོ་ཡང༌། །གཅིག་ནས་གཅིག་བརྒྱུད་ཤེས་པར་བྱ། །ཞེས་བྱ་བ་དང༌། ཧེ་བཛྲ་ལས། ཆུ་བོའི་རྒྱུན་ནི་རབ་འབབ་དང༌། །མར་མེའི་རྩེ་མོ་རབ་བཅིངས་ལྟར། །ཞེས་བྱ་བ་ལ་སོགས་པ་སྟེ ༑ མཉམ་གཞག་རྒྱུན་པར་ལས་དང་པོ་པ་ལ་མེད་ཀྱང་རྗེས་སྤྱོད་ ¡Ã ལ་སོགས་ ( ¡¶ ཞ ¡· རྫོགས་ ) པ་དང༌། དང་པོའི་སེམས་ཀྱི་འཕེན་པས་རྒྱུན་ཡིན་ཏེ། སྤྱོད་འཇུག་ལས ༑ རྒྱུན་མི་འཆད་པར་དུ་མ་ཞིག །ནམ་མཁའ་མཉམ་པར་རབ་ཏུ་འབྱུང༌། ། ཞེས་བཤད་པ་ལྟ་བུའོ། །འབྲས་བུ་ཞེས་བྱ་བ་ནི་རྒྱུ་ལ་གནས་པ་མངོན་དུ་གྱུར་ པ་སྟེ། མེ་ཏོག་སྨིན་པ་ལྟ་བུའོ། །དེ་བས་ན་སེམས་ཉིད་སངས་རྒྱས་སུ་གྲུབ་པ་དེ་ལ་འབྲས་བུ་ཞེས་བྱ་སྟེ། སཾ་པུ་ཊ་ལས། བརྒྱད་ཀྱི་རང་འདོད་རྣལ་འབྱོར་ ¡Ã པའི་ ( ¡¶ ཞ ¡· མའི་ ) ། །སངས་རྒྱས་རྣམས་ཀྱི་འབྲས་བུའི་དོན། །ཞེས་སོ། །འབྲས་བུ་དེ་རྒྱུད་ཇི་ལྟར་ཡིན་ཞེ་ན། རྟག་ཏུ་རྒྱུན་མི་འཆད་པའི་ཕྱིར་རྒྱུད་དེ། མདོ་སྡེ་རྒྱན་ ལས ༑ རང་བཞིན་དང་ནི་རྒྱུན་མི་འཆད། །རྒྱུད་གྱིས་དེ་དག་རྟག་པ་ཉིད། །ཅེས་གསུངས་ཏེ། ཆོས་ཀྱི་སྐུ་ནི་རང་བཞིན་གྱིས་རྟག་སྟེ་འགྱུར་བ་མེད་པས་རྒྱུད་དོ། ། ལོངས་སྤྱོད་རྫོགས་པའི་སྐུ་ནི་ཆོས་ཀྱི་ལོངས་སྤྱོད་རྒྱུན་མི་འཆད་པའི་ཕྱིར་རྒྱུད་ དོ ༑ ༑ སྤྲུལ་པའི་སྐུ་ནི་མཛད་པ་ཕྲིན་ལས་ཀྱི་འཁོར་ལོ་རྒྱུན་མི་འཆད་པའི་ཕྱིར་རྒྱུད་ དོ ༑ ༑ དེ་ལྟར་དཔལ་སཾ་པུ་ཊ་ལས། རྟག་པར་མཛད་པ་ཞེས་བྱ་བ་དང༌། བཅོམ་ལྡན་འདས་ནི་འོག་མིན་ལས་འཕོ་བར་མི་མཛད་དོ་ཞེས་གསུངས་པ་དང༌། སེམས་ ཅན་གྱི་རྒྱུད་དུ་རྒྱུ་ཞིང་ཆོས་ཀྱི་སྒོ་དུ་མ་རྗེས་སུ་བསྐོར་ཞིང་རྣམ་པར་དག་པའི་བཅོམ་ལྡན་འདས་འཁོར་བར་ལྷན་ཅིག་ཏུ་བཞུགས་སོ།

།ཞེས་གསུངས་པ་ལྟ་བུའོ ༑ ༑ དེ་ལྟར་ན་སྤྱིར་དུ་རྒྱུད་ཅེས་བྱ་བའི་དོན་དེ་བསྟན་ནས། ¡Ã ད་ནི་ ( ¡¶ ཞ ¡· བཞི་པ་ ) བྱེ་བྲག་སོ་སོའི་རྒྱུད་ཀྱི་ ¡Ã དོན་དེ་ ( ¡¶ ཞ ¡· རྒྱུད་དེའི་མཚན་ ) འཆད་དོ། །དེ་ལ་རྒྱུད་ཀྱི་མཚན་ནི་ཕལ་ཆེར་ཡང་དག་པར་རྫོགས་པའི་སངས་རྒྱས་ཀྱི་ཡེ་ཤེས་འབྲས་བུའི་རྒྱུད་ལས་བརྩམས་ཏེ་བཏགས་ནས། དེ་ནས་ཐབས་རྒྱུད་དང་རྒྱུ་རྒྱུད་ལ་ཤུགས་ལ་སྦྱར་བའི་མཚན་མང་སྟེ། ¡Ã དཔེར་ན་ ( ¡¶ ཞ ¡· - དཔེར་ན་ ) དཔལ་ཀྱེའི་རྡོ་རྗེ་དང་ ༑ གསང་བ་འདུས་པ་དང༌། ཡང་དག་པར་སྦྱོར་བ་ལ་སོགས་པ་ལྟ་བུའོ། ། ཡང་ཁ་ཅིག་ནི་འབྲས་བུའི་རྒྱུད་དང་ཐབས་རྒྱུད་ལས་བཏགས་ཏེ་དཔེར་ན་དེ་ཁོ་ན་ཉིད་འདུས་པ་ལྟ་བུའོ། ཁ་ཅིག་ནི་ཐབས་རྒྱུད་ཁོ་ནའི་མིང་ཡིན་ཏེ། དཔེར་ན་ལེགས་པར་གྲུབ་པར་བྱེད་པ་ལྟ་བུའོ། །དེ་བཞིན་དུ་ཁ་ཅིག་ནི་ཞུ་བ་པོའི་མིང་སྟེ་དཔུང་བཟང་ལྟ་བུའོ། ཁ་ཅིག་ནི་གཞུང་ཚད་ཀྱི་མིང་སྟེ་རྒྱུད་འབུམ་ཕྲག་ལྔ་པ་ལྟ་བུ་སྟེ་དེ་ལ་སོགས་པ་དུ་མ་ཡོད་དོ། །དེ་ལ་རེ་ཞིག་ཀྱེའི་རྡོ་རྗེའི་རྒྱུད་ཅེས་བྱ་བ་ ནི ༑ རྒྱུད་ལས། ཧེ་ནི་སྙིང་རྗེ་ཆེན་པོ་ཉིད། །བཛྲ་འང་ཤེས་རབ་བརྗོད་པར་བྱ། ། ཐབས་དང་ཤེས་རབ་བདག་ཉིད་རྒྱུད། །དེ་ནི་ང་ཡིས་བཤད་ཀྱིས་ཉོན། །ཞེས་གསུངས་ཏེ། ཀྱེ་ནི་སྙིང་རྗེ་སྟེ་ཐབས་ཡིན་ལ། རྡོ་རྗེ་ནི་དེ་འདྲ་བས་ཤེས་རབ་བོ། ། དེ་བས་ན་སངས་རྒྱས་རྣམས་ཀྱི་དམིགས་པ་མེད་པའི་ཐུགས་རྗེ་ཆེན་པོ་དང་ཤེས་རབ་ཆེན་པོ་ཟུང་དུ་ཆུད་པ་ལ་ཀྱེའི་རྡོ་རྗེ་སྟེ་འབྲས་བུའི་རྒྱུད་དོ། །དེ་ལྟ་བུའི་འབྲས་བུ་ཐོབ་པར་བྱེད་པའི་ལམ་ཡང་ཀྱེའི་རྡོ་རྗེ། རྒྱུའང་ཀྱེའི་རྡོ་རྗེ་ཡིན་དགོས་ ( ¡¶ ཞ ¡· དགོངས་ ) པས་རྒྱུད་གསུམ་ཀ་ཀྱེའི་རྡོ་རྗེ་སྟེ། ཕྱོགས་ཀྱི་གླང་པོས། ཤེས་རབ་ ཕ་རོལ་ཕྱིན་གཉིས་མེད། །ཡེ་ཤེས་དེ་ནི་དེ་བཞིན་གཤེགས། །བསྒྲུབ་བྱ་ ( ¡¶ ཞ ¡· བྱེད་ ) དེ་དོན་སྦྱོར་བ་ཡིས། །གཞུང་དང་ལམ་ཡང་དེ་སྒྲ་ཡིན། །ཞེས་བཤད་པ་ལྟ་བུའ ༑ ༑ དེ་ལ་རྒྱུ་རྒྱུད་ཀྱེའི་རྡོ་རྗེའི་ཐབས་ཤེས་རབ་གཉིས་སུ་མེད་པ་གང་ཡིན་ཞེ་ན

༑ དཔལ་གསང་བ་འདུས་པའི་རྒྱུད་ཕྱི་མ་ལས། ཤེས་རབ་ཐབས་ནི་སྙོམས་འཇུག་པ། །རྣལ་འབྱོར་ཞེས་ནི་བཤད་པ་ཡིན། །གང་ཞིག་དངོས་མེད་ཤེས་རབ་སྟེ ༑ ༑ ཐབས་ནི་དངོས་པོའི་མཚན་ཉིད་དོ། །ཞེས་གསུངས་པས། ¡Ã དངོས་པོ་ ( ¡¶ ཞ ¡· - དངོས་པོ་ ) གཟུང་བ་དང་འཛིན་པ་ལ་སོགས་པ་སྣ་ཚོགས་སུ་སྣང་བས་ཐབས་ཏེ་འཁོར་བ། དེ་ཉིད་ངོ་བོ་སྤྲོས་པ་མཐའ་དག་དང་བྲལ་བས་ཤེས་རབ་སྟེ་ མྱ་ངན་ལས་འདས་པའོ། །དེ་གཉིས་དབྱེར་མེད་པ་ནི་རྒྱུའི་རྒྱུད་ཀྱེའི་རྡོ་རྗེའོ། ། ཡང་གསུངས་པ། འཁོར་བ་སྤངས་ནས་གཞན་དུ་ནི། །མྱ་ངན་འདས་པ་རྟོགས་མི་ འགྱུར ༑ ༑ འཁོར་བ་གཟུགས་དང་སྒྲ་ལ་སོགས། །འདི་རྣམས་ཆོས་ནི་མྱ་ངན་འདས ༑ ༑ ཞེས་གསུངས་པ་ལྟ་བུའོ། །ཐབས་ཀྱི་རྒྱུད་ཀྱི་ཐབས་ཤེས་རབ་གཉིས་སུ་མེད་པ་ནི། རྒྱུའི་རྒྱུད་དེ་ལྟར་གནས་པའི་བྱིན་གྱིས་བརླབས་ཀྱིས་བྱུང་བ་ལམ་སྨིན་གྲོལ་གྱི་ཏིང་ངེ་འཛིན་འཁོར་ ( ¡¶ ཞ ¡· - འཁོར་ ) དང་བཅས་པའོ། །དེའང་སྨིན་ ལམ་དབང་ནི་ཐབས། གྲོལ་ལམ་བསྐྱེད་རྫོགས་གཉིས་ནི་ཤེས་རབ། དབང་ ལའང་བུམ་དབང་ནི་ཐབས། དབང་ཕྱི་མ་གསུམ་ནི་ཤེས་རབ། གྲོལ་ལམ་ལའང་ བསྐྱེད་པའི་རིམ་པ་ནི་ཐབས། རྫོགས་རིམ་ནི་ཤེས་རབ། བསྐྱེད་རིམ་ལའང་ཡབ་ ཀྱི་ཆ་ནི་ཐབས། ཡུམ་གྱི་ཆ་ནི་ཤེས་རབ། རྫོགས་པ་ལའང། དམིགས་བཅས་ནི་ཐབས ༑ དམིགས་མེད་ནི་ཤེས་རབ་བོ། །དེ་དག་རེ་རེ་ལའང་ཐབས་ཤེས་རབ་དུ་མ་ཞིག་ཡོད་དེ་མདོར་ན་ཐབས་ཤེས་རབ་ཏུ་མ་གྱུར་པ་གང་ཡང་མེད་དོ། །དེ་དག་གི་དབྱེ་བ་མང་པོ་དང་དེ་དག་ལ་ཐབས་ཤེས་རབ་དུ་མའི་ལུང་ནི་ཡི་གེ་མངས་པས་འཇིགས་པས་མ་བྲིས་སོ། །དེ་ལྟར་ན་ལམ་ཐབས་རྒྱུད་ཀྱེའི་རྡོ་རྗེའོ། །འབྲས་ བུའི་རྒྱུད་ཀྱི་ཐབས་ཤེས་རབ་གཉིས་སུ་མེད་པ་ནི་གཟུགས་ཀྱི་སྐུ་ལ་ཐབས་ཞེས་བྱ ༑ ཆོས་ཀྱི་སྐུ་ལ་ཤེས་རབ་ཅེས་བྱ། གཟུགས་སྐུའང་ཡབ་ཀྱི་ཆར་སྣང་བ་ནི་ཐབས་ཡིན་ལ། ཡུམ་གྱི་ཆར་སྣང་བ་ནི་ཤེས་རབ་བོ། །ཆོས་སྐུའང་དམིགས་པ་ལས་གྲོལ་བའི་ཤེས་རབ་དབྱིངས་ནི་ཤེས་རབ་ཡིན་ལ།

སྙིང་རྗེ་ཆེན་པོ་ཡེ་ཤེས་ཀྱི་ཆ་ནི་ཐབས་སོ། །དེ་ལྟར་ཡང་སཾ་པུ་ཊ་ལས། སྤྲོས་པ་མེད་པའི་རང་བཞིན་ཉིད ༑ ༑ ཤེས་རབ་ཅེས་ནི་ཡོངས་སུ་གྲགས། །ཡིད་བཞིན་ནོར་ལྟར་མ་ལུས་པའི།། ཐམས་ཅད་དོན་མཛད་སྙིང་རྗེ་ཆེ། །དམིགས་མེད་གནས་སུ་ཤེས་རབ་དང༌། ། དམིགས་མེད་སྙིང་རྗེ་ཆེན་པོ་ཉིད། །བློ་དང་ལྷན་ཅིག་གཅིག་གྱུར་པས། །ནམ་ མཁའ་ནམ་མཁའ་ཇི་ལྟ་བུའོ། །ཞེས་གསུངས་པ་ལྟ་བུའོ། །དེ་ལྟར་ན་འབྲས་བུ འང་ཀྱེའི་རྡོ་རྗེ་ཡིན་པས་ཀྱེའི་རྡོ་རྗེའི་རྒྱུད་དོ། ། ( ¡¶ ཞ ¡· + ལྔ་པ་ནི་ ) དེ་ལྟ་བུའི་རྒྱུད་དེ་ཉིད་རྒྱལ་པོ་ཡིན་ཏེ། རྒྱུའི་རྒྱུད་རང་བཞིན་གྱིས་རྣམ་པར་རྟོག་པས་མི་རྫི་བས་ རྒྱལ་པོ། ཐབས་རྒྱུད་དེས་གློ་བུར་གྱི་རྣམ་རྟོག་སྤོང་བས་རྒྱལ་པོ། འབྲས་བུའི་རྒྱུད་གློ་བུར་དུའང་རྣམ་རྟོག་གིས་མི་རྫི་བས་རྒྱལ་པོའོ། །དཔལ་ཡང་དག་པར་སྦྱོར་བའི་རྒྱུད་ཆེན་པོ་ཞེས་བྱ་བའི་མཚན་གྱི་དོན་ཡང་དེ་ཉིད་ཡིན་ཏེ། རྒྱུད་དེ་ཉིད་ལས་ཡང་དག་པར་སྦྱོར་བ་ནི་ཐབས་དང་ཤེས་རབ་ཀྱི་བདག་ཉིད་ཅན་ནོ། །ཞེས་འབྱུང་བས་དེ་ཀྱེའི་རྡོ་རྗེ་དང་འདྲའོ། །ཆེན་པོ་ནི་རྒྱལ་པོ་དང་འདྲའོ། །དེ་བས་ན་བཤད་པ་སྔ་མ་བཞིན་སྦྱར་བར་བྱ་སྟེ་རྒྱུད་གསུམ་གྱི་བདག་ཉིད་ཅན་དུ་ཤེས་པར་བྱའོ ༑ ༑ འཕགས་པ་མཁའ་འགྲོ་མ་རྡོ་རྗེ་གུར་ཞེས་བྱ་བའི་རྒྱུད་ཀྱི་རྒྱལ་པོའི་བརྟག་པ་ཞེས་བྱ་བའི་མཚན་གྱི་དོན་ནི། འཕགས་པ་ནི་སྡིག་པ་མི་དགེ་བའི་ཆོས་ལས་རིང་དུ་འཕགས་པའོ། །མཁའ་འགྲོ་མ་ནི་ཆོས་ཀྱི་སྐུ་ནམ་མཁའ་ལྟར་མི་རྟོག་པར་ནམ་མཁའ་དང་མཉམ་པས་མཁའ་འགྲོ་མའོ། །གཟུགས་ཀྱི་སྐུ་གཉིས་རྫུ་འཕྲུལ་གྱིས་སངས་རྒྱས་ཀྱི་ཞིང་ཐམས་ཅད་དུ་ཁྱབ་པར་འགྲོ་བས་མཁའ་འགྲོ་མའོ ༑ ༑ རྡོ་རྗེ་ནི་སྟོང་པ་ཉིད་དེ་ཤེས་རབ་ཀྱི་རང་བཞིན་ནོ། །གུར་ནི་དྲ་བ་ལས་འབྲེལ་བ་སྟེ། ཕན་ཚུན་ཁྱབ་པར་གྱུར་པའི་ཡེ་ཤེས་ཏེ། དེ་བཞིན་གཤེགས་པའི་ ཐུགས་རྗེ་ཆེན་པོ་སྟེ་ཐབས་སོ། །དེ་ལྟར་ན་ཐབས་ཤེས་རབ་གཉིས་སུ་མེད་པ་སྟེ་

འབྲས་བུ་ཡང་དག་པར་རྫོགས་པའི་སངས་རྒྱས་སོ། །རྒྱུད་ནི་རྒྱུད་གསུམ་སྟེ་སྔ་མ་ དང་འདྲའོ། །འཕགས་པ་མཁའ་འགྲོ་མ་ནི་ཐབས་དང༌། འབྲས་བུའི་རྒྱུད་ལ་སྦྱར་ རོ ༑ ༑ བརྟག་པ་ཞེས་བྱ་བ་ནི་རྡོ་རྗེ་རྩེ་མོ་ལས། ཆོས་ཀུན་གཟུགས་ཅན་མ་ཡིན་ ཡང་ ༑ ༑ གཟུགས་ཅན་དུ་ཡང་འདི་རུ་བསྟན། །ཆོས་ཀུན་ཚིག་གི་སྤྱོད་ཡུལ་མིན།། སྒྲ་རུ་ཡང་ནི་འདིར་བསྟན་ཏཽ། །དེ་བས་ངོ་མཚར་ཆོས་འདི་རུ། །རྟོག་པས་བརྟག་ པ་ཡིན་པར་བརྗོད། །ཅེས་གསུངས་ཏེ། སངས་རྒྱས་ཀྱི་གཟུགས་ཀྱི་སྐུ་ནས་རྗོད་བྱེད་སྒྲའི་རྒྱུད་ ( ¡¶ ཞ ¡· རྒྱུན་ ) ཡན་ཆད་ལ་བྱའོ། །རྒྱལ་པོ་ནི་སྔ་མ་དང་འདྲའོ། །དེ་ལྟར་ན་བརྗོད་བྱ་དོན་གྱི་རྒྱུད་ནི་སེམས་ཉིད་གཉིས་སུ་མེད་པའི་ཡེ་ཤེས་ཐབས་ཤེས་རབ་ཟུང་དུ་འབྲེལ་བ་རྒྱུད་གསུམ་ ( ¡¶ ཞ ¡· - གསུམ་ ) གྱི་བདག་ཉིད་ཅན་ནོ། ། ( ¡¶ ཞ ¡· + སྤྱི་དོན་ ) གཉིས་པ་དེ་རྗོད་པར་བྱེད་པའི་སྒྲའི་རྒྱུད་ལ་གཉིས། རྗོད་བྱེད་སྒྲ་ རྒྱུད་དངོས་དང༌། དེ་ལས་བྱུང་བའི་དགོས་འབྲེལ་ལོ། །དང་པོ་ལ་གཉིས་ཏེ། ཐུན་ མོང་ལས་གྲགས་པ་དང༌། ཐུན་མོང་མ་ཡིན་པ་ལས་གྲགས་པའོ། །དང་པོ་ལ་གསུམ་སྟེ། རྗོད་བྱེད་མཚན་ཉིད་པ་དང༌། བཏགས་པ་བ་དང༌། དེ་ལ་བརྩད་ 

( ¡¶ ཞ ¡· རྩོད་ ) པ་སྤངས་བའོ། །དང་པོ་བརྗོད་ ¡Ã བྱ་རྒྱུད་གསུམ་ ( ¡¶ ཞ ¡· བྱའི་དོན་ ) སྟོན་པར་བྱེད་པའི་མིང་དང་ཚིག་དང་ཡི་གེའི་རྣམ་པ་རང་རང་གི་རྣམ་པར་རིག་པ་ལ་སྒྲ་སྤྱི་དང་དོན་སྤྱི་འདྲེས་པའི་རྣམ་པར་སྣང་བ་གང་ཡིན་པའོ། །གཉིས་པ་བཏགས་པ་བ་ནི་དེ་ཉིད་རྒྱུད་ལ་སྐྱེ་བར་བྱེད་པའི་གྲོགས་སུ་གྱུར་པ་སྟེ་སངས་རྒྱས་རྣམས་ ( ¡¶ ཞ ¡· - རྣམས་ ) ཀྱི་གསུངས་ནས་འཆད་པ་པོའི་ངག་གི་བར་དང་གླེགས་བམ་དུ་བྲིས་པའི་བར་རོ། །དེ་ལ་བཏགས་པ་བར་ཇི་ལྟར་འདོགས་ཤེ་ན། རང་གི་སྒྲ་རྒྱུད་བློ་ལ་སྒྲ་ ( ¡¶ ཞ ¡· - སྒྲ་ ) གནས་པའི་འབྲས་བུ་ཡིན་པའི་ཕྱིར་དང༌། གཞན་མ་རྟོགས་པ་ལ་སྒྲའི་རྒྱུད་བློ་ལ་སྐྱེད་པའི་རྒྱུ་ཡིན་པའི་ཕྱིར་གདགས་པར་བྱའོ། ། དོན་ ( ¡¶ ཞ ¡· - དོན་ ) གསུམ་པ་རྩོད་སྤང་ནི་གལ་ཏེ་སྒྲའི་རྒྱུད་ཀྱང་རང་གི་རྣམ་པར་རིག་པ་ལས་འཇོག་ན། མཛོད་ལས། ཆོས་ཀྱི་ཕུང་པོ་བརྒྱད་ཁྲི་དག །གང་རྣམས་ ཐུབ་པས་གསུངས་དེ་དག །ཚིག་གམ་མིང་ཡིན་དེ་དག་ནི། །གཟུགས་སམ་འདུ་བྱེད་དག་ཏུ་འདུས། །ཞེས་བཤད་པ་དང་འགལ་ལོ་ཞེ་ན། འགལ་མོད་དེ་ལྟར་སྨྲ་བ་ནི་བྱེ་བྲག་སྨྲ་བའམ་མདོ་སྡེ་པའི་གྲུབ་པའི་མཐའ་ཡིན་ཏེ། ཁོ་བོ་ཅག་ཀུན་རྫོབ་ཏུ་རྣམ་པར་རིག་པ་ཙམ་དུ་འདོད་པའི་ཕྱིར་རོ། །གལ་ཏེ་རྣམ་རིག་ལ་སྣང་བའི་མིང་ཚིག་གི་རྣམ་པ་རྗོད་བྱེད་ཀྱི་རྒྱུད་ཡིན་ན་ཐམས་ཅད་རྒྱུད་དུ་འགྱུར་རོ་ཞེ་ན། མ་ཡིན་ཏེ་མིང་ཚིག་གི་རྣམ་པ་ ¡Ã ནི་རྗོད་བྱེད་ ( ¡¶ ཞ ¡· - ནི་རྗོད་བྱེད་ ) ཙམ་ལ་བྱའི། རྒྱུད་ནི་བརྗོད་བྱ་རྒྱུད་གསུམ་སྟོན་ན་ཡིན་ནོ། །གལ་ཏེ་འོ་ན་རྟོག་པའམ་བྱ་བའི་རྒྱུད་མང་པོ་ཞིག་རྒྱུད་མི་སྟོན་པས་རྒྱུད་མ་ཡིན་ལ། ཕ་རོལ་དུ་ཕྱིན་པའང་རྒྱུད་ སྟོན་པས་རྒྱུད་དུ་འགྱུར་རོ་ཞེ་ན། དེ་ལྟར་ཆ་གཅིག་ནས་ཡིན་ཏེ། རྒྱུད་སྟོན་པས་རྒྱུད་བླ་མའི་བསྟན་བཅོས་ཞེས་བྱ་བ་ལྟ་བུའམ། བྱ་སྤྱོད་མང་པོའང་འོད་ཟེར་ཅན་གྱི་རྟོག་པའམ་ཆོ་ག་ཞིབ་མོ་ཞེས་བྱ་བ་ལ་སོགས་པ་བཏགས་པ་ལྟ་བུའོ།

།འོ་ན་གོང་དུ་རྒྱུད་སྡེ་བཞིར་ཕྱེ་བ་དང་འགལ་ལོ་ཞེ་ན། གཙོ་ཆེར་ཐབས་རྒྱུད་ཆོ་ག་ཞིབ་མོ་ཞེས་བཏགས་ཀྱི་དེ་དག་གིས་ཀྱང་མཐར་རྒྱུད་ཀྱི་དོན་སྟོན་པས་རྒྱུད་ཡིན་ནོ། ། དོན་ ( ¡¶ ཞ ¡· - དོན་ ) གཉིས་པ་ཐུན་མོང་མ་ཡིན་པ་ལས་གྲགས་པའི་རྗོད་བྱེད་སྒྲ་རྒྱུད་མཚན་ཉིད་པ་ནི། རྡོ་རྗེ་རྩེ་མོ་ལས། སེམས་ཅན་ཀུན་ལ་ཕན་པའི་ཕྱིར། །སངས་རྒྱས་རྣམས་ཀྱི་ཆོས་ལོངས་སྤྱོད། །སངས་རྒྱས་ཐུགས་རྗེས་བསྐུལ་ནས་ནི། །རྒྱུད་དོན་ཡང་དག་འབྱུང་བ་ཡིན། །ཞེས་འབྱུང་བས། འོག་མིན་གྱི་བྱང་ཆུབ་སེམས་དཔའ་ཆོས་ཀྱི་ལོངས་སྤྱོད་ནི་རྗོད་བྱེད་སྒྲ་རྒྱུད་མཚན་ཉིད་པ་ཡིན་ལ། དེ་ལས་བྱུང་བས་བཏགས་པ་བ་ནི་ད་ལྟའི་རྣལ་འབྱོར་པའི་བློ་ལ་སྣང་བ་དང༌། གླེགས་བམ་གྱི་བར་ནི་རྗོད་བྱེད་སྒྲའི་རྒྱུད་བཏགས་པ་བའོ། །གང་ལྟར་ཡང་རུང་སྟེ། མཚན་ཉིད་པ་དང་བཏགས་པ་བ་གཉིས་ནི་རྗོད་བྱེད་ཀྱི་རྒྱུད་ཀྱི་ཚོགས་སོ། །སྤྱི་དོན་གཉིས་པ་དེ་ལས་བྱུང་བའི་དགོས་འབྲེལ་ལ་བཞི་སྟེ། བརྗོད་བྱ། དགོས་པ། དགོས་པའི་དགོས་པ། འབྲེལ་བའོ། །དང་པོ་བརྗོད་བྱ་ནི་རྒྱུད་གསུམ་མོ། །རང་དང་གཞན་རྒྱུད་གསུམ་ཉམས་སུ་ལེན་པ་ནི་དགོས་པའོ། །རྒྱུད་གསུམ་པོ་མཐར་ཕྱིན་པ་ནི་དགོས་པའི་ཡང་དགོས་པའོ། །གལ་ཏེ་རྗོད་བྱེད་སྒྲ་རྒྱུད་མཚན་ཉིད་པ་དང་ཐབས་རྒྱུད་ལ་ཁྱད་པར་གང་ཡོད། དགོས་པའི་དགོས་པ་དང་འབྲས་བུའི་རྒྱུད་ལ་ཁྱད་པར་གང་ཡོད་ཅེ་ན། རྗོད་བྱེད་ཅེས་བྱ་བ་ནི་གསུམ་ཀའི་སྤྱི་རང་གི་ བློ་རྣམ་རིག་ལ་སྣང་བ་ཡིན་ལ། རྒྱུད་གསུམ་པོ་ཉིད་ནི་བརྗོད་བྱ་ཡིན་ཞིང༌། འབྲས་བུ་ཉིད་མངོན་དུ་གྲུབ་པ་ནི་དགོས་པའི་ཡང་དགོས་པ་ཡིན་ནོ། །ཡང་ན་སྟོང་པ་དང་སྙིང་རྗེ་གཉིས་སུ་མེད་པ་ནི་བརྗོད་བྱ་ཡིན་ལ་དེ་ཉམས་སུ་བླངས་པས་གནས་སྐབས་ཀྱི་འབྲས་བུ་གྲུབ་པ་ནི་དགོས་པ་ཡིན་ལ། འབྲས་བུའི་རྒྱུད་མངོན་པར་གྲུབ་པ་ནི་དགོས་པའི་ཡང་དགོས་པ་ཡིན་ནོ། །འབྲེལ་བ་ནི་ཐམས་ཅད་ཐབས་དང་ཐབས་ལས་བྱུང་བ་རྒྱུ་འབྲས་སུ་འབྲེལ་ལོ།

།དེ་ལྟར་ན་རྒྱུད་ཀྱི་མཚན་དོན་གྱི་སྒོ་ནས་དགོས་འབྲེལ་ཡན་ལག་བཞི་བསྟན་པ་འདི་ཉིད་ཀྱིས་ཀྱང་རྒྱུད་བཤད་དུ་རུང་བས་རྒྱུད་བཤད་ཐབས་ཀྱི་མན་ངག་ཏུའང་འགྱུར་ལ། གཙོར་ནི་ རྒྱུད་ཀྱི་དགོས་པ་དང་བརྗོད་བྱ་བསྟན་ཏོ། །རྒྱུད་ཀྱི་མཚན་གྱི་དོན་གྱི་ལེའུ་སྟེ་ གཉིས་པའོ།། །། ད་ནི་སྤྱི་དོན་གསུམ་པ་རྒྱུད་བཤད་ཐབས་ཀྱི་མན་ངག་བཤད་པར་བྱའོ།། དེ་ལ་བཤད་ཐབས་ཀྱི་མན་ངག་དཔག་ཏུ་མེད་ཀྱང༌། བླ་མ་གོང་མ་རྣམས་ཀྱི་ གསུང་ནི་དྲུག་སྟེ། ཉན་པ་པོས་རྟོགས་སླ་ཞིང་གཟུང་བདེ་བར་བྱ་བའི་ཕྱིར། གདམས་ངག་དྲུག་གིས་བཤད་པ་དང༌། སེམས་ཀྱི་རྒྱུད་རྒྱུན་ཆགས་གཅིག་གཏན་ལ་དབབ་པ་ལ་དགོངས་ནས་རྒྱུད་གསུམ་གྱིས་བཤད་པ་དང༌། དབང་ལམ་དུ་བྱ་བའི་ཕྱིར་བདག་ཉིད་ཅན་གསུམ་གྱིས་བཤད་པ་དང༌། འབྲས་བུ་སྐུ་བཞི་ལམ་དུ་བྱ་བའི་ཕྱིར་གནས་པ་བཞིས་བཤད་པ་དང༌། སྤང་བྱ་མི་མཐུན་པའི་ཕྱོགས་སྤོང་བ་ལ་དགོངས་ནས་དག་པ་གསུམ་གྱིས་བཤད་པ་དང༌། སྣོད་མ་ཡིན་པ་ལ་གབ་ཅིང་གསང་བའི་དོན་དུ་གསང་བ་རྣམ་པ་བདུན་གྱིས་བཤད་པ་སྟེ་དྲུག་ཏུ་གསུངས་སོ ༑ ༑ དེ་ལ་དང་པོ་གདམས་ངག་དྲུག་ནི། དཔལ་ནམ་མཁའ་དང་མཉམ་པའི་རྒྱུད་ལས་གསུངས་པ། ཡི་གེ་དང་ནི་ཡན་ལག་དང༌། །བསྡུས་པའི་དོན་དང་སྤྱི་ཡི་དོན ༑ སྦས་དང་དེ་བཞིན་མཐར་ཐུག་པར། །རྒྱུད་རྣམས་ངེས་པ་ཁོ་ན་ཡིན། ། ཞེས་གསུངས་ཏེ། བརྗོད་པར་བྱ་བའི་དོན་བཤད་པར་བྱ་བའི་མན་ངག་གསུམ་དང་ ༑ དེ་རྗོད་པར་བྱེད་པའི་སྒྲ་འཆད་ལུགས་ཀྱི་མན་ངག་གསུམ་མོ། །དང་པོ་ནི་སྤྱིའི་དོན་དུ་བཤད་པ་དང༌། སྦས་པའི་དོན་དུ་བཤད་པ་དང༌། མཐར་ཐུག་པའི་དོན་དུ་བཤད་པའོ། །སྒྲ་གསུམ་ནི་ཡན་ལག་གི་དོན་གྱིས་བཤད་པ་དང༌། ཡི་གེའི་དོན་གྱིས་བཤད་པ་དང༌། མན་ངག་གི་དོན་གྱིས་བཤད་པའོ། །དེ་ལ་སྤྱིའི་དོན་ཞེས་བྱ་བ་ནི་ ( ¡¶ ཞ ¡· + རང་གཞན་ ) གཉིས་ཀྱི་ཐུན་མོང་དུ་གྱུར་པ་ལ་བྱའོ། །དེ་ལའང་གཉིས ༑ ཕ་རོལ་ཏུ་ཕྱིན་པའི་ཐེག་ ¡Ã པ་ ( ¡¶ ཞ ¡· པའི་ ) སྤྱི་དང༌།

བསྐྱེད་པའི་རིམ་ ¡Ã པ་ ( ¡¶ ཞ ¡· པའི་ ) སྤྱིའོ། །དང་པོ་ནི་ཕ་རོལ་ཏུ་ཕྱིན་པའི་ཐེག་པ་ནས་བཤད་པའི་བྱང་ཆུབ་ཀྱི་ཕྱོགས་ཀྱི་ཆོས་སུམ་ཅུ་རྩ་བདུན་དང་ལམ་ལྔ་དང༌། ས་བཅུ་ལ་སོགས་པ་ཉིད་གསང་སྔགས་ཀྱིའང་བརྗོད་བྱར་བསྟན་པས་ཐུན་མོང་བ་ཡིན་པས་སྤྱིའོ། ། བསྐྱེད་ ( ¡¶ ཞ ¡· + པའི་ ) རིམ་ ( ¡¶ ཞ ¡· + པའི་ ) སྤྱི་ནི་ལམ་བསྐྱེད་པའི་རིམ་པ་ནི་ཕྱག་རྒྱ་ཆེན་པོ་མཆོག་སྒྲུབ་པས་ཀྱང་བསྒོམ་ལ། ཐུན་མོང་གི་དངོས་གྲུབ་སྒྲུབ་པས་ ཀྱང་བསྒོམ་པར་བྱ་བས་ཐུན་མོང་ཡིན་པས་སྤྱིའོ། །དེ་ལྟར་སློབ་དཔོན་ཟླ་གྲགས་ཀྱིས་ཀྱང་བཤད་པ། མདོ་སོགས་ལ་ནི་རབ་ཞུགས་པའི། །འགྱོད་པ་ཟློག་པར་ བྱེད་པ་དང༌། །བསྐྱེད་པའི་རིམ་དང་ལྡན་པ་དག །སྤྱི་ཡི་དོན་ནི་ཡིན་པར་བཤད།། ཅེས་བཤད་པ་ལྟ་བུའོ། །སྦས་པའི་དོན་ཞེས་བྱ་བ་ནི་གོང་མའི་ཡུལ་དུ་གྱུར་ལ་ འོག་མའི་ཡུལ་དུ་མ་གྱུར་པས་སྦས་པའོ། །དེ་ལ་གོང་མ་ནི་མཆོག་དབང་གསུམ་ཐོབ་པའི་ཡུལ་ཡིན་ལ། འོག་མ་བུམ་དབང་ཐོབ་པའི་ཡུལ་མ་ཡིན་པས་སོ། །དེ་ནི་ཐབས་ལས་མིང་བཏགས་ན་རང་བྱིན་བརླབ་ཀྱི་ཐབས་དང༌།དཀྱིལ་འཁོར་འཁོར་ལོའི་ཐབས་ཞེས་བྱ་ལ། བྱང་ཆུབ་ཀྱི་སེམས་ལས་མིང་བཏགས་ན་ཀུན་རྫོབ་བྱང་ཆུབ་སེམས་དང༌། དོན་དམ་བྱང་ཆུབ་སེམས་ཞེས་བྱ་སྟེ། ཧེ་བཛྲ་ལས། དཀྱིལ་ འཁོར་འཁོར་ལོའི་ཐབས་དང་ནི། །རང་བྱིན་བརླབ་པའི་རིམ་པས་ཀྱང༌། །ཀུན་རྫོབ་དོན་དམ་ཚུལ་ཅན་གྱི། །བྱང་ཆུབ་སེམས་ནི་རབ་ཏུ་བསྐྱེད། །ཅེས་སོ། །ཡེ་ཤེས་ལས་མིང་བཏགས་ན་རང་བྱུང་དང་ལྷན་ཅིག་སྐྱེས་པའི་ཡེ་ཤེས་ཞེས་བྱའོ། ། མདོར་ན་གསང་བ་ཆོས་ཀྱི་དཀྱིལ་འཁོར་གསུམ་ལ་བློ་གཏད་དེ་བཤད་པ་མཐའ་ དག་སྦས་དོན་ནོ། །དེ་ལ་དཀྱིལ་འཁོར་གསུམ་ནི། དཀྱིལ་འཁོར་ལུས་སུ་ གསུངས་པ་སྟེ། །དཀྱིལ་འཁོར་བྷ་གར་བརྗོད་པ་ཉིད། །དཀྱིལ་འཁོར་བྱང་ཆུབ་སེམས་ཉིད་དེ། །དཀྱིལ་འཁོར་གསུམ་ནི་བརྟག་པར་བྱ། །ཞེས་གསང་བ་འདུས་པ་དང་སཾ་པུ་ཊའི་རྒྱུད་ཕྱི་མ་གཉིས་ཀ་ལས་བཤད་དོ།

།དེ་ལ་ལུས་རགས་པ་དང་ཕྲ་བ་རྩའི་གནས་ལུགས་ཀྱི་བར་སྟོན་པ་ནི་ལུས་ཀྱི་དཀྱིལ་འཁོར་རོ། །ཐབས་ཤེས་རབ་ཀྱི་རྡོ་རྗེ་པདྨ་དང་རྩ་ ( ¡¶ ཞ ¡· + དང་ ) ཡི་གེའི་གནས་ལུགས་སྟོན་པ་ནི་བྷ་གའི་དཀྱིལ་འཁོར་རོ། །ཀུན་རྫོབ་ཐིག་ལེའི་གནས་ལུགས་དང་དོན་དམ་དེ་ལས་སྐྱེས་པའི་བདེ་སྟོང་གི་ཏིང་ངེ་འཛིན་གྱི་ཚུལ་སྟོན་པ་ནི་བྱང་ཆུབ་སེམས་ཀྱི་དཀྱིལ་འཁོར་རོ། །དེ་དག་གི་ཞར་ལ་དེ་རྣམས་ལ་མངོན་དུ་བྱེད་པ་སྙིང་པོ་ཡེ་ཤེས་རླུང་ སྟོན་པ་རྣམས་ཀྱང་སྦས་པར་བསྡུའོ། །མཐར་ཐུག་པ་དེ་ནི་དེ་ལས་གོང་དུ་བོགས་དབྱུང་དུ་མེད་པས་མཐར་ཐུག་པ་སྟེ་སངས་རྒྱས་ཀྱི་ཡེ་ཤེས་སོ། །དེའང་རྣམ་པ་གཉིས་ཏེ་སྤྱིའི་དོན་གྱི་མཐར་ཐུག་སྟེ་ཐེག་པ་ཐུན་མོང་བ་ལས་གྲགས་པ་ས་བཅུ་གཅིག་པ་དང༌། སྦས་དོན་གྱི་མཐར་ཐུག་སྟེ་གསང་སྔགས་ལས་གྲགས་པའི་ས་བཅུ་གཉིས་པ་དང་ས་བཅུ་གསུམ་པའོ། །གལ་ཏེ་མཐར་ཐུག་གཉིས་མི་རིགས་ཏེ་སྤྱི་དོན་མཐར་ཐུག་ཡིན་ན་དེའི་སྟེང་དུ་ཡོན་ཏན་བསྐྱེད་དུ་མེད་པས་སྦས་དོན་གྱི་མཐར་ཐུག་མི་རིགས་ལ། ཡོན་ཏན་བསྐྱེད་དུ་ཡོད་ན་སྤྱི་དོན་ལ་མཐར་ཐུག་མི་རིགས་སོ་ཞེ་ན། ཡོན་ཏན་བསྐྱེད་དུ་ཡོད་དེ་ཕ་རོལ་ཏུ་ཕྱིན་པ་ལས་ཆོས་ཀྱི་དབྱིངས་ཀྱི་ཡེ་ཤེས་ཞེས་བྱ་བའང་མ་བཤད་ལ། གསང་སྔགས་ལས་བཤད་པ་དང་ ༑ ཕ་རོལ་ཏུ་ཕྱིན་པ་ལས་སྐུ་གསུམ་ལས་མ་བཤད་ལ། གསང་སྔགས་ལས་སྐུ་བཞི་དང་ལྔ་ ( ¡¶ ཞ ¡· + ལྔར་ ) བཤད་པ་དང༌། དེ་བཞིན་དུ་དཔལ་སཾ་པུ་ཊ་ལས། གང་དག་བསམ་གྱིས་མི་ཁྱབ་པའི་གནས་མངོན་དུ་མ་བྱས་པ་དེ་དག་ནི་བདེ་བར་གཤེགས་པ་སྟེ་སངས་རྒྱས་ཡིན་ལ། མཚན་གཞི་མཚོན་པར་བྱེད་པ་ནི་རྡོ་རྗེ་ སེམས་དཔའ་ཡང་དག་པར་གསུངས་སོ། །ཞེས་བྱ་བ་འདིས་ཕར་ཕྱིན་པས་ཤེས་ བྱ་གཞན་སྣང་ལས་མི་མཁྱེན་ཏེ། ཕར་ཕྱིན་པས་གཟུགས་ཀྱང་གཞན་ལ་རྣམ་པ་ ཐམས་ཅད་མཁྱེན་པའི་ཡེ་ཤེས་ཀྱང་གཞན་ཞེས་གསུངས་ལ། གསང་སྔགས་ ཀྱིས་ཐམས་ཅད་རང་སྣང་དུ་མཁྱེན་ཞེས་པ་འདིས་ཁྱད་པར་ཡོད་དོ།

།འོ་ན་སྤྱི་ དོན་ལ་མཐར་ཐུག་ཏུ་བཞག་པ་མི་འཐད་སྙམ་ན་འཐད་དེ་ཉེ་བའི་བྱ་བ་བྱས་པའི་ཕྱིར་རོ། །དཔེར་ན་མངོན་རྟོགས་རྒྱན་ལས། ས་དགུ་འདས་ནས་ཡེ་ཤེས་ནི། ། གང་གིས་སངས་རྒྱས་སར་གནས་པ། །དེ་ནི་བྱང་ཆུབ་སེམས་དཔའི་ས། །བཅུ་པ་ཡིན་པར་ཤེས་པར་བྱ། །ཞེས་ས་བཅུ་པ་ལ་སངས་རྒྱས་ཀྱི་སར་བཤད་པ་དང་ འདྲའོ ༑ ༑ གལ་ཏེ་སློབ་དཔོན་ཟླ་གྲགས་ཀྱིས། འོད་གསལ་བ་ནི་རབ་ ¡Ã སྟོན་ དང་ ( ¡¶ ཞ ¡· སྟོང་བ ) ། །ཟུང་དུ་འཇུག་པ་རྟོགས་པ་སྟེ། །རྣམ་པ་གཉིས་སུ་མཐའ་བཤད་དེ། །མཐའ་ནི་མུར་ཐུག་པ་ཡི་ཚིག །ཅེས་སྟོང་པ་ཉིད་ལ་སོགས་པ་སྟོན་པ་ལ་མཐར་ཐུག་ཏུ་བཤད་པས་དེ་དང་འགལ་ལོ་སྙམ་ན་མི་འགལ་ཏེ་སྟོང་ཉིད་དང་ཟུང་འཇུག་ལ་ལུགས་གཉིས་ཡོད་དེ། དེའི་མཐར་ཐུག་པ་ཐོབ་པ་དང༌། སྤྱི་བློའི་ཡུལ་དུ་བསྟན་པའོ། །སྤྱི་བློའི་ཡུལ་མིང་གི་རྗེས་སུ་འབྲངས་པ་ཙམ་དེ་ནི་ཕར་ཕྱིན་པ་དང་སྤྱིའི་དོན་དུ་བཤད་ལ། མཐར་ཐུག་ཐོབ་པ་ནི་སངས་རྒྱས་ཉིད་དོ། །འོད་གསལ་བ་ནི་ཆོས་ཀྱི་སྐུ་ལ། ཟུང་འཇུག་ནི་ལོངས་སྤྱོད་རྫོགས་པའི་སྐུའོ། །དེ་ལྟར་བརྗོད་པར་བྱ་བའི་དོན་གསུམ་ལ་ཕྱེ་ན་དྲུག་ཏུ་བསྟན་ཏོ། །དེ་རྗོད་པར་བྱེད་པའི་སྒྲ་གསུམ་ནི་ཡན་ལག་གི་དོན་གྱིས་བཤད་པ་དང༌། ཡི་གེའི་དོན་གྱིས་བཤད་པ་དང་ ༑ མན་ངག་གི་དོན་གྱིས་བཤད་པའོ། །དང་པོ་ཡན་ལག་གི་དོན་ནི། འཇིག་རྟེན་ལས་གྲགས་པའི་སྒྲ་དང་བཅས། བསྟན་བཅོས་ལས་གྲགས་པའི་སྒྲ་དང་བཅས། སྒྲ་འཇུག་པའི་རྒྱུ་མཚན་ངེས་པའི་ཚིག་དང་བཅས་པས་གཞུང་དངོས་ཀྱི་དོན་ལ་འཇུག་པའོ། །དེའང་འཇིག་རྟེན་ལས་གྲགས་པ་ནི་དཔེར་ན་རྒྱུ་མེ་ཤེལ་ལས་གྲུབ་པ་ནམ་མཁའ་ལ་གནས་པ་ཚ་བའི་འོད་ཅན་དེ་ལ་ཉི་མ་ཞེས་གྲགས་ སོ ༑ ༑ དེ་ཉིད་ལ་བསྟན་བཅོས་ལས་བཅུ་གཉིས་པ་དང༌། འོད་ཟེར་བརྒྱ་པ་དང༌། པདྨོའི་གཉེན་དང༌། སྣང་བྱེད་དང༌། ཉིན་མོར་བྱེད་པ་ལ་སོགས་པ་གྲགས་པ་བཞིན་ནོ། །སྒྲ་འཇུག་པའི་རྒྱུ་མཚན་ཡང་ཉི་མ་ལ་ཁྱིམ་ ( ¡¶ ཞ ¡· བུ་ ) བཅུ་གཉིས་ཡོད་པས་དང༌།

པདྨོ་ཁ་འབྱེད་པས་དང༌། མུན་པ་སེལ་བ་མཚན་མོ་འཇོམས་པ་ལ་སོགས་པ་ཡོན་ཏན་དུ་མ་ཡོད་པས་དེ་དང་དེ་དག་ཏུ་གདགས་སུ་རུང་བས་ཚུལ་འདི་ལྟ་བུ་རྣམས་ལ་ཡན་ལག་གི་དོན་གྱིས་བཤད་དོ། །ཡི་གེའི་དོན་གྱིས་བཤད་པ་ནི་གཞུང་དངོས་ ( ¡¶ ཞ ¡· + དོན་ ) ལ་རྒྱུ་མཚན་མི་རྙེད་ཀྱང་ཡི་གེ་སོ་སོར་བྱས་ནས་རྒྱུ་མཚན་རྙེད་པ་སྟེ་དེའང་སྒྲས་དྲངས་ཏེ་བཤད་པ་དང༌། ཡི་གེ་ཁ་ཕྲལ་ཏེ་སྒྲས་དྲངས་པས་བཤད་པ་དང༌། ཡི་གེའི་དབྱིབས་མཐུན་པས་བཤད་པའོ། །དང་པོ་ནི་དཔེར་ན། ན་ག་ཧ་ཨ་ཤྭ་ ( ¡¶ ཞ ¡· ན་ ) ཞེས་བྱ་བ་ལྟ་བུ་དངོས་སུ་བཤད་དུ་མི་བཏུབ་པ་ལ་སྒྲས་དྲངས་ན། ན་ནི་ན་ར་སྟེ་མི། ག་ནི་ག་བི་སྟེ་བ་ལང་ཞེས་བྱ་བ་ལྟ་བུའོ།། གཉིས་པ་ནི་དཔེར་ན། ཧཱུྃ་ཞེས་བྱ་བ་ནི་གཟུང་བ་དང་འཛིན་པ་དང་བྲལ་བའོ། ། ཞེས་གསུངས་པ་ལ་དངོས་སུ་གཟུང་འཛིན་དང་བྲལ་བའི་དོན་བཤད་དུ་མི་བཏུབ་པས་ཡི་གེ་ཕྲལ་ཏེ་སྒྲར་དྲངས་ནས་བཤད་ན། ¡Ã ཧ་གྲ་ ( ¡¶ ཞ ¡· ཧ་གྷ་ ) ཧ་ཀ་ནི་གཟུང་ བའི་ཡུལ་དུ་འགྲོ་ལ ¡Ã ཧ། གྲ་ ( ¡¶ ཞ ¡· གྷུ་ ) ཧ་ཙ་ནི་འཛིན་ ¡Ã པའི་སེམས་སུ་འགྲོ་ལ ༑ ( ¡¶ ཞ ¡· པ། ) ཞབས་སྐྱེད་ཨཱུ་ལ་རྐྱེན་ཤ་བྱིན་པས་ ¡Ã ཤཱུ་སྟེ་ཤཱུནྱཏཱ་ ( ¡¶ ཞ ¡· ཤུ་ནྱ་ ) སྟོང་པར་འགྲོ། ཀླད་ཀོར་མ་ཡིན་ ( ¡¶ ཞ ¡· བྱིན་ ) པས་མ་ནོ་ཏ་སྟེ་ཡིད་ཡིན་པས་གཟུང་འཛིན་གཉིས་ཀྱིས་སྟོང་པའི་ཡིད་ཙམ་ཞེས་བྱ་བ་ལྟ་བུར་འགྲོ་བ་ལ་བྱའོ། །དབྱིབས་མཐུན་པས་བཤད་པ་ནི་དཔེར་ན་ཨེ་བཾ་མ་ཡ་ནི་ལྟེ་བ་དང་སྙིང་ག་དང་མགྲིན་པ་དང་སྤྱི་བོའོ་ཞེས་གསུངས་པ་ལྟ་བུ་དངོས་སུ་བཤད་དུ་མི་བཏུབ་པ་ལ་དབྱིབས་ཀྱིས་མཚོན་ནས་ཨེ་གྲུ་གསུམ་དང་དབྱིབས་མཐུན་པས་ལྟེ་བ་སྤྲུལ་པའི་འཁོར་ལོར་བརྗོད་པ་ལྟ་བུའོ། །བསྡུས་པའི་དོན་གྱི་མན་ངག་གིས་བཤད་པ་ནི ༑ འཇིག་རྟེན་ལས་གྲགས་པའི་སྒྲ་ཡང་མ་ཡིན་བསྟན་བཅོས་ལས་གྲགས་པའང་མ་ཡིན་ཡི་གེས་བཤད་དུའང་མི་བཏུབ་སྟེ། སངས་རྒྱས་རྣམས་ཀྱིས་བརྡ་མཛད་པ་སྟེ་དཔེར་ན་ཆང་ལྟ་བུ་ལ་འཇིག་རྟེན་ན་ཆང་ཞེས་གྲགས།

བསྟན་བཅོས་ལས་མྱོས་བྱེད་ཅེས་གྲགས་ཏེ་འདིར་མ་ད་ན་ཞེས་པ་ལྟ་བུ་དེ་བཞིན་གཤེགས་པའི་བརྡར་གྱུར་པ་ལས་སོ་སོ་སྐྱེ་བོ་རྣམས་ཀྱི་སྤྱོད་ཡུལ་མ་ཡིན་པ་སྟེ། དེ་བཞིན་དུ་ཚོད་མ་ལ་མ་ལ་ཨྀན་དྷ་ནཾ་དང༌། བཟའ་བ་ལ་ཏྲི་པི་ཏ་ཞེས་བརྗོད་པ་ལྟ་བུ་སྟེ། ཟླ་བ་གྲགས་པས། ཀོ་ཊ་ཀྵ་ལ་སོགས་པའི་སྒྲ། །འཇིག་རྟེན་བསྟན་བཅོས་ལ་མེད་ཅིང་ ༑ ༑ དེ་བཞིན་གཤེགས་པའི་བརྡར་གྱུར་པ། །འདི་ནི་ཇི་བཞིན་མིན་པར་བརྗོད ༑ ༑ ཅེས་གསུངས་པ་ལྟ་བུའོ། །དེ་ལྟར་ན་བརྗོད་བྱའི་དོན་གསུམ་མམ་ཕྱེ་ན་དྲུག་པོ་དེ་རེ་རེ་ལའང་རྗོད་བྱེད་ཀྱི་སྒྲ་འཆད་ལུགས་གསུམ་གྱིས་འཆད་པ་ཡིན་ ནོ ༑ ༑ དེ་དག་ཀྱང་དཔེར་བརྗོད་སྤྱིར་བསྟན་པ་དང༌། རྒྱུད་ཇི་ལྟར་འཆད་པའི་ཚུལ་བསྟན་པའོ། །དང་པོ་ནི་ཕར་ཕྱིན་པ་སྤྱི་ལྟ་བུ་ལ། ཡན་ལག་གི་དོན་གྱིས་བཤད་ པ་ནི། དང་པོར་བྱམས་པ་བསྒོམ་པར་བྱ་ཞེས་བྱ་བ་ལྟ་བུ། དང་པོ་ནི་གོ་རིམ། བྱམས་པ་ནི་བདེ་བ་དང་བདེ་བའི་རྒྱུ་དང་ཕྲད་པར་འདོད་པ། བསྒོམ་པར་བྱ་བ་ནི་རྣལ་འབྱོར་པས་དེ་ལྟ་བུ་ལ་བསམ་པར་བྱ་ཞེས། འཇིག་རྟེན་དང་བསྟན་བཅོས་ལས་གྲགས་པའི་སྒྲ་ཇི་བཞིན་པའོ། །ཡི་གེའི་དོན་ནི། ཡི་དྭགས་བསྟན་ལ་རབ་ཏུ་གནས

༑ ༑ ཞེས་བྱས་པ་ན། ཕ་རོལ་ཏུ་ཕྱིན་པ་དག་ཀྱེ་མ་ཡི་དྭགས་ལ་ཇི་ལྟར་སྡོད་སྙམ་པ་ལ། ཡི་དྭགས་ནི་པྲེ་ཏ་ལ་པྲ་ཏེ་རབ་ཏུའོ། །ཨེ་ཏ་ནི་སོང་བ་སྟེ་རྣམ་པར་རྟོག་པ་རབ་ཏུ་སོང་བ་ལ་སྡོད་པའོ། །དེ་བཞིན་དུ་ཕ་རོལ་ཏུ་ཕྱིན་པ་ཉིད་གང་ཡིན་སྙམ་ན། པཱའ་ར་ནྀ་མཆོག་ཏེ་སངས་རྒྱས་སོ། །མི་ཏཱ་ནི་འགྲོ་བ་སྟེ་སངས་རྒྱས་ཀྱི་སར་འགྲོ་བ་ཞེས་བྱ་བ་ལྟ་བུའོ། །མན་ངག་གི་དོན་གྱིས་བཤད་པ་ནི། སྙིང་སྟོབས་སྐུ་དང་རྡུལ་ངག་བཅས། །མུན་པ་སེམས་ཀྱི་རང་བཞིན་ལས། །ཞེས་བརྗོད་པ་ན། ཀྱེ་མ་སྔགས་པ་ནི་མུ་སྟེགས་གྲངས་ཅན་འཛིན་པའོ་སྙམ་པ་ལ། རྡུལ་མུན་པ་སྙིང་ སྟོབས་ཀྱི་བརྡས་ ( ¡¶ ཞ ¡· བརྡ་ ) ནི་ཀུན་བརྟགས་པ་དང་གཞན་དབང་དང་ཡོངས་གྲུབ་ཡིན་ནོ་ཞེས་བཤད་པ་དང༌། དེ་བཞིན་དུ་ཕྱག་བཅུ་གཉིས་བསྒོམ་ཞེས་བྱས་པ་ན་ཕྱག་བཅུ་གཉིས་ཀྱིས་ཅི་བྱ་སྙམ་པ་ལ་དེ་ནི་རྟེན་ཅིང་འབྲེལ་བར་འབྱུང་བ་བཅུ་གཉིས་ཤེས་པར་བྱ་བའི་དོན་ཡིན་ནོ་ཞེས་བྱ་བ་ལྟ་བུ་སྟེ། མདོར་ན་ཕ་རོལ་ཏུ་ཕྱིན་པ་བ་དག་གསང་སྔགས་ལ་ཞུགས་པ་ན་ ( ¡¶ ཞ ¡· ནི་ ) རིམ་པ་གཉིས་ཀྱི་དོན་མ་གོ་བས་འདི་ལྟ་བུ་ལམ་མ་ཡིན་ནམ་སྙམ་པའི་འགྱོད་པ་བསལ་བའི་ཕྱིར། ཁ་ཅིག་ ནི་སྒྲ་ཇི་བཞིན། ཁ་ཅིག་ནི་ཡི་གེའི་སྒོ་ནས། ཁ་ཅིག་ནི་བརྡ་ཡིན་ནོ་ཞེས་བསྟན་པའི་སྒོ་ནས་དེ་དག་གཞུག་པའི་ཕྱིར་བསྟན་པ་ནི་ཕ་རོལ་ཏུ་ཕྱིན་པའི་སྤྱི་དོན་ ཏེ ༑ ཡེ་ཤེས་རྡོ་རྗེ་ཀུན་ལས་བཏུས་པ་ལས། ངོ་མཚར་གྱི་རྡོ་རྗེའི་ཚིག་ཐོས་ནས་ སློབ་དཔོན་རྩེ་མོའི་གསུང་རབ་དཔེ་བསྡུར་མ་པོད་དང་པོ། མེས་པོའི་ཤུལ་བཞག་ལས། ས་སྐྱ་བཀའ་འབུམ། PAGE 132PAGE 131

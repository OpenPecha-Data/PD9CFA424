པ་ལ་གྲོངས་པ་མི་མངའ་བས།དེ་ཁྱོད་ཀྱིས་མ་ཤེས་པས་ཉེས་པ་དང་བཅས་པའོ།། ཡང་ཁྱོད་ཀྱིས་དེ་བཞིན་གཤེགས་པའི་གསང་བའི་གནས་སྦུབས་སུ་ནུབ་པ་སྤྱོད་པ་དམན་པ་ལ་བསྟན་ཏོ། །དེས་དེ་བཞིན་གཤེགས་པའི་སྐུ་ལ་མཆི་མའི་ཐིགས་པས་མ་རུང་བར་བྱས་སོ། །དེ་ལྟར་ཁྱོད་ཉེས་པ་བརྒྱད་བྱས་པ་ཡིན་པས་སོང་ཞིག ཀུན་དགའ་བོས་སྨྲས་པ། དེ་བཞིན་གཤེགས་པས་ནི་སྒྲ་གཅན་འཛིན་ནི་ངའི་བུ། ཀུན་དགའ་བཽ་ནི་ངའི་གོ་རེ་ལོང༌། དགེ་སློང་དགེ་འདུན་རྣམས་ལ་གཏད། ཅེས་བདག་ཁྱེད་ལ་གཏད་པ་ཡིན་ནོ། །འོད་སྲུངས་ཀྱིས་སྨྲས་པ། ཁྱོད་ཉིད་ལ་ཕན་པའི་ཕྱིར་དང་བསྟན་པ་རྒྱས་པར་བྱ་བའི་ཕྱིར་ཆད་པས་བཅད་དོ། །དེའི་ཚེ་མ་འགགས་པས་སྨྲས་པ། ཀུན་དགའ་བོ་ནི་མང་དུ་ཐོས་པ་ཅན་ཡིན་ན། དེ་མིན་ན་ཇི་ལྟར་བཤད་པར་བྱ། འོད་སྲུངས་ན་རེ། ཡོན་ཏན་དང་ལྡན་པ་ཡོད་མོད། སོ་སོ་སྐྱེ་བོ་དང་མི་གཅིག་གོ །སླར་ཀུན་དགའ་བོ་ལ་འོད་སྲུངས་ཀྱིས་སྨྲས་པ། ཁྱོད་ཀྱིས་དགྲ་བཅོམ་པ་ཐོབ་པ་ན་འདིར་ཤོག་ཅིག དེ་ནས་ཀུན་དགའ་བོས་དགེ་འདུན་ལ་ཕྱག་བཙལ་ཏེ་མཆི་མ་ཕྱུང་ནས་ཡུལ་བྲྀ་ཛི་ག་ལ་བ་དེར་སོང་ངོ༌། །དེར་ཀུན་དགའ་བོས་ཁྱིམ་བདག་གི་བུ་བཞི་རབ་ཏུ་ཕྱུང་པ་སྟེ། གདམས་ངག་བྱིན་པས་དེ་དག་གིས་དགྲ་བཅོམ་པ་ཐོབ་བོ། །དེ་ནས་ཁྱིམ་བདག་བྲྀ་ཛིའི་བུས་མཁན་པོ་དགྲ་བཅོམ་པ་ཡིན་ནམ་མ་ཡིན་མངོན་པར་ཤེས་པས་བལྟས་པ་ན་མ་ཡིན་པར་མཐོང་ངོ༌། །སྐལ་བ་ཡོད་དམ་མེད་བལྟས་པ་ན་ཡོད་པར་མཐོང་ངོ༌། །དེས་མཁན་པོ་ལ་སླར་གདམས་པ། གཽ་ཏམ་ཁྱོད་ནི་བག་མེད་མ་མཛད་པར། །མྱ་ངན་འདས་ལ་ཐུགས་བསྐྱེད་མཛད་ནས་ཀྱང༌། །ཤིང་དྲུང་སྟུག་པོར་ཡང་དག་གནས་མཛོད་ཅིག 

༑ རིང་པོར་མི་ཐོགས་ཞི་བ་ཐོབ་པར་འགྱུར། །ཞེས་གདམས་སོ། །དེ་ནས་ཀུན་དགའ་བོས་དེའི་ལྟར་མཉན་ནས། ཞག་བདུན་སྔས་ནང་ཚངས་ཅན་ལ་མ་བརྟེན་པར་བདག་མེད་པའི་བསམ་གཏན་ལ་གནས་པས་དགྲ་བཅོམ་པ་ཐོབ་པར་གྱུར་ཏོ། །དེ་ནས་བདག་ནི་དགྲ་བཅོམ་པ་རྣམས་ཀྱི་ནང་དུ་འགྲོ་བ་ལ་དབང་ཞེས་བསམས་ཏེ་བི་མ་ལ་ཡཱ་ནིའི་ཕུག་ག་ལ་བ་དེར་སོང་སྟེ། རྒན་པ་རྣམས་ལ་ཕྱག་བཙལ་ཏེ་རང་གི་གནས་སུ་འདུག་གོ །དེའི་ཚེ་ཀུན་དགའ་བཽ་རྒྱང་ ¡Ã མ་ ( ¡¶ ཞ ¡· རིང་མོ་ ) ནས་འོངས་པ་མཐོང་བས་དགེ་སློང་མང་པོས་དེ་བཞིན་གཤེགས་པའང་རྗེས་སུ་དྲན་པར་གྱུར་ཏོ། །དེ་ནས་དགེ་འདུན་འདུས་ཏེ་འོད་སྲུངས་ཀྱིས་སྨྲས་པ། ཚེ་དང་ལྡན་པ་དག་མ་འོངས་པ་ན་དགེ་སློང་དག་དྲན་པ་ཉམས་པ་བྱུང་བས་དེ་དག་གི་མདོ་ཙམ་མ་ཡིན་པ་སྡེ་སྣོད་གསུམ་རྒྱས་པར་འཛིན་མི་ནུས་སོ། །དེའི་ཕྱིར་སྔ་དྲོ་མདོ་ཙམ་བཟླས་པར་བྱའོ། །ཕྱི་དྲོ་སྡེ་སྣོད་གསུམ་པོ་བཟླས་པར་བྱའོ།། དགེ་སློང་རྣམས་ཀྱིས་དེ་བཞིན་ནོ་ཞེས་གསོལ་ཏེ། ཆོས་ཀྱི་ཁྲི་བཤམས། དེར་འོད་སྲུངས་བཞུགས་ཏེ། དེ་ལ་བསྐོར་ ¡Ã ནས་འདུག་སྟེ་མདུན་དུ་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· ཏེ་མངོན་དུ་ ) བདར་རོ། །དེ་ནས་འོད་སྲུངས་ཀྱིས་ཀུན་དགའ་བོ་ལ་སྨྲས་པ། ཚེ་དང་ལྡན་པ་ཀུན་དགའ་བོ་ཁྱོད་ནི་རྣམ་པར་གསལ་བ་ཡིན་ན་མདོ་ཟློས་པར་དད་དམ། ཀུན་དགའ་བོས་སྨྲས་པ་དད་དོ། །དེ་ནས་འོད་སྲུངས་ཀྱིས་ཐོག་མར་བྱས་ཏེ། བླ་གོས་རྣམས་སྟན་དུ་བཏིང་སྟེ་ཁྲི་བཤམས་པ་དང༌། ཀུན་དགའ་བོས་ཀྱང་རྒན་པ་ལ་ཕྱག་བཙལ་ཏེ། སེམས་ཅན་ཐམས་ཅད་ལ་སྙིང་བརྩེ་བ་སེམས་ལ་བཞག་སྟེ་ཁྲི་ལ་འདུག་གོ །འདུག་ནས་ཀྱང་སྔོན་བཅོམ་ལྡན་འདས་ཀྱིས་མདོ་གང་དུ་བཤད་པའི་དྲན་པའི་ཏིང་ངེ་འཛིན་ལ་སྙོམས་པར་ཞུགས་སོ། །དེའི་ཚེ་ལྷ་རྣམས་ཀྱིས། འཕགས་པའི་འདི་ཡང་དོགས་མེད་པར། །སེང་གེ་སྒྲ་བཞིན་ཆོས་ཀྱང་འཆད། །ཞི་བའི་ཆོས་འདི་བཤད་པ་ལ། །དད་པས་ཆོས་འདི་མཉན་པར་གྱིས། །ཞེས་བྱ་བ་ལ་སོགས་པ་སྨྲས་ཏེ།

།ཆོས་ལ་མཉན་པའི་ཕྱིར་ལྷགས་སོ། །ཡང་འོད་སྲུངས་ཀྱིས་སྨྲས་པ། ཆོས་འདི་ཐམས་ཅད་ནང་ན་མཆོག་གྱུར་པ། །རྒྱལ་བས་གསུངས་པ་ཚེ་ལྡན་ཁྱོད་ཀྱིས་བཤད། །སྟོན་པས་གོང་དུ་བཤད་པ་བཞིན། །འཇིག་རྟེན་ཕན་འདོད་ ¡Ã སྡེ་སྣོད་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· སྟོན་པས་ ) ཤོད། །ཅེས་གསོལ་བ་བཏབ་བོ། །དེ་ནས་ཀུན་དགའ་བོས་སྟོན་པ་དྲན་པ་དང་བཅས་པས་གནས་རྣམས་ལ་ཕྱག་འཚལ་ཏེ ༑ དང་པོ་མངོན་པར་བྱང་ཆུབ་པའི་ཕྱོགས་སུ་བལྟས་ཏེ། ཐལ་མོ་སྦྱར་བ་བྱས་ནས ༑ འདི་སྐད་བདག་གིས་ཐོས་པའི་དུས་གཅིག་ན། བཅོམ་ལྡན་འདས་ཡུལ་འཁོར་མོ་འཇིག་ ( ¡¶ ཞ ¡· ཞིག་ ) ན་བཞུགས་ཏེ། དགེ་སློང་གི་དགེ་འདུན་ཆེན་པོ་དང་ཡང་ཐབས་ཅིག་གོ །བཅོམ་ལྡན་འདས་ཀྱིས་བཀའ་སྩལ་པ། དགེ་སློང་འདི་ལ་འཕགས་པའི་ ¡Ã སྡུག་བསྔལ་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· - སྡུག་བསྔལ་ ) བདེན་པ་ཞེས་བྱ་བ་འདི་ནི་སྔོན་མ་ཐོས་པའི་ཆོས་སོ་ཞེས་སྨྲས་སོ། །དེའི་ཚེ་ལྷ་དག་འདོད་ཆགས་དང་བཅས་པ་རྣམས་ཀྱིས་སྨྲས་པ། ཀྱེ་མའོ་འཇིག་རྟེན་འདི་ཡང་ནི། །མི་རྟག་ཕྱིར་ན་ཁྱད་པར་མེད། །གང་གི་རིན་ཆེན་གཏེར་དེ་ཡི། །ཡོན་ཏན་རྒྱ་མཚོ་དག་ནི་སྐེམས། ། བདག་གིས་དེ་ལས་མངོན་ཐོས་ཏེ། །ཆོས་ནི་ཐར་པར་བྱེད་པ་ལ། །འདི་སྐད་ཐོས་ཞེས་སྦྱོང་བར་བྱེད། །འདི་ནི་དེ་རིང་བཤད་པ་ཡིན། །ཞེས་སྨྲས་སོ། །དེ་ནས་ཀུན་ཤེས་ཀཽཎྜི་ཎྱས་སྨྲས་པ། འདི་ནི་ང་ལ་བཤད་དོ་ཞེས་བྱ་བ་ལ་སོགས་པ་སྨྲས་སོ། ། དེ་ནས་ཐམས་ཅད་སྟན་ལས་ལངས་ཏེ། སྟོན་པའང་ཐོས་པའི་ལམ་དུ་གྱུར་ཏོ། ། ཞེས་ཀྱང་སྨྲས་སོ། །གཞན་ཡང་ཆོས་ཀྱི་འཁོར་ལོ་གང་དུ་གསུང༌། གཉིས་པའང་དེ་ཉིད་དུ་སྟེ་བདེན་པ་བཞིའོ། །གསུམ་པའང་དེ་ཉིད་དུ་སྟེ་གཟུགས་ནི་བདག་མ་ཡིན་པའོ་ཞེས་བྱ་བ་ལ་སོགས་པ་བཤད་དོ། །དེ་བཞིན་དུ་བཅོམ་ལྡན་འདས་ཀྱི་གནས་གང་དང་གང་དུ་མདོ་གང་དང་གང་བཤད་པ་དེ་དག་ཐམས་ཅད་བཤད་དེ། གང་ཕུང་པོ་ལྔ་སྟོན་པ་དེ་དང་དེ་ཕུང་པོ་ལྔའི་སྡེ་དང༌། དེ་བཞིན་དུ་སྐྱེ་མཆེད་ཀྱི་སྡེ་དང་ ༑ གང་རྟེན་ཅིང་འབྲེལ་བར་འབྱུང་བའི་སྡེ་དང༌།

གང་འཕགས་པའི་བདེན་པའི་སྡེ་དང༌། གླེང་གཞིའི་སྡེ་དང༌། བཅོམ་ལྡན་འདས་ཀྱི་བཀའ་མང་ ( ¡¶ ཞ ¡· མངའ་ ) བ་རྣམས་བཅོམ་ལྡན་འདས་ཀྱི་སྡེ་དང༌། ཉན་ཐོས་ཀྱི་ ¡Ã བཀའ་མང་ ( ¡¶ ཞ ¡· མངའ་ ) བ་ཉན་ཐོས་ཀྱི་སྡེ་དང༌།བྱང་ཆུབ་ཀྱི་ཕྱོགས་ཀྱི་ཆོས་སུམ་ཅུ་རྩ་བདུན་སྟོན་པ་ལམ་གྱི་སྡེ་དང༌། བསླབ་པ་སྟོན་པ་བསླབ་པའི་སྡེ་དང༌། ཚིགས་སུ་བཅད་པ་རིང་པོའི་སྡེ་དང༌། བར་པ་ལ་བར་བའི་སྡེ་དང༌། ཚིགས་སུ་བཅད་པ་གཅིག་ནས་བཅུའི་བར་ལ་གཅིག་ནས་བསྐྱེད་པའི་སྡེ་ཞེས་མིང་དུ་བཏགས་སོ། །དེ་ལྟར་མདོ་སྡེ་རྣམས་བསྡུས་པ་ལ་ལྷ་རྣམས་ཀྱིས་ཀྱང་བསྔགས་པ་བྱིན། དགྲ་བཅོམ་པ་རྣམས་ཀྱིས་ཀྱང་མཐུན་པར་བསྔགས་སོ། །དེ་ནས་འོད་སྲུངས་ཀྱིས་སྨྲས་པ། ཁྱོད་ཀྱིས་དེ་སྐད་དུ་ཟད་དམ། ཀུན་དགའ་བོས་དེ་སྙེད་དུ་ཟད་དོ་ཞེས་སྨྲས་ཏེ་ཁྲི་ལས་བབས་སོ། །ཡང་འོད་སྲུངས་ཀྱིས་ཁྲི་ལ་འདུག་སྟེ། ཉེ་བ་འཁོར་ལ་སྨྲས་པ། ཨུ་པ་ལི་ཁྱོད་ནི་བཅོམ་ལྡན་འདས་ཀྱིས་འདུལ་བའི་འཛིན་པར་གསུངས་ན་འདུལ་བ་སྨྲ་བར་དད་དམ། སྨྲས་པ་དད་དོ་ཞེས་ཁྲི་ལ་འདུག་གོ །དེ་ནས་གླེང་གཞི་ནི་ཝཱ་རཱ་ཎ་སཱི་སྟེ། དགེ་སློང་དག་ཤམ་ཐབས་ཟླུམ་པོར་བགོ་བར་བྱའོ། གཉིས་པ་ཆོས་གོས་ཟླུམ་པོར་བགོ་བར་བྱའོ། །གསུམ་པ་གྲོང་ཀ་ལནྟ་ཀར་འཁྲིག་པ་སངས་ པའོ ༑ ༑ དེ་བཞིན་དུ་ཐམས་ཅད་བཤད་དེ་ཕམ་པའི་སྡེ་དང༌། ལྷག་མའི་སྡེ་དང༌། མ་ངེས་པའི་ཆོས་གཉིས་དང༌། སྤང་བའི་ལྟུང་བ་སུམ་ཅུ་དང༌། ལྟུང་བྱེད་དགུ་བཅུ་ཐམ་པ་དང༌། བསླབ་པའི་ཆོས་མང་པོ་དང༌། རྩོད་པ་ཞི་བར་བྱེད་པའི་ཆོས་བདུན་བཅས་སོ། །འདི་དག་ནི་གནང་བའོ། །འདི་དག་ནི་བཀག་པའོ་ཞེས་བྱ་བ་འདུལ་བ་སྨྲས་ཏེ་ཁྲི་ལས་བབས་སོ། །དེ་ནས་འོད་སྲུངས་ཀྱིས་མ་འོངས་པའི་སེམས་ཅན་ཟབ་མོའི་དོན་ལ་མོས་པར་བྱ་བའི་ཕྱིར་བདག་ཉིད་ཀྱིས་མ་ལྟ་བུ་བཤད་དོ་སྙམ་སྟེ་ཁྲི་ལ་འདུག་སྟེ་སྨྲས་པ། ཚེ་དང་ལྡན་པ་དག་ཤེས་བྱའི་མཚན་ཉིད་ཡོངས་སུ་སྒྲོགས་པ་འདི་ནི་མ་ལྟ་བུའོ།

།འདི་ལྟ་སྟེ། དྲན་པ་ཉེ་བར་བཞག་པ་བཞི་དང༌། ཡང་དག་པར་སྤོང་བ་བཞི་དང༌། ཞེས་བྱ་བ་ལ་སོགས་པ་དང༌། སྟོང་པ་ཉིད་དང༌། མཚན་མ་མེད་པ་དང༌། སྨོན་པ་མེད་པ་ལ་སོགས་ཏེ། ཆོས་ཀྱི་ཕུང་པོ་དང་ལྡན་པ་འདི་དག་ནི་མ་ལྟ་བུའོ། །ཞེས་བྱ་བ་ནས་བརྩམས་ཏེ། མ་ལྟ་བུའི་སྡེ་སྣོད་རྒྱས་པར་བསྡུས་སོ། །དེ་ལྟར་སྡེ་སྣོད་རིམ་པོ་ཆེ་གསུམ་བསྡུས་པ་རེ་རེ་ལ་ཡང་དགྲ་བཅོམ་པ་ཐམས་ཅད་ཀྱིས་འདི་ནི་བདག་ཅག་ལ་བཤད་དོ། །ཞེས་བྱ་བ་ལ་སོགས་པས་མཐུན་པར་བསྟན་ཅིང་ལྷ་རྣམས་ཀྱིས་ཀྱང་རྗེས་སུ་མཐུན་པར་བསྔགས་པ་བྱས་སོ། །དེ་ནས་ཡང་གནས་བརྟན་འོད་སྲུངས་ཀྱིས་སྨྲས་པ། ཀྱེ་ཧོ་འགྲོ་ལ་ཕན་པའི་ཆོས་གཙུག་ལག །སྟོབས་བཅུའི་བཀའ་ལ་ཚད་ཡོད་མ་ཡིན་ཀྱང་ ༑ ༑ འཇིག་རྟེན་ངན་པས་མུན་པ་དག་ཏུ་སྟོན། །ཡང་དག་བསྟན་པ་བློ་རིག་གསལ་བའི་སྒྲོན། །ཞེས་སྨྲས་ཏེ། དེས་བསམས་པ། སྟོན་པའི་བཀའ་བཞིན་བྱ་བ་ནི་བྱས། སེམས་ཅན་མ་ནུས་པ་ལ་ཕན་གདགས་པ་སུས་ནུས། ད་ནི་མྱ་ངན་ལས་འདའ་སྙམ་སྟེ། ངོ་ཚ་མེད་ལ་ཆད་པས་བཅད། །ངོ་ཚ་ཅན་ལ་ཕན་ཡང་བཏགས།། བདག་ལ་ཕན་ཡང་ཞི་བྱས་ཀྱི། །བདག་ནི་མྱ་ངན་འདའ་དུས་བབ། །ཅེས་སྨྲས་ཏེ། ཀུན་དགའ་བོ་ལ་བསྙེན་ཏེ་སྨྲས་པ། ཚེ་དང་ལྡན་པ་ཀུན་དགའ་བོ་ཤེས་པར་གྱིས་ཤིགབཅོམ་ལྡན་འདས་ཀྱིས་བདག་ལ་བསྟན་པ་གཏད་དེ་མྱ་ངན་ལས་འདས་སོ།། ད་ནི་བདག་ཀྱང་མྱ་ངན་ལས་འདའ་བས་སྟོན་པའི་བསྟན་པ་ཁྱོད་ཀྱིས་སྐྱོངས་ཤིག ཁྱོད་ཀྱིས་ཀྱང་རྒྱལ་པོའི་ཁབ་ན་དེད་དཔོན་ཞིག་གི་བུ་རྩྭ་ཤ་ན་ཀའི་གོས་ཀྱིས་དཀྲིས་པ་ཞིག་འབྱུང་བས། དེའི་མིང་ཡང་རྩྭ་ཤ་ན་ཀའི་གོས་ཅན་ཞེས་བྱ་བར་གདགས་སོ། །དེ་ནི་ད་ལྟར་རྒྱ་མཚོ་ལ་སོང་སྟེ་དེ་ཕྱིར་ལོག་པ་དང་སངས་རྒྱས་ཀྱི་བསྟན་པ་ལ་ལོ་ལྔར་མཆོད་སྦྱིན་བྱེད་པས། དེ་རབ་ཏུ་ཕྱུང་ལ་དེ་ལ་བསྟན་པ་གཏོད་ཅིག ཅེས་སྨྲས་ནས། ད་ནི་བདག་འདའ་བར་བྱ་སྙམ་སྟེ། སྐུ་གདུང་གི་མཆོད་རྟེན་བརྒྱད་དང་ཚེམས་མཆེ་བ་བཞི་ལ་ཡང་རྫུ་འཕྲུལ་གྱིས་སོང་སྟེ་མཆོད་པ་བྱས་སོ།

།དེ་ནས་གནས་བརྟན་འོད་སྲུངས་ཀྱིས་རྒྱལ་པོ་མ་སྐྱེས་དགྲ་ལ་སྦྲན་པ་དང་དེ་གཉིད་དུ་སོང་བས་མ་ཚོར་རོ། །དེ་ནས་བདག་ཉིད་རི་བྱ་རྐང་དུ་སོང་སྟེ། དེ་བཞིན་གཤེགས་པས་གནང་བའི་ཕྱག་དར་ཁྲོད་ཀྱི་ལྷུང་བཟེད་དང་ཆོས་གོས་དང་བཅས་ཏེ། རི་ཀུ་ཀུ་པཱ་ཏིའི་རྩེ་མོ་ལ་བྲག་ཚལ་པ་བཞི་ཡོད་པའི་དབུས་སུ་འདུག་ནས་སྨོན་ལམ་བཏབ་པ། བདག་གསོན་བཞིན་དུ་འདིར་གནས་པར་གྱུར་ཅིག དེ་ནས་བདག་གིས་ལུས་ལ་མེས་མི་ ( ¡¶ ཞ ¡· - མི་ ) ཚིག་པ་ལ་སོགས་པ་གནོད་པ་མེད་པར་གྱུར་ཅིག དེ་ནས་བྲག་བཞི་པོ་འདིས་འགེབས་པར་གྱུར་ཅིག གལ་ཏེ་རྒྱལ་པོ་མ་སྐྱེས་དགྲ་འོངས་ན་དེ་ལ་བསྟན་ཏེ་ཡང་འགེབས་པར་གྱུར་ཅིག དེ་ནས་ཡང་གསོན་བཞིན་དུ་གནས་ཏེ་སངས་རྒྱས་བྱམས་པའི་བསྟན་པའི་ཚེ། འཁོར་འདུས་པ་དང་པོ་དང་གཉིས་པ་དང་གསུམ་པའི་ཚེའང་འདིར་གཤེགས་པར་གྱུར་ཅིག དེའི་ཚེ་དགེ་སློང་རྣམས་ལ་བདག་གི་ལུས་འདི་སྟོན་པར་གྱུར་ཅིག དེའི་ཚེ་དགེ་སློང་མང་པོ་ཕྱག་དར་ཁྲོད་ལ་བརྟེན་ཏེ་དགྲ་བཅོམ་པ་ཐོབ་པར་གྱུར་ཅིག དེ་ནས་བདག་གིས་ལུས་འདི་རྫུ་འཕྲུལ་གྱིས་པུར་སྦྱོང་བར་གྱུར་ཅིག ཞེས་སྨོན་ལམ་བཏབ་སྟེ་མྱ་ངན་ལས་འདས་སོ། །དེའི་ཚེ་ལྷ་རྣམས་ཀྱིས་ཀྱང་མཆོད་པ་བྱས་སོ། །དེའི་ཚེ་ས་ཆེན་པོ་གཡོས་པར་གྱུར་པ་དང༌། དེས་རྒྱལ་པོ་མ་སྐྱེས་དགྲ་གཉིད་སད་པ་དང༌། དེས་བསམས་པ། འཕགས་པ་འོད་སྲུངས་མྱ་ངན་ལས་འདས་ངེས་སྙམ་སྟེ་གནས་བརྟན་ཀུན་དགའ་བོ་ག་ལ་བ་དེར་སོང་ངོ༌། །དེ་ནས་གནས་བརྟན་ཀུན་དགའ་བོས་རི་ཀུ་ཀུ་བཱ་ཏ་ག་ལ་བ་དེར་རྒྱལ་པོ་ཁྲིད་དེ་འོངས་ནས་འཕགས་པ་འོད་སྲུངས་ཀྱི་པུར་བསྟན་ཏེ། དེ་ནས་རྒྱལ་པོས་འདི་ཞུགས་ལ་བཞེན་པར་བགྱིའོ་ཞེས་སྨྲས་སོ། །ཀུན་དགའ་བོས་སྨྲས་པ། རྒྱལ་པོ་ཆེན་པོ་འདི་མི་ནུས་ཏེ་སྨོན་ལམ་གྱི་ཁྱད་པར་འདི་ལྟ་བུ་བཏབ་པའི་ཕྱིར་རོ་ཞེས་སྨྲས་སོ། །དེ་ནས་ཡང་རི་བཞིས་བཀབ་པར་གྱུར་ཏོ། །དེ་ནས་རྒྱལ་པོས་ཀུན་དགའ་བོ་ལ་སྨྲས་པ

༑ བཅོམ་ལྡན་འདས་དང་འཕགས་པ་འོད་སྲུངས་ཀྱིས་བདག་ལ་མ་སྦྲོན་ན། འཕགས་པ་ཁྱོད་མྱ་ངན་ལས་འདའ་བའི་ཚེ་བདག་སྦྲོན་ཅིག་ཅེས་གསོལ་ནས་ཡོན་བདག་ཏུ་ཁས་བླངས་སོ། །འཕགས་པ་འོད་སྲུངས་ཀྱིས་གཞི་བཟུང་སྟེ་དགྲ་བཅོམ་པ་ལྔ་བརྒྱས་ཆོས་བསྡུས་པ་སྟེ་བསྡུ་བ་དང་པོའོ།། །།དེ་ནས་གནས་བརྟན་ཀུན་དགའ་བོ་འཁོར་དང་བཅས་ཏེ་སྨྱིག་མའི་ཚལ་ན་འདུག་གོ །དེའི་ཚེ་རྩྭ་ཤ་ན་ཀའི་གོས་ཅན་རྒྱ་མཚོ་ལས་ཕྱིར་ལོག་སྟེ། ཀུན་དགའ་བོ་ག་ལ་བ་དེར་སོང་སྟེ་ཕྱག་བཙལ་ནས། འཕགས་པ་བཅོམ་ལྡན་འདས་གང་ན་བཞུགས། བདག་ལོ་ལྔའི་དུས་སྟོན་བགྱིད་པར་འཚལ་ལོ་ཞེས་ཞུས་སོ། ཀུན་དགའ་བོས་སྨྲས་པ། དེ་བཞིན་གཤེགས་པ་མྱ་ངན་ལས་འདས་སོ། །དེས་དེ་ཐོས་པ་དང་བརྒྱལ་ཏེ་ས་ལ་བགྱེལ་ཏེ། དེ་སངས་པ་དང་ལོ་ལྔའི་དུས་སྟོན་བྱས་སོ ༑ ༑ དེ་ནས་གནས་བརྟན་ཀུན་དགའ་བོས་དེ་ལ་སྨྲས་པ། བུ་ཁྱོད་སངས་རྒྱས་ཀྱི་བསྟན་པ་ལ་རབ་ཏུ་བྱུང་ཞིག དེས་ཀྱང་དེ་བཞིན་ནོ་ཞེས་སྨྲས་ཏེ་རབ་ཏུ་བྱུང་ངོ་ ༑༑ དེའང་ཐོས་པ་འཛིན་པ་ཞིག་སྟེ་མྱུར་དུ་ཆོས་ཐམས་ཅད་ཁོང་དུ་ཆུད་པར་གྱུར་ཏོ ༑ ༑ དེ་ནས་ཀུན་དགའ་བོ་སྨྱིག་མའི་ཚལ་ན་འདུག་པའི་ཚེ་དགེ་སློང་ཞིག་གིས་སྨྲས་པ། ལོ་བརྒྱའི་བར་དུ་གང་འཚོ་བས། །ཆུ་ལ་ཆུ་སྐྱར་མཐོང་བ་ན། །བདག་གི་འཚོ་བ་ཉིན་གཅིག་དགེ །ཞེས་སྨྲས་པ་དང༌། ཀུན་དགའ་བོས་དེ་ཐོས་ཏེ། དེ་ལ་སྨྲས་པ། བུ་བཅོམ་ལྡན་འདས་ཀྱིས་ནི་དེ་ལྟར་མ་གསུངས་ཏེ། ལོ་བརྒྱའི་བར་དུ་གང་འཚོ་བས། །སྐྱེ་དང་འཇིག་པ་མ་མཐོང་བས། །གང་ཟག་གཉིས་པ་སྟོན་པ་ཡི ༑ ༑ ས་འདི་རྟག་པར་བཤད་པ་སྟེ། །མ་དད་ཞེ་སྡང་བློ་ཅན་དང༌། །དད་པ་ལོག་པར་འཛིན་པའོ། །མདོ་སྡེ་ལོག་པར་འཛིན་པ་ནི། །བ་ལང་འདམ་དུ་བྱིང་བ་བཞིན ༑༑ ཞེས་བྱ་བ་ལ་སོགས་པ་བྱས་པས་དེས་ཀྱང་རང་གི་མཁན་པོ་ལ་སྨྲས་སོ།། མཁན་པོས་སྨྲས་པ། །ཀུན་དགའ་བོ་ཡང་རྒས་པར་གྱུར། །དེའི་དྲན་པ་ཉམས་པར་འགྱུར། །སེམས་ཅན་རྒ་བས་གཙེས་པ་ནི། །ནུས་པ་དང་ཡང་བློ་ཡང་རྙིང༌།

། ཞེས་བརྗོད་པ་དང༌། དེས་ཀྱང་སླར་གནས་བརྟན་ཀུན་དགའ་བོ་ལ་སྨྲས་སོ། ། ཀུན་དགའ་བོས་བསམས་པ། འདི་ལ་ནི་བདག་ཉིད་ཀྱིས་མི་ཕན་ལ། ཤཱ་རིའི་བུ་ལ་སོགས་པ་དང༌། དེ་བཞིན་གཤེགས་པས་ཀྱང་ལོ་སྟོང་གི་བར་དུ་བཞུགས་པས་ཀྱང་ཕན་པར་མི་འགྱུར་གྱིས། དེ་བས་ན་བདག་ཀྱང་མྱ་ངན་ལས་འདའ་སྙམ་སྟེ། ཡང་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· ཡ་ ) རབ་དེ་དག་སྔར་ཡང་འདས། །གསར་བུ་དག་ནི་བདག་མི་མཐུན ༑ ༑ བདག་གི་གཅིག་པུ་ཏིང་འཛིན་ལ། །སྒོ་ང་སྐྱེས་བཞིན་གནས་སུ་འདུག། ཅེས་སྨྲས་ཏེ། རྩྭ་ཤ་ན་ཀའི་གོས་ཅན་ལ་སྨྲས་པ། བུ་བཅོམ་ལྡན་འདས་ཀྱིས་གནས་བརྟན་འོད་སྲུངས་ལ་བསྟན་པ་གཏད་དོ། །དེས་ཀྱང་ཁོ་བོ་ལ་གཏད་དོ། ། ངས་ཁྱོད་ལ་གཏད་ཀྱིས།ཁྱོད་ཀྱིས་ཀྱང་གྲོང་ཁྱེར་ ¡Ã མ་ཏུ་ར་ཏི་ ( ¡¶ ལུ ¡· མདུད་རིར་ ) མུནྟོ་ཞེས་བྱ་བར་ཡོན་བདག་འབྱུང་བས། དེར་ལྷ་ཁང་གྱིས་ཤིག དེར་དྲི་འཚོང་མ་གུཔྟ་ལ་བུ་ཨུ་པ་གུཔྟ་ཞེས་བྱ་བ་འབྱུང་པས། དེ་རབ་ཏུ་བྱུང་ཞིག དེ་བཅོམ་ལྡན་འདས་ཀྱིས་ལུང་བསྟན་ཏེ། མྱ་ངན་ལས་འདས་ནས་ལོ་བརྒྱ་ན་མཚན་དང་མི་ལྡན་པ་ཞེས་བྱ་བ་སངས་རྒྱས་ཀྱི་བསྟན་པ་ལ་བྱ་བ་བྱེད་དོ། །ཞེས་གསུངས་པས་དེ་ལ་གཏོད་ཅིག རྩྭ་ཤ་ན་ཀའི་གོས་ཅན་གྱིས་སྨྲས་པ། མཁན་པོ་དེ་བཞིན་དུ་བགྱིའ ོ༑ ༑ དེ་ནས་ཀུན་དགའ་བོས་བསམས་པ། ད་ནི་བདག་མྱ་ངན་ལས་འདའ་བར་བྱ་ན་ཡུལ་གང་དུ་མྱ་ངན་ལས་འདའ་ན། དེ་དང་ཡུལ་གཞན་རིང་བསྲེལ་ལ་རྩོད་པ་འབྱུང་བས་ཡུལ་གཉིས་ཀྱི་ཆུའི་སྟེང་དུ་མྱ་ངན་ལས་འདས་ལ། རིང་བསྲེལ་བགོ་བར་བྱ་སྙམ་སྟེ་དེར་གཤེགས་སོ། །དེའི་ཚེ་རྒྱལ་པོ་མ་སྐྱེས་དགྲ་ལ་རྨི་ལམ་ངན་པ་བྱུང་སྟེ། སད་ནས་རྩྭ་ཤ་ན་ཀའི་གོས་ཅན་གྱི་དྲུང་དུ་སོང་སྟེ། འཕགས་པ་ཀུན་དགའ་བོ་གར་གཤེགས་ཞེས་སྨྲས་པ་དང༌། དེས་སྨྲས་པ། བཅོམ་ལྡན་འདས་ཀྱི་གཅུང་པོ་འདི། །གསུང་ལས་ཀྱང་ནི་མངོན་པར་ཤེས། །ཡངས་པ་ཅན་གྱི་གྲོང་ཁྱེར་དུ། །མངོན་དུ་ཡང་ནི་དེ་རུ་སོང༌། །དེ་ནས་རྒྱལ་པོས་དེ་མྱ་ངན་ལས་འདའ་བར་ཤེས་ཏེ་དཔུང་དང་བཅས་ཏེ་དེར་དོང་ངོ༌།

༑ ༑ དེས་དྷི་དྷི་ཀ་ལའོ། །དེས་ཀྲྀཥྞ་པ་ལའོ། །དེས་སུ་དརྴ་ན་ལ་བསྟན་པ་གཏད་དེ་མྱ་ངན་ལས་འདས་སོ། །དེའི་ཚེ་སངས་རྒྱས་མྱ་ངན་ལས་འདས་ནས་ལོ་བརྒྱ་དང་བཅུ་ ( ¡¶ ལུ ¡· + གཉིས་ ) ལོན་ནོ། །དེའི་དུས་སུ་ཡངས་པ་ཅན་གྱི་དགེ་སློང་དག་གིས་རང་གི་ལག་པས་ས་བརྐོར་ ( ¡¶ ལུ ¡· + མི་ ) རུང་ངོ༌། །ཞེས་བྱ་བ་ལ་སོགས་པ་མི་རུང་བའི་གཞི་བཅུ་བྱས་སོ། །དེའི་ཚེ་ན་ཡངས་པ་ཅན་ཉིད་ན་ཀུན་དགའ་བོའི་སློབ་མ་ཐམས་ཅད་འདོད་ཅེས་བྱ་བ་དགྲ་བཅོམ་པ་རྣམ་པར་གྲོལ་བ་ལ་བསྒོམས་པ ༑ འདོད་པ་ཆུང་བ་ཞིག་ཡོད་དེ། དེའི་སློབ་མ་ཡ་ཤ་ཞེས་བྱ་བ་ཞིག་གྲོང་ ¡Ã ཁྱེར་ནོར་ཅན་ ( ¡¶ ལུ ¡· མདའ་ ) ན་ཡོད་དོ། །དེ་སློབ་དཔོན་ཅན་དུ་འོངས་པའི་ལམ་དུ་མི་རུང་བའི་གཞི་བཅུ་ཐོས་སོ། །དེས་སློབ་དཔོན་ལ་སྨྲས་པ། འདི་ལྟ་བུ་དག་རུང་ངམ ༑ སློབ་དཔོན་གྱིས་སྨྲས་པ། འདི་དག་མི་རུང་སྟེ་གནས་འདི་དང་འདིར་བཅས་སོ་ཞེས་སྨྲས་སོ། །དེས་སྨྲས་པ། དེ་ཡལ་བར་འདོར་རམ། སློབ་དཔོན་གྱིས་སྨྲས་པ། མི་འདོར་རོ། ཁྱོད་ཀྱིས་གྲོགས་ཚོལ་ཅིག ཁོ་བོ་ཡང་གྲོགས་ཚོལ་དུ་འགྲོའོ་ཞེས་སྨྲས་ནས། དེ་བསམ་གཏན་ལ་ཞུགས་ནས། ཡུལ་ཤལ་པ་ན་གནས་བརྟན་ཤ་ཏ་ཞེས་བྱ་བ་ཡོད་པ་ཅན་དུ་ཕྱིན་ཏོ། །དངོས་པོ་རྒྱས་པར་བརྗོད་དེ ༑ ཤ་ཏ་ཡང་དེ་བཞིན་དུ་གྲོགས་ཚོལ་དུ་སོང་ངོ༌། །དེའི་ཚེ་དེའི་སློབ་མའི་ཉེ་གནས་ཀྱི་དྲུང་དུ། ཡངས་པ་ཅན་གྱི་དགེ་སློང་འོངས་ཏེ་དེ་དག་ལ་སྨྲས་པ། མཁན་པོ་གར་སོང༌། དེས་སྨྲས་པ། གྲོགས་ཚོལ་དུ་སོང༌། ཡངས་པ་ཅན་པས་སྨྲས་པ། ཅིའི་དོན་དུ། དེས་སྨྲས་པ། ཁྱེད་ལ་ཆད་པས་བཅད་པའི་ཕྱིར་རོ། །ཡངས་པ་ཅན་པས་སྨྲས་པ། ངེད་ཀྱིས་ཅི་བྱས། དེས་ཞིབ་པར་སྨྲས་སོ། །ཡངས་པ་ཅན་པས་སྨྲས་པ། དེས་ཅི་ཉེས། དེས་ཞིབ་མོར་སྨྲས་ཏེ་བསྟན་པ་ལ་གནོད་པའི་ཕྱིར་རོ ༑ ༑ དེ་ནས་ཡངས་པ་ཅན་པ་དག་སྐྲག་ནས་གྲོས་བྱས་པ། ཁ་ཅིག་ན་རེ་རང་ཅག་ཀྱང་གྲོགས་བཙལ་ལོ་ཞེས་ཟེར་རོ། ཁ་ཅིག་ན་རེ་རྩོད་པར་འགྱུར་བས་བྲོས་པར་བྱ་ཞེས་ཟེར་རོ།

ཁ་ཅིག་ན་རེ་བཟོད་པར་གསོལ་ལོ་ཞེས་ཟེར་རོ། ཁ་ཅིག་ན་རེ་སློབ་མ་འདི་དག་ལ་བསུག་སྦྱིན་ཞེས་ཟེར་ཏེ། དེ་ལ་མཐུན་ནས་བསུག་བྱིན་ནས་སོང་ངོ༌། །དེ་ནས་ཚེ་དང་ལྡན་པ་ཡ་ཤ་འོངས་ནས། དེའི་སློབ་མ་དག་གིས་སྨྲས་པ། མཁན་པོ་དེ་དག་ལ་སྐྲག་པར་བགྱིད་མི་འཚལ་ཏེ་བཟློག་ཟིན་ཏོ། ། མཁན་པོས་སྨྲས་པ། ཁྱོད་ཀྱིས་བསུག་ཟོས་སམ། །དེ་དག་ཅང་མི་ཟེར་བར་འཁོད་དོ། །དེ་ནས་འཕགས་པ་ཡ་ཤས་སྔོན་སྨྲས་པའི་གྲོགས་རྣམས་ལ་སྤྲིངས་པ། གྲོགས་པོ་དག་བསླབ་པ་ལ་ལྷོད་པའི་ཕྱོགས་འཕེལ་གྱི་རིངས་པར་འོངས་པར་རིགས་སོ། །བསྟན་པའི་དགོས་པ་ལ་མ་བཤོལ་ཅིག ཡང་ཚིགས་སུ་བཅད་དེ་སྨྲས་པ། བགས་ཀྱིས་བྱ་ལ་རིངས་པར་དང༌། །རིངས་པར་བྱ་ལ་བགས་ཀྱིས་ བྱེད ༑ ༑ ཚུལ་བཞིན་བྱེད་པ་རབ་ཏུ་སྤངས ( ¡¶ ལུ ¡· ལངས ) ། ། དེ་ནི་བྱིས་པའི་སྡུག་བསྔལ་རྙེད། །དེ་ནི་གྲགས་པའི་སྐལ་བ་མེད། །མཛའ་བོ་ལ་ནི་དེས་སྤངས་པའོ།། དེ་ཡི་དོན་ཡང་མི་འཕེལ་ཏེ། །ཟླ་བ་ཁམ་པ་ཇི་བཞིན་ནོ། །གང་གིས་དགོས་པ་བགས་ཀྱིས་བྱེད། །དེ་ལྟར་བློ་ཆེན་རིངས་པར་བྱེད། །ཚུལ་བཞིན་མ་ཡིན་རབ་ཏུ་སྤངས ( ¡¶ ལུ ¡· ལངས ) ། །མཁས་པ་དེས་ནི་བདེ་བ་རྙེད། །གྲགས་པར་དེ་གྱུར་སྐལ་བ་ཅན། །གྲོགས་པོ་ལས་ཀྱང་སྤང་མི་འགྱུར། །དེ་ཡི་དོན་ཀྱང་འཕེལ་བས་ཏེ། ། དཔེ་རུ་ཟླ་བ་ཚེས་པ་འདྲ། ། ( ¡¶ ལུ ¡· + ཚུལ་བཞིན་བྱེད་པ་རབ་ཏུ་ལངས། ། ) ཞེས་སྨྲས་ཏེ། ཡ་ཤ་འདུ་ཁང་དུ་ཞུགས་ནས་བས་མཐར་སོང་ནས་བསམ་གཏན་བཞི་པ་ལ་སྙོམས་པར་ཞུགས་ནས་གཎྜཱི་བརྡུངས་སོ། །དེའི་ཚེ་དགྲ་བཅོམ་པ་བདུན་བརྒྱར་གཅིག་གིས་མ་ལངས་པ་འདུས་ཏེ། ཐམས་ཅད་ཀྱང་གནས་བརྟན་ཀུན་དགའ་བོའི་སློབ་མ་ཤ་སྟག་གོ །དེའི་ཚེ་ཚེ་དང་ལྡན་པ་ཀུག་ཙ་སོ་མི་ནི་ཞེས་བྱ་བ་དེ་འགོག་པ་ལ་ཞུགས་པས་མ་ ( ¡¶ ཞ ¡· - མ་ ) ཐོས་སོ། །དེ་ནས་སྙོམས་པར་ཞུགས་པ་ལས་ལངས་པ་དང༌། ལྷ་དག་གིས་སྨྲས་པ། འཕགས་པ་མྱུར་དུ་བཞུད་ཅིག བསྟན་པ་ལ་རྨ་འབྱིན་པའི་དགེ་སློང་སུན་དབྱུང་བའི་ཕྱིར་དགེ་འདུན་ཐམས་ཅད་འདུས་སོ།

།འཕགས་པ་དེའང་སོང་སྟེ། དེ་ལྟར་ན་དགེ་སློང་བདུན་བརྒྱ་འདུས་པར་གྱུར་ཏོ། །དེ་ནས་འཕགས་པ་ཡ་ཤས་ཐོག་མར་དངོས་པོ་བཅུ་པ་བརྗོད་དེ་བཅའ་བ་དག་པར་བཅས་སོ། །སྡེ་སྣོད་ལྷག་མ་རྣམས་ཀྱང་དག་པར་བྱས་ཏེ་བསྡུས་སོ། །དགྲ་བཅོམ་པ་བདུན་བརྒྱས་ཡང་དག་པར་བཟླས་པ་ཞེས་བྱས་ཏེ་ཆོས་བསྡུ་བ་གཉིས་པའོ།། །།ཆོས་བསྡུ་བ་གཉིས་པོ་ཉིད་ནི་འདུལ་བ་ཕྲན་ཚེགས་ཀྱི་གཞི་ལས་ཇི་ལྟར་འབྱུང་བ་བཞིན་བྲིས་པ་ཡིན་ནོ། ། ( ¡¶ ཞ ¡·¡¶ ལུ ¡· + དེའི་དུས་ ) རྒྱལ་པོ་མྱ་ངན་མེད་ཡིན་ཞེས་གཏམ་དུ་གྲག་གོ །དེ་ནས་ཡང་དུས་ཕྱིས་དགེ་སློང་ལྷ་ཆེན་པོའི་རིང་ལ་སྟོན་པའི་དགེ་སློང་གི་དགེ་འདུན་རྩ་བའི་སྡེ་བཞི་ལ་སྡེ་པ་ཐ་དད་པ་བཅོ་བརྒྱད་དུ་གྱེས་ཏེ་འདི་ལྟར། ཤར་དང་ནུབ་དང་གངས་རི་དང༌། །འཇིག་རྟེན་འདས་པར་སྨྲ་བ་དག ༑ རྟག་པར་སྨྲ་པའི་སྡེ་པ་རྣམས། །ལྔ་ཚན་ཕལ་ཆེན་དགེ་འདུན་སྡེ། །བཞི་པ་དང་ནི་འོད་སྲུངས་སོ། །ས་སྟོན་དང་ནི་ཆོས་སྲུང་དང༌། །མང་ཐོས་གོས་དམར་སློབ་མ་དང༌། །རྣམ་པར་ཕྱེ་སྟེ་སྨྲ་བ་རྣམས། །ཐམས་ཅད་ཡོད་པར་སྨྲ་བའི་སྡེ། ། འཇིགས་བྱེད་ཚལ་གནས་རྒྱལ་བྱེད་གནས། །གཙུག་ལག་ཁང་ཆེན་གནས་བརྟན་པ ༑༑ སར་སྒྲོགས་རི་དང་བསྲུང་བ་པ། ། ¡Ã གནས་མ་བུ་ཡི་སྡེ་པ་རྣམས། ། ( ¡¶ ཞ ¡· - གནས་མ་་་་་་་པ་རྣམས། ། ) ཀུན་གྱིས་བཀུར་བ་རྣམ་པ་གསུམ། །ཡུལ་དོན་སློབ་དཔོན་བྱེ་བྲག་གིས། །ཐ་དད་རྣམ་པ་བཅོ་བརྒྱད་གྱུར། །ཞེས་ལྟ་བ་དང་སྤྱོད་པ་མི་མཐུན་པ་དུ་མ་བྱུང་བར་གྱུར་ཏེ། དེའི་ཚེ་སངས་རྒྱས་མྱ་ངན་ལས་འདས་ནས་ལོ་སུམ་བརྒྱ་ལོན་ནོ། །དེ་ལྟར་མི་མཐུན་པ་དེ་དག་དག་པར་བྱ་བའི་ཕྱིར་ཁ་ཆེའི་ཡུལ་གྱི་བྱེ་བྲག་ཀུན་པ་ན་ཞེས་བྱ་བའི་དགོན་པ་ཛཱ་ལནྡྷ་རར་ཡོན་བདག་རྒྱལ་པོ་ཀ་ནི་ཀས་བྱས་ནས། གནས་བརྟན་དགྲ་བཅོམ་པ་དགེ་སློང་སེང་གེ་ཞེས་བྱ་བས་གཞི་བཟུང་སྟེ། དགྲ་བཅོམ་པ་ལྔ་བརྒྱ། བྱང་ཆུབ་སེམས་དཔའ་གཟུངས་ཐོབ་པ་ལྔ་བརྒྱ་སོ་སོ་སྐྱེ་བོ་པཎྜི་ཏ་ལྔ་བརྒྱ་སྟེ་སྟོང་ལྔ་བརྒྱ་འདུས་ནས་ཆོས་ཐམས་ཅད་དག་པར་བྱས་སོ།

།དེ་དག་ཡན་ཆད་ནི་ཐོས་པ་འཛིན་པ་ཡིན་ཡང༌། མ་འོངས་པའི་སྐྱེ་བོ་བློ་རྨོངས་པར་གྱུར་པ་དག་ ( ¡¶ ཞ ¡· + འབྱུང་བས་ ) དེ་བས་ཀྱང་ལྟ་བ་དང་སྤྱོད་པ་མི་མཐུན་པ་དུ་མ་བྱུང་བས་དེ་སྤང་བར་བྱ་བའི་ཕྱིར། གླེགས་བམ་ལ་ཡི་གེར་བྲིས་སོ ༑ ༑ འདི་སྐད་དུ་བཅོམ་ལྡན་འདས་ལ་ཉེ་བ་འཁོར་གྱིས་ཞུས་པ། བཅོམ་ལྡན་འདས་འདུལ་བ་ཡི་གེར་མི་བྲི། འདུལ་བ་དང་འདྲ་བ་ཡི་གེར་མི་བྲི་ཞེས་གསུངས་ན ༑ མ་འོངས་པའི་སེམས་ཅན་དག་དྲན་པ་ཉམས་པར་གྱུར་ན་ཇི་ལྟར་བསྒྲུབ་པར་བགྱི ༑ བཀའ་སྩལ་པ། ཨུ་པ་ལི་དྲན་པ་ཉམས་ན་ལོ་མ་ལ་ཡི་གེ་བྲིས་ཏེ་བཅང་བར་བྱའོ། །ཞེས་བྱ་བའི་དུས་སུ་གྱུར་ཏོ། །འདི་ནི་གནས་བརྟན་སེང་གེས་གཞི་བཟུང་སྟེ་དགེ་སློང་སྟོང་ལྔ་བརྒྱས་བསྡུས་པ་ཞེས་བྱ་བ་སྟེ་བསྡུ་བ་གསུམ་ ( ¡¶ ཞ ¡· བསྡུས་ ) པའོ།། །།འདི་ནི་ ( ¡¶ ཞ ¡· + ཆོས་ ) བླ་མ་བརྒྱུད་པའི་གཏམ་བརྒྱུད་ལ་བརྟེན་ཏེ་བྲིས་པ་ཡིན་ནོ། །དེ་ལྟར་ན་ཆོས་བསྡུ་བ་གསུམ་པོ་འདི་ནི་འཕགས་པ་ཉན་ཐོས་ཀྱི་ལུགས་ལས་ཇི་ལྟར་གྲགས་པ་ལྟར་དུ་བྲིས་པ་ཡིན་ཏེ། འཕགས་པ་ཉན་ཐོས་ཀྱི་ལུགས་དང༌། རྟེན་དེ་ལས་དགེ་སློང་གི་དངོས་པོ་ནི། སངས་རྒྱས་ཀྱི་བསྟན་པའི་རྩ་བ་ལྟ་བུར་གྱུར་པའི་ཕྱིར། དེ་ལྟར་ཐུབ་པ་ཆེན་པོས་ཆོས་ཀྱི་ཕུང་པོ་བརྒྱད་ཁྲི་བཞི་སྟོང་དེ་སྙེད་ཅིག་གསུངས་པ་དེ་དག་སྟོན་པའི་སྲས་རྣམས་ཀྱིས་ལེགས་པར་བསྡུས་པའོ། །བར་བར་དུ་ཆོས་ལ་ལྷད་ཞུགས་པ་རྣམས་ཀྱང་དག་པར་ཞུས་དག་བྱས་ཏེ། སངས་རྒྱས་ཀྱི་བསྟན་པ་རྒྱས་པར་བྱས་པ་ཡིན་ནོ། །བཀའ་རྣམས་ཀྱི་འབྱུང་བའི་རིམ་པའོ།། །།བཀའ་དེ་དག་ལ་བརྟེན་ནས་གཅིག་ནས་གཅིག་ཏུ་བརྒྱུད་པས། ལེགས་པར་བསླབས་ཤིང་སྦྱངས་ནས། ལེགས་པར་རྟོགས་པའི་སློབ་དཔོན་རྣམས་ཀྱིས་བཀའ་དོན་གསལ་བར་བྱེད་པའི་བསྟན་བཅོས་རྣམས་ཀྱང་རིམ་གྱིས་འཇིག་རྟེན་དུ་ཤར་བར་གྱུར་ཏོ། །དེ་ལ་བཀའི་རྣམ་པར་འགྲེལ་པ་ལ་ནི་གཉིས་ཏེ། བཀའ་སྤྱིའི་འགྲེལ་པ་དང༌། སོ་སོའི་རྣམ་པར་འགྲེལ་པའོ།

།དང་པོ་ནི་མུ་སྟེགས་བྱེད་ཀྱི་རྒོལ་བ་ངན་པ་ཚད་གཅད་པའི་ཕྱིར་དང༌། ཆོས་ཀྱི་རྗེས་སུ་འབྲང་པ་རྣམས་ཡིད་ཆེས་པར་བྱ་བའི་སྒྲའི་བསྟན་བཅོས་དང་རྟོག་གེའི་ཚུལ་ལོ། །གལ་ཏེ་དེ་གཉིས་ཕྱི་རོལ་པའི་མ་ཡིན་ནམ་སྙམ་ན། བདེན་ཏེ་དེ་ནི་ལུགས་སྔོན་དུ་གྲགས་པས་ཏེ། དེ་ཉིད་ལ་བརྟེན་ཏེ་བསྟུན་ནས་ཡང་དག་པར་ན་སྒྲ་དང་རིག་པ་ཁྱད་པར་དུ་འདི་ལྟར་འཕགས་སོ་ཞེས་སྟོན་པ་བདེ་བར་གཤེགས་པའི་སྒྲ་དང་ཚད་མ་སྟེ། དགྲ་བཅོམ་པ་ཆོས་སྐྱོབ་དང༌། གནས་བརྟན་དབྱིག་གཉེན་དང༌། བཙུན་པ་ཕྱོགས་ཀྱི་གླང་པོ་དང༌། སློབ་དཔོན་ཆོས་ཀྱི་གྲགས་པ་ལ་སོགས་པ་རྣམས་ཀྱིས་ཚད་མའི་ཚུལ་མཛད་དོ། །སློབ་དཔོན་ཟླ་གྲགས་དང༌། ཙནྜ་གོ་མི་ལ་སོགས་པ་རྣམས་ཀྱིས་སྒྲའི་བསྟན་བཅོས་མཛད་དོ། །སོ་སོའི་རྣམ་པར་འགྲེལ་པ་ནི། ནང་རིག་པའི་དོན་གཏན་ལ་འབེབས་པ་སྟེ། སྡེ་སྣོད་གསུམ་འབྱེད་པའོ། །དེའང་འཕགས་པ་ཉན་ཐོས་ཀྱི་ལུགས་ལས། བཙུན་པ་ཆོས་སྐྱོབ་དང༌། དབྱིག་བཤེས་དང༌། དབྱངས་སྒྲོགས་དང༌། སངས་རྒྱས་ལྷ་ཞེས་བྱ་བ་བྱེ་བྲག་ཏུ་སྨྲས་པའི་གཙོ་བོ་བཞི་དང༌། གཞན་ཡང་བཙུན་པ་རཱ་མ་དང༌། གོ་ཆ་མཐའ་ཡས་ལ་སོགས་པ་དང༌། མདོ་སྡེ་པའི་བཙུན་པ་དཔལ་ལེན་དང༌། གཞོན་ནུ་ལེགས་དང༌། གནས་བརྟན་བཙུན་པ་ལ་སོགས་པའང་ཡིན་ནོ། །ཐེག་པ་ཆེན་པོའི་ཚུལ་གསལ་བར་མཛད་པ་ནི། འཕགས་པ་ཀླུ་སྒྲུབ་དང༌། འཕགས་པ་ལྷ་དང༌། བྷ་བྱ་དང༌། སངས་རྒྱས་བསྐྱངས་དང་ ༑ ཟླ་གྲགས་དང༌། ཡེ་ཤེས་སྙིང་པོ་དང༌། ཞི་བ་འཚོ་ལ་སོགས་པ་དབུ་མའི་ཚུལ་ངོ་བོ་ཉིད་མེད་པར་སྟོན་པ་གསལ་བར་མཛད་པ་དང༌། འཕགས་པ་ཐོགས་མེད་དང༌། དབྱིག་གཉེན་དང༌། བློ་བརྟན་དང༌། ཡོན་ཏན་འོད་དང༌། ཕྱོགས་ཀྱི་གླང་པོ་དང༌། ཆོས་ཀྱི་གྲགས་པ་དང༌། ཤངྐར་ནནྟ་དང༌། ཤེས་རབ་འབྱུང་གནས་སྦས་པ་དང༌། ཤཱནྟྀ་པ་ལ་སོགས་པ་རྣམས་རྣམ་རིག་སེམས་ཙམ་གསལ་བར་མཛད་པ་ཡིན་ནོ། །སློབ་དཔོན་འདི་དག་གིས་ཕྱོགས་གང་དང་གང་གསལ་བར་མཛད་པ་དང༌། འདི་དག་གི་གཏམ་རྒྱུད་རྣམས་ནི་གཞན་དུ་གོ་བར་བྱའོ།

།ཐེག་པ་ཆེན་པོ་གསང་སྔགས་ཀྱི་འབྱུང་བའི་ཚུལ་ཡང་གཞན་དུ་གོ་བར་བྱ་བས་འདིར་རེ་ཞིག་བཞག་གོ །དེ་ལྟར་བསྟན་བཅོས་མཁན་པོ་དེ་རྣམས་ཀྱིས་རྒྱ་གར་གྱི་ཡུལ་དང་ ༑ ཁ་ཆེ་དང༌། རྒྱ་ནག་དང༌། ལི་ཡུལ་དང༌། བལ་ཡུལ་ལ་སོགས་པ་རྣམས་སུ་སངས་རྒྱས་ཀྱི་བསྟན་པ་རྒྱས་པར་མཛད་པ་ཡིན་ནོ།། །།ཁ་བ་ཅན་གྱི་ཁྲོད་འདིར་སངས་རྒྱས་ཀྱི་བསྟན་པ་ཇི་ལྟར་རྒྱས་པར་གྱུར་སྙམ་ན། ལྷ་མོ་དྲི་མ་མེད་པས་ཞུས་པ་ལས། གདོང་དམར་ཅན་གྱི་ཡུལ་དུ་སྟོན་པ་མྱ་ངན་ལས་འདས་ནས་ལོ་ཉིས་སྟོང་ལྔ་བརྒྱ་ན་དམ་པའི་ཆོས་རྒྱས་པར་འགྱུར་རོ་ཞེས་གསུངས་སོ། །དེའང་འདི་ལྟར་རྟོགས་པར་བྱ་སྟེ། བོད་ཀྱི་སྐྱེ་བོ་འདི་དག་ནི་ཁ་ཅིག་ན་རེ་རྒྱལ་པོ་སྐྱ་བསེང་གི་དམག་མི་སྙིང་རྗེ་ཆེ་བ་རྣམས་རི་བྲག་ཏུ་བྲོས་པར་གྱུར་པ་ལས་འཕེལ་ལོ་ཞེས་ཟེར་རོ། ཁ་ཅིག་ན་རེ་སྤྲེའུ་བྱང་ཆུབ་སེམས་དཔའ་ཞིག་ལ་བྲག་སྲིན་མོ་ཞིག་གིས་འདོད་པ་བསྟེན་པར་གསོལ་བ་བཏབ་པ་ལས་འཕེལ་ལོ་ཞེས་ཟེར་ཏེ། གང་ལྟར་ནའང་སྙིང་རྗེ་དང་ལྡན་པ་ཐེག་པ་ཆེན་པོའི་རིགས་ཅན་ནོ། །དེ་ལྟར་འཕེལ་བ་ལས་དུས་རེ་ཞིག་ན་རྒྱལ་པོ་བྱང་ཆུབ་སེམས་དཔའི་རིགས་ཅན་སྐལ་བ་དང་ལྡན་པ་ལྷ་ཐོ་ཐོ་རི་ཞེས་བྱ་བ་འཁོར་དང་བཅས་པ་སྨིན་པར་བྱ་བའི་ཕྱིར་སངས་རྒྱས་དང་བྱང་ཆུབ་སེམས་དཔའ་རྣམས་ཀྱིས་ཐུགས་རྗེའི་བྱིན་གྱི་རླབས་ཀྱིས་སྤྲུལ་ཏེ། གསེར་གྱི་མཆོད་རྟེན་ཁྲུ་གང་པ་ཞིག་དང༌། ( ¡¶ ཞ ¡·¡¶ ལུ ¡· + ཡི་གེ་དྲུག་པ་དང༌། ) པང་ཀོང་ཕྱག་རྒྱ་པ་ཞིག་ནམ་མཁའ་ལས་རྒྱལ་པོའི་ཁབ་ཀྱི་སྟེང་དུ་བབས་སོ། །དེ་སྡུག་ཅིང་མཛེས་པས་གཉེན་པོ་གསང་བ་ཞེས་བཏགས་ནས་གསེར་སྐྱེམས་ཀྱི་མཆོད་པས་བཀྲ་ཤིས་པར་གྱུར་ཏོ། །རྒྱལ་པོའི་མནལ་ལམ་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· ལམ་ ) དུ་ད་སྟེ་གདུང་རབས་བདུན་ན་འདི་ཤེས་པར་འགྱུར་རོ་ཞེས་བྱ་བ་ཡང་བྱུང་ངོ༌། །དེ་ནས་གདུང་རབས་བདུན་ནས་འཕགས་པ་སྤྱན་རས་གཟིགས་ཀྱི་རྣམ་འཕྲུལ་གྱིས་རྒྱལ་པོ་སྲོང་བཙན་སྒམ་པོ་ཞེས་བྱ་བར་སྤྲུལ་ཏེ་བྱོན་ནས།

དགེ་བ་བཅུ་ལས་བརྩམས་ཏེ་ཁྲིམས་བཅས། སྒྱུར་བྱེད་དམ་པ་སམྦྷོ་ཊ་ཞེས་བྱ་བས་བོད་ཀྱི་སྐད་དང་མཐུན་པར་ཡི་གེའང་བྱས་ཏེ། མདོ་དཀོན་མཆོག་སྤྲིན་ལ་སོགས་པ་འགའ་ཞིག་ཀྱང་བསྒྱུར་རོ། །རྒྱལ་པོའི་བཙུན་མོ་ཀོང་ཇོ་དང་ཁྲི་བཙུན་གྱིས་ཀྱང་མཆོད་གནས་དེ་བཞིན་གཤེགས་པའི་སྐུ་གཟུགས་སྤྱན་དྲངས་སོ། །རྗེ་འབངས་རྣམས་ཀྱིས་གཙུག་ལག་ཁང་དུ་མ་ཡང་བརྩིགས་ཏེ་དམ་པའི་ཆོས་ཀྱི་སྲོལ་བཏོད་ ( ¡¶ ལུ ¡· གཏོད་ ) དོ། །དེ་ནས་གདུང་རབས་ལྔར་ཡང་ཀ་ཆུ་ལ་སོགས་པའི་ལྷ་ཁང་བཞེངས་ཏེ་ཆོས་མ་ནུབ་པར་བྱས་སོ། །དེ་ནས་ཆོས་ཀྱི་རྒྱལ་པོ་ཁྲི་སྲོང་ལྡེ་བཙན་ཞེས་བྱ་བའི་སྐུ་རིང་ལ། ཟ་ཧོར་གྱི་མཁན་པོ་བོ་དྷི་ས་ཏྭ་དང་ ༑ ཨུརྱན་གྱི་མཁན་པོ་པདྨ་སཾ་བྷ་ཝ་ལ་སོགས་པ་སྤྱན་དྲངས་ནས་བོད་ཀྱི་ལྷ་སྲིན་གདུག་པ་ཅན་རྣམས་བཏུལ། སྒྱུར་བྱེད་དམ་པ་བཻ་རོ་ཙ་ན་རཀྵི་ཏ་དང༌། ལྕེ་ཁྱི་འབྲུག་ལ་སོགས་པས་བཀའ་དང་བསྟན་བཅོས་དུ་མ་བསྒྱུར། དཔལ་བྲག་དམར་ལ་སོགས་པ་གཙུག་ལག་ཁང་བརྒྱ་རྩ་བརྒྱད་ཀྱང་བརྩིགས་ཏེ་དམ་པའི་ཆོས་རྒྱས་པར་མཛད་དོ། །དེའི་སྲས་ཁྲི་ལྡེ་སྲོང་བཙན་ཞེས་བྱ་བའི་སྐུ་རིང་ལ་སྐར་ཆུ་ ( ¡¶ ཞ ¡· ཆུང་ ) ལ་སོགས་པའི་གཙུག་ལག་ཁང་ཀྱང་བཞེངས། རྒྱ་གར་གྱྀ་མཁན་པོ་ཛྙཱ་ན་སེ་ན་ལ་སོགས་པ་སྤྱན་དྲངས། སྒྱུར་བྱེད་དམ་པ་ཡེ་ཤེས་སྡེ་ལ་སོགས་པས་སྔར་མ་བསྒྱུར་བ་རྣམས་ཀྱང་བསྒྱུར། བསྒྱུར་བ་རྣམས་ཀྱང་སྐད་གསར་བཅད་ཀྱིས་དག་པར་བྱས་ཏེ་རྒྱས་པར་མཛད་དོ། །དེའི་སྲས་ཁྲི་གཙུག་ལྡེ་བཙན་གྱི་སྐུ་རིང་ ¡Ã ལ་འོན་ཅང་རྡོ་ ( ¡¶ ཞ ¡· འོན་འཇང་ ¡¶ ལུ ¡· ལའང་འུ་ཤང་ ) ལ་སོགས་པའི་གཙུག་ལག་ཁང་སྟོང་བཞེངས། འཛམ་བུའི་གླིང་སུམ་གཉིས་ལ་མངའ་མཛད། རྒྱ་གར་གྱི་མཁན་པོ་ཛི་ན་མི་ཏྲ་ལ་སོགས་པ་སྤྱན་དྲངས། སྒྱུར་བྱེད་དམ་པ་བནྡེ་དཔལ་བརྩེགས་དང་ཀླུའི་རྒྱལ་པོ་རྒྱལ་མཚན་ལ་སོགས་པས་བཀའ་དང་བསྟན་བཅོས་དུ་མ་བསྒྱུར་ཏེ་བསྟན་པ་དར་ཤིང་རྒྱས་པར་མཛད་དོ། །དེ་ཡན་ཆད་བོད་འབངས་སྐྱིད་ཀྱི་ཉི་མ་བདེ་ཞིང་བཀྲ་ཤིས་པའི་དུས་ཡིན་ནོ།།

།།དེའི་མཆེད་ཟླ་རྒྱལ་པོ་གླང་དར་མ་ཞེས་བྱ་བ་ཞིག་བྱུང་སྟེ། དེའི་ཐུགས་སྔོན་མི་དགེ་བ་ལ་གོམས་པའི་སྟོབས་ཀྱིས་སངས་རྒྱས་ཀྱི་བསྟན་པ་ལ་ཞེ་སྡང་བར་གྱུར་ཏེ། དཀོན་མཆོག་གི་དབུ་འཕང་སྨད་དེ་བསྟན་པ་བསྣུབས་སོ། །ད་དུང་ལྷ་ཁང་རྣམས་ཀྱང་བཞུགས་ཤིང་བསྟན་པའི་མེ་རོ་ཅུང་ཟད་ཙམ་ལུས་པ་ལ། ། བྱང་ཆུབ་སེམས་དཔའ་དཔལ་གྱི་རྡོ་རྗེ་ཞེས་བྱ་བས་སྡིག་སྤྱོད་ཀྱི་རྒྱལ་པོ་བཀྲོངས་ནས་ཉིད་ལིའི་ཡུལ་དུ་རྫུ་འཕྲུལ་གྱིས་གཤེགས་སོ། །རྒྱལ་པོ་དེའི་སྲས་ཁྲི་གནམ་ལྡེ་འོད་སྲུངས་ཞེས་བྱ་བ་དང༌། དེའི་མཆེད་ཟླ་ཡུམ་བརྟེན་གྱིས་རྒྱལ་སྲིད་སོ་སོར་བགོས་ཏེ། དེ་གཉིས་ཀྱི་སྐུ་རིང་ལ་མདོ་སྨད་ཀྱི་ཁམས་རྨ་ལུང་རྡོ་རྗེ་བྲག་དན་ཏིག་གི་ཡང་དགོན་ན། གཙང་རབ་གསལ་དང༌། གཡོ་དགེ་འབྱུང་དང༌། དམར་བན་ཤཱཀྱ་མུ་ནེ་ཞེས་བྱ་བའི་དགེ་སློང་གསུམ་བཞུགས་སོ། །དེའི་ཚེ་མདོ་སྨད་དེ་ཉིད་ན་བོན་པོའི་རིགས་སུ་གྱུར་ཀྱང་མོས་པ་སྤྱོད་པའི་བྱང་ཆུབ་སེམས་དཔའ་དགོངས་པ་གསལ་ཞེས་བྱ་བ་ཞིག་བྱུང་སྟེ། དེས་སངས་རྒྱས་ཀྱི་བསྟན་པ་ལ་དད་དེ་སྔར་གྱི་དགེ་སློང་གསུམ་དང༌། རྒྱའི་ཧ་ཤང་ཀེང་ཝང་དང༌། གིམ་ཕག་དང༌། ཌཱ་པོ་དང་བཅས་པ་ལས་བསྙེན་པར་རྫོགས་པར་བྱས་སོ། །དེ་དག་ལ་སོགས་ཏེ་མདོ་སྨད་ཀྱི་ཁམས་སུ་བསྟན་པ་ཅུང་ཟད་དར་བར་གྱུར་ཏོ། །དེ་ནས་རྒྱལ་པོ་འོད་སྲུངས་ཀྱི་སྲས་ཁྲི་དཔལ་འཁོར་བཙན་གྱི་སྐུ་རིང་ལ། སྨན་ལུང་ལ་སོགས་པ་གཙུག་ལག་ཁང་གི་དབེན་ས་བརྒྱད་བཙུགས་སོ། །དེའི་དུས་སུ་གཙང་པ་མི་གཉིས་དང༌། དབུས་པ་མི་གཉིས་ཀྱིས་མདོ་སྨད་དུ་ཕྱིན་ཏེ་སྡོམ་པ་བླངས་སོ། །ལོ་སྟོན་རྡོ་རྗེ་དབང་ཕྱུག་དང༌། ཚོང་གེ་ཤེས་རབ་སེང་གེ་དང༌། སླུ་མེས་ཚུལ་ཁྲིམས་ཤེས་རབ་དང་

༑ འབྲི་རྗེ་ཡེ་ཤེས་ཡོན་ཏན་རྫིང་ཀར་པ་རྣམས་ཏེ། དེ་དག་གི་དབུས་གཙང་དུ་བསྟན་པ་འོངས་སོ། ཁ་ཅིག་ན་རེ་བླ་ཆེན་དགོངས་པ་རབ་གསལ་ལས་བླངས་ཟེར ༑ ཁ་ཅིག་ན་རེ་ཐུལ་པ་ཡེ་ཤེས་རྒྱལ་མཚན་ལས་བླངས་ཟེར། དེའི་ཚེ་མཁན་པོས་སྤྲིངས་པ། ལོ་སྟོན་མཐུ་ཆེས་བསྟན་པ་སྲུངས། །སླུ་མེས་བཙུན་པས་མཁན་པོ་གྱིས། །ཚོང་གེ་བློ་རྣོ་སྟོན་པར་བསྐོ། །འབྲིང་དུ་སྐྱོང་མཁས་གནས་བརྟན་གྱིས།། ཞེས་གདམས་ཀྱང་སྔ་སོར་བྱས་ནས་བསྟན་པ་རྒྱས་པར་དགོངས་ཏེ་སོ་སོར་གཙུག་ལག་ཁང་བརྩིགས་ཏེ་རྒྱསཔར་བྱས་སོ། །རྒྱལ་པོ་དེའི་སྲས་གཉིས་ཏེ། ཁྲི་བཀྲ་ཤིས་བརྩེགས་པ་དཔལ་དང༌། ཁྲི་སྐྱིད་ལྡེ་ཉི་མ་མགོན་ནོ། །སྔ་མས་ནི་གཡོར་གཙང་གི་མངའ་མཛད་དོ། །གཅུང་མངའ་རིས་སྟོད་དུ་གཤེགས་སོ། ། གཅེན་ལ་སྲས་གསུམ་སྟེ། དཔལ་ལྡེ། འོད་ལྡེ། སྐྱིད་ལྡེའོ། །གཅུང་ལ་གསུམ་སྟེ། ལྡེ་གཙུག་མགོན། བཀྲ་ཤིས་མགོན། དཔལ་གྱི་མགོན་ནོ། །འོད་སྲུངས་ནས་བརྩམས་ཏེ། དེ་ཡན་ཆད་ཀྱི་གངས་ཅན་འདི་ན་གཙུག་ལག་ཁང་དང༌། གླེགས་བམ་བཞུགས་མོད། བཤད་པའི་བཀའ་ཉམས་པས་གཟུ་ལུམ་དུ་མ་བྱུང་སྟེ། སངས་རྒྱས་ཀྱི་བསྟན་པའི་གཟུགས་བརྙན་ལ་སྤྱོད་པ་ཡིན་ནོ། །དེ་ནས་འདིར་བསྟན་པ་རྒྱས་ཤིང་དག་པར་བྱ་བའི་ཕྱིར། བྱང་ཆུབ་སེམས་དཔའ་ཐུགས་རྗེ་དང༌། སྨོན་ལམ་གྱི་སྟོབས་ཅན་ཞིག་སའྀ་བདག་པོ་བཀྲ་ཤིས་མགོན་གྱི་སྲས་སུ་སྐུ་བལྟམས་པ་མཚན་ཡང་ཁྲི་ ¡Ã དཔལ་སྲོང་ངེ་ ( ¡¶ ཞ ¡· ལྡེ་ཁྲི་སྲོང་ངེ་ ¡¶ ལུ ¡· ལྡེ་སྲོང་བཙན་ ) ཞེས་བྱའོ། །དེའི་མཆེད་ཀྱང་ཁྲི་དཔལ་འཁོར་ལྡེ་ཞེས་བྱ་སྟེ། དེའང་སྨོན་ལམ་དང་ལྡན་པའོ། །བྱང་ཆུབ་སེམས་དཔའ་དེ་རབ་ཏུ་བྱུང་སྟེ་མཚན་ཡང་ཡེ་ཤེས་འོད་ཅེས་བྱ་བ་ཡིན་ལ། དེ་ནི་སྙིང་རྗེ་ཆེན་པོ་པདྨ་དཀར་པོ་དང༌། འཇམ་དཔལ་གྱི་རྒྱུད་ཆེན་པོ་ནས་ལུང་བསྟན་ཏེ། ཁ་བ་ཅན་དུ་རྒྱལ་པོའི་རིགས། །ཡེ་ཤེས་འོད་ཅེས་བྱ་བ་འབྱུང༌། །ཞེས་པ་དང༌། བྱང་ཕྱོགས་སུ་འོད་ཀྱི་མཐས་བརྒྱན་པ་ཅན་གྱི་དགེ་སློང་འབྱུང་བར་འགྱུར་རོ།

།དེས་ཀྱང་བསྟན་པ་རྒྱས་པར་བྱེད་དོ། །ཞེས་གསུངས་སོ། །དེ་བས་ན་སྔོན་གྱི་སྨོན་ལམ་དང༌། ཐུགས་རྗེའི་དབང་གིས་ནི་ལྷ་རིགས་བྱང་ཆུབ་སེམས་དཔའི་སྤྲུལ་པ་སྟེ། བླ་མ་བྱང་ཆུབ་སེམས་དཔའ་ཡེ་ཤེས་འོད ༑༑ མངའ་རིས་སྟོད་འདིར་གདུལ་བྱའི་དཽན་དུ་བྱུང༌། །ཞེས་བྱའོ། །དེས་དཔལ་ཐོ་ལིང་གི་གཙུག་ལག་ཁང་བཞེངས། བནྡེ་རིན་ཆེན་བཟང་པོ་ལ་སོགས་པ། བློ་གསལ་བའི་ཁྱེའུ་ཉི་ཤུ་རྩ་གཅིག་ལེགས་པར་སྦྱར་བའི་སྒྲ་སྐད་སློབ་ཏུ་བཅུག་ནས། རྒྱ་གར་གྱི་མཁན་པོ་ཤྲདྡྷཱ་ཀ་ར་ཝརྨ་ལ་སོགས་པ་དང༌། རིན་ཆེན་བཟང་པོ་དཔོན་དང་གཡོག་ཏུ་བཅས་པ་སྤྱན་དྲངས་ནས། སྔར་ཆོས་མ་བསྒྱུར་བ་ ( ¡¶ ཞ ¡· + རྣམས་བསྒྱུར་བ་ ) དང༌། བསྒྱུར་བ་རྣམས་ཞུས་དག་བྱས། བསྟན་པ་རྒྱས་པར་མཛད་དོ། ། དེའི་མཆེད་ཀྱི་རྒྱལ་སྲིད་བཟུང་སྟེ། དེས་ཀྱང་ཁ་ཆར་གྱི་གཙུག་ལག་ཁང་བཙུགས་སོ ༑ ༑ དེའི་སྲས་རྒྱལ་པོ་ལྷ་སྡེས་ཀྱང་དམ་པའི་ཆོས་དུ་མ་བསྒྱུར་བའི་ཡོན་བདག་མཛད་དོ། །དེའི་སྲས་བཙུན་པ་བྱང་ཆུབ་འོད་ཏེ། དེས་ཀྱང་རྒྱ་གར་གྱི་མཁན་པོ་དཱི་པཾ་ཀ་ར་ལ་སོགས་པ་སྤྱན་དྲངས་ནས། སྒྱུར་བྱེད་རིན་ཆེན་བཟང་པོའི་སྐུ་ཚེའི་སྨད་དེ་དགེ་བའི་བློ་གྲོས་དང༌། བཤེས་གཉེན་བཟང་པོ་ལ་སོགས་པས་དམ་པའི་ཆོས་བསྒྱུར་ཅིང་བཤད་དོ། །དེའི་ཚེ་རྒྱལ་སྲིད་སྐྱོང་བ་རྒྱལ་པོ་འོད་ལྡེས་མཛད་དེ། མཁན་པོ་ཛྙཱ་ན་ཤྲཱི་ལ་སོགས་པ་སྤྱན་དྲངས་སོ། །དེའི་དུས་སུ་རུ་བཞིའི་ས་འདིར། ཆོས་སྐྱོང་བའི་རྒྱལ་པོའི་བཀས་བསྐུལ་བ་མ་ཡིན་པར། སྔོན་གྱི་སྨོན་ལམ་གྱི་སྟོབས་ཀྱིས་རྒྱ་གར་གྱི་མཁན་པོ་སྨྲྀ་ཏི་ཛྙཱ་ན་ལ་སོགས་པ་དང༌། སྒྱུར་བྱེད་དམ་པ་ཏིང་ངེ་འཛིན་བཟང་པོ་དང༌། ཤཱཀྱ་ཡེ་ཤེས་དང༌། ལྷས་བཙས་ལ་སོགས་པ་དུ་མ་བྱུང་སྟེ། བསྟན་པ་དར་ཞིང་རྒྱས་པར་མཛད་དོ། །དེའི་སྲས་རྒྱལ་པོ་ཙེ་ལྡེའི་སྐུ་རིང་ལ་ནི། རྒྱ་གར་གྱི་མཁན་པོ་ཆེན་པོ་པུ་ནྱ་ཤྲཱི་དང༌། ཞི་བ་བཟང་པོ་ལ་སོགས་པ་སྤྱན་དྲངས་སོ། །དེའི་ཚེ་ལོ་ཙཱ་བ་དགའ་རབ་རྡོ་རྗེ་དང༌།

ཆོས་བརྩོན་དང༌། བློ་ལྡན་ཤེས་རབ་ལ་སོགས་པ་བརྒྱ་ཐམ་པ་ཡང་བྱུང་སྟེ་དག་པར་བྱས་པ་ཡིན་ནོ། ། དེ་ལྟར་ཁ་བ་ཅན་གྱི་ཁྲོད་འདིར་ཡང༌། རྒྱལ་བློན་བྱང་ཆུབ་སེམས་དཔའ་རྣམས་དང་ ༑ པཎྜི་ཏ་དང༌། ལོ་ཙཱ་བ་སྐྱེས་ཆེན་ཁྱད་པར་དུ་འཕགས་པ་རྣམས་ཀྱིས་ལེགས་པར་བྱས་པ་ཡིན་པས། གཟུ་ལུམ་དང༌། རང་དགའི་ཆོས་མ་ཡིན་ནོ། ། རྣམ་པར་སྤྲོས་པས་ཆོག་གོ། །།ད་ནི་ཞར་ལས་འོངས་པ་འདི་ཡང་བརྗོད་པར་བྱ་སྟེ། སངས་རྒྱས་མྱ་ངན་ལས་འདས་ནས་དམ་པའི་ཆོས་ལོ་དུར་གནས། དེང་སང་ཡན་ཆད་ལོ་དུ་སོང་སྙམ་ན། འཛམ་བུའི་གླིང་འདིར་སྟོན་པའི་བསྟན་པ་ལོ་ལྔ་སྟོང་ཞིག་གནས་ཏེ། དེ་ལ་ལོ་ལྔ་བརྒྱ་པ་ཕྲག་བཅུར་བྱས་ཏེ། ལྔ་བརྒྱ་པ་དང་པོ་ལ་དགྲ་བཅོམ་པའི་འབྲས་བུ་ཐོབ་པ་མང༌། གཉིས་པ་ལ་ཕྱིར་མི་འོངས་པ་ཐོབ་པ་མང༌། གསུམ་པ་ལ་རྒྱུན་དུ་ཞུགས་པའི་འབྲས་བུ་ཐོབ་པ་མང་སྟེ། འབྲས་བུའི་དུས་གསུམ་མོ། །བཞི་པ་ལ་ལྷག་མཐོང་སྒོམ་པ་མང༌། ལྔ་པ་ལ་ཏིང་ངེ་འཛིན་སྒོམ་པ་མང༌། དྲུག་པ་ལ་ཚུལ་ཁྲིམས་ལ་གནས་པ་མང་སྟེ། སྒྲུབ་པའི་དུས་གསུམ་མོ། །བདུན་པ་ལ་མངོན་པ་ལ་སྤྱོད་པ་མང༌། བརྒྱད་པ་ལ་མདོ་སྡེ་ལ་སྤྱོད་པ་མང༌། དགུ་པ་ལ་འདུལ་བ་ལ་སྤྱོད་པ་མང་སྟེ། ལུང་གི་དུས་གསུམ་མོ། །བསྟན་པ་གཤའ་མའོ། །བཅུ་པ་ལ་ནི་རྟགས་དང་མཚན་མ་ལ་བརྟེན་པས་ཏེ། འདི་ལ་ནི་ལྔ་བརྒྱ་ཐ་མའོ་ཞེས་འབྱུང་ངོ༌།། འདི་འདས་ནས་འཛམ་བུའི་གླིང་དུ་བསྟན་པ་མི་གནས་སོ། །སྔར་ལོ་དུ་ལོན་པ་ཡང་འདི་ལྟར་བརྩི་བར་བྱ་སྟེ། སྤྱིར་རྒྱ་གར་གྱི་ལུགས་ལ་བོད་དང་རྒྱ་ནག་བཞིན་ལོ་སྐོར་བཅུ་གཉིས་སུ་བརྩི་བ་ནི་མེད་དེ། ལུགས་ཀྱི་བསྟན་བཅོས་ལས་རྒྱལ་པོ་རེས་ལོ་ཚིགས་བཅོས་ནས། གཅིག་གཉིས་ཞེས་བྱ་བ་ལ་སོགས་པའི་བར་ལ་ལོ་ཚིགས་འདི་མ་བཅོས་ཀྱི་བར་དུ་བརྩིའོ།། །།ཁ་ཅིག་ལ་ལོ་དྲུག་ཅུར་བརྩི་བ་ཡོད་དེ། དེ་ཐུན་མོང་ལ་གྲགས་པ་མ་ཡིན་ནོ ༑༑ ལོ་ཚིགས་བཅོས་ཞེས་བྱ་བ་རྒྱལ་པོ་བསོད་ནམས་དང༌། གྲགས་པ་འདོད་པ།

གཏོང་ཕོད་ཆེ་བ་ཞིག་གིས་མངའ་རིས་སུ་གཏོགས་སོ་ཅོག་གི་བུ་ལོན་ཐམས་ཅད་རྒྱལ་པོའི་དཀོར་མཛོད་ཀྱིས་བཅལ་ནས་རིན་མེད་པར་བྱས་ཏེ། སྔར་གྱི་ཟོང་དུ་སྤྱོད་པ་རྣམས་བོར་ནས། ཕྱིས་སྤྱོད་པ་རྣམས་གསར་དུ་བྱས་ཏེ། བདག་གིས་མིང་བཏགས་ནས་སྤྱོད་དོ། །དེ་ནས་བཟུང་སྟེ་ཡང་བརྩི་བ་ཡིན་ནོ། །འདི་ཡི་གཏམ་ནི་ལུགས་ཆེན་པོ་ཞེས་བྱ་བ་ལས། གཡུལ་བརྟན་འོག་ཏུ་ཉིས་སྟོང་ལོན། །དགའ་བྱེད་འོག་ཏུ་བརྒྱད་བརྒྱ་སྟེ། །ཟླ་སྲུང་ཉིས་བརྒྱ་སུམ་ཅུ་གཅིག །རྗེད་མངས་བདུན་བརྒྱ་ཉི་ཤུ་བཞི། །བལ་གནས་བརྒྱད་བརྒྱ་བཅུ་རྩ་བཞི། །གོ་ཆ་ཉིས་བརྒྱ་བཞི་བཅུ་གཉིས། །ཞེས་འབྱུང་སྟེ། ཡུལ་དབུས་ན་སྒྲ་ངན་གྱི་སྲས་ལྔ་བརྒྱའི་གཅེན་འཐབ་པར་དཀའ་བ་ཞེས་བྱ་བ་བལྟམས་པའི་དུས་སུ་རྩོད་དུས་ཀྱི་མགོ་བཟུང་ངོ༌། །དེའི་དགྲ་ཟླ་རྒྱལ་པོ་སྐྱ་བསེང་གི་སྲས་ལྔའི་གཅེན་གཡུལ་ངོར་བརྟན་པ་ཞེས་བྱ་བས་ལོ་བཅོས་ནས་ལོ་ཉིས་སྟོང་དུ་ཉེ་བའི་ཚེ་སངས་རྒྱས་ཤཱཀྱ་ཐུབ་པ་བྱོན་ནོ། །དེ་བཞིན་གཤེགས་པའི་ལོ་ནི་རྒྱའི་ལུགས་བརྩི་བས་བྱི་བའི་ལོ་ཡིན་ལ། བརྒྱད་ཅུ་རྩ་གཉིས་བྱའི་ལོ་ལ་འདས་སོ། །ཞེས་ཟེར་ཏེ་འདི་ནི་མ་དག་པ་ཡིན་ནོ།། འོ་ན་ཇི་ལྟར་བྱ་ཞེ་ན། རྒྱ་གར་གྱི་ལོ་དྲུག་ཅུའི་མིང་དུ་བྱས་ན་རབ་ཏུ་སྐྱེ་བ་ལ་ལྷུམས་སུ་ཞུགས་ནས་འབྱོར་པ་ལ་བལྟམས་སོ། །རྒྱ་དང་བོད་ཀྱི་ལོའི་མིང་ནི་མེ་མོ་ཡོས་ལ་ལྷུམས་སུ་ཞུགས་ནས། ས་ཕོ་འབྲུག་ལ་སྐུ་བལྟམས་སོ། །ཆུ་ཕོ་སྟག་ལ་བྱང་ཆུབ་ནས་མེ་མོ་ཕག་གི་ལོ་རྫོགས་ནས། ས་ཕོ་བྱི་བའི་ལོ་མགོ་རྩོམ་པའི་བར་མྱ་ངན་ལས་འདས་སོ། །དེ་ནས་ས་ཕོ་བྱི་བ་ནས་བརྩམས་ནས། བརྒྱ་སུམ་ཅུ་རྩ་བདུན་ན་རྒྱལ་པོ་དགའ་བྱེད་བྱུང་ངོ༌། །ཞེས་རྟོག་གེ་འབར་བ་ལས་འབྱུང་ངོ༌། ། རྒྱལ་པོ་དེས་ལོ་བཅོས་ནས་ལོ་བརྒྱད་བརྒྱ་ན་རྒྱལ་པོ་ཟླ་སྲུང་བྱུང་ངོ༌། །དེས་ལོ་བཅོས་ནས་ཉིས་བརྒྱ་སུམ་ཅུ་རྩ་གཅིག་ན་རྒྱལ་པོ་རྗེད་མངས་བྱུང་ངོ༌། །དེས་ལོ་བཅོས་ནས་བདུན་བརྒྱ་ཉི་ཤུ་བཞི་ལོན་པ་ན།

ལོ་དེའི་འཕྲོ་ནས་བཟུང་སྟེ་བལ་པོས་ཀྱང་བརྩིས་ཏེ། དེ་ནས་ལོ་ ¡Ã བརྒྱད་བརྒྱ་ ( ¡¶ ཞ ¡· - བརྒྱད་བརྒྱ་ ) ¡Ã སུམ་ཅུ་ ( ¡¶ ཞ ¡· བརྒྱ་བརྒྱད་ཅུ་རྩ་ ¡¶ ལུ ¡· བཅུ་ ) བཞི་ལོན་པ་ན་རྒྱལ་པོ་འོད་ཟེར་གོ་ཆ་བྱུང་ངོ༌། །དེས་ལོ་བཅོས་ནས་ ¡Ã ཉིས་བརྒྱ་བཞི་བཅུ་ ( ¡¶ ལུ ¡· བདུན་བརྒྱ་ཉི་ཤུ་ ) རྩ་གཉིས་ལོན་པ་ན། བོད་ཀྱི་རྒྱལ་པོ་ཁྲི་གཙུག་ལྡེ་བཙན་གྱི་སྐུ་རིང་ལ་བརྩིས་ཏེ། བྲན་བློན་ཆེན་པོ་ཞང་ཁྲི་གསུམ་རྗེ་སྟག་སྙ་ལ་སོགས་པ་བརྒྱ་དྲུག་སྤ་བཀོང་ནས་མཇལ་དུམ་ཆུས་གསོན་ཏེ། ཆུ་ཕོ་སྟག་གི་ལོ་ལ་རྒྱ་དང་མཇལ་དུམ་བགྱིས། ཆུ་མོ་ཡོས་བུའི་ལོ་ལ་ཧོར་དང་མཇལ་དུམ་བགྱིས་ཏེ། ཆུ་ཕོ་སྟག་ལོ་འདི་ཕན་ཆད་སངས་རྒྱས་མྱ་ངན་ལས་འདས་ཚུན་ཆད་སྤྱིར་སྡོམ་ན་ལོ་ཉིས་སྟོང་དགུ་བརྒྱ་ལྔ་བཅུ་རྩ་ལྔ་ལོན་ཏེ། སླན་ཆད་ཀྱང་དེ་བཞིན་དུ་བརྩིའོ། །མེ་ཕོ་འབྲུག་གི་ལོ་ལ་བྲུ་ཞ་ཡུལ་གྱི་མདུན་ས་ཤིབ་པེ་ཅོག་ལ་ཚལ་དུ། སྟོད་ཀྱི་ཆོས་ཀྱི་གཞི་འཛིན་པ་བནྡེ་ཆོས་ཀྱི་བློ་གྲོས་དང༌། ཞང་སྐྱིད་སུམ་རྗེ་ལ་སོགས་པས་བསྡུས་པའི་དུས་སུ་བརྩིས་ན། མྱ་ངན་ལས་འདས་ནས་ལོ་ཉིས་སྟོང་དགུ་བརྒྱ་དྲུག་ཅུ་རྩ་དགུ་ལོན་ནོ། །ཤིང་མོ་གླང་གི་ལོ་ལ་གཙང་དང་གཡོན་རུའི་བཙད་པོ་ཁྲི་དཔལ་འཁོར་བཙན་གཙང་གྲོམ་པ་ལྷ་རྩེ་ན་བཞུགས་ནས་ཡབ་ཀྱིས་མདད་ ( ¡¶ ལུ ¡· ལྷ་ ) མཛད་པའི་དུས་སུ་བརྩིས་ན། ལོ་སུམ་སྟོང་སུམ་ཅུ་རྩ་བརྒྱད་ལོན་ནོ། །དེའི་དུས་སུ་རྒྱལ་པོ་མགོན་སྤྱན་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· སྐྱོབས་ ) འཕན་ཡུལ་བྲག་མཁར་ན་བཞུགས་སོ། །ཆུ་ཕོ་འབྲུག་གི་ལོ་ལ་ཅོག་ལ་ཡུལ་སྦེག་ ( ¡¶ ལུ ¡· བསྲེག་ ) མཁར་དུ་སྟོད་སྨད་ཀྱི་ཡབ་མཆེད་ ¡Ã བྱིང་ ( ¡¶ ཞ ¡· བྱོན་ ཏེ་ ) གདན་འཛོམ་པའི་དུས་སུ་མོལ་བ་ཆེན་པོ་མཛད། རུམ་ཡུལ་པ་སྒམ་གྱི་དབེན་ས་གསར་དུ་བཙུགས་པའི་དུས་སུ་བརྩིས་ན་ལོ་སུམ་སྟོང་ཆིག་བརྒྱ་ཉི་ཤུ་རྩ་ལྔ་ལོན་ནོ། །མེ་ཕོ་འབྲུག་གི་ལོ་ལ་ལྷ་བླ་མ་ཡེ་ཤེས་འོད་ཡབ་སྲས་དབེན་གནས་པ་སྒམ་བྱམས་སྙོམས་གླིང་དུ་གདན་འཛོམ་མོ།

།ལྷ་སྲས་གཅུང་ལྷ་འཁོར་རབ་ཏུ་གཤེགས ༑ གུ་གེའི་ཡུལ་ན་གནས་པའི་འབངས་ཞང་ཞུང་ཞིག་གིས་གཅེན་གྱི་འཁོས་ཆེན་པོ་མཛད་པའི་དུས་སུ་བརྩིས་པ་ན། སངས་རྒྱས་མྱ་ངན་ལས་འདས་ནས་ལོ་སུམ་སྟོང་ཆིག་བརྒྱ་བཞི་བཅུ་ཞེ་དགུ་ལོན་ཏེ་སླན་ཆད་ཀྱང་འདི་བཞིན་དུ་བརྩིའོ ༑ ༑ དེ་ལྟར་བརྩིས་པ་ལས་ལོ་བཞི་བསྐྱེད་པ་ནི་རྒྱ་གར་གྱི་རྒྱལ་པོ་རྣམས་ཀྱིས་བརྩིས་པ་དང༌། བཙད་པོ་ཁྲི་གཙུག་ལྡེ་བཙན་གྱིས་བརྩིས་པ་ནི་མཐུན། མར་ཡུལ་དུ་བནྡེ་ཆོས་ཀྱི་བློ་གྲོས་ཀྱིས་བརྩིས་པ་མན་ཆད་མ་མཐུན་པས་གཞུང་སྔ་མ་བཙན་པར་བགྱིས་སོ། །དེ་ནས་སྟོད་དུ་མངའ་བདག་འོད་ལྡེ་བཙན་གྱིས། བཙན་པོ་བཀྲ་ཤིས་ཁྲི་ལྡེ་བཙན་ཕྱག་ནས་སྤྱན་དྲངས་པའི་ཚེ། སྟོད་ཀྱི་ཆོས་ཉན་པའི་ར་ས་འཛིམས་སུ་བརྩིས་པས་ལོ་སུམ་སྟོང་ཆིག་བརྒྱ་དགུ་བཅུ་ཐམ་པ་ལོན་ནོ ༑ ༑ དེ་ནས་ས་མོ་ཕག་གི་ལོ་ལ་ལོ་ཙཱ་བ་ཆེན་པོ་བློ་ལྡན་ཤེས་རབ་འདས་ཏེ། ལྕགས་ཕོ་སྟག་གི་ལོ་ལ་འདུ་བ་ཆེན་པོ་བྱས་པའི་དུས་སུ་བརྩིས་ན་ལོ་སུམསྟོང་ཉིས་བརྒྱ་བཞི་བཅུ་རྩ་གསུམ་ ( ¡¶ ཞ ¡· དྲུག་ ) ལོན་ནོ། །མེ་མོ་ཕག་གི་ལོའི་མཇུག་ཏུ། ན་ལ་རྩེ་གནས་པོ་ཆེའི་གཙུག་ལག་ཁང་དུ་ཆོས་ལ་འཇུག་པའི་སྒོ་འདི་བྲིས་ཏེ། དེའི་དུས་སུ་བརྩིས་པ་ན་སངས་རྒྱས་མྱ་ངན་ལས་འདས་ནས་ཕག་གི་ལོ་འདི་ཡན་ཆོད་ལོ་སུམ་སྟོང་སུམ་བརྒྱ་ཐམ་པ་ལོན་ནོ། ཁ་ཅིག་ལ་འདི་ལས་ལོ་གསུམ་གྱིས་མང་བ་ཡོད་དེ། དེ་ནི་ལོའི་མིང་དང་མི་མཐུན་པས་འདི་བཞིན་བརྩིའོ། །དེང་སང་ནི་སངས་རྒྱས་ཀྱི་བསྟན་པ་ཆོས་མངོན་པའི་མཇུག་ཏུ་ཉེ་བ་ཡིན་ཏེ། ད་ནི་སངས་རྒྱས་ཀྱི་བསྟན་པ་ལོ་སྟོང་ལྷག་པ་ཙམ་དུ་རྣལ་མར་གནས་སོ། །ཞར་ལས་འོངས་པ་རྫོགས་སོ།། །།འདིར་སྨྲས་པ། བརྩེ་ཆེན་ལྡན་པ་སྟོན་པས་ལེགས་གསུངས་དགྲ་བཅོམ་དང་ ༑ བློ་གསལ་རྣམ་བཤད་མཛད་དང་ས་བདག་སྒྱུར་བྱེད་དང༌། ལེགས་བཤད་ལ་གོམས་སྐྱེ་བོ་མཁས་གཞན་དག་གིས་ཀྱང༌། །བསྟན་པ་དྲི་མ་མེད་པ་ཉི་བཞིན་གསལ་བར་བྱས།

༑ ༑ པ་ཕ་གཉིས་ལ་ཡ་རའོ། །བ་ལ་ཡ་ར་ལ་གསུམ་སྟེ། །མ་ལ་ཡ་ར་ཟ་ལ་ལ ༑ ༑ ར་ལ་ཀ་ག་ང་ཇ་ཉ། །ཏ་ད་ན་བ་མ་ཙ་ཛ། །ལ་དང་དེ་ལྟར་བཅུ་བཞི་གདགས། །ལ་ལ་ཀ་ག་ང་ཅ་ཇ། །ཏ་ད་པ་བ་ཧ་བཅུ་གདགས། །ས་ལ་ཀ་ག་ང་ཉ་ཏ ༑ ༑ ད་ན་པ་བ་མ་ཙ་ར། །ལ་སྟེ་དེ་ལྟར་བཅུ་གསུམ་གདགས། །ཧ་ལ་ར་ནི་གདགས་པའོ། །ཝ་ནི་ཀུན་ལ་གདགས་རུང་ཡང༌། །སྐབས་ཀྱི་བཅོ་ལྔ་དག་ལ་འཇུག ༑ ཝ་བཏགས་གུག་སྐྱེད་མི་འཇུག་པས། །དེས་ན་ཝ་ནི་ཡན་ལག་ཡིན། ། བོད་ལ་ཝ་ནི་འར་བསྡུས་པས། །དེས་ན་མ་ནིང་བརྗོད་ཀྱང་རུང༌། །དེ་དག་གདགས་ཡིག་ཉིས་བརྩེགས་ཏེ། །གསུམ་བཏགས་པ་ཡང་བརྗོད་པར་བྱ། །རྐ་རྒ་རྨ་ལ་ཡ་ཡིན་ཏེ། །སྐ་སྒ་སྤ་སྦ་སྨ་ལྔ་ལ། །ཡ་ར་གཉིས་གཉིས་གདགས་པར་བྱ། ། སྒྲ་ཞེས་སྐབས་འགའ་དག་ཏུ་འབྱུང༌། །དེ་ལྟར་ཉིས་བརྩེགས་ལྔ་བཅུ་བརྒྱད། ། སུམ་བརྩེགས་ཡི་གེ་བཅུ་བཞིའོ། །མདོར་ན་ཕ་མ་སུམ་ཅུ་ལ། །ཆ་ཐ་ཚ་ཞ་འ་ཤ་ཨ ༑ ༑ ཡི་གེ་བདུན་ལ་བརྩེགས་པ་མེད། །དེ་ལྟར་བརྩེགས་པ་བདུན་ཅུ་གཉིས། ། ཝ་ཡི་དོན་ནི་སྔར་བཤད་པས། །དེ་ནི་བརྩེགས་པར་ཡང་མི་བགྲང༌།། །།ད་ནི་འཕུལ་བ་བརྗོད་པར་བྱ། །ཡི་གེའི་ཕ་མ་སུམ་ཅུ་ལ། །ག་ད་བ་མ་འ་ཞེས་བྱ། །འཕུལ་བར་བྱེད་པའི་ཡི་གེ་ལྔ། །འ་ཧ་ཨ་གསུམ་འཕུལ་བ་མེད། །ཝ་ནི་ཡོད་དང་མེད་པའང་ཡིན། །ར་ལ་གཉིས་ནི་ཕྱེད་འཕུལ་ཏེ། །རྐྱང་པ་མི་འཕུལ་བརྩེགས་པ་འཕུལ། །དེ་ལ་ཡི་གེ་གས་འཕུལ་བ། །ཅ་ཉ་ཏ་ད་ན་ཙ་ཞ། །ཟ་ཡ་ཤ་ས་བཅུ་གཅིག་གོ །ཀ་ག་ང་པ་བ་མ་དྲུག །ད་ཡིས་འཕུལ་བར་བྱེད་པ་ཡིན། །ཀ་ག་ཅ་ཏ་ད་ཙ་ཞ། །ཟ་ཤ་ས་སྟེ་དེ་ལྟར་བཅུ། །ཡི་གེ་བ་ཡིས་འཕུལ་བ་ཡིན། ཁ་ག་ང་ཆ་ཇ་ཉ་ཐ། །ད་ན་ཚ་ཛ་བཅུ་གཅིག་ནི། །མ་ཡིས་འཕུལ་བར་བྱེད་པ་ཡིན། །ག་ཁ་ཆ་ང་ཐ་ད་ཕ། །བ་ཚ་ཛ་སྟེ་བཅུ་པོ་ནི། །འ་ཡིས་འཕུལ་བར་བྱེད་པ་ཡིན། །རྐྱང་པ་འཕུལ་བ་འ་ཡི་མཐའ། །དགོས་ཀྱི་གུག་སྐྱེད་ཅན་ལ་མིན། །དེ་ལྟར་རྐྱང་པ་འཕུལ་བ་ཡི། །ཡི་གེ་བཞི་བཅུ་ཞེ་བརྒྱད་ཡོད་དོ།། །།ད་ནི་བརྩེགས་པའི་འཕུལ་བ་བརྗོད། །གས་འཕུལ་བ་ལ་བརྩེགས་པ་མེད

༑༑ དས་འཕུལ་ཀྱ་ཀྲ་གྱ་གྲ་དང༌། །པྲ་པྱ་བྲ་བྱ་མྲ་མྱ་བཅུ། །བ་ཡིས་འཕུལ་བའི་ཡི་གེ་ལ། །ཀྱ་ཀྲ་ཀླ་གྱ་ཟླ་གླ་གྲ། །རྐ་རྒ་རྔ་རྗ་རྙ་རྟ་རྡ། །རྣ་རྩ་རྫ་རླ་ལྟ་ལྡ་སྐ། །སྒ་སྔ་སྙ་སྟ་སྡ་སྣ་སྩ། །སྲ་སླ་དེ་ལྟར་སུམ་ཅུ་དང༌། །རྐྱ་རྒྱ་སྐྱ་སྐྲ་སྒྱ་སྒྲ་དྲུག །སུམ་བརྩེགས་བས་འཕུལ་སུམ་ཅུ་དྲུག །མས་འཕུལ་ཁྱ་ཁྲ་གྱ་གྲ་བཞི། །འགྲོན་ཞེས་བྱ་བ་སྲིད་པ་ཙམ། །འས་འཕུལ་བ་ལ་ཁྱ་ཁྲ་གྱ། །གྲ་ཏྲ་ཕྲ་ཕྱ་བྱ་བྲ་དགུ། །དེ་ལྟར་བརྩེགས་འཕུལ་དྲུག་ཅུ་སྟེ། །འཕུལ་བའི་ཡི་གེ་བརྒྱ་རྩ་བརྒྱད། །བ་གླ་མ་དྲ་གཉིས་ན་དགོས་ཆུང་བས། །བརྒྱ་རྩ་དྲུག་ཡིན་མདོར་བསྡུ་ན། །ལུས་ཀྱི་ཡི་གེ་ཉིས་བརྒྱ་དང༌། །བརྒྱད་ཀྱིས་ལྷག་པར་རབ་ཏུ་ལྡན། །དེ་ལ་ཡན་ལག་བཞི་ལྡན་པས ༑ ༑ ཆིག་སྟོང་དང་ནི་བཞི་བཅུར་འགྱུར། །ག་ང་ད་ན་བ་ལ་ར། །ལ་ས་ཞེས་བྱའི་ཡི་གེ་དགུ། །མཐའ་རྟེན་ཀུན་ལ་རབ་འཇུག་པས། །དགུ་སྟོང་སུམ་བརྒྱ་དྲུག་ཅུར་འགྱུར། །ས་ནི་ག་ང་བ་མ་སྟེ། །བཞི་ལ་མཐའ་རྟེན་གཉིས་པར་འཇུག །དེ་ལ་ལྔ་སྟོང་ཉིས་བརྒྱ་ཡོད། །ན་ར་ལ་གསུམ་པོ་ལ་ཡང༌། །ད་དྲག་དེ་བཞིན་གསུམ་པར་འཇུག །དེ་ལ་གསུམ་སྟོང་བརྒྱ་ཉི་ཤུ། །དེ་དག་བསྣན་པས་ཆིག་ཁྲི་དང༌། ། བདུན་སྟོང་དྲུག་བརྒྱ་བརྒྱད་ཅུར་འགྱུར། །དེ་ལ་མཐའ་རྟེན་མེད་པ་དང༌། །ཝ་བཏགས་བཅོ་ལྔ་ཨ་ཡིག་བདུན། །བགླ་བདྲོན་ཞེས་བྱ་བ་ལ་གསུམ། །སྐབས་ཀྱིས་འཇུག་པས་དེ་དག་བསྣན། །དེ་དག་བརྗོད་དུ་རུང་ཚད་ཀྱི། །ཡི་གེའི་གྲངས་ནི་མདོར་བསྡུ་ན། ཁྲི་ཁྲག་གཅིག་དང་སྟོང་ཕྲག་བརྒྱད། །བདུན་བརྒྱ་བཞི་བཅུ་ཞེ་ལྔ་ལྡན། །དེ་དག་ཐམས་ཅད་བོད་སྐད་ལ། །མི་དགོས་གང་དང་གང་དགོས་པ། ། རང་གི་བློ་ཡིས་དཔྱད་ཅིང་སྦྱར། ཁོ་བོས་འདི་ལྟར་རྣམ་བསམས་ནས། །བོད་ཀྱི་སྐད་ལ་གང་དགོས་པ། །སྟོང་ཕྲག་བརྒྱད་ཀྱིས་འདུས་པར་བསམ། །སྔོན་གྱི་བྱང་ཆུབ་སེམས་དཔའ་ཡིས། །བོད་སྐད་ཡི་གེར་བསྡུས་པ་ལས། །ཡི་གེའི་བྱེ་བྲག་དབྱེ་བ་དང༌། །གྲངས་ཀྱི་དབྱེ་བསྡུ་དེ་ལྟར་བཤད།

།ཡི་གེ་བྱེ་བྲག་དང་དབྱེ་བསྡུ་བསྟན་པ་སྟེ་ལེའུ་དང་པོའོ།། །།གང་གིས་འབྱུང་གནས་མི་ཤེས་པས། །ལོག་པར་སྨྲ་བ་དུ་མ་མཐོང༌། །དེ་ཕྱིར་དག་པར་བརྗོད་ཤེས་ཕྱིར། །ཡི་གེའི་འབྱུང་གནས་བརྗོད་ཐབས་བཤད། ཁོག་པ་མགྲིན་པ་རྐན་དང་ལྕེ། །སྣ་དང་སྤྱི་བོ་སོ་དང་མཆུ། །སོ་སོ་འམ་ནི་འདུས་པ་ལས། །སྒྲ་རྣམས་ཐམས་ཅད་འབྱུང་བར་བཤད། ཁོག་པ་མགྲིན་པ་ཐུན་མོང་གནས ༑༑ ལྷག་མ་སོ་སོར་ངེས་པ་སྟེ། །བོད་སྐད་གནས་རྣམས་ཐམས་ཅད་ལས། ། འབྱུང་བའི་སྒྲ་ཡང་ཡོད་པ་ཡིན། །འ་ཧ་ཨ་ཡི་ཡི་གེ་གསུམ། ཁོག་པ་མགྲིན་པའི་གནས་ནས་འབྱུང༌། །ཀ་ཁ་ག་ངའི་ཡི་གེ་གསུམ། །རྐན་ཕུག་ལྕེ་རྩེའི་ཆ་ལས་འབྱུང་ ༑ ༑ ཡ་ཞ་ཅ་ཆ་ཇ་ཉ་ཤ །བདུན་པོ་ལྕེ་དཀྱིལ་རྐན་དབུས་ལས། །ཏ་ཐ་ད་ན་ར་ལ་དྲུག །ལྕེ་རྩེ་རྐན་མཐའི་རྙིལ་ལས་འབྱུང༌། །ཙ་ཚ་ཛ་ཟ་ས་ལྔ་པོ། །ལྕེ་རྩེ་སོ་ལས་འབྱུང་བའོ། །པ་ཕ་བ་མ་ཝ་དང་ལྔ། །མཆུ་ལས་རབ་ཏུ་འབྱུང་བ་ཡིན། །ང་ཉ་ན་མའི་ཡི་གེ་བཞི། །རང་རང་གནས་དང་སྣ་ལས་འབྱུང༌། །ལྷག་མ་གནས་རིམ་ཅི་བཞིན་དུ། །ཅུང་ཟད་ཙམ་གྱིས་དྲག་ཅེས་བརྗོད། །ཨི་ནི་རྐན་ཕུག་ལྕེ་རྩེའི་ཟུར ༑ ༑ ཨུ་ནི་མཆུ་སྨད་ཨེ་རྐན་ཕུག །ཨོ་ནི་མཆུ་སྟོད་སོ་ལས་འབྱུང༌། །ཡན་ལག་རྣམས་ཀྱང་དེ་ལྟར་འབྱུང༌། །དྲག་ལ་སྤྱི་བོའི་ཆ་ཅུང་ཟད། །ཡོད་ཅེས་སྔོན་གྱི་མཁས་རྣམས་སྨྲ། །དེ་ལྟར་ཡི་གེའི་གནས་ཤེས་ནས། །རང་རང་གནས་ནས་ཕྱུང་སྟེ་བརྗོད། །ཡི་གེའི་མདོ་ཞེས་བྱ་བ་ལས། །འདི་དང་འབྱུང་གནས་མི་འདྲ་ཡོད། ། དེ་ནོར་འདི་བདེན་གང་ཡིན་པ། །རང་གི་ཁ་ལྕེ་ཡིད་ལ་དྲིས། །དེ་བཞིན་བོད་ཀྱི་ཡི་གེ་ཡི། །བཀླག་ཐབས་ནོར་བ་དུ་མ་བྱས། །ཀ་ཁ་ཕ་མ་ཀུན་གྱིས་ཤེས། །གང་ཡིན་རང་གི་བློ་ཡིས་དཔྱོད། །དེ་ལྟར་འབྱུང་གནས་ཤེས་བྱས་ནས། །ད་ནི་སྒྲ་ཡི་འབྱིན་ཐབས་བརྗོད། །དེ་ཡང་ཡི་གེའི་གོ་རིམ་བཞིན། །བསླབས་ན་དག་པར་འགྱུར་བ་ཡིན། །དང་པོར་ཀ་ཁ་སུམ་ཅུ་བསླབ། །འབྱུང་གནས་བློ་ལ་ཞེན་པར་བྱ ༑༑ དེ་ནས་དེ་ལ་གུག་སྐྱེད་ནི།

།བཏགས་ཏེ་ཨི་ལ་གུ་གུ་བརྗོད། །ཨུ་ལ་ཞབས་ཀྱུ་ཨེ་འགྲེང་བུ། །ཨོ་ལ་ན་རོ་བརྗོད་ཅིང་སྦྱར། །དེ་བཞིན་འབྱུང་གནས་ཤེས་པ་ཡིས ༑ ཁ་ལྕེའི་འགྲོ་ཚུལ་ཡིད་ལ་འཆགས། །དེ་ནས་བརྩེགས་པའི་ཡི་གེ་བསླབ། ། བཏགས་པའི་བརྡར་ནི་སྟེ་ཞེས་བྱ། །གང་དང་གང་ལ་གང་བཏགས་པ། །གོ་རིམ་བཞིན་དུ་གནས་ནས་འབྱུང༌། །དེ་ལ་ར་མགོ་ལྕེ་རྩེ་འདར། །ལ་མགོ་ལྕེ་ཀློད་རྒྱ་ཡང་བསྐྱེད། །ས་མགོ་སོ་ཡི་བར་ནས་འབྱུང༌། །ལྷག་མ་རྣམས་ཀྱང་དེ་བཞིན་ཤེས ༑༑ གལ་ཏེ་མི་ཕྱིན་གྱུར་པ་ནི། །ཡི་གེ་ཕྲལ་ཞིང་བརྗོད་པར་བྱ། །དཔེར་ན་གྲ་ལ་ག་རའམ། །སྒྲ་ལ་སྒ་ར་ཇི་བཞིན་དུ། །དེ་ནས་འཕུལ་བ་བསླབ་པར་བྱ། ། འས་འཕུལ་ཐོག་མར་རྐན་ནས་དབྱུང༌། །དས་འཕུལ་ལྕེ་རྩེ་ཀློད་པ་ལས། །བ་མས་ཕུལ་ལ་ཐོག་མར་ནི། །མཆུ་བཙུམས་པ་ལ་ཁྱད་པར་སྣ། །འས་འཕུལ་གྲེ་བའི་ཕུག་ནས་དབྱུང༌། །འཕུལ་ཤིང་བརྩེགས་པའི་ཡི་གེ་ཡང༌། །གོ་རིམ་བཞིན་དབྱུང་མ་ཕྱིན་ན། །འདི་ཡང་ཕྲལ་ཞིང་བཀླག་པ་སྟེ། །མཁྲ་ལ་མཁའ་ར་ཇི་བཞིན་ནོ ༑ ༑ སྦྱོར་དང་ཕྲལ་ཞིང་ནན་ཏན་བསྐྱེད། །དེ་ཡི་ཕྱི་ནས་ཤིན་ཏུ་སླ། །དེ་ནས་དེ་དག་རྣམས་ལ་ཡང༌། །གུག་སྐྱེད་བཞི་པོ་སྦྱར་ཞིང་བཀླག །བརྩེགས་སམ་འཕུལ་ལམ་འཕུལ་ཞིང་བརྩེགས། །ཚེག་བར་གཅིག་ན་ཡི་གེའི་གྲངས། །དུ་ཡོད་དེ་དག་ཐམས་ཅད་ཀྱིས། །གུག་སྐྱེས་མཉམ་པོར་ཐོབ་པ་ཡིན། །དཔེར་ན་སྒྲུ་ལ་སྒུ་རུའམ ༑ ༑ སྦྲེ་ལ་སྦེ་རེ་ཇི་བཞིན་ནོ། །དེ་ལྟར་མ་ཤེས་ཐམས་ཅད་ལ། །གུག་སྐྱེད་བཅས་པ་ཕྲལ་ཞིང་སྦྱར། །དེ་དག་ལྕེ་ཡི་ཟུར་ཕྱིན་ནས། །དེ་ནས་དེ་ལ་མཐའ་བརྟེན་བསླབ། །མཐའ་བརྟེན་གྱིས་ནི་ཡི་གེ་ལ། །བཞག་ཅེས་བྱ་བ་བརྗོད་པར་བྱ ༑ ༑ ཡང་ན་ལུས་དང་མཐའ་བརྟེན་གྱིས། །བར་དུ་ལ་ཞེས་བྱ་བ་གཞུག །སྦྱོར་ཞིང་ཀློག་པའི་དུས་ན་ནི། །མཐའ་བརྟེན་ཟུར་ཕྱིན་རབ་ཏུ་བསླབས། །དེ་བས་སྦྱོར་བའི་དུས་ཀྱི་ཚེ། །ལུས་ཀྱི་ཡི་གེ་ཟུར་མི་ཕྱིན། །དེ་ཕྱིར་དེ་ལ་ཕན་བྱའི་ཕྱིར།། ནོར་པ་དེ་ཡི་འཕྲང་བཤད་བྱ། །དེ་ལ་རང་གི་ཡུལ་གྱི་སྐད།

།མ་དག་པ་ཡི་ནོར་བ་མཐོང་ ༑༑ དབུས་པས་ར་ཡི་མགོ་ཅན་དང༌། །རྐྱང་པར་འདྲ་ཞིང་ཡ་སྟ་དང༌། །ར་སྟ་ཡང་ནི་ནོར་བ་ཡོད། །གཙང་པས་མས་འཕུལ་འས་འཕུལ་དུ། །ནོར་ཅིང་ར་ཡི་མགོ་ཅན་དང༌། །གས་འཕུལ་དས་འཕུལ་ནོར་བ་ཡོད། །རྐན་ཕུག་དང་ནི་ལྕེ་རྩེ་ཡ ༑ ༑ ཡི་གེ་སྦྱར་བ་བཀླག་པར་དཀའ། །དེ་བས་ཡུལ་རྣམས་ཕལ་ཆེར་དུ། །དེ་ནི་ནོར་བ་དག་ཏུ་བརྗོད། །མདོར་ན་སྒྲ་ཡི་འབྱུང་གནས་ལས། །འགལ་འདུའི་ཡི་གེ་བཀླག་པར་དཀའ། །གནས་གཅིག་ནས་ནི་འབྱུང་མོད་ཀྱང༌། །དྲག་ཞན་འགལ་བར་ནོར་བ་ཡོད། །དབུས་གཙང་གཉིས་ཀའི་ཡུལ་མཐིལ་གྱི། །ཐ་མལ་པ་ཡི་སྐད་རྣམས་ལ། །མཐའ་བརྟེན་རྫོགས་པ་རབ་ཏུ་ཉུང༌། །དེ་བས་བསྡུ་ཚེ་མཐའ་བརྟེན་གཟབ ༑ ༑ ད་ནི་རབ་ཏུ་བཤད་པར་བྱ། །བོད་ཀྱི་སྐད་ལ་མི་དགོས་ཀྱང༌། །ནོར་བའི་འཕྲད་ཉིད་བསྟན་པའི་ཕྱིར། །འདོགས་འབྲེལ་མེད་པ་འགའ་ཡང་བརྗོད། ། ཀེ་ཀྱེ་ཁེ་ཁྱེ་གེ་གྱེ་དང༌། །ངེ་ཉེ་ཀི་ཀྱི་ཁི་ཁྱི་དང༌། །གི་གྱི་ངི་ཉི་དགེ་དགྱེ་ནོར། ། གུག་འགྲེང་ཡ་དང་གནས་གཅིག་ཕྱིར། །ཀྲ་ཏྲ་ཁྲ་ཐྲ་གྲ་དྲ་དྲུག །ཀྲི་ཏྲི་ཁྲི་ཐྲི་དྲི་ལ་སོགས། །གུག་སྐྱེད་བཞི་ཀ་སྦྱར་བ་ཡང༌། །དེ་དག་ཕན་ཚུན་ནོར་བར་འགྱུར། །དེ་བཞིན་བསྐྲ་བསྟྲ་བསྲ་སྤྲ་དང༌། །བསྒྲ་བསྦྲ་བསྟྲ་བ བཀྲ་བཏྲ་བགྱ། །འགྲ་འདྲ་འགྱ་མགྱ་འབྲ་དགྲ་དྷྲ། །འཁྲ་འཐྲ་འཕྱ་མཁྱ་འཕྲ་མཐྲ་དང༌། །སྐྲ་སྟྲ་སྲ་སྒྲ་ ་ པྲ། །དེ་ལྟར་དེ་དག་ཐམས་ཅད་ལ། །གུག་སྐྱེད་བཞི་དང་མཐའ་བརྟེན་དགུ ༑ ༑ སྦྱར་བ་བཀླག་པ་རབ་ཏུ་དཀའ། །པྱ་དབྱ་སྦྱ་བཤའ་བཅའ་དང་ནི། །གུག་སྐྱེད་མཐའ་བརྟེན་བཅས་པ་དཀའ། །ཀྐ་སྐ་རྐ་དཀའ་གྒ་སྒ་རྒ་དགའ། །དྡ་སྔ་རྔ་དངའ་དཉའ་གཉའ་སྙ་རྙ། །ཏྟ་སྟ་རྟ་གཏའ་རྡ་སྡ་གདའ། །ནྣ་སྣ་རྣ་གནའ་མྨ་སྨ་རྨ་དམའ ༑ ༑ ཙྩ་རྩ་གཙའ་བཙའ་ཀྱ་དཀྱ་རྐྱ། །མཉའ་དམྱ་རྨྱ་བརྙ། བསྙ་སྨྱ་རྨྱ། རྒྱ་དགྱ ༑ བཀྱ་བརྐྱ་བསྐྱ་བསྒྱ་བགྱ་བཀའ་བརྐ་བསྐ། །མངའ་རྔ་སྔ་རྣམས་ནོར་བར་འགྱུར ༑ ༑ གཙང་པས་དས་འཕུལ་ར་མགོ་རྣམས་སུ་ནོར། །དབུས་པས་འཕུལ་དང་ར་ས་རྐྱང་པ་ནི།

།ནོར་གྱུར་གུག་སྐྱེད་མཐའ་བརྟེན་བཅས་དང་འདྲ། །དེ་བས་གུག་སྐྱེད་མཐའ་བརྟེན་བཅས་པ་བསླབས། །ཀླ་གླ་རླ་སླ་ལླ་བླ་བརླ། །འདི་ལ་གུག་སྐྱེད་མཐའ་རྟེན་བསླབ། །མི་ཉི་མྱི་མྱ་ཉ་ནྱ་དྲུག །སླ་སྟེ་ནོར་མེད་དེ་བས་བསླབ ༑ ༑ མཁའ་འཁའ་མགའ་འགའ་མངའ་དངའ་འདིར། །མདའ་འདའ་ལ་སོགས་འས་འཕུལ་དང༌། །མས་འཕུལ་བ་ནི་ནོར་མི་བྱ། །འཕྱ་འཆའ་འབྱ་འཇ་སྨྲ་དམྲ་སོགས། །འདི་དག་ཀུན་ལའང་གུག་སྐྱེད་དང༌། །མཐའ་བརྟེན་སྦྱར་ཞིང་ལེགས་པར་བསླབ། །དེ་བཞིན་ཡི་གེ་བཀླག་དཀའ་བ། །རང་རང་བློ་གྲོས་དག་གིས་སྦྱར། །དེ་ལྟར་སྦྱོར་ཀློག་སློབ་པའི་ཚེ། །ནན་ཏན་བསྐྱེད་ན་ཕྱི་ནས་སླ། ། སྦྱོར་ཀློག་ལྷག་ཆད་མེད་གྱུར་ན། །ཚིག་གི་བར་དུ་རྐྱང་སྡུད་བྱ། །དེ་ཚེ་སྒྲ་ལ་རབ་ཏུ་འབད། །ཤིན་ཏུ་དལ་བ་ཡིད་ལ་བཞག །ཚིག་སྡུད་འོག་མར་འདྲེས་དགོས་ཏེ།། དཔེར་ན་དགྲ་བཅོམ་སྒྲ་བཅོམ་བཞིན། །དེ་ལྟར་བཀླག་ན་འབྱོངས་པའི་ཚེ། །དེ་ལ་ངེས་པ་མང་དུ་སྐྱེ། །ཚིག་སྡུད་ནོར་བ་མེད་གྱུར་ཅིང༌། །ཤིན་ཏུ་མྱུར་བ་གྱུར་པའི་ཚེ། །མིང་གི་མཚམས་སུ་བསྡུ་བར་བྱ། །ཕྲད་ཀྱི་ཚིག་རྣམས་རྐྱང་པར་བཀླག། དེ་ལྟར་རབ་ཏུ་ཤེས་གྱུར་ནས། །ཕྲད་ཚིག་སྔ་མའི་ཁོངས་སུ་བསྡུ། །དེ་ལྟར་གཉིས་སྡུད་གསུམ་སྡུད་ཚེ། །མཐའ་བརྟེན་ཚིག་རྣམས་རྫོགས་པར་བརྗོད། །གལ་ཏེ་མཐའ་བརྟེན་མ་གཟབ་ན། །དེ་ཡི་ཟུར་རྣམས་ཆག་པར་འགྱུར། །དེ་བས་མཐའ་བརྟེན་དཀའ་བ་ཡི། །ཚིག་རྣམས་ཀྱང་ནི་སྦྱར་བར་བྱ། །སངས་རྒྱས་བཅོམ་ལྡན་འདས་ཞེས་དང༌། །དེ་བཞིན་གཤེགས་པ་དགྲ་བཅོམ་དང༌། །ཡང་དག་རྫོགས་པའི་སངས་རྒྱས་དང༌། །སློབ་དཔོན་བྱང་ཆུབ་སེམས་དཔའ་སོགས།། བརྗོད་པ་རབ་ཏུ་དཀའ་བ་ཡིན། །མིང་མཐར་ཝ་སྒྲ་དབྱུང་དཀའ་སྟེ། །ར་ལ་ང་མཐར་དེ་ནོར་ཞིང༌། །གཞན་དག་ལ་ནི་ར་ནོར་འགྱུར། །ང་པ་ག་ད་ལ་སོགས་པའི ༑ ༑ གོང་འོག་ཚིག་འཕྲོ་འགལ་འདུ་དང༌། །བྱིན་ནོ་འགྱུར་རོ་བྱེད་དོ་དང༌། ། འགལ་ལོ་བསམ་མོ་ལ་སོགས་པ། །བསྟབས་པའི་ཟུར་ཡང་འབྱུང་དཀའ་བས།

། དེ་ཕྱིར་ལྕེ་ལ་གང་མི་བདེ། །དབྱུང་བར་དཀའ་བའི་ཚིག་རྣམས་ཀྱི། །ཟུར་རྣམས་རབ་ཏུ་དབྱུང་བར་བྱ། །དེ་ལྟར་གཉིས་གསུམ་སྡུད་པ་ལས། །སྒྲ་ཟུར་ཕྱིན་པ་མ་ནོར་ཚེ། །སྐད་དང་རྗེས་སུ་མཐུན་པ་ཡི། །ཀློག་གི་སྐད་ལ་བསྐྱོན་ཏེ་བཀླག ། ཚིགས་བཅད་རྣམས་ལ་བཅད་ཅིང་བཀླག །ལྷུག་པ་རྣམས་ལ་བསྲིང་བར་བྱ། །དེ་ཚེ་ཤིན་ཏུ་མགྱོགས་གྱུར་པས། །ཤིན་ཏུ་ཐེག་པ་གལ་ཡང་ཆེ། །ཀློག་སྐད་སློབ་པ་ཐོག་མའི་ཚེ། །བཀླག་པ་སླ་ལ་བསླབ་པར་བྱ། །དེ་ནས་དལ་བར་བྱ་བ་དང༌། ། དབུགས་ཐག་བསྲིང་བ་ཉིད་ལ་བསླབ། །ཟུར་ཕྱིན་མ་ནོར་ལྷག་ཆད་མེད། །སྒྲ་སྐད་མཉམ་ལ་ཐོགས་རྡུགས་མེད། །དལ་ཞིག་མགྲིན་པ་རབ་ཏུ་སྙན། །ཀློག་མཐར་ཕྱིན་པའི་ཚད་ཡིན་ནོ། །ངག་གིས་དེ་ལྟར་མ་ཤེས་ན། །ཡིད་ཀྱིས་ཤེས་པས་ཅི་ཞིག་ཕན། །ཟས་ཉིད་ཁ་ཡིས་མ་ཟོས་ན། །ཐོས་པས་འགྲངས་ལ་མི་ཕན་ཞིང་ ༑ ༑ དེ་ཕྱིར་དང་པོ་ལེགས་པར་བསླབ། །མ་དག་པ་ཞིག་སྔར་ཤེས་ནས། ། སྦྱོར་བ་འམ་ནི་རྐྱང་ཀློག་ཏུ། །བཤིག་ནས་བསླབས་ན་མཁས་པར་འགྱུར། །ཡི་གེའི་འབྱུང་གནས་དང༌། འབྱིན་ཐབས་དང༌། བཀླག་ཐབས་བསྟན་པའི་ལེའུ་སྟེ་གཉིས་པའོ།། །།དེ་ནས་སྔགས་དང་རྒྱ་སྐད་ཀྱི། །ཡི་གེའི་བཀླག་ཐབས་བརྗོད་པར་བྱ། ། འདིར་བརྗོད་རྣམས་ལ་ངག་བྱང་ན། །ཡི་གེ་ཀུན་ལ་ཐོགས་མེད་འགྱུར། །གང་གིས་བཀླག་ཐབས་མི་ཤེས་པ། །རིགས་སྔགས་ལོག་པར་བཟླས་གྱུར་ན། །དད་མེད་རྣམས་ལ་དོན་མེད་འགྱུར། །དད་པ་ཡོད་ཀྱང་དངོས་གྲུབ་རིང༌། །དེ་ལ་བཀླག་ཐབས་མདོར་བསྡུ་ན། །ཨཱ་ལི་ཀཱ་ལི་གུག་སྐྱེད་བཅས། །དེ་རྣམས་སློབ་དཔོན་ངག་བཞིན་དུ། ཁ་མི་འདྲིས་པར་ཡུན་རིང་བསླབ། །བཀླག་ཐབས་རིམ་བཞིན་མི་སློབ་པར། །སྔགས་འབྲུ་རྣམས་ལ་འདི་ཅི་ཞེས། །ཟེར་བའི་རྒྱུ་མེད་འབྲས་འདོད་པའི། །རྨོངས་རྣམས་ཡུན་རིང་དོན་མེད་འགྱུར། །དེས་ན་བོད་ཀྱི་ཡི་གེ་ཡི། །སྒྲ་འབྱིན་ཐབས་གཞན་སྔར་བཤད་རྣམས། །ཤེས་ན་རྒྱ་སྐད་ཡི་གེ་ཡང༌།། ཕལ་ཆེར་མཐུན་པས་དག་པར་འགྱུར།། །།ད་ནི་རྒྱས་པར་བཤད་པར་བྱ།

༑ ༑ ྉྤ་ཞེས་བྱ་བ་མཆུ་ཅན་ནོ། །ཡི་གེ་དེ་རྣམས་ཟུར་ཕྱིན་ན། །རྒྱ་སྐད་ཐམས་ཅད་ཤེས་པར་འགྱུར།། །།ད་ནི་སྒྲ་ཡི་འབྱིན་ཐབས་བརྗོད། །སྐད་ནི་རྒྱ་སྐད་ཡིན་མོད་ཀྱང༌། །བོད་ཀྱི་སྐད་དང་རྗེས་མཐུན་པར། །བརྗོད་པར་སླ་བ་རབ་ཏུ་བྲི། །རིགས་ནི་སྤྱོད་པས་སྟོན་པར་བྱེད། ཁ་ཟས་ལུས་ཀྱིས་སྟོན་པར་བྱེད། །བརྩེ་བ་ཏབ་ཏབ་བྱེད་པས་སྟོན ༑ ༑ ཡུལ་ནི་སྐད་ཀྱིས་སྟོན་པར་བྱེད། །དེ་སྐད་ཟེར་བའི་ཚིག་བདེན་པས། ། སྐད་གདངས་ཡུལ་གྱི་སྐད་ཀྱིས་འབྱུང༌། །གལ་ཏེ་སྐད་གདངས་བཅོས་པ་ན། ། རྣར་མི་སྙན་ཅིང་བརྗོད་ནོར་འགྱུར། །རྒྱ་སྐད་ཕལ་ཆེར་ལྕེ་རྩེ་ལས། །འབྱུང་ཞིང་བརྗོད་པ་རབ་ཏུ་མྱུར། །སྐད་གདངས་མགྲིན་པ་ཕྲ་ཞིང་སྙན། །བོད་སྐད་ཕལ་ཆེར་ལྕེ་དབུས་ལས། །འབྱུང་ཞིང་བརྗོད་པ་དལ་བ་ཡིན། །མགྲིན་པ་སྦོམ་འགྱུར་ཁ་ཆེ་དང༌། །ཨུརྻན་ཡུལ་ཡང་དེ་དང་འདྲ། །འོན་ཀྱང་དེ་གཉིས་རྒྱ་སྐད་ཀྱི། ། ལུགས་སུ་དེ་དག་ལེགས་པར་སྨྲ། །བོད་ལ་ཁྱད་པར་སྐད་ཐུང་ཞིང༌། །རིང་ཐུང་ཁྱད་པར་འགའ་ཡང་མེད། །དེ་ཕྱིར་བོད་ཀྱི་རྒྱ་གར་སྐད། །ལེགས་ལོབས་ཁ་ཆེའི་སྐད་འདྲ་འབྱུང༌། །དེ་བས་སྐད་གདངས་མ་བཅོས་པར། །རིང་ཐུང་ཁྱད་བརྗོད་བསླབ་ལ་འབད། །དེ་ལ་འ་མེད་ཐུང་བ་མང༌། །ཐུང་དང་དྲག་པོར་ནོར་མི་བྱ། ། དབྱངས་ཀྱི་ཡི་གེ་རྣམ་དྲུག་ལ། །འ་བཏགས་རྣམས་ནི་རིང་པར་ཀློག །རྀ་རཱྀ་ལྀ་ལཱྀ་མ་ནིང་བཞི། །ཨི་རི་ཨི་ལི་ལྟ་བུར་བརྗོད། །དེ་ཡང་འ་བཏགས་རིང་བར་བསླབ། ཁ་ཆེ་ཨི་ཨེ་ལུགས་གར་སྨྲ། །ཨེ་ཨཽ་བསྒྲེངས་དང་བསྟོད་པ་གཉིས། །འ་མེད་ཀྱང་ནི་རིང་བར་བཀླག །ཨཻ་ཨེ་འིར་བཀླག་པ་སྟེ། །བསྒྲེང་ཞིང་བཀུག་པ་ཡིན་ཕྱིར་རོ ༑ ༑ ཨཽ་ཨོའུ་གང་དུ་བྲིས་ཀྱང་རུང༌། །ཨོ་འུར་བརྗོད་པ་བསྟོད་སྨད་ཕྱིར། །སྣ་ལྡན་རྣམ་པར་བཅད་པ་སྤངས། །རིང་བའི་སྒྲར་ནི་བརྗོད་པར་བྱ། །དེ་ལྟར་དབྱངས་ཡིག་བཅུ་དྲུག་གི །རིང་ཐུང་ཁྱད་པར་ལེགས་ཤེས་ནས། །གསལ་བྱེད་སུམ་ཅུ་རྩ་བཞི་ཡི། །བརྗོད་པ་བསླབ་ཅིང་མིག་འདྲིས་བྱ།

།བརྩེགས་པ་རྣམས་ཀྱང་དེ་འདྲ་སྟེ། །དེ་ལྟར་བརྗོད་ན་རབ་ཏུ་མཛེས།། །།ད་ནི་བརྩེགས་པའི་ཡི་གེ་བརྗོད། །རྒྱ་གར་ལུགས་ལ་གསལ་བྱེད་ཀྱི། །ཡི་གེ་ཐམས་ཅད་བརྩེགས་ཡོད་པས། །ཆིག་སྟོང་ཆིག་བརྒྱ་ལྔ་བཅུ་དྲུག །དེ་ལ་གུག་སྐྱེད་བཅུ་གཉིས་ཀྱིས། །ཆིག་ཁྲི་སུམ་སྟོང་བརྒྱད་བརྒྱ་དང༌། །བདུན་ཅུ་རྩ་གཉིས་རབ་ཏུ་ལྡན། །དེ་སྟེང་སྔར་གྱིས་མནན་པ་ཡོད། །དེ་ལྟར་ཡི་གེ་མང་ཡོད་ཀྱང༌། ། རྒྱ་ལུགས་བཀླག་བསླབ་ཅི་ཕྱིར་ན། །བརྩེགས་པའི་སྟེང་མ་གང་དུ་སྦྱོར། །དཔེར་ན་ཏིཥྛི་ཏིས་ཐྲ་བཞིན། །གལ་ཏེ་གོང་དུ་མི་སྦྱར་བར། །སྔར་བསྟན་འབྱུང་གནས་ཤེས་གྱུར་པས། །ཕྲལ་ཞིང་གནས་ནས་རིམ་བཞིན་དབྱུང༌། །དེས་ན་མི་ཤེས་འགའ་ཡང་མེད། །གུག་སྐྱེད་ཅན་ནི་བྱུང་གྱུར་ནའང༌། །ཕྲལ་ཏེ་གུག་སྐྱེད་ཀུན་ལ་སྦྱར ༑ ༑ དཔེར་ན་ཏྱུ་ལ་ཏ་ཡུའམ། ། ་ལ་ཤི་ཐི་ཇི་བཞིན་ནོ། །གལ་ཏེ་ཕྲལ་ཏེ་ཀློག་པ་ན། །ཅི་ཕྱིར་ཕྲལ་ནས་མི་འབྲི་ཞེས། །རྒོལ་ན་བོད་སྐད་ཀྱང་མཚུངས་ཞེས ༑ ༑ དེ་གཉིས་ཁྱད་པར་མེད་པར་བསྒྲེ། །མྱུར་དུ་བརྗོད་ཕྱིར་ཡིག་ཉུང་ཕྱིར། ། སྒྲ་གནས་བསྡུས་ཕྱིར་འདིར་བརྩེགས་ན། །དེས་ན་འདི་ཡང་དེ་དང་འདྲ། །དེ་ཕྱིར་ཕྲལ་ཞིང་མྱུར་བར་བཀླག །དེས་ན་རྒྱ་གར་སྒྲ་གནས་ཀུན། །བསྡུས་པས་ཨ་ལས་ཨ་བྱུང་ཞིང༌། །ཨེ་ལས་ཨི་བྱུང་ཨོ་ལས་ཨུ། །དེ་ལྟར་འབྱུང་གནས་བསྡུས་པ་ཡོད ༑ ༑ གསང་སྔགས་ཀྱི་ནི་རྒྱུད་རྣམས་ལ། །ཡི་གེ་མང་པོ་བརྩེགས་པ་ཡོད། །དེ་ནི་ས་བོན་ཡིན་པའི་ཕྱིར། །དུས་གཅིག་མི་བརྗོད་ཕྲལ་ཏེ་བཀླག། །།ད་ནི་བླུན་པོའི་མི་རྣམས་ལ། །བསྟན་ཕྱིར་བོད་ལ་མེད་པ་ཡི། །ཡིག་འབྲུ་བརྩེགས་པ་བཀླག་པའི་ཐབས། །བོད་ཀྱི་ལུགས་ཀྱིས་འགའ་ཞིག་བརྗོད། །ཙ་ཚ་ཛ་ལ་ཡ་སྟ་བཏགས། །བོད་ཀྱི་ཅ་ཆ་ཇ་བཞིན་བཀླག །གུག་སྐྱེད་ཅན་ཡང་དེ་བཞིན་ཏེ། །དཔེར་ན་ཙྱ་ལ་ཙེ་ཡ་བཞིན། །ཏ་ཐ་ད་སོགས་ཡ་སྟ་འབྱུང༌། །ཨེ་སྒྲ་ལྡན་པས་ཕྲལ་ཞིང་བཀླག །དཔེར་ན་ཏྱ་ལ་ཏེ་ཡ་འམ། །ཐྱ་ལ་ཐེ་ཡ་ཇི་བཞིན་ནོ།། ཛྙ་ཞེས་བྱ་བའི་ཡི་གེ་ལ།

།རྒྱ་གར་འགྱར་ཀློག་ཁ་ཆེ་གཉའ། །བོད་ཀྱི་གཉའ་ཞེས་བྱ་བར་ཀློག ཁ་ཆེའི་ལུགས་སུ་བཀླག་ན་བདེ། །ཥྞ་ཥྟ་ཥྛ་སོགས་ཡི་གེ་ལ། །ཤ་ལོག་ཤ་རྣལ་གང་བྱུང་ཡང༌། །ས་དང་འདྲ་བར་བྱ་བ་སྟེ། །སྣྲ་སྟྲ་ ཞེས་བརྗོད་པས་འགྲུབ། །གལ་ཏེ་དེ་ལྟར་མ་ཐོབ་ན། །ཕྲལ་ཏེ་བཀླགས་པས་ཐམས་ཅད་འགྲུབ ༑ ༑ སྟྱཻ་དྷ་ལྟ་བུ་ལ། སྟྱཻ་ཡད་དྷར་བཀླགས་པས་འགྲུབ། །དེ་ལ་སོགས་པ་བསམ་མི་ཁྱབ། །སྔགས་ཀྱི་བཀླགས་ཐབས་མི་ཤེས་ན། །འདི་མང་བྲིས་པས་ཅི་ཞིག་ཕན། །དེ་བས་སྔར་བརྗོད་ཐབས་ལ་འབད། །དེ་ནས་གཞན་ཡང་རབ་བཤད་བྱ། །ཨཱ་ལི་ཀཱ་ལི་གུག་སྐྱེད་བཅས། །ལེགས་ཤེས་ཁ་ཆེ་ནས་བྱུང་ཞིང༌། ། ལྕེ་བདེར་འགྱུར་བའི་ཡི་གེ་བྲི། ། རྦ་ཛྙནྶ་ཎྜ་ཌ་ཀྵཿསྠ་རྵ་ནྐ་ ཿཏྲ་ ཌྒ ཙྪ། ཡ་རྐྲ་ཏྟྱ་རྫ་ནཱ་ དྷྲ། ཤངྐུ་ རྠ་ ཉྫུ་ཉྫུ་རྡྷྭ། སྟྭནྟྱ་ཀྲོ་སྨྲོ་སྤྲོ་དྡྷ་ ་ཌཱ། ། ཀནྠི་རྦྷྱ་ཥྐ་བྫིམ་ཕཱ། ཡ ཌྷྱ་པྲེངྐ༌། ངྐ་ཚེཙྪར། རེངྐ་ སྲེ་ཀླི་ རྠཿ འདི་རྣམས་ལེགས་པར་ཟུར་ཕྱིན་ན། །ལྕེ་བདེ་བར་ནི་འགྱུར་བ་ཡིན། །དེས་ན་གཎྜཱིའི་བསྟོད་ཚོགས་ལས། །ཚེག་ཀློག་སྦྱངས་ན་མཁས་པར་འགྱུར། །འདིར་མང་བརྗོད་པས་ཅི་ཞིག་བྱ། །བཀླག་ཐབས་ལེགས་པར་མི་སློབ་པར། །ཡི་གེར་བྲིས་པས་ཆོག་ཤེས་པའི། །རྨོངས་རྣམས་དོན་མེད་མ་གྱུར་ཏམ། །དེ་བས་ལེགས་པར་སློབ་འདོད་ན། །ཐོག་མར་ཨཱ་ལི་ཀཱ་ལི་ལ། །སློབ་དཔོན་གྱི་ནི་ངག་བཞིན་དུ། ། རིང་ཐུང་ཕྱེད་པར་ལེགས་པར་བསླབ། །དེ་ལ་གུག་སྐྱེད་བཅས་ཤེས་ན། །དོན་ཡོད་ཞགས་པའི་གཟུངས་སོགས་ལ། །ཤིན་ཏུ་དལ་བར་བྱས་ནས་ནི། །རིང་ཐུང་ཁྱད་ཕྱེད་བྱ་ཞིང་བསླབ། །མདོར་ན་སློབ་དཔོན་བརྗོད་པ་བཞིན། །བོད་ཀློག་ཤེས་པས་ཞག་བཅུའམ། །ཟླ་ཕྱེད་ཉི་ཤུ་རྩ་གཅིག་ཏུ། །བསླབས་ན་ཐམས་ཅད་ཤེས་པར་འགྱུར། །སྐད་གདངས་མ་བཅོས་དལ་བ་དང༌། །རིང་ཐུང་ཁྱད་པར་ལེགས་ཕྱེད་ཅིང༌། །སྒྲ་ཡི་ཟུར་ཕྱིན་མི་ཐོགས་པ། །སྔགས་ཀྱི་ཀློག་མཐར་ཕྱིན་པ་ཡིན ༑ ༑ སྔགས་བརྗོད་རྣམས་ལ་རྣམ་ཀུན་ཏུ། །གལ་ཆེ་བ་ནི་ས་མཚམས་ཡིན། །

དཔེ་ལ་ནན་ཏན་བསྐྱེད་དེ་བསླབ། །རྒྱ་ཡིག་ཤེས་ན་གཅིག་པ་རྟ། །ཀློག་པ་བསླབ་ན་མི་ཤེས་པས། །ས་མཚམས་གང་དང་གང་ཡིན་པ། །རང་ཉིད་ཀྱིས་ཀྱང་ལེགས་ཤེས་འགྱུར། །རྒྱ་གར་གྱི་སྐད་དང་སྔགས་ཀྱི་བཀླག་ཐབས་བསྟན་པའི་ལེའུ་སྟེ་གསུམ་པའོ།། །།དེ་ལྟར་འདི་ནི་ཁོ་བོས་འབད་དེ་བྲིས་པ་གང༌། །མཁས་པ་སློབ་པར་འདོད་པའི་བྱིས་པ་གཞུག་ཕྱིར་ཡིན། །འོན་ཀྱང་ཆེན་པོ་རྣམས་ཀྱིས་ལན་འགའ་གཟིགས་གྱུར་ནའང༌། །དེ་ཡང་དོན་མེད་མི་འགྱུར་བློ་གྲོས་སྐྱེ་བར་འགྱུར་སྙམ་དང་ ༑ ༑ དེ་ཉིད་ལེགས་པར་ཤེས་ན་ཀློག་གི་སློབ་དཔོན་ཤེས་ལ་རབ་ཏུ་མཁས་པར་འགྱུར། །དེ་ཕྱིར་མཁས་པ་རྣམས་ཀྱང་ཁེངས་པ་དོར་ནས་འདི་བྲིས་ལེགས་པར་སློབ་གྱུར་སྙམ། །བརྩེ་བས་འདི་བྲིས་པ་ལས་དུབ་པར་གྱུར་པའི་བསོད་ནམས་གང་ཡོད་པ། །དེས་ནི་འགྲོ་བ་མཐའ་དག་བྱང་ཆུབ་སེམས་ལྡན་སྤྱོད་པ་མཆོག་ལ་གནས་ཀྱི་བར། །གང་དང་གང་བརྗོད་དེ་དག་དོན་གསལ་དོན་ལ་བརྗོད་པའི་ཟུར་ཕྱིན་རབ་ཏུ་མཁས། །ལེགས་སྦྱར་རྒྱ་མཚོའི་བློ་གྲོས་ཡངས་པའི་གཟུངས་ཀྱི་རྩེ་མོ་མཆོག་ནི་ཐོབ་པར་ཤོག །ཡི་གེའི་བཀླག་ཐབས་བྱིས་པ་བདེ་བླག་ཏུ་འཇུག་པ་ཞེས་བྱ་བ། །དགེ་བསྙེན་བསོད་ནམས་རྩེ་མོས་སྦྱར་བའོ།། །། རྐེད་སླས་དམ་པ་དང༌། དཔལ་ས་སྐྱའི་དབེན་གནས་དམ་པར་སྦྱར་བའོ། །ཕག་ལོ་རྟའི་ཚེས་བཅུ་གཅིག་ལ་ཚར་བར་བགྱིས་སོ། །ཞུ་བ་པོ་ཞང་སྟོན་ཕྱར་བུ་པས་བགྱིས་སོ།། །།ཨོཾ་ཡེ་དྷརྨཱ་ཧེ་ཏུ་པྲ་བྷ་ཝཱ་ཧེ་ཏུནྟེ་ཥཱནྟ་ཐཱ་ག་ཏོ་ཧྱ་ཝ་ད་ཏ། ཏེ་ཥཱཉྩ་ཡོ་ནི་རོ་དྷ་ཨེ་ཝཾ་བཱ་དཱི་མ་ཧཱ་ཤྲ་མ་ཎ་ཡེ་སྭཱ་ཧཱ། ཨོཾ་སུ་པྲ་ཏིཥྛ་བཛྲཱ་ཡེ་སྭཱ་ཧཱ།། །། སློབ་དཔོན་རྩེ་མོའི་གསུང་རབ་དཔེ་བསྡུར་མ་པོད་ གསུམ་ པ། མེས་པོའི་ཤུལ་བཞག་ལས། ས་སྐྱ་བཀའ་འབུམ། PAGE 498PAGE 499 EMBED CorelDRAW.Graphic.9 EMBED CorelDRAW.Graphic.9 EMBED CorelDRAW.Graphic.9 EMBED CorelDRAW.Graphic.9 EMBED CorelDRAW.Graphic.9 EMBED CorelDRAW.Graphic.9

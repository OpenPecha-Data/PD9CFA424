དེའི་སུལ་ལམ། གངྒཱ་ལ་སོགས་པའི་ཆུ་འགྲམ་མོ། །འཛམ་བུའི་གླིང་ན་ཡོད་པའི་ཡུལ་ཉི་ཤུ་རྩ་བཞི་ན་ཡོད་པའི་དུར་ཁྲོད་བརྒྱད་དོ། །དེར་རྣལ་འབྱོར་པ་དཀྱིལ་འཁོར་བསྒོམ་པ་འདིས་མཚོན་ནས་དབང་བསྐུར་བ་མདོར་བསྟན་པར་ཡང་འགྲོ་སྟེ། དེ་ཁོ་ན་ཉིད་སྟོན་པའི་སྔོན་དུ་དབང་འགྲོ་དགོས་ལ། དེ་ལ་ཡང་དཀྱིལ་འཁོར་སྔོན་དུ་འགྲོ་དགོས་སོ། །དེ་ལ་གནས་སྔོན་དུ་འགྲོ་དགོས་པས། དཀྱིལ་འཁོར་གྱི་སྔོན་དུ་འགྲོ་བ་གནས་བསྟན་པའོ། །ལེའུ་དང་པོ་བསྟན་ཏོ།། དེ་ལྟར་ལེའུ་དང་པོས་མདོར་བསྟན་ནས། གཉིས་པ་ལ་སོགས་པས་རྒྱས་པར་བཤ

ད་པ་ལས། དེ་ལ་ལེའུ་དང་པོ་གསང་བ་མཆོག །ཅེས་པ་ལ་སོགས་པ་དང་༌། རི་བོ་ལ་སོགས་པས་དཀྱིལ་འཁོ ར་བྲི་བའི་གནས་བསྟན་ལ། དེ་རྒྱས་པར་བཤད་པའི་ཚུལ་གྱིས་གནས་དང་འབྲེལ་བའི་དཀྱིལ་འཁོར་བྲི་བ་ལ་སོགས་པ་བསྟན་ཏེ། ལེའུ་དང་པོར་ལམ་གྱི་གསང་བ་དང༌། གསང་བ་བསྒྲུབ་པ་ལ་སོགས་པ་དེའི་དེ་ཁོ་ན་ཉིད་ནི་ལེའུ་བཞི་པའི་རིམ་གཉིས་དེ་ཁོ་ན་ཉིད་བཞི་བཞི་བརྒྱད་དུ་བཤད་དོ ༑ ༑ དེ་ལྟ་བུའི་དེ་ཁོ་ན་ཉིད་སྟོན་པ་ལ། དབང་སྔོན་དུ་འགྲོ་དགོས་པས། ལེའུ་གསུམ་པས་དབང་བཤད་ལ། དེ་ཡང་སྔོན་དུ་དཀྱིལ་འཁོར་བྲི་བས་གཉིས་པར་དཀྱིལ་འཁོར་བསྟན་ཏོ། །དེ་ལ་ཡང་དོན་ལྔ་ལས་དང་པོ་ས་ ( ¡¶ ཞ ¡· - ས་ ) གཟུང་བ་བསྟན་ཏེ། དེ་ཡང་དང་པོ་དཀྱིལ་འཁོར་ ( ¡¶ ཞ ¡· + གྱི་ ) བའི་ལྕི་བ་ལ་སོགས་པ་ཐུན་མོང་དང༌། ཐུན་མོང་མ་ཡིན་པའི་རྫས་གཉིས་ཀྱིས་བྱུགས་ལ། དེ་ནས་རང་ཉིད་ཀྱང་སློབ་དཔོན་མཚན་ཉིད་ཅེས་བྱ་བ་ལ་སོགས་པའི་ཚུལ་གྱིས་རྒྱན་གྱིས་བརྒྱན་ཅིང་བདག་ཉིད་ལྷར་བསྐྱེད་དེ། དེ་ནས་ཧེ་རུ་ཀ་དྲན་པར་བྱ་ཞེས་པ་ལུས་ལ་ལྷ་རྣམས་དགོད་པར་བྱ་ཞེས། འདུ་བའི་འབྲེལ་བའམ་རྩ་རྐྱང་གི་ལུགས་ལྟར་བཀོད་ལ། དེ་ནས་སྙིང་གའི་འཁོར་ལོར་དགོད། ཅེས་པ་ཡེཤེས་ཀྱི་དཀྱིལ་འཁོར་དགུག་གཞུག་བྱའོ། །དེ་ཉིད་གོ་བགོས་པའོ། །དེ་ནས་གཉིས་པ་ལ་སོགས་པ། ཞེས་བྱ་བ་ལ་སོགས་པས་ན་སྲུང་བའི་ ¡Ã འཁོར་ལོ་ ( ¡¶ ཞ ¡· དཀྱིལ་འཁོར་ ) བྱ་བ་སྟེ། འདིའི་བྱེད་ལུགས་ནི་ནག་པོ་ ( ¡¶ ཞ ¡· + པའི་ ) དཀྱིལ་འཁོར་ཆོ་ག་ལྟར་བྱ་གསུང༌། དེ་ནས་དཀྱིལ་འཁོར་གྱི་འཁོར་ལོ་ནམ་མཁའ་ལ་སྤྱན་དྲངས་ལ ༑ མཆོད་པ་ཙམ་ནི་ས་ཡོངས་སུ་གཟུང་བའོ། །དེ་ནས་སེམས་མེད་ཅེས་བྱ་བ་ལ་སོགས་པས་ནི། དོན་གཉིས་པ་ཐིག་གི་བྱ་བ་བསྟན་ཏོ། །དེའི་དབུས་སུ་པདྨ་ཞེས་བྱ་བ་དང༌། སྔར་བསྟན་པའི་རོ་བསྲེགས་པའི་སོལ་བའི་ཕྱེ་མས། ཞེས་བྱ་བ་ལ་སོགས་པ་དོན་ཕྱོགས་གཅིག་ཏུ་བསྡེབས་པས་དོན་གསུམ་པ་རྡུལ་ཚོན་བསྟན་ཏེ། ཐལ་ཆེན་དང་རྡོ་རྒྱུས་ཀྱིས་དཀར་པོ་དང༌།

ཐལ་ཆེན་དང་བ་བླས་སེར་པོའོ། །ཐལ་ཆེན་དང་སོ་ཕག་གིས་དམར་པོའོ། །ཐལ་ཆེན་དང་དུར་ཁྲོད་ཀྱི་ཤིང་ལས་ལྗང་གུ་བྱའོ། །དེ་ལྟ་བུའི་ཚོན་གྱིས་དཀྱིལ་འཁོར་བྲིས་ལ། ལྟེ་བར་དཀྱིལ་འཁོར་དགོད་ཅེས་བྱ་བ་ལ་སོགས་པས་ལྷ་དགོད་པ་བསྟན་ནོ། །དེ་ནས་བུམ་པ་ཞེས་བྱ་བ་ལ་སོགས་པས་བུམ་པ་ལྷག་པར་གནས་པ་བསྟན་ཏེ། དེ་ཡང་སུ་མ་ཏི་ནི་བུམ་པ་དགུར་བཞེད་ལ། དེ་གཙོ་བོ་ཡབ་ཡུམ་བུམ་པ་གཅིག་ལ་སོགས་པར་བཞེད། བགྲེས་པོ་རྣམས་ནི་བུམ་པར་བཅུར་བཞེད་དེ། སྒོ་མཚམས་མ་བརྒྱད། གཞན་གཙོ་བོ་ཡབ་ཡུམ་ལ་སོགས་པ་རྣམས་ལ་བུམ་པ་གཅིག ལས་ཐམས་ཅད་དང་བཅུར་བཞེད། ཡང་ལྷ་ཞུ་ཞིང་བཟླས་པ་བྱ་བ་ཡང༌༑ དཀྱིལ་འཁོར་མངལ་སྐྱེས་ཀྱི་ཚུལ་གྱིས་ནམ་མཁའ་ལ་དམིགས་ལ། དེ་དག་སྙོམས་འཇུག་བྱེད་པར་དམིགས་ལ། དེ་ལས་བྱུང་བའི་བྱང་ཆུབ་ཀྱི་སེམས་བུམ་པར་བྱུང་བར་བསམ་ཞིང་བཟླས་གསུང༌། བགྲེས་པོ་རྣམས་ཕྱག་ལེན་དུ་དཀྱིལ་འཁོར་ཆོ་ག་ལྟར་མཛད་གསུང༌། གསཻར་དང་དངུལ་ཞེས་བྱ་བ་ལ་སོགས་པས་ནི་ལྔ་པ་དཀྱིལ་འཁོར་སྒྲུབ་པ་བསྟན་ཏེ། ལྷ་གནས་ཀྱི་ལྷ་སྙིང་གར་བསྡུས་པ་དེ་སྤྲོས་ལ་ནམ་མཁའ་ལ་དམིགས་ཏེ། རྡུལ་ཚོན་ལ་བསྟིམ་སྟེ་སྦྱངས་ནས ༑ མཆོད་པ་རྒྱས་པས་མཆོད་པ་བྱའོ། །རྒྱུད་འདིའི་ཐོག་ཏུ་འདི་ཙམ་གང་ཡིན་ཞེས། བླ་མ་སུ་མ་ཏི་གསུང་ལ། ལག་ཏུ་ལེན་པའི་ཆོ་ག་ཐམས་ཅད་ནག་པོ་དཀྱིལ་འཁོར་ཆོ་ག་ལྟར་བྱ་ཞེས་བླ་མ་བགྲེས་པོ་རྣམས་གསུང་ངོ༌། །དེ་ལྟར་བསྟན་པའི་དཀྱིལ་འཁོར་དང་འབྲེལ་བའི་ལེའུ་གསུམ་པས་དབང་བསྟན་ལ། དེ་ལ་ཡང་དྲུག་ལས་དང་པོ་དབང་ནོད་པའི་བླ་མ་མཉེས་དགོས་པས། བླ་མ་མཉེས་པར་བྱ་བ་དང༌། དྲིལ་དབྱངས་ཞེས་བྱ་བ་ལ་སོགས་པས་དཀྱིལ་འཁོར་ཡང་མཆོད་ཅིང་མཉེས་པར་བྱ་བ་བསྟན་ཏེ། དེའི་བུ་ཕུག་ཅེས་བྱ་བ་ལ་སོགས་པས་ནི། དོན་གཉིས་པ་དང་པོ་ལ་ལས་ཐམས་ཅད་པའི་བུམ་པའི་ཆུས་བསང་གཏོར་བྱས་ནས། གདོང་གཡོགས་བཅིངས་ཏེ་མེ་ཏོག་ཕྲེང་བ་ལག་ཏུ་བྱིན་ནས། ཁྱོད་ཅི་ལ་དགའ་ལ་སོགས་པ་ནག་པོ་ཆོ་ག་ལྟར་བྱ་སྟེ།

འདི་དག་ནི་མཇུག་གི་ཆོ་ག་དངོས་སུ་བསྟན་ལ། འདིས་མཚོན་ནས་བུམ་དབང་བཅུ་ཤེས་པར་བྱ་སྟེ། ཞེས་བླ་མ་སུ་མ་ཏི་གསུང༌། བླ་མ་ཉིད་བུམ་དབང་དོན་དུ་བཞེད། དེ་ནས་སློབ་དཔོན་ལེགས་པར་ནི་ཞེས་བྱ་བ་ལ་སོགས་པ་གསུམ་པ་གསང་བའི་དབང་བསྐུར་བ་སྟེ། དེ་ཡང་བླ་མ་ཕྱག་རྒྱ་དང་སྦྱོར་བ་ལས་བྱང་ཆུབ་ཀྱི་སེམས་བྱུང་སྟེ ༑ པདྨ་ལས་དངོས་སུ་ལྕེས་ལེན་པ་ནི་དབང་པོ་རབ་བོ། །དབང་པོ་འབྲིང་ནི་མཐེབ་སྲིན་གྱིས་སྟེར་ཞིང་ལེན་ནོ། །རྟོག་པ་ཆེ་བ་དབང་པོ་ཐ་མས་ ¡Ã ཆང་ ( ¡¶ ཞ ¡· ཞོ་ )( ¡¶ ལུ ¡· གོ་བ་ ) ལ་སོགས་པ་དང་སྦྱར་ཏེ་ལྕེས་བླང་ངོ༌། །དེ་ནས་སློབ་མ་ཞེས་བྱ་བ་ལ་སོགས་པས་ནི་གསང་བའི་དབང་གི་རྗེས་ལ། སློབ་མ་བསྲུང་བའི་ཕྱིར་བྱང་ཆུབ་ཀྱི་སེམས་ལ་ཨོཾ་ཨཱཿཧཱུྃ་བཟླས་ཏེ། གནས་གསུམ་དུ་ཐིག་ལེ་བྱས་ལ་བསྲུང་བའོ། །ཕྱི་ཉིན་ཞེས་པའང་དབང་གི་རྗེས་ཉིད་ལ་བྱ་གསུང་ངོ༌། ། བླ་མ་ལ་ལ་དེ་ལྟར་བཞེད་དོ། །བླ་མ་ཉིད་ནི་བགྲེས་པོ་རྣམས་ཀྱི་རྗེས་སུ་འབྲངས་ནས་ཕྱི་ཉིན་ཞེས་བྱ་བ་ལ་སོགས་པ་པྲ་སེ་ནའི་སྔོན་དུ་སློབ་མ་བསྲུང་བར་བྱ་གསུངས་ཏེ། གསང་དབང་གི་ཕྱིར་ཉིན་ནམ་ནུབ་གཉིས་པ་ལ་གསུང༌། ཐིག་ལེ་བསྲུང་ལུགས་འདྲ། གདོང་གཡོགས་དང་གསང་དབང་གི་དུས་ཀྱིའམ། ཡང་ན་ཤེས་རབ་ཡེ་ཤེས་ཀྱི་དུས་སུ་གདོང་གཡོགས་བཅིངས་པའམ། དགྲོལ་བའོ ༑ ༑ དཀྱིལ་འཁོར་བསྟན་ལ་ཕྱག་རྒྱའི་ལུས་ཀྱི་དཀྱིལ་འཁོར་དང་རྩ་བསྟན་པ་སྟེ། ཕྱག་རྒྱ་ནི་གསང་བའི་ཡུམ་དེ་ཉིད་དམ། ཡང་ན་གཞན་ལ་བྱའོ། །ལྷའི་གནས་རྣམས་ ( ¡¶ ཞ ¡· - རྣམས་ ) བསྟན་པ་ནི། སྐད་ཅིག་དང་དགའ་བའི་དབྱེ་བ་བསྟན་པའོ། །དེ་ནས་སློབ་མ་དང་སྙོམས་འཇུག་བྱེད་དུ་བཅུག་ལ། མཉམ་པའི་རོ་ལྷན་ཅིག་སྐྱེས་པ་སློབ་མའི་སྤྱོད་ཡུལ་དུ་ཉམས་སུ་མྱང་བར་བྱའོ། །ཨཱ་ལི་ཀཱ་ལི་བརྗོད་པར་བྱ་བ་ཡང་བླ་མ་ལ་ལ་གསུང་ན་གཞན་དུ་ཤེས་པར་བྱའོ། །སློབ་དཔོན་དང་ཞེས་བྱ་བ་ལ་སོགས་པས་ལྔ་པ་སློབ་དཔོན་ལ་ཡོན་དབུལ་བར་བསྟན་ཏོ། །དེ་ལྟར་ངེས་པ་ཞེས་བྱ་བ་ལ་སོགས་པས།

བླ་མ་ལ་ཡོན་ཕུལ་བའི་ཕན་ཡོན་བསྟན་ཏོ། །སྔགས་པས་ཞེས་བྱ་བ་ལ་སོགས་པས་ལྡོག་པའི་སྒོ་ནས་དཀྱིལ་འཁོར་དུ་མ་ཞུགས་པའི་ཉེས་དམིགས་བསྟན་ཏོ། །རྣལ་འབྱོར་འདི་ནི་ཞེས་བྱ་བས་ནི་མ་བླ་ན་མེད་པའི་བདེ་བ་མཆོག་གི་རྒྱུད་ཀྱི་ཆེ་བ་བསྟན་ཏོ། ། དགོད་དང་ཞེས་བྱ་བ་ལ་སོགས་པ་མི་དང་ལྷ་ལ་སྦྱར་བ་གཉིས་སུ་གསུངས་ཏེ་གཞན་དུ་གསལ་ལོ། །དེ་ལྟར་དབང་བསྐུར་བ་དེ་དང་དེ་ཁོ་ན་ཉིད་བསྟན་པ་དང་འབྲེལ་ཏེ། དེ་བས་ན་ལེའུ་དང་པོས་བསྐྱེད་རྫོགས་གཉིས་མདོར་བསྟན་པ་དང་ ༑ བཤད་པའི་དབང་དུ་བྱས་ཏེ་ལེའུ་བཞི་པས་དེ་ཁོ་ན་ཉིད་རྒྱས་པར་བཤད་དོ། །དེ་ལས་དང་པོ་བསྐྱེད་པའི་རིམ་པའི་དབང་དུ་བྱས་པ་དེ་ཁོ་ན་ཉིད་བཞི་ལས། དེ་ནས་རྐང་པ་གཉིས་ཀྱིས་དཀྱིལ་འཁོར་གྱི་དེ་ཁོ་ན་ཉིད་བསྟན་ཏེ། རྣལ་འབྱོར་མ་ནི་སོ་བདུན་ཡིན་ལ། ས་ལ་ཞེས་པས་མཚོན་ནས་རླུང་ནས་གཞལ་ཡས་ཁང་བསྐྱེད་པའི་བར་གྱིས་བསྟན་ཏོ། །བརྩོན་ཆེན་མ་ལ་སོགས་པས་ནི་ལྷའི་དེ་ཁོ་ན་ཉིད་བསྟན་སྟེ། རྣལ་འབྱོར་མ་སོ་བདུན་ལས་འདིར་སྐུ་གསུང་ཐུགས་ཀྱི་རྩ་ ( ¡¶ ལུ ¡· - རྩ་ ) ཉི་ཤུ་རྩ་བཞི་སྟེ། བརྩོན་འགྲུས་ཆེན་མོ་ནི་མས་དགོད་པ་ནི་ཀུན་སྤྱོད་ལྟར་རིམ་པ་བཟློག་པ་སྟེ། དགོས་པ་ནི་བླ་མ་མེད་པར་འཇུག་པ་དགག་པའོ། །རང་མིང་ཞེས་བྱ་ལ་སོགས་པ་སྔགས་ཀྱི་དེ་ཁོ་ན་ཉིད་བཟླས་པའི་སྔགས་བསྟན་ཏོ། །དཔའ་བོ་ཉིད་འགྱུར་ཞེས་བྱ་བས་ཡེ་ཤེས་ཀྱི་དེ་ཁོ་ན་ཉིད་བསྟན་ཏེ། ཡབ་ཡུམ་གནས་པར་ ( ¡¶ ལུ ¡· ལྔ་ ) བྱིན་གྱིས་བརླབ་པ་ལ་སོགས་པ་བྱེད་པ་སྟེ་སྙོམས་པར་འཇུག་པའི་བདེ་བས་མཆོད་པའོ། །ཡང་རྫོགས་པའི་རིམ་པའི་དེ་ཁོ་ན་ལྟར་ ( ¡¶ ཞ ¡· ཉིད་ ) བཤད་ན་ས་ལ་རྣམ་པར་རོལ་པ་ཞེས་བྱ་བ་ལ་སོགས་པ་ནི་ལུས་ཉིད་རྟེན་གྱི་དཀྱིལ་འཁོར་དུ་གྲུབ་པར་བསྟན་ཏེ ༑ རྐང་པའི་མཐིལ་དུ་གནས་པའི་རླུང༌། །རྩོམ་ཆེན་གཞུ་ཡི་དབྱིབས་ཀྱིས་ནི ༑ ༑ ཞེས་བྱ་བ་ལ་སོགས་པ་དཀྱིལ་འཁོར་གྱི་དེ་ཁོ་ན་ཉིད་དོ། །བརྩོན་ཆེན་ཞེས་བྱ་བ་ལ་སོགས་པས་ནི་རྣལ་འབྱོར་མ་སོ་བདུན་གྱིས།

ནང་དུ་རྩ་གསུམ་དུ་འདུས་པར་བསྟན་ཏེ། དེ་ལྷའི་དེ་ཁོ་ནའོ། །རང་མིང་ཞེས་བྱ་བ་ལ་སོགས་པས་རྩ་ཉི་ཤུ་རྩ་གསུམ་ལ་སོགས་པ་དགག་པའི་ཡིག་འབྲུ་བསྟན་ཏེ། དོན་གྱིས་བྱང་སེམས་རང་ཡིག་འབྲུའི་རྣམ་པར་ཡོད་པ་དེ་ལྟར་དམིགས་ཏེ་སྔགས་ཀྱི་དེ་ཁོ་ནའོ ༑ ༑ དཔའ་བོ་སྟེ་འགྱུར་ཞེས་བྱ་བ་ལ་སོགས་པས་ནི། རྩ་རྣམས་བྱང་ཆུབ་ཀྱི་སེམས་སུ་གནས་པ་དེ་བསྒོམ་པའི་སྟོབས་ཀྱི་འཕེལ་བར་འགྱུར་བ་ལས་དགའ་བ་བཞིའི་ཡེ་ཤེས་སྐྱེ་བ་ནི། ཡེ་ཤེས་ཀྱི་དེ་ཁོ་ནའོ། །དེ་བས་ན་ལེའུ་འདི་ལྟ་བུའི་རྫོགས་པའི་རིམ་པ་བཤད་པས། ལེའུ་གསུམ་པ་ཤེས་རབ་ཡེ་ཤེས་བསྟན་པའི་དོན་བཤད་པས་དབང་བཞི་པའི་དོན་བཤད་པར་ཡང་འགྲོ་གསུང་ངོ༌།། །། ད་ནི་ལེའུ་ལྔ་པ་སྔར་དབང་སྔོན་དུ་སོང་བའི་འབྲས་བུའི་གསང་བ་བསྒྲུབ་པ་ལ་འཇུག་ལ། དེ་ཡང་གཉིས་ཏེ་འབྲས་བུའི་གསང་བ་དངོས་དང་དེ་ལ་འཇུག་ པའོ ༑༑ འབྲས་བུ་ནི་མཆོག་གི་སིངྔྷི་དང་ཐུན་མོང་གི་སིངྔྷི་ལས་ཀྱི་ཚོགས་སོག་པ་རྣམས་སོ། །དེ་བསྒྲུབ་པ་ལ་འཇུག་པ་ལའང་གཉིས་ཏེ། བསྒྲུབ་པ་འཇུག་པ་དངོས་དང༌། དེའི་ཡན་ལག་སྔགས་བཏུ་བའོ། །དེའང་འབྲས་བུ་ལམ་གྱི་ཚོགས་བསྒྲུབ་པ་ལ་ནི་སྔགས་དགོས་ལ། དེ་ཡང་དག་པར་བྱ་དགོས་པས་ལྷག་ཆད་སྤང་བའི་ཕྱིར་སྔགས་བཏུ་བ་བསྟན་ཏེ། དེ་ཡང་ལེའུ་དང་པོ་སྔགས་བཟླས་ཞེས་བྱ་བ་ལ་སོགས་པའི་རྒྱས་པར་བཤད་པའོ། །ལེའུ་ལྔ་པ་འདི་ཡང་འགྲོ་ཞེས་གསུང་ངོ༌། །བཏུ་བའི་སྔགས་ཀྱང་ས་སྟེང་ན་ནི་རིན་ཆེན་བཞི་ཞེས་པས་ན། རྣམ་པ་བཞི་སྟེ་ཡབ་ཡུམ་གྱི་རྩ་སྔགས་གཉིས་སུ་མེད་པ་དང༌། སྙིང་པོ་གཉིས་སུ་མེད་པ་དང༌། ཉེ་སྙིང་གཉིས་ ( ¡¶ ལུ ¡· + གཉིས་ ) སུ་མེད་པ་དང༌། གོ་ཆ་གཉིས་སུ་མེད་པ་དང་བཞི་ཡིན་ཏེ། འདི་དག་གིས་དངོས་གྲུབ་རྣམས་འགྲུབ་པར་བྱེད་པས་རིན་པོ་ཆེ་བཞི་ཡིན་ནོ། །ཧཱུྃ་བཞི་པ་ནི་གྲུབ་པའི་བར་ཆད་སེལ་བར་བྱེད་པ་ཡིན་ཏེ། དེ་རང་གིས་མི་འགྲུབ་པས་གྲོགས་ཡིན་པས་རིན་ཆེན་དུ་མ་གྲགས་གསུང་

༑ རྩ་བའི་སྔགས་བཏུ་བ་ནི་ལེའུ་ལྔ་པས་སྔགས་ཀྱི་ལུས་དང༌། བདུན་པས་ཡན་ལག་བཏུ་བ་དང༌། ཉི་ཤུ་རྩ་ལྔ་པས་བཏུས་པའི་སྔགས་བསྟན་གསུང༌། སྙིང་པོ་ལ་སོགས་པ་བཏུ་བ་ནི་དྲུག་པས་ཟུར་ཙམ་དང༌། བརྒྱད་པས་བསྟན་ཏོ།། ཉེ་སྙིང་བཏུ་བ་ནི་ལེའུ་བཅུ་གཉིས་པ་ལ་སོགས་པས་བསྟན། གཞུང་ལས་ཐམས་ཅད་ཅེས་པ། ཨཱ་ལི་ཀཱ་ལི་སྟེ་ཀུན་སྤྱོད་ལས། ཨཱ་ལི་ཀཱ་ལི་མ་གཏོགས་པ ༑ ༑ འདི་ལྟར་ཆོས་གཞན་ཡོད་མ་ཡིན། །ཞེས་གསུངས་སོ། །ཆོས་འབྱུང་བཞི་གྲུ་གསུམ་མོ། །དེར་སྣ་ཚོགས་པདྨ་བྲིས་ལ། མི་ཤིགས་པས་ཏེ་ཨ་བཀོད་ལ། ཆོ་ག་བཞིན་ཞེས་པས་ཨཱ་ལི་ཀཱ་ལི་དགོད་པ་མཚོན་ཡང་མཆོད་ ( ¡¶ ཞ ¡· མཚོན་ ) པའོ། །རྟེན་ཆོས་འབྱུང་བ་བརྟེན་པ་ཨཱ་ལི་ཀཱ་ལི་བསྡུས་པའོ། །བསྒྱུར་ཞེས་པ་ནི་བསྒྱུར་གཞི་ཀཱ་ལི་སྒྱུར་བྱེད་ཨཱ་ལི་སྟེ། ཀ་ལྟ་བུ་ལ་ཨ་བཞག་པས་ཀ་བྱ་བ་སྲོག་ཅན་རང་གཅིག་ཡིན་ལ། ཨཱ་རིང་པོ་བཞག་པས་ཀཱ་རིང་པོར་འགྱུར་བ་ལ་སོགས་པའོ། །བཞི་བཞི་ནི་བཞི་ཚན་བཞི་སྟེ། སྒྱུར་བྱེད་ཨཱ་ལི་བཅུ་དྲུག་གོ ༑ ཨཱ་ལིས་བསྒྱུར་བའི་ཀཱ་ལིས། བགེགས་ལ་སོགས་པ་ ¡Ã ཐམས་ཅད་ ( ¡¶ ཞ ¡· - ཐམས་ཅད་ ) ཐུབ་པས་ཐུབ་པ་ ¡Ã བདུན་གྱིས་ ( ¡¶ ལུ ¡· གཅིག་གིས་ ) བསྟན་པ་ནི། བསྒྱུར་གཞི་ཀཱ་ལི་ཀ་ཚན་ལ་སོགས་པ་བདུན་ནོ། །དཔའ་བོ་ནི་གཙོ་བོ་སྟེ་ཡབ་ཡུམ་གྱི་རྩ་སྔགས་བཏུ་བ་སྟེ། ཕན་ཡོན་ནི་འདོད་པའི་དོན་སིངྔྷི་བསྒྲུབ་པའོ། ། བཞི་པ་ཡི་ནི་ཞེས་བྱ་བ་ལ་སོགས་ཀྱིས་ནི། རྩ་བའི་སྔགས་ཀྱི་ལུས་བཏུ་བ་བསྟན་ཏོ། །དེ་ནས་ལེའུ་བདུན་པ་དང་སྦྱར་བ་སྟེ། ཡན་ལག་བཏུ་བ་བསྟན་ཏོ།། དེ་ནས་ལེའུ་དྲུག་ ( ¡¶ ལུ ¡· བདུན་ ) པས་ནི་དཔའ་བོའི་གོ་ཆ་བཏུ་བ་བསྟན་ལ། བཅོམ་ལྡན་འདས་ཀྱི་སྙིང་པོ་དང༌། ཉེ་བའི་སྙིང་པོ་ཡང་མདོར་བསྟན་པ་ཙམ་སྟེ ༑ རྒྱས་པར་ནི་ལེའུ་བཅུ་གཉིས་པ་ལ་སོགས་པར་འཆད་དོ། །འདིར་བསྟན་པ་ཙམ་བྱེད་པ་ཡང་རྩ་བའི་སྔགས་དང་ཕྱོགས་གཅིག་ཏུ་བསྟན་པར་བྱ་བའི་ཕྱིར་གསུངས་སོ། །དེ་ནས་བརྒྱད་པ་དང་སྦྱར་ཏེ་སྙིང་པོའི་སྔགས་དང༌།

ཡུམ་གྱི་གོ་ཆ་བསྟན་ཏོ། །རིན་ཆེན་བཞི་ནི་རྩ་སྔགས་གཉིས་སུ་མེད་པ་ལ་སོགས་པ་སྟེ་སྔར་བརྗོད་པ་ཉིད་དོ། །ད་ནི་ལེའུ་དགུ་པས་ལེའུ་དང་པོར་འདོད་པའི་དོན་ཀུན་སྒྲུབ་བྱེད་ལ་སོགས་པ་དང༌། སྔགས་བཟླས་པས་བསམ་གཏན་ལ་སོགས་འབྲས་བུའི་གསང་བ་བཤད་པ་དེའི་རྒྱས་བཤད་འབྲས་བུའི་གསང་བ་ལ་འཇུག་པ་དངོས་བསྟན་ཏེ། དེ་ཡང་རིན་ཆེན་བཞིས་འབྲས་བུ་སྒྲུབ་པ་ལ་འཇུག་པ་ལས་དང་པོ་རྩ་བའི་སྔགས་ཀྱིས་སྒྲུབ་པ་ལ་འཇུག་པ་བསྟན། དེ་ཡང་ཐུན་མོང་སྒྲུབ་པ་ལ་འཇུག་པའོ། །དེ་ལ་ཡང་གཉིས་ཏེ། འཇིག་རྟེན་མནན་པ་དང་ལས་ཀྱི་ཚོགས་གཉིས་སུ་གསུང༌། ལྷ་ཐམས་ཅད་ཅེས་པ་ནས་གསོད་པར་བྱེད་དོ ༑ ༑ ཞེས་བྱ་བའི་བར་གྱི་འཇིག་རྟེན་མནན་པ་བསྟན་ཏེ། དབང་པོ་རབ་བསམ་བཏན་ཙམ་མམ། འབྲིང་སྔགས་དང་གཉིས་སམ། ཐ་མས་འཁྲུལ་འཁོར་དང་གསུམ་མམ། དེ་བས་ཐ་ཤལ་གྱིས་སྦྱིན་སྲེག་དང་བཅས་པས་ལྷ་རྣམས་ཀྱི་མངལ་འཕོ་ཞིང༌། ¡Ã མངལ་འཇིག་ ( ¡¶ ཞ ¡· འཇིགས་ ) པ་ལ་སོགས་པས་འཇིག་རྟེན་ཐམས་ཅད་འདུལ་ལོ། །སྤྲིན་རྣམས་ཞེས་པ་ལ་སོགས་པ་ནི་ཙཀྲ་ལ་བརྟེན་པ་དང་མ་བརྟན་པའི་ལས་དུ་མ་བསྟན་ཏོ།། །།ལེའུ་བཅུ་པས་ནི། ལེའུ་དང་པོ་འབྲས་བུའི་གསང་བ་བཤད་པ་ལས། མཆོག་ཐུན་མོང་གཉིས་ཀ་བསྒྲུབ་པར་བཤད་དོ། །དེ་བས་ན་སྐུ་གསུམ་ལམ་དུ་བྱེད་པས་ན་ལམ་གྱི་གསང་བ་བཤད་པར་ཡང་འགྲོ་གསུང་ངོ༌། །རིན་ཆེན་བཞི་པོས་ནི་ ¡Ã བསྙེན་པའི་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· སྙིང་ ) ལས་སྦྱོར་བསྒྲུབ་པའི་ལས་སྦྱོར་རོ། །སྐུ་གསུམ་ལ་ཡང་གཉིས། རྟོགས་པ་འཆར་བའི་དབང་དུ་བྱས་པ་དང༌། ལམ་དུ་བྱེད་པའི་དབང་དུ་བྱས་པའོ། ། རྟོགས་པ་འཆར་བ་ན་དང་པོར་ཆོས་སྐུ་འཆར། དེ་ནས་ལོངས་སྐུ། དེ་ནས་སྤྲུལ་སྐུའོ ༑ ༑ ལམ་དུ་བྱེད་པ་ན་དང་པོ་སྤྲུལ་སྐུ་ལམ་དུ་བྱེད། དེ་ནས་ལོངས་སྐུ། དེ་ནས་ཆོས་སྐུའོ། །དེ་ཡང་རིམ་པ་གཉིས་ལས་བསྐྱེད་པའི་རིམ་པའི་དབང་དུ་བྱས་ནས་ཀུན་སྤྱོད་དམ། ལཱུ་ཧི་པ་ལྟར་བྱང་ཆུབ་ཀྱི་ཕྱོགས་དྲན་པ་ཆོས་སྐུ།

ལུས་ཀྱི་དཀྱིལ་འཁོར་ལོངས་སྐུ། གོ་ཆ་ལ་སོགས་པ་སྤྲུལ་སྐུར་བྱ་བའམ་དེ་ལྟར་ལམ་གྱི་སྐུ་གསུམ་པོར་ཆོས་སྐུ་དང་པོར་ ¡Ã བཏང་ངོ་ ( ¡¶ ཞ ¡· ལྟར་རོ ) ། །ཡང་ན་ཀཾ་པ་ལ་ལྟར་ཐུགས་ཀྱི་དཀྱིལ་འཁོར་ལ་སོགས་པ་གང་གིས་གཙོ་བྱས་པ་ནི་གཙོ་བོ་ཡིན་ལ། གཉིས་པོ་ཕལ་པར་བྱས་པའི་ཙཀྲ་གསུམ་ལ་སྐུ་གསུམ་གཞག་པ་འམ། ཨ་བྷི་ངྷ་ན་ལྟར། ཡི་གེ་བཅུ་གཉིས་པའི་སྒྲུབ་ཐབས་ལྟར་རང་ལ་སྐུ་གསུམ་དུ་བཞག་པའོ་གསུང༌། དེ་ཡང་ཕྱག་རྒྱ་བཞི་དང་སྦྱར་བ་དང༌། རིམ་པ་བཞི་དང་སྦྱར་བའི་ཕྱོགས་གཉིས་གསུང༌། དང་པོ་ལྟར་ན་སྤྲུལ་པ་མཚོན་པར་བྱེད་པ་ལམ་རྐྱང་གསུང༌། ལཽངས་སྐུ་ཆོས་ཀྱི་ཕྱག་རྒྱ་བསྒོམས་པ་གསུང༌། ཆོས་སྐུ་མ་ཧཱ་མུ་དྲཱ་རྟོགས་པ་ལ་གསུང༌། ¡Ã ཨ་བྷི་ངྷ་ ( ¡¶ ཞ ¡· ཏི་ས་ ) ན་ནས་བཤད་པ་གཞན་དུ་སྣང་སྟེ། རྡོ་རྗེ་དྲིལ་བུ་པའི་རིམ་པ་ལྔ་པ་ལྟར་ཤེས་པར་བྱ་བའམ། སཾ་པུ་ཊ་ལྟར་ཤེས་པར་བྱ་བའམ། རིམ་པ་བཞི་དང་སྦྱར་ན་ལོངས་སྐུ་སྤྱི་བོ་ནས་སྙིང་གའི་བར་གྱིས་བདེ་བའི་ཡེ་ཤེས། སྤྲུལ་པ་སྙིང་ག་ནས་ལྟེ་བའི་བར་རོ། ། ཆོས་སྐུ་ལུས་ཐམས་ཅད་ལ་ཁྱབ་པ་ལྷན་ཅིག་སྐྱེས་པ་ལ་བྱ་གསུང༌། དེ་ལྟ་བུའི་སྐུ་གསུམ་ལམ་དུ་བྱས་པས་ཧེ་རུ་ཀ་ཡི་དངོས་གྲུབ་ཐོབ་ཅེས་བྱའོ། །གསང་སྔགས་འདིའི་སྙིང་པོ་ཞེས་བྱ་བ་ལ་སོགས་པས་ནི། སྔགས་ཀྱི་སྒོ་ནས་ཐུན་མོང་སྒྲུབ་པ་བསྟན་ཏོ།། །།ལེའུ་བཅུ་གཅིག་པས་ནི་ཐུན་མོང་གི་སིངྔྷི་སྒྲུབ་པའི་ཐབས་སྐྱེ་བ་བདུན་པ་བརྟག་པ་བསྟན་ཏེ། དེ་ཡང་ལེའུ་འདིས་བརྟག་པ་

¡Ã དངོས་ ( ¡¶ ལུ ¡· དང་པོ་ ) བསྟན། ¡Ã ང་གཅིག་ ( ¡¶ ཞ ¡· བཞི་ཞེས་ ¡¶ ལུ ¡· བཞི་རྩ་དགུ་ ) པས་རྒྱས་པར་བཤད། ( ¡¶ ལུ ¡· + བཅུ་ ) བཞི་པས་སྐྱེ་བ་གཅིག་པ་སྒྲུབ་པ་བསྟན་ ཏོ ༑༑ ༑༑ ལེའུ་བཅུ་གཉིས་པས་ནི་ཉེ་བའི་སྙིང་པོའི་ལས་སྒྲུབ་པ་སྟོན་ལ། དེ་ཡང་ཕལ་ཆེར་འཁོར་ལོ་ལ་བརྟེན་ཏེ་སྒྲུབ་པ་སྟོན་ཏོ། །ལྷག་མ་གཞུང་ཉིད་དུ་ཤེས་པར་བྱའོ། །འཁོར་ལོ་འདི་དག་ལ་ཡི་གེ་འདི་དགོད་པ་ནི་ཇོ་བོ་བྷ་དནྟ་ཞི་བ་དང༌། དྲག་པོ་ལ་མགོ་འོག་ཏུ་སྟོན་ལ། རྒྱས་པ་ལ་སོགས་པ་མགོ་བོ་ཕྱིར་སྟོན་པར་གསུང༌། ཇོ་བོ་བགྲེས་པོ་རྣམས་ཞི་བ་ལ་སོགས་པ་ལ་མགོ་ཕྱིར་སྟོན་ལ་དྲག་པོ་ལ་ནང་དུ་སྟོན་པར་བཞེད་དོ། །ཙཀྲ་ཐམས་ཅད་ཀྱི་རབ་ཏུ་གནས་པ་ནི ༑ འཁོར་ལོའི་ལྟེ་བ་ལ་བཀོད་པའི་ཡི་གེ་ལས་འཁོར་ལོ་བསྐྱེད་པ། ཡེ་ཤེས་ཀྱི་འཁོར་ལོ་སྤྱན་དྲངས་ཏེ་བསྟིམ་ལ། མཆོད་བསྟོད་བྱས་པ་ཙམ་གྱིས་ཆོག་གསུང་ ༑༑ ༑༑ ལེའུ་བཅུ་གསུམ་པས་ནི་དཔའ་མོའི་གོ་ཆ་གཉིས་ཀྱིས་ལས་བསྒྲུབ་པ་བསྟན་ཏོ། །འདི་ཡང་ཕལ་ཆེར་ནི་འཁོར་ལོ་ལ་བརྟེན་པའོ། །འཁོར་ལོ་ཇི་ལྟར་བྱ་བའི་རིམ་པ་ནི་གཞན་དུ་གསལ་བས་འདིར་མ་བྲིས་སོ།། །། ལེའུ་བཅུ་བཞི་པས་ནི་སྔར་གྱི་ཚོགས་ཀྱི་སིངྔྷི་མྱུར་དུ་འགྲུབ་པར་བྱ་བའི་ཕྱིར། སྐྱེ་བ་བདུན་པ་བསྒྲུབ་པ་བསྟན་ཏོ། །འདི་ཡང་ཨ་བྷི་ངྷ་ན་ནས་འབྱུང་བའི་ཆོ་གའི་སྒྲུབ་ཐབས་ལྟར་བསྒོམས་ཏེ། སྔོན་དུ་བསྙེན་པ་ལ་རྟགས་ཕྱུང་ལ། དེ་ནས་ཀཾ་པ་ལ་ལྟར་དཀྱིལ་འཁོར་རྫོགས་པར་ཆུ་རིས་བྱས་ལ། དེའི་ལྷོ་ཕྱོགས་སུ་བསྒྲུབ་བྱའི་མི་སྐྱེ་བ་གཅིག་ ( ¡¶ ཞ ¡· བདུན་ ) པ་ལ་སོགས་པའི་གཟུགས་འབྲས་ཕུབ་ལས་བྱ་བ་ལ་སོགས་པའི་ཆོ་ག་ལོགས་སུ་ཤེས་པར་བྱའོ། །ད་ནི་དེ་ལྟར་བསམ་གཏན་གྱི་གསང་བ་དང་འབྲས་བུ་གསང་བ་ལ་འཇུག་པའི་ཡན་ལག་རིགས་ལྡན་མ་མཆོད་པ་སྟོན་པ་ནི། ལེའུ་བཅུས་བསྟན་པ་ལས། དེ་ཡང་ལེའུ་བཅོ་ལྔ་པས་ནི་རིགས་ལྡན་མར་མཆོད་པ་ལ། རིགས་ལྡན་མ་ངོ་ཤེས་དགོས་ལ། ངོ་ཤེས་པ་ལ་བརྡ་དགོས་

( ¡¶ ཞ ¡· འདོགས་ ) པས་ཡི་གེ་སོ་སོའི་བརྡ་བསྟན་ཏེ། ངོ་མ་ཤེས་པར་མཆོད་ན་ནི་སྒྲུབ་པ་པོ་གདོན་ལ་སོགས་པས་བར་ཆད་དུ་འགྱུར་རོ། ། དེ་ཡང་ཡི་གེ་རེ་རེའི་བརྡས་ནི་སྤྱིར་རིགས་ལྡན་མའི་རྣལ་འབྱོར་པ་ཙམ་དུ་ངོ་ཤེས་པར་བྱེད་དོ།། །།ལེའུ་བཅུ་དྲུག་པས་ནི་སོ་སོའི་རིགས་ལྡན་ངོ་ཤེས་པར་བྱ་བའི་ཕྱིར་རིགས་བདུན་དུ་ཕྱེ་ལ། དེ་རེ་རེའང་ཁ་དོག་ལ་སོགས་པ་ལྔ་ལྔར་ཤེས་པར་བྱའོ། །རིགས་བདུན་གྱི་ཡང་ཕག་མོ་རྡོ་རྗེ་སེམས་དཔའི་རིགས་དང་ ༑ ཡང་འོག་ནས་དམར་ཞིང་སེར་ཞེས་བྱ་བ་ལ་སོགས་པས། རྡོ་རྗེ་སེམས་དཔའི་རིགས་བསྟན་པ་གཉིས་གཅིག་ཏུ་བྱས་ལ་རྡོ་རྗེ་སེམས་དཔའི་རིགས་ཏེ། སྔ་མ་ནི་རྒྱུ་རྡོ་རྗེ་འཛིན་པའི་དབང་དུ་བྱས་པ་ཡིན་ལ། ཕྱི་མ་ནི་འབྲས་བུ་རྡོ་རྗེ་འཛིན་གྱི་དབང་དུ་བྱས་པའོ་གསུང༌། ཁ་དོག་ལ་སོགས་མཚན་ཉིད་ལྔ་པོའང་འོག་མ་འོག་མ་ལ་མེད་པ་རྣམས་རིགས་ཀྱི་རིགས་པ་ཞེས་བྱ་བ་ལ་སོགས་པས་རིགས་དྲུག་ཏུ་བྱས་པའི་རྟགས་བསྟན་པ་སྟེ། དེ་ཡང་རྡོ་རྗེ་སེམས་དཔའི་རིགས་ཀྱི་རྟགས་ཚང་ལ། ལྷག་མ་ལྔ་ནི་སོ་སོར་རང་རང་གི་ཁ་དོག་དང༌། ཕྱག་མཚན་འཆིང་བ་དང༌། སྔགས་ཟློས་པ་ལ་སོགས་པ་ལས་ཤེས་པར་བྱའོ།། །།ལེའུ་བཅུ་བདུན་པས་ནི་རིགས་ལྡན་མ་རྣམས་ཀྱི་འཇིག་རྟེན་དང་འཇིག་རྟེན་ལས་འདས་པ་རྣམས་ཀྱི་མིང་གི་རྣམ་གྲངས་ཀྱི་སྒོ་ནས་མཚན་ཉིད་བསྟན་ཏོ། ། འཇིག་རྟེན་ལས་འདས་པའི་མིང་གི་རྣམ་གྲངས་ནི་བརྒྱད་དེ། འཇིག་རྟེན་ལས་འདས་པ་ ¡Ã ལོ་ཀ་ལ་ ( ¡¶ ཞ ¡· ཁོ་ན་ལས་ ¡¶ ལུ ¡· ལ་ ) གྲགས་པ་དང༌། འཇིག་རྟེན་ལས་འདས་པའི་སྤྱོད་པ་དང་ལྡན་པས་སོ། །འཇིག་རྟེན་པའི་མིང་གི་རྣམ་གྲངས་ནི་བདུན་ནོ། །དེ་ཡང་ལོ་ཀ་ལ་གྲགས་ཤིང༌། ལོ་ཀ་པའི་སྤྱོད་པ་དང་ལྡན་པའོ། ། བདུན་པོའི་མཚན་ཉིད་སོ་སོར་བཤད་པའི་མངོན་སུམ་ཞེས་བྱ་བ་ལ་སོགས་པས་བཤད་དོ། །མཁའ་འགྲོ་མ་རྣམས་ཀྱི་མཚན་ཉིད་བརྒྱད་བཤད་པ་ནི། མཆོད་པའི་རྟེན་བྲིས་སྐུ་ལ་སོགས་པའི་འབྱོར་པ་དང་མི་ལྡན་པ་ལ།

རྣལ་འབྱོར་མ་སུམ་ཅུ་རྩ་བདུན་ལ་མཆོད་པའི་རྟེན། སྡུད་བྱེད་བརྒྱད་ཀྱི་མཚན་མ་བརྒྱད་བྲིས་ལ་མཆོད་པའོ། །སྡུད་བྱེད་ལའང་ལུགས་གསུམ་སྟེ། པྲ་ཛྙཱ་རཀྵི་ཏས་ཀང་ཀ་ནའི་འགྲཻལ་པ་མ་ཚང་བའི་དགོངས་པ་ཁ་བསྐངས་པ་དང༌། བྷ་དནྟ་གསུངས་པ་ནཱ་རོ་པའི་དགོངས་པ་གཉིས་ཡོད་དེ་ལོགས་སུ་ཤེས་པར་བྱའོ།། བླ་མ་བགྲེས་པོ་རྣམས་ནི་གཟུགས་ཅན་མས་སྐུའི་ཕྱོགས་ཀྱི་བཞིའི་བསྡུམ་ ( ¡¶ ཞ ¡· བསྡུས་ ) ལ། ལཱ་མཱས་མཚམས་ཀྱི་བཞི་བསྡུས། ཡོངས་གྱུར་མས་གསུང་གི་ཆས་བཞི་བསྡུས། བྱིས་བཅས་མས་ཕྱོགས་ཀྱི་བཞི་བསྡུས། མི་ལྡོག་མས་ཐུགས་ཀྱི་བཞི་བསྡུས། ཨེ་ཧི་ཀཱ་ལ་མོས་མཚམས་ཀྱི་བཞི་བསྡུས་སོ། །ལྷག་མ་ཧེ་རུ་ཀའི་རྣལ་འབྱོར་མ་མོ་ཞེས་བསྟན་པ་དང༌། ¡Ã ཙུམྦི་ཀཱ་མ་སྨོས་པས་ཙུམྦ་ཀཱས་ ( ¡¶ ཞ ¡· ཙུ་པི་ཀ་མ་སྨོས་པས་ཙུ་པི་ཀས་ ) སྙིང་པོའི་རྣལ་འབྱོར་མ་བཞི་བསྡུས་པ་ལས་མེད ༑ ཧེ་རུ་ཀས་སྒོ་མཚམས་མ་བརྒྱད་བསྡུས་པའོ། །འོ་ན་འདས་པའི་མིང་གི་རྣམ་གྲངས་ན། སྡུད་པ་མོ་ཡོད་པས་སྡུད་པ་མོས་སྡུད་པ་མེད་དམ། ཞེ་ན། སྡུད་པ་མོ་དང་ཕག་མོ་དང་ཧེ་རུ་ཀ་ནི་དོན་གཅིག་སྟེ་རྣམ་གྲངས་ཡིན་ནོ་གསུང་ངོ༌།། དེ་བས་ན་དེས་སྒོ་མཚམས་མ་བརྒྱད་བསྡུས་ ( ¡¶ ཞ ¡· གསུངས་ ) སོ། །ལེའུ་ ¡Ã དང་པོས་ནི་ཙུམྦི་ཀཱས་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· བཅོ་བརྒྱད་པས་ནི་ཙུ་པི་ཀས་ ) བསྡུས་པའི་མཁའ་འགྲོ་མ་ནང་ཚན་གྱིས་ཕྱེས་བས་རིགས་དྲུག་ཏུ་བསྟན་ཏོ།། །།བཅུ་དགུ་པས་ ¡Ã ཙུམྦི་ཀཱས་ ( ¡¶ ཞ ¡· ཙུ་པི་ཀས་ ) བསྡུས་པའི་ལཱ་མཱ་ལ་ནང་མཚན་གྱིས་ཕྱེ་བའི་རིགས་ལྔར་བསྟན་ཏོ། །གཉིས་ཀྱིས་མཚོན་ནས་ཁཎྜཱ་རོ་ཧཱ་ ( ¡¶ ཞ ¡· ཧི་ ) ལ་སོགས་པ་རྣལ་འབྱོར་མ་ ¡Ã ཉེར་ལྔ་ ( ¡¶ ཞ ¡· གསུམ་སྔ་མ་ )( ¡¶ ལུ ¡· སོ་ལྔ་ ) ལ་ཡང་རིགས་དྲུག་དང་ལྔར་ཕྱེ་བས་བསྟན་པར་ཤེས་པར་བྱ་སྟེ། མཁའ་འགྲོ་རྒྱ་མཚོ་ལས་བསྟན་ཏོ་གསུང༌།། །།ལེའུ་ཉི་ཤུ་པས་ནི་དེ་ལྟར་རིགས་ལྡན་གྱི་མཚན་མ་ཁ་དོག་གི་མཚན་མ་རྣམས་ཀྱིས་རིགས་ངོ་ཤེས་པ་དང༌། བརྡའི་ལན་གྱིས་རྣལ་འབྱོར་མ་མཉེས་པར་མ་བྱས་ན།

བྱིན་གྱིས་རློབ་པར་མི་འགྱུར་བས་བརྡ་བསྟན་ཏོ། ། དེ་ཡང་ལེའུ་འདི་ལས་ནི་ལག་པའི་བརྡ་གཙོ་བོར་བྱས་ཏེ་བསྟན་ཏོ།། །། ལེའུ་ཉེར་གཅིག་པས་ནི་ལག་པའི་བརྡ་དང་དེ་གཞན་ལ་སྟོན་པ་གཙོ་བོར་བྱས་ནས་བསྟན་ཏོ།། །།ལེའུ་ཉེར་གཉིས་པས་ནི་ལག་པའི་བརྡ་འབའ་ཞིག་ཏུ་མ་ཟད་དེ། ལྕེ་དང་སྤྱི་བོ་ ( ¡¶ ཞ ¡· + སྟོན་པ་ ) ལ་སོགས་པ་ཡན་ལག་དུ་མ་གཙོར་བྱས་ནས་བསྟན་ཏོ།། །།ལེའུ་ཉེར་གསུམ་པས་ནི་དེ་ལྟར་ངོ་ཤེས་ནས་མཉེས་པ་དང༌། རྣལ་འབྱོར་མ་རྣམས་ཀྱང་གཅིག་ལ་ཆགས་པར་ཤེས་པར་བྱ་སྟེ་ ( ¡¶ ལུ ¡· + མ་ ) མཆོད་ནས་སིངྔྷི་མི་འབྱུང་བས། ཆགས་པའི་རྒྱུ་སོ་དང་སེན་མོ་བརྡབ་པ་ལ་སོགས་པ་དང༌། ཆགས་པའི་རྟགས་དངོས་བཞིན་འགྱུར་བ་ལ་སོགས་པ་བསྟན་ཏེ། དེ་ལྟ་བུར་གསལ་བ་ཞེས་བྱ་བའི་གོས་ནི། རྣལ་འབྱོར་མ་རྣམས་ལ་སྙོམས་པར་ཞུགས་པ་ལས་བདེ་བ་མི་རྟོག་པ་སྐྱེ་སྟེ། དེ་ཡང་དང་པོ་པའི་ཡེ་ཤེས་རེག་བྱ་བདེ་བ་དང་བཅས་པ་སྐྱེ། དེ་བསྒོམས་པ་ལས་ཆོས་ཀྱི་སྐུའི་རྟོགས་པ་སྐྱེ་ལ། དེ་ལས་ལོངས་སྐུ་དང་སྤྲུལ་སྐུ་འབྱུང་ངོ༌། །འདིའི་མཚན་མ་ཞེས་པས་མཚོན་པར་བྱེད་པ་དེ་ཕྱིས་ཕྲ་མོ་དང་ཐིག་ལེས་མཚོན་པ་སྟེ། དཔའ་བོ་ཡིན་ན་རྡོ་རྗེ་ནོར་བུར། རྣལ་འབྱོར་མ་ཡིན་ན་པདྨར། ཉི་མའི་དཀྱིལ་འཁོར་ཁ་སྦྱོར་བསྒོམས་ལ། དེའི་བར་དུ་རྡོ་རྗེ་ཕྲ་མོ་ལ་སོགས་པ་བསྒོམས་ལ། དེ་ལ་དམིགས་པས་དང་པོ་རེག་བྱ་བདེ་བ། དེ་ནས་མི་རྟོག་པ་བདེ་བ་དང༌། དེ་བསྒོམས་པ་ལས་མཐར་ཆོས་སྐུ་འགྲུབ་བོ། །སོགས་ ( ¡¶ ཞ ¡· + ཨེ་ ) ཞེས་བྱ་བ་ནི་མཚན་མ་ལ་སོགས་པ་ཁོངས་སུ་བསྡུ་བ་ཡང༌། འཁོར་ལོ་དང༌། པདྨ་དང༌། རིན་པོ་ཆེ་དང༌། སྣ་ཚོགས་རྡོ་རྗེ་དང༌། དུང་ལ་སོགས་པ་བསྡུའོ། །མཆོག་གཡོན་ཞེས་བྱ་བ་ནི་རྣལ་འབྱོར་མ་ལས་དང་པོ་དཔེའི་ཡེ་ཤེས་ཕྱག་རྒྱ་ཆེན་པོའི་ཡེ་ཤེས་སྐྱེ་ལ། དེ་ལས་དོན་གྱི་ཡེ་ཤེས་ཕྱག་རྒྱ་ཆེན་པོའི་རྟོགས་པ་སྐྱེ་བས་མཆོག་གོ་གསུང་ངོ༌།། །།ལེའུ་ཉེར་ ( ¡¶ ཞ ¡· བཅུ་ ) བཞི་པ་ལས་ནི་ལེའུ་

¡Ã དག་པའི་ ( ¡¶ ལུ ¡· བཅོ་ལྔ་པའི་ ) ཡི་གེ་རེ་རེའི་བརྡ་བསྟན་པའི་འཕྲོ་ཡི་གེ་འདུས་པའི་བརྡ་བསྟན་ཏེ། སློབ་དཔོན་མེད་པར་འཇུག་པ་དགག་པའི་དོན་དུ་དཀྲུགས་ཏེ་བསྟན་པའོ། །སྡེ་ཚན་བཞི་ཞེས་བྱ་བ་ནི། །གཟུགས་དང་ཁ་དོག་སྟོན་པའི་སྡེ་ཚན ༑ མཁའ་འགྲོ་མ་དང་ལཱ་མཱ་གཉིས་ཀྱི་མཚན་ཉིད་བསྟན་པ་དང༌། དེ་ནས་ལག་པའི་བརྡའི་སྡེ་ཚན་དང༌། ངག་གི་བརྡའི་སྡེ་ཚན་བཞི་གསུང༌། དེ་ལྟར་ལེའུ་བཅུས་རིགས་ལྡན་མཆོད་པའི་རྒྱས་བཤད་སྟོན་ཏོ།། །།ཉི་ཤུ་ལྔ་པས་ནི། འབྲས་བུའི་གསང་བ་ལ་འཇུག་པ་དང༌། དེའི་ཡན་ལག་ཏུ་ལེའུ་ལྔ་པས་རྩ་བའི་སྔགས་ཀྱི་ལུས་བསྟན་ལ། ¡Ã བདུན་པས་ ( ¡¶ ཞ ¡· གཅིག་ ) ཡན་ལག་བསྟན་པའི་འཕྲོས་འདིར་རྐང་པ་བརྒྱད་ཀྱི་རྣམས་དང༌། ཕཊ་བརྒྱད་ཀྱང་ཐུ་ལ། བཏུས་པའི་རྩ་བའི་སྔགས་རང་ཡང་བསྟན་ཏེ། འདི་ལྟར་འཁྲུགས་པ་ཡང་སློབ་དཔོན་མེད་པར་འཇུག་པ་དགག་པའི་ཕྱིར་རོ།། །།ལེའུ་ཉེར་དྲུག་པས་ནི་གོང་དུ་བསྟན་པའི་ཚུལ་གྱིས་རིགས་ལྡན་མ་རྣམས་ངོ་ཤེས་པར་བྱས་ཤིང་དཔའ་བོ་དང་ ༑ རྣལ་འབྱོར་མ་གཅིག་ལ་གཅིག་རྗེས་སུ་ཆགས་པར་བསྟན་ཡང༌། ཚུལ་ཇི་ལྟར་བསྟེན་པར་བྱ་བ་མ་བསྟན་པས་བསྟེན་པའི་ཚུལ་བསྟན་པ་དང༌། ལམ་གྱི་གྲོགས་ཐུན་མོང་མ་ཡིན་པའི་ཚིག་བརྒྱད་བསྟན་ཏོ། །དེ་ཡང་ཕོ་ཉ་མོ་དེ་གྲུབ་སྟེར་ཡན་ཆད་ཀྱིས་ཕོ་ཉ་མོ་གསུམ་པོ་རང་བསྟན་ཏོ། །མཐོང་ཞེས་བྱ་བ་ལ་སོགས་པས་སྔགས་སྐྱེས་ཀྱི་ཕོ་ཉ་མོ་བསྟེན་པའི་ཚུལ་བསྟན་ཏེ། དེ་ཡང་ཟླ་བ་ཕྱེད་དུ་འཇམ་རླུང་སྦྱང་ཞིང་བསླབས་ཏེ། མཐོང་བ་ནི་སྐར་ཁུང་ལ་སོགས་པ་ནས ༑ ཕྱག་རྒྱའི་གཟུགས་མཐོང་བ་ཙམ་གྱིས་བདེ་བ་ལ་བསླབ་ཅིང་སྦྱང་ངོ༌། ། ¡Ã དེ་ནས་ལུས་ལ་རེག་ཅིང་ཞག་གསུམ་སྦྱང་ངོ༌། ། ( ¡¶ ཞ ¡· - དེ་ནས་་་་་་་སྦྱང་ངོ༌། ། ) དེ་ནས་འོ་བྱ་ཞིང་ཞག་གསུམ་སྦྱང་ངོ༌། །དེ་ནས་ལུས་ལ་འཁྱུད་པའི་བདེ་བ་ལ་བསྒྲུབ་ ( ¡¶ ཞ ¡· བསླབ་ ) ཅིང་རླུང་སྦྱང་ངོ༌། །དེ་ནས་གནས་ཀྱི་ཁྱད་པར་ཞེས་བྱ་བ ༑ རྡོ་རྗེ་དང་པདྨ་བསྲུབས་ཏེ་ཐིག་ལེ་བྱུང་བ་ཡང་ཡང་སོ་སོར་དྲངས་ཏེ།

ལུས་ལ་ཁྱབ་པར་བྱ་ཞིང་ཞག་གསུམ་དུ་བསླབ་ཏེ། བདེ་བས་བརྒྱལ་བ་འབྱུང་ངོ་ ༑ ༑ མ་དང་སྲིང་མོ་ཞེས་པའི་དོན་ནི་སེམས་བརྟན་པ་ཐོབ་པས་ཐ་མལ་པའི་མ་ལ་སོགས་པའི་བར་རྟོག་པ་མེད་པར་བསླབ་པའམ། ཡང་ན་ལས་དང་པོ་པས་རང་གི་རིགས་གང་ཡིན་པའི་བཙུན་མོ། ཌོ་བི་ནཱི་ལ་སོགས་པ་ཡོན་ཏན་ལྔ་དང་ལྡན་པར་བྱ་ཏེ་སྤྱད་པའོ། །ཡོན་ཏན་ལྔ་ནི་ཕན་པར་བྱེད་པ་དང༌། བརྩེ་བ་ཆེ་བ་དང༌། ངག་ཉན་པ་དང༌། བདེ་བ་བསྐྱེད་པ་དང༌། བསྒྲུབ་པའི་བྱ་བ་བྱེད་པའོ ༑ ༑ གང་ཞིག་ཅེས་བྱ་བ་རྐང་པ་གཉིས་ཀྱིས་ཕྱག་རྒྱ་དེ་ཡང་དང་པོར་དམ་ཚིག་ལ་བསླབ་པ་གོམས་པར་བྱས་ལ་བསྟེན་པར་བསྟན་ཏོ། །དེ་ནས་ཞེས་བྱ་བ་ལ་སོགས་པ་ནི་བསྲུང་བའི་དམ་ཚིག་བསྟན་ཏོ། །དམ་ཚིག་རྣམས་ཀྱང་བསྐྱེད་པའི་བར་གྱིས་གཡོན་གྱིས་ཀུན་ཏུ་སྤྱོད་པ་གཙོར་བྱས་ཏེ། དམ་ཚིག་སྤྱི་བསྟན་ཏོ། །ལྷ་གནས་ ( ¡¶ ཞ ¡· གཞན་ ) ཞེས་བྱ་བ་ལ་སོགས་པ་ཚིག་རྐང་རེ་རེ་ཐུན་མོང་མ་ཡིན་པའི་དམ་ཚིག་བརྒྱད་བསྟན་ཏོ། །བློ་ཆུང་ཞེས་བྱ་བ་ལ་སོགས་པས་སྔགས་སྐྱེས་ཀྱི་ཕོ་ཉ་མོའི་གསང་རྩ་རྒྱས་པར་བྱ་བའི་ཐབས་བསྟན་ཏོ། །དག་དང་ཞེས་བྱ་བས་ཕོ་ཉ་མོ་བསྟན་པ་དང༌། དམ་ཚིག་སྐྱོང་བའི་བསྒྲུབ་པ་པོ་བསྐྱེད་པ་ལ་སློབ་པ་དང༌། རྫོགས་པ་ལ་སློབ་པ་དང༌། བསྐྱེད་རྫོགས་གཉིས་ཀ་ལ་སློབ་པ་གསུམ་བསྟན་ཏོ།། །།ཉེར་བདུན་པས་ནི་དེ་ལྟར་དམ་ཚིག་སྐྱོང་བ་དེ་ཡང༌། སྤྱོད་པ་དང་ལྡན་པ་གཅིག་གིས་དམ་ཚིག་བསྟེན་པ་ཡིན་པས་སྤྱོད་པ་བསྟན་ལ། དེ་ལའང་བཞི་ལས་འདིར་སྤྲོས་བཅས་སྤྱོད་པ་བསྟན་ཏེ། དེ་ལ་ཡེ་ཤེས་རྒྱུས་སྐྱེས་བྱ་བ་ཡན་ཆད་ཀྱིས་ནི། སྤྱོད་པའི་གྲོགས་ཕོ་ཉ་མོ་བཟུང་བ་བསྟན་ཏོ། །བསམ་པའི་ཞེས་བྱ་བ་ལ་སོགས་པས་བསམ་པ་ལ་སོགས་པའི་ཁྱད་པར་བསྟན་ཏེ། བསམ་པའི་ཁྱད་པར་དང༌། ལུས་ཀྱི་ཁྱད་པར་དང༌། ངག་གམ་སྔགས་ཀྱི་ཁྱད་པར་དང༌། དམ་ཚིག་གི་ཁྱད་པར་དང༌། ¡Ã དགོས་པའི་ཁྱད་པར་དང༌། ( ¡¶ ཞ ¡· - དགོས་པའི་ཁྱད་པར་དང༌། ) ཁ་དོག་དང་གོས་ལ་སོགས་པའི་ཁྱད་པར་དང༌།

མཆོད་ཕྱིར་ཐོགས་པ་ལ་སོགས་པའི་ཁྱད་པར་དང༌། གསང་བར་བྱ་བ་དང༌། གཙང་སྦྲ་ལ་སོགས་པའི་ཁྱད་པར་དང་བརྒྱད་དོ། །རྣལ་འབྱོར་པས་ནི་ཞེས་བྱ་བ་ལ་སོགས་པས་ནི། སྤྱི་དོན་གསུམ་པར་སྤྱོད་པའི་ཕན་ཡོན་བསྟན་ཏོ ༑ ༑ དེ་ལྟར་རྟག་ཏུ་ཞེས་བྱ་བ་ལ་སོགས་པས་ནི། བསྒྲུབ་པའི་བྱ་བ་གཉིས་ལས་སངས་རྒྱས་ལ་མཆོད་པ་ལ་སོགས་ཚོགས་ཀྱི་འཁོར་ལོ་དང༌། མཆོད་ནས་དམ་ཚིག་ཅན་ལ་གཏོར་མ་སྦྱིན་པ་བསྟན་ཏོ། །དེ་ལ་ཡང་དང་པོར་ཆོ་ག་བསྟན། དེ་ནས་དུས་བསྟན། དེ་ནས་གཏོར་མ་དབུལ་བ་བསྟན། ¡Ã དེ་ནས་བསྒྲུབ་པ་པོ་བསྟན། ( ¡¶ ཞ ¡· - དེ་ནས་བསྒྲུབ་པ་པོ་བསྟན། ) དེ་ནས་གཏོར་མ་བཏང་བའི་ཕན་ཡོན་བསྟན ༑ དེ་ནས་དུས་དང་ཆ་ལུགས་བསྟན་ཏེ་དེ་ལྟར་དྲུག་ཏུ་གནས་སོ། ། ¡Ã གཉིས་ ( ¡¶ ལུ ¡· ཉེར་ཤུར་རྩ་བརྒྱད་ ) པ་ནི་དེ་ལྟར་གོང་དུ་བསྟན་པའི་སྤྲོས་བཅས་ཀྱི་སྤྱོད་པའི་བྱ་བ་དང་བྱ་བ་མ་ཡིན་པའི་བྱེ་བྲག་བསྟན་ཏེ། དེ་ཡང་དོན་དྲུག་ཏུ་གནས་ཏེ། མ་དད་ཐམས་ཅད་སུན་འབྱིན་བྱེད། །ཅེས་པའི་བར་གྱིས་བདག་གཞན་གཉིས་ཀ་མཆོད་པའི་བྱེ་བྲག་བསྟན་ཏེ། མཆོད་པའི་དགོས་པ་ནི་མྱ་ངན་ལས་འདས་པ་ཐོབ་པར་འགྱུར་བའི་དགོས་པའོ། །མཆོད་པའི་ཡུལ་ནི་བདག་ཉིད་དང་གཞན་ཡང་བྱང་ཆུབ་སེམས་དཔའ་དང༌། ཉན་ཐོས་ལ་སོགས་པ་དང༌། རྟེན་དང་བརྟེན་པར་བཅས་པ་དང༌། རྣལ་འབྱོར་མ་དང་འཇིག་རྟེན་པའི་འབྲེལ་བ་ལ་སོགས་པ་རྣམས་ཏེ། དེ་ལ་ནི་ཟས་ལ་སོགས་པས་མཆོད་པ་བྱའོ། །ཡང་རྣལ་འབྱོར་མ་རྣམས་ལ་ནི་བཟའ་བས་འདུས་པ་ཞེས་བྱ་ ¡Ã བ་སྙོམས་འཇུག་གི་བདེ་བས་ཀྱང་ ( ¡¶ ཞ ¡· བས་ ) མཆོད་པ་བྱའོ། །ཡང་བྱ་བ་མ་ཡིན་པའི་བྱེ་བྲག་འཇིག་རྟེན་པའི་འབྲེལ་བ་རྣམས་དང་བདེ་སྡུག་ཐུན་མོང་དུ་སྤྱད་པར་བྱ་བ་ཡིན་གྱི ༑ བྱ་བ་སྔགས་དང་ཕྱག་རྒྱ་ལ་སོགས་པ་ནི། སྦྱིན་པར་མི་བྱ་ཞེས་བསྟན་ཏོ།། དག་བྱེད་ཅེས་བྱ་བ་ལ་སོགས་པས་ནི་དོན་གཉིས་པ་དམ་ཚིག་གི་བྱེ་བྲག་བཟའ་བའི་དམ་ཚིག་བདུད་རྩི་ལྔ་ ¡Ã བསྟན་ཏེ། བཟའ་བའི་དམ་ཚིག་བདུད་རྩི་ལྔ་བསྟེན་

༑ ༑ དེ་གསུམ་ཡང་རྡོ་རྗེ་ ¡Ã སེམས་དཔའི་ ( ¡¶ ཞ ¡· སློབ་དཔོན་གྱི་ ) རིགས་གཅིག་ཏུ་བྱའོ། །བ་ལང་ཤ་ཞེས་བྱ་བ་ལ་སོགས་པ་ནི་དོན་ལྔ་པ་སྦྱིན་སྲེག་གི་བྱེ་བྲག་བསྟན་ཏེ། དེ་ཡང་དང་པོར་ཕྱིའི་དྲག་པོའི་སྦྱིན་སྲེག་བསྟན། དེ་བཞིན་ཞེས་བྱ་བ་ལ་སོགས་པས་ནི་ནང་གི་སྦྱིན་སྲེག་ཁ་ཟས་སམ་ཚོགས་ལ་ལོངས་སྤྱོད་པའི་ཚུལ་བསྟན་ཏོ། །ད་ནི་ཕོ་ཉ་ཞེས་བྱ་བ་ལ་སོགས་པས་ནི་བླ་ན་མེད་པའི་སྦྱིན་སྲེག་བསྟན་པར་འགྲོ་ལ། དོན་དྲུག་པ་ཕོ་ཉ་བསྟེན་ ( ¡¶ ཞ ¡· བསྙེན་ ) པ་དང༌། དེའི་དགོས་པ་བསྟན་ཏོ། །འདི་ཐམས་ཅད་ཀྱིས་ཀྱང་ལམ་གྱི་གསང་བའི་ཡན་ལག་བསྟན་པར་ཤེས་པར་བྱའོ།། །།ལེའུ་ཉེར་དགུ་པ་འདིས་ནི་ལམ་གྱི་གསང་བ་ལྷན་ཅིག་སྐྱེས་པའི་ཡེ་ཤེས་ཀྱི་གྲོགས་སམ་ཡན་ལག ཕོ་ཉ་མོའི་མཚན་ཉིད་དང༌། དེ་བསྟེན་པའི་ཚུལ་ལས་དང་པོ་པའི་དབང་པོ་མིན་པ་ཅི་ནས་ཀྱང༌། ལས་ཀྱི་ཕྱག་རྒྱ་ལ་བརྟེན་ནས་དཔེའི་ཡེ་ཤེས་མྱོང་བར་བྱ་ལ། དེ་ལ་བརྟེན་ནས་དོན་གྱི་ཡེ་ཤེས་རྟོགས་ ( ¡¶ ལུ ¡· + པའི་ཁྱད་ ) པར་བསྟན་ཏེ། དེ་ཡང་གསང་ལ་བསྟེན་པའོ ༑ ༑ ནུས་པ་ཐོབ་པས་ནི་ཕྱོགས་ལས་རྣམ་པར་རྒྱལ་བའི་སྤྱོད་པའི་དུས་སུ་མ་གསང་བར་བསྟན་ཏོ། །ནུས་པ་མ་ཐོབ་པར་འཇིག་རྟེན་པའི་མངོན་དུ་སྤྱོད་པ་སྤྱད་ན། འཇིག་རྟེན་པས་འཕྱ་བར་འགྱུར་བས་རྣལ་འབྱོར་པ་རང་ཉིད་ཀྱི་སྐྱེ་བོ་མི་གནས་པ་ལ་སོགས་པར་བསྟན་ཏོ། །རྣལ་འབྱོར་ཆེ་མཆོག་ཅེས་པ་ལ་སོགས་པས། དཔལ་ལྡན་བདེ་མཆོག་འཁོར་ལོའི་རྣལ་འབྱོར་བཞི་ལ་གནས་པའི་རྣལ་འབྱོར་ཆེན་པོའི་དེའི་ཆེ་བ་བསྟན་ཏོ། །དེ་ནས་ལེའུ་སུམ་ཅུ་པས་སུཾ་བྷ་ནི་བཏུ་བའི་ཆོ་ག་བསྟན་ཏོ། །ལེའུ་ ¡Ã བཅུ་གསུམ་པ་ ( ¡¶ ཞ ¡· སོ་གཅིག་ ¡¶ ལུ ¡· ལེའུ་སུམ་ཅུ་གསུམ་པ་ ) སོགས་ཀྱིས་ནི་ལེའུ་དང་པོའི་གསང་བ་སྒྲུབ་པ་བཤད་པའི་སྐབས་སུ་བར་མ་དབུགས་དབྱུང་བ་ཞེས་བྱ་བ་ལ་སོགས་པ་བཟའ་བའི་དམ་ཚིག་བསྟེན་པ་བསྟན་པ་ནི། མདོར་བསྟན་པ་ཡིན་ལ། ཡང་ལེའུ་དང་པོར་ལམ་གྱི་གསང་བ་བཞག་པ་ཡང་གསང་བའི་བཤད་པའི་ལེའུ་འདི་དག་ལས་རྟོགས་པར་བྱའོ།

།དེ་ཡང་སུམ་ཅུ་ ¡Ã སོ་གཅིག་ ( ¡¶ ཞ ¡· རྩ་གཉིས་ ) པ་ལས་གསུམ་ལས། གྲུབ་པར་འགྱུར་ཞེས་བྱ་བའི་བར་གྱིས་བཟའ་བའི་རྫས་བསྟན། དུང་དང་མུ་ཏིག་ཅེས་བྱ་བ་ལ་སོགས་པས་བཟའ་བ་འདོད་པ་བརྩད་པའི་སྐྱོན་སྤང་བ་བསྟན ༑ བརྡ་ནི་ཞེས་བྱ་བ་ལ་སོགས་པས་ནི་སྔར་བསྟན་པའི་རྫས་རྣམས་ཀྱི། ཚོགས་ཀྱིས་མཆོད་པ་ལ་སོགས་པའི་དུས་སུ་ལག་པའི་མཆོད་པ་ཇི་ལྟར་བྱ་བའི་ཆོ་ག་བསྟན་ཏོ། །ལེའུ་སོ་གསུམ་པ་ལ་ཡང་དོན་གསུམ་ལས། འགྲུབ་པར་འགྱུར་ཞེས་བྱ་བའི་བར་གྱིས་ནི། བཟའ་བའི་དམ་ཚིག་གི་རྫས་བསྟེན་པའི་ཕན་ཡོན་དང་བཅས་པ་བསྟན་ཏོ། །ཀྲྀཥྞ་ཞེས་བྱ་བ་ལ་སོགས་པས་ནི་སྔར་བསྟན་པའི་རྫས་རྣམས་དང༌། ཁྱི་མོ་ནག་མོའི་ཐུགས་ལ་སོགས་པའི་བསྲེག་རྫས་བྱ་སྟེ། རྩ་བའི་སྔགས་ལ་བསྙེན་པ་སྔོན་དུ་སོང་ནས་རོ་གཙང་མ་བླངས་ཏེ་རོ་ལངས་བསྒྲུབ་པ་བསྟན་ཏོ། །བ་ལང་ལྷས་ཞེས་བྱ་བ་ལ་སོགས་པས་ནི་ལེའུ་དང་པོར་ཧེ་རུ་ཀ་ཡང་དག་སྦྱོར་བ་ལ་སོགས་པས་མདོར་བསྟན་པའོ། །རྒྱས་པར་བཤད་པའི་དབང་དུ་བྱས་ཏེ། བརྟེན་པ་བསྐྱེད་པ་མངལ་སྐྱེས་ཀྱི་ཚུལ་དུ་རྒྱུ་རྡོ་རྗེ་འཛིན་པ་བསྐྱེད་པ་དང༌། འབྲས་བུ་རྡོ་རྗེ་འཛིན་པ་གཉིས་སུ་ཕྱེ་སྟེ་བསྟན་ཏོ ༑ ༑ ལེའུ་སོ་བཞི་པ་ནས་རྩ་བདུན་པའི་བར་ལེའུ་བཞི་ནི་དྲང་དོན་དུ་ཕྱིའི་སྦྱིན་སྲེག་དང༌། བཟླས་པ་ལ་སོགས་པ་བསྟན་པ་ལས་ལམ་གྱི་གསང་བ་སྒྲུབ་པའི་ཡན་ལག་བསྟན་ལ། ངེས་དོན་དུ་ཕྱག་རྒྱ་བཞི་བཤད་པས་ནི་ལེའུ་དང་པོའི་མཆོད་གཞི། དཔལ་ཆེན་པོ་ལ་སོགས་པའི་ལམ་གྱི་གསང་བའི་རྫོགས་རིམ་རྒྱས་པར་འཆད་དོ།། །།དེ་ཡང་ལེའུ་སོ་བཞི་པས་ནི་དྲང་དོན་དུ་ཕྱིའི་ ¡Ã སྦྱིན་སྲེག་བསྟན་ལ། ངེས་དོན་དུ་བདེ་བ་སྔོན་དུ་སོང་བའི་སྟོང་པ། བདེ་བ་མི་རྟོག་པ་བསྒོམ་པ་ཕྱག་རྒྱ་ཆེན་པོ་བསྟན་ཏོ། །དེ་ལ་དྲང་དོན་དུ་བཤད་ན། ཐེ་ཚོམ་མེད་པ་ཡན་ཆད་ཀྱིས་སྦྱིན་སྲེག་གི་སྔོན་དུ་བསྐྱེད་པའི་རིམ་པ་ལྷ་སོ་བདུན་རྫོགས་པར་བསྒོམ་པར་ ( ¡¶ ཞ ¡· རིམ་པ་ལྷ་གསུམ།

༑ དེའི་བདེ་བས་ཀྱང་དཀྱིལ་འཁོར་ཡང་བདེ་བར་བསྒོམ་པ་དང༌། དེ་ནས་དཀྱིལ་འཁོར་ཡང་རིམ་གྱིས་བསྡུ་བ་བཤད་དོ། །ཆགས་བྱས་ལ་ཞེས་པས་ཡབ་ཡུམ་འོད་དུ་ཞུ་བ་བསྟན་ཏོ། །ཐིག་ལེ་དམར་སྐྱ་བདེ་བའི་རང་བཞིན་དུ་བསྒོམ་པའོ ༑ ༑ དེ་ནས་མེས་བསྲེགས་ཏེ་ཞེས་བྱ་བ་ལ་སོགས་པས་ནི། དེ་ལྟ་བུའི་བདེ་བ་མི་རྟོག་པ་ལ་བློ་གནས་པས་ཕུང་པོ་ལྔ་དང༌། ཡང་ཕུང་པོ་ལྔ་དང༌། འབྱུང་བ་བཞིའི་རྟོག་པ་སྟེ། རྟོག་པ་སྲེག་པ་བསྟན་ཏོ། །དེ་ནས་འབར་ཞེས་བྱ་བ་ལ་སོགས་པས་ནི། བདེ་བའི་མི་རྟོག་པས་བློ་གཡེངས་ན་ཡང་དཀྱིལ་འཁོར་བློས་སྤྲོས་ཏེ་བསྒོམ་པར་བསྟན་ཏོ། །ཡང་དཀྱིལ་འཁོར་ལ་ཤེས་པ་ཆེར་སྤྲོས་ཏེ་སོང་ཡང་ ༑ བདེ་བ་མི་རྟོག་པ་བསྒོམ་མོ། །ཆོ་གའི་རིམ་པའི་ཞེས་བྱ་བའི་དོན་ཡང་ཐིག་ལེ་ལ་དམིགས་ཏེ། བདེ་བའི་མི་རྟོགས་པ་བསྒོམས་པས་དཀྱིལ་འཁོར་ལ་སོགས་པ་ཐམས་ཅད་བདེ་བ་མི་རྟོག་པར་བྱ་བར་བསྟན་ཏོ། །ཞིབ་ཚེགས་སུ་ནི་གཞུང་ཉིད་དུ་ཤེས་པར་བྱའོ།། །།ལེའུ་སོ་ལྔ་པས་ནི་དྲང་དོན་དུ་སྔགས་ཀྱི་བཟླས་པ་བསྟན་ལ། ངེས་དོན་དུ་ནི་ཆོས་ཀྱི་ཕྱག་རྒྱ་རིམ་པ་བཞི་བསྒོམ་པ་བསྟན་ཏོ། །དེ་ཡང་དེ་ནས་ ¡Ã ཞེས་བྱ་བ་རྐང་པ་གསུམ་གྱིས་དྲང་དོན་དུ་བཟླས་པ་དང༌། ངེས་དོན་ཆོས་ཀྱི་ཕྱག་རྒྱ་རྫོགས་རིམ་གྱི་སྔོན་དུ་བསྐྱེད་པའི་རིམ་པ་བསྒོམ་པ་བསྟན་ཏོ། །ལས་ནི་ ( ¡¶ ཞ ¡· - ཞེས་བྱ་་་་་་་ཏོ། །ལས་ནི་ ) ཞེས་བྱ་བ་ལ་སོགས་པ་ནི་བསྐྱེད་པའི་རིམ་པའི་དེ་ཁོ་ན་ཉིད་མཆོད་པའི་བར་ ( ¡¶ ཞ ¡· དབང་ ) དུ་བྱས་ནས། དེ་ནས་སྙོམས་པར་ཞུགས་པ་ལས་བྱུང་བའི་བྱང་ཆུབ་སེམས་དེ་རིན་པོ་ཆེ་བཞི་གང་ཡང་རུང་བའི་སྔགས་སུ་བསམས་ལ། དེ་ནས་སྔགས་འཁོར་བའི་ལམ་དུ་བཟླས་པ་བྱས་ཏེ། ལས་བཞིའི་པོའི་བྱེ་བྲག་གིས་ཁ་དོག་དང་དབྱིབས་རྣམས་ཤེས་པར་བྱའོ། །འདིར་ནི་ཞི་བ་དང་དྲག་པོའི་སྔགས་ཀྱི་བཟླས་ལུགས་གཉིས་ལས་མེད་དེ། དེ་རྩུབ་ཅིང་དྲག་པ་མ་ཡིན་ནོའི་བར་གྱིས་ཞི་བ་གཙོར་ཤེས་པར་བྱའོ།

།ཞི་བ་ལ་ནི་ཞེས་པ་རྐང་པ་གཉིས་བྷ་དནྟའི་དཔེ་ལ་མེད་ལ། བླ་མ་བགྲེས་པོ་རྣམས་ཀྱི་རྒྱ་དཔེ་ལ་ནི་སྣང་ངོ༌། །ས་བོན་ཕྲེང་བ་ཞེས་བྱ་བ་ལ་སོགས་པས་ནི་དྲག་པོའི་སྔགས་བཟླས་ལུགས་གཙོར་སྟོན་ཏེ། དེ་ཡང་དབུགས་ལ་མདའ་ནི་སྦྱར་བར་བྱའི་བར་གྱིས་ཚར་བཅད་པ། གསད་པའི་ལས་བསྒྲུབ་པ་བསྟན་ལ། དུས་མིན་ཞེས་པ་མན་ཆད་ཀྱིས་གསད་པའི་བསྒྲུབ་བྱ་དེ་རྗེས་སུ་གཟུང་བ་བསྟན་ཏེ། དེ་ཡང་སྤྱི་བོར་ཁ་ཊྭཱཾ་ག་བསྒོམ་པ་དང༌། ཐོད་པ་བསྒོམ་པའི་མན་ངག་གཉིས་སུ་ཤེས་པར་བྱའོ། །ངེས་པའི་དོན་དུ་བཤད་ན་རིམ་པ་བཞི་ལས་ནི་ཞེས་བྱ་བས་དཀྱིལ་འཁོར་བསྡུས་ནས་རླུང་གཡོན་ནས་བཅུག་སྟེ། ལྟེ་བའི་བར་དུ་བཅུག་ནས། ལྟེ་བ་ལ་སོགས་པ་འཁོར་ལོ་བཞི་དང༌། རྩ་གསུམ་དང་བཅས་པ་གསལ་བར་དམིགས་ཏེ། བརྒྱུད་རིམ་བསྟན་ཏོ། ། གཏུམ་མོའི་མེ་བདག་གི་འཁོར་ལོ་རྣམས་ལས་བརྒྱུད་དེ་སྣ་གཡས་ནས་སྤྲོས་པ་དང་ ༑ ཏ་ཐཱ་ག་ཏའི་ཤངས་བུག་གཡོན་པར་འཇུག་པ་ལ་སོགས་པ་ནས་བདག་གི་སྣ་བུག་གཡོན་པར་ཡེ་ཤེས་ཀྱི་བདུད་རྩི་འཇུག་པའི་བར་ནི་སྔགས་ཀྱི་རིམ་པའོ ༑ ༑ དུས་མིན་འཆི་ཞེས་བྱ་བ་རྐང་པ་གཉིས་ཀྱི་སྣ་བུག་གཡོན་ནས་ཞུགས་པ་ནས་བཟུང་སྟེ། ལྟེ་བར་ཕྱིན་གྱི་བར་ལ་དགའ་བ་བཞི་སྐྱེ་བ་ནི་ཡེ་ཤེས་ཀྱི་རིམ་པ་སྟེ་དེ་བསྟན་ཏོ། །ཐོད་པ་ཞེས་པ་རྐང་པ་གཉིས་ཀྱིས། བདག་གི་རྒྱུད་ལ་སངས་རྒྱས་ཀྱི་སྐུ་དང་ཡེ་ཤེས་འགྲུབ་སྟེ། སྐུ་དང་ཡེ་ཤེས་ཟུང་དུ་འཇུག་པ་ནི་གསང་བའི་རིམ་པ་སྟེ་དེ་བསྟན་ཏོ། །འདི་དག་གི་ཡང་མངོན་པར་རྟོགས་པ་ཇི་ལྟ་བ་ནི་དཔྱིད་ཀྱི་ཐིག་ལེ་ལས་ཤེས་པར་བྱའོ། །ཞེས་བླ་མ་གསུང་ངོ༌།། །། ལེའུ་སོ་དྲུག་པས་ནི་དམ་ཚིག་གི་ཕྱག་རྒྱ་དང༌། ལས་ཀྱི་ཕྱག་རྒྱ་དང་གཉིས་བསྟན་ཏེ། དེ་ཡང་དག་པའི་ཡུལ་ནི་དེ་བཞིན་དུ། །ཞེས་བྱ་བའི་བར་གྱིས་ནི་དམ་ཚིག་གི་ཕྱག་རྒྱ་ལྟར་ན་འོད་ཀྱི་རིག་མ་དང༌། ལས་ཀྱི་ཕྱག་རྒྱ་ལྟར་ན་རྐྱང་མའི་གནས་ལ་བྱིན་གྱིས་བརླབ་པ་ཀུན་ཏུ་སྤྱོད་པ་ལྟར་ཤེས་པར་བྱའོ།

།དཔའ་བོ་མཆོད་པའི་ཞེས་བྱ་བ་ལ་སོགས་པས་ནི་དམ་ཚིག་གི་ཕྱག་རྒྱ་ལྟར་ན་སྔགས་ལས་ཞི་བ་ལ་སོགས་པའི་ལས་བསྒྲུབ་པ་བསྟན་ཏོ། །ལས་ཀྱི་ཕྱག་རྒྱ་དང་ལྡན་ན་དགའ་བ་བཞིའི་མངོན་པར་རྟོགས་པ་བསྐྱེད་པའི་ཚུལ་བསྟན་ཏོ།། །། ལེའུ་སོ་བདུན་པས་ནི་རྐྱང་མ་བདེ་བླག་ཏུ་མ་རྙེད་ན་དབང་དུ་བྱ་བའི་ཐབས་བསྟན་ཏེ། ཇི་སྲིད་འཆི་བར་དབང་དུ་འགྱུར། །ཞེས་བྱ་བ་ཁ་ཟས་ཀྱི་སྒོ་ནས་བྱ་བ་མ་ཡིན་པ་བསྟན་ཏོ། །ཏིལ་ནག་ཅེས་བྱ་བ་ལ་སོགས་པ་བྱུགས་པས་དབང་དུ་བྱ་བ་བསྟན། བ་ལང་ཤ་ཞེས་བྱ་བ་ལ་སོགས་པས་སྦྱིན་སྲེག་གིས་དབང་དུ་བྱ་བ་བསྟན། འཇིག་རྟེན་གསུམ་ན་ཞེས་བྱ་བས་དེ་ལྟ་བུའི་ཐབས་ཀྱང་གཞན་ལ་སྦ་བ་བསྟན་ཏོ།། །།ལེའུ་སོ་བརྒྱད་པས་ནི་རང་དོན་དུ་སྤྱོད་པ་གསུམ་ལས་སྤྲོས་མེད་ཀྱི་སྤྱོད་པ་བྱ་ ༼ བ ༽ དང༌། བྱ་བ་མ་ཡིན་པའི་ཆ་བསྟན་ཏོ། །དེ་ཡང་ལྟུང་བ་ཞེས་བྱ་བའི་བར་གྱིས་ལྡོག་པའི་སྒོ་ནས་བྱ་བ་མ་ཡིན་པ་བསྟན་ཏོ། །ཧེ་རུ་ཀ་དཔལ་ཞེས་བྱ་བ་ལ་སོགས་པས་འཇུག་པའི་སྒོ་ནས་བྱ་བའི་ཆ་བསྟན་ཏོ།། རྟག་ཏུ་དཔའ་བོ་ཆེན་པོ་ཞེས་བྱ་བས་དེ་ལྟར་གནས་པའི་ཕན་ཡོན་བསྟན་ ཏོ ༑༑ ༑༑ ལེའུ་སོ་དགུ་ནས་ཞེ་བརྒྱད་ ( ¡¶ ལུ ¡· གཉིས་ ) ཀྱི་བར་གྱིས་ནི། ལམ་གྱི་གསང་བ་བསྐྱེད་རྫོགས་གཉིས་ལ་གནས་པའི་མཚན་མའི་རྟགས་བསྟན་ཏེ། དེ་ཡང་ལེའུ་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· + སུམ་ཅུ་རྩ་ ) དགུ་པའི་ ( ¡¶ ཞ ¡· + སྒྲུབ་པའི་ ) སྒྲུབ་པ་པོ་ཧ་བརྒྱད་ཀྱི་ཆོ་ག་དང་ལྡན་པས་དུར་ཁྲོད་ལ་སོགས་པར་འཇིགས་པ་མེད་པར་གནས་ལ ༑ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· + དེ་ལ་ ) ཐུན་མོང་གི་དངོས་གྲུབ་མཆོག་གི་སིངྔྷི་སྐུ་གསུམ་འགྲུབ་པར་བསྟན་ཏོ།། །།ལེའུ་བཞི་བཅུ་པས་ནི་ལམ་གྱི་གསང་བ་བསྒྲུབ་པའི་ལས་ཀྱི་ཕྱག་རྒྱ་རྐྱང་མ་དབང་དུ་བྱ་བ་བསྟན་ཏེ། དེ་ཡང་སྔར་ལེའུ་ ¡Ã སོ་བདུན་ ( ¡¶ ལུ ¡· ཉེར་གསུམ་ ) པར་བསྟན་པའི་དབང་དུ་བྱས་ནས་དེའི་འབྲེལ་བར་རྗེས་སུ་ཆགས་པར་བྱ་བ་ཡིན་ལ། འདིར་ནི་སྔར་དབང་དུ་མ་བྱས་པ་ཞིག་བསྟན་ཏོ།

།དེ་ཡང་ཤ་ལྔ་ལ་སོགས་པའི་སྦྱིན་སྲེག་གིས་རིགས་ལྔའི་བུ་མོ་དབང་དུ་བྱ་བ་བསྟན་ཏོ།། །།ཞེ་གཅིག་པས་ནི། སུཾ་བྷ་ནིའི་ལས་ཀྱིས་ཚོགས་ལས་ཆེན་བཅུ་གཉིས་བསྟན་ཏེ། འདི་ཡང་འཁོར་ལོ་ལ་བརྟེན་པའི་ལས་ཚོགས་ཏེ། དེ་ཡང་དཔལ་ནཱ་རོ་ཏས་ཀཾ་པ་ལ་དང་མཐུན་པར། འཁོར་ལོ་བཞིས་ལས་སོ་གསུམ་འགྲུབ་པར་བཞེད་དོ། །མཁའ་འགྲོ་ལས་བྷ་བས་འཁོར་ལོ་ལྔ་བཞེད་དོ། །མཁའ་འགྲོ་རྒྱ་མཚོ་ལས། ལས་བཅུ་གཉིས་ལ་ཙཀྲ་བཅུ་གཉིས་ཡོད ༑ འདི་ཡང་དབང་པོ་རབ་སེམས་ལ་བརྟན་པ་ཐོབ་པའི་དབང་དུ་བྱས་ཏེ་དྲན་པ་ཙམ་གྱིས་འགྲུབ་བོ། །ཤ་ ( ¡¶ ལུ ¡· ཤར་ ) དང་བྱང་སེམས་ ( ¡¶ ལུ ¡· ཞེས་ ) བྱ་བ་ལ་སོགས་པ་ནི། དེ་ལྟར་བུའི་དངོས་གྲུབ་ཀྱང་རྣལ་འབྱོར་མ་རྣམས་ཀྱིས་རྩོལ་བས་མཁའ་འགྲོ་མ་རྣམས་བསྟན་ཏོ།། །།དེ་ཡང་སྔར་ལེའུ་ ¡Ã བཅུ་བདུན་ ( ¡¶ ལུ ¡· བཅོ་བརྒྱད་ ) པར་ནི། རྣལ་འབྱོར་མ་བརྒྱད་ ¡Ã ཀྱིས་སོ་བདུན་ ( ¡¶ ལུ ¡· ཀྱི་ས་བོན་ ) བསྡུས་པར་བསྟན་ལ། འདིར་ནི་རྣལ་འབྱོར་མ་དྲུག་གིས་བསྡུས་པར་དོན་གྱིས་བསྟན་ཀྱང་འགལ་བ་མེད་ཅིང་གཅིག་ཏུ་བསྡེབས་ཏེ་གོ་བར་བྱའོ ༑༑ ༑༑ ལེའུ་ཞེ་གཉིས་པས་ནི་རྣལ་འབྱོར་མ་རྣམས་བསྟན་པའི་དངོས་གྲུབ་ཀྱི་རྟགས་བསྟན་ཏེ། དེ་ཡང་དང་པོར་བདུད་རྩི་ ( ¡¶ ལུ ¡· + ལྔ་ ) དང་ཤ་ལ་སོགས་པའི་བཟའ་བ་རྣམས་ཀྱིས་བདེ་བ་སྐྱེ་བའི་ཐབས། ལུས་ཟུངས་རྒྱས་པར་བྱ་བའི་ཐབས། རྐྱང་མ་གཉིས་ཀས་བསྟེན་པའི་ཐབས་བསྟན་ཏོ། །ཐོག་མཐའ་ཞེས་བྱ་བ་ལ་སོགས་པས་ནི་རྐྱང་མ་ལས་དང་པོར་དཔེའི་ཡེ་ཤེས་བདེ་བ་ལས་མི་རྟོག་པ་རྟོགས་པར་བྱ་བ་དང༌། དེ་ནས་དོན་གྱི་ཡེ་ཤེས་རྟོགས་པ་བདེ་བ་མཐའ་གསུམ་གྱིས་མ་རེག་པ་ཉམས་སུ་མྱོང་བར་བྱ་བ་བསྟན་ཏོ། །ཐོག་མ་ཐ་མ་ཞེས་བྱ་བ་སོགས་ནི་རྐྱང་མ་བསྟན་པའི་ཐོག་མ་དང་ཐ་མར། ཧ་ཧ་ལ་སོགས་པ་ཐུང་ངུ་བརྒྱད་བརྗོད་ ¡Ã པར་ཤེས་པར་ ( ¡¶ ཞ ¡· པས་བརྙེས་པས་ ) བསྟན་ཏོ། ། གཟུགས་དང་ཞེས་བྱ་བ་ལ་སོགས་པས་ནི། རྐྱང་མ་བསྟེན་པ་ལས་བདེ་བ་སྐྱེས་པའི་རྟགས་གཟུགས་དང་ལས་ཞེས་བྱ་བ་ཕྱིའི་རྟགས་དང༌།

ནང་གི་རྟགས་གཉིས་སུ་བསྡུས་ལ་རྒྱས་པར་བྱས་ན་རེ་རེ་ལའང་ལྔ་ལྔ་སྟེ་བཅུའོ། །དེ་ཡང་ཐུན་མོང་དུ་འགྲོ་བ་གཞན་པ་བརྒྱད་སོ་སོར་བགྲངས་ནའོ། །དེ་ནི་ཐམས་ཅད་ཀྱི་དབང་དུ་བྱས་པའོ། །དུ་བར་བྱེད་པ་རྟགས་སུ་བསྔགས་ཞེས་བྱ་བ་ལ་སོགས་པ་ནི་ཕྱིའི་གསང་བ་ཤེས་རབ་རྐྱང་མ་སྦྱང་བ་ལྔ་བསྟན་ཏོ། །ནང་གི་གསང་བ་ལྔ་ཡང་ཐབས་དང་འདྲ་བར་ཤེས་པར་བྱའོ། །དེ་ལྟར་ན་གཉིས་ལ་རྟགས་ཉི་ཤུའོ།། །།ལེའུ་ཞེ་གསུམ་པ་ནི་འབྲས་བུའི་གསང་བ་རྒྱས་པར་བཤད་པ། ཡི་གེ་བདུན་ ( ¡¶ ཞ ¡· བརྒྱ་ ) པའི་ལས་ཀྱི་ཚོགས་བསྟན་ཏེ། དེ་ཡང་དང་པོར་ཤློ་ཀ་གཅིག་གིས་སྔོན་དུ་ཡི་གེ་བདུན་པ་ལ་བསྙེན་པ་སྔོན་དུ་སོང་བའི་རྣལ་འབྱོར་པ་ལ། མཁའ་འགྲོ་མ་ཐམས་ཅད་ཀྱི་སྤྱན་གྱིས་གཟིགས་ཤིང་བྱིན་གྱིས་རློབ་ལ། དེ་ཡང་གནས་ཁྱད་པར་ཅན་དུ་གནས་ན་བྱིན་གྱིས་རློབ་པས་གནས་མདོར་བསྟན་ཏོ། །ལམ་གྱི་ཞེས་པ་ལ་སོགས་པས་ལམ་གྱི་ཁྱད་པར་བཤད་དོ། །ཕྱག་རྒྱ་ཞེས་བྱ་བ་ལ་སོགས་པས་སྒྲུབ་པ་པོའི་ཆ་ལུགས་བསྟན་ཏོ།། སྙིམ་པ་མེ་ཏོག་ཅེས་བྱ་བ་ལ་སོགས་པས་ཡི་གེ་བདུན་པ་ལ་སྔོན་དུ་བསྙེན་པ་འབུམ་ཕྲག་གཅིག་བྱ་སྟེ། ཡང་དེའི་རྗེས་ལ་ཁྲི་ཕྲག་གཅིག་བཟླས་པ་བྱས་པས། ཀ་ར་བཱི་རའི་མེ་ཏོག་དམར་པོ་ལ་སྔགས་བཟླས་ལ་དངོས་གྲུབ་ཀྱི་ལྟས་བརྟག་པ་བསྟན་ཏོ། །གནས་ཀྱི་ཞེས་བྱ་བ་སོགས་པས་ངག་གི་དབང་ཕྱུག་ལ་སོགས་པ་བསྟན། ཡུལ་དང་ཞེས་སོགས་བྱ་བས་མི་སྣང་བ་བསྒྲུབ་པ་བསྟན་ཞེས་བྱ་བ་ལ་སོགས་པ་ནི། སྔགས་ཀྱི་འོད་ཟེར་ཤིང་མྱ་ངན་མེད་ཀྱི་རྩེ་ལ་སྤྲོས་ཏེ། སྔགས་ཀྱི་རྡུལ་ཕྲ་རབ་རི་རབ་ལྟ་བུར་ལྕི་བར་བསམ་པ་ལས་ཤིང་ཆག་ཅིང་ལྟུང་བར་འགྱུར་རོ། །དེས་གཞན་ལྟུང་བར་བྱ་རྒྱུ་ལྟུང་བར་ཡང་བྱའོ། །བདག་གི་ལུས་ཞེས་བྱ་བ་ལ་སོགས་པས་ནི། ཡི་གེ་བདུན་པའི་བསྙེན་པ་སྔོན་དུ་སོང་བས་རྨི་ལམ་བརྟག་པ་བསྟན་ཏོ། །དེ་བཞིན་སྔགས་བཟླས་ཞེས་བྱ་བ་ལ་སོགས་པས་རལ་གྲི་ལ་སོགས་པ་ལྔ་ལ་སྔོན་དུ་སྦྱང་བ་བྱ་སྟེ།

དེ་ནས་ཁྱེའུ་དང་བུ་མོ་གཙང་མ་ལ་ཁྲུས་བྱས་ཏེ། ཡི་གེ་བདུན་

( ¡¶ ཞ ¡· བརྒྱ་ ) པ་བཟླས་པས་དམིགས་ཡུལ་དེ་བལྟར་གཞུག་གོ །དེ་ལྟར་ཡང་ཉིན་པར་ཁྱིམ་ཤིན་ཏུ་མུན་པ་ནག་པ་ལ་བྱའོ།། མཐེ་བོང་གི་ ¡Ã དཔུང༌། ( ¡¶ ལུ ¡· དུས་སུ་ ) མཐེ་བོང་སྣག་ཚ་དང་རྒྱ་སྐྱེགས་ལ་སོགས་པས་བསྐུས་ལ་སྦྱངས་ཏེ། དེ་ནས་བུམ་པ་གཙང་མར་བདུད་རྩི་ལྔ་ཅུང་ཟད་བླུགས་པར་ཡི་གེ་བདུན་པའང་དེ་ལ་བཟླས། དེའི་སྟེང་དུ་མར་མེ་བཞག་ལ་བལྟར་གཞུག དེ་ནས་དང་པོའི་ཞེས་བྱ་བ་ལ་སོགས་པས་ནད་གསོ་བར་བསྟན ༑ དེ་ཟླ་བའི་དབུས་སུ་ཞེས་བྱ་བ་ལ་སོགས་པས་ཁྱད་པར་དུ་དུག་ལ་སོགས་པ་ཞི་བར་བྱ་བ་བསྟན་ཏོ། །མཚན་མོ་ཞེས་བྱ་བ་ལ་སོགས་པས་ནི་སྦྱིན་སྲེག་གི་སྒོ་ནས་རྒྱས་པ་ལ་སོགས་པའི་ལས་བསྒྲུབ་པ་བསྟན་ཏོ། །དཀར་པོ་ཞེས་བྱ་བ་ལ་སོགས་པས་ནི། ལག་པ་ལ་ཡི་གེ་བདུན་པ་བཀོད་པའི་སྦྱོར་བས། སྔར་ཚར་བཅད་པ་རྗེས་སུ་བཟུང་བ་དང༌། ཁྱད་པར་དུ་དུག་སྦྲུལ་གྱིས་ཟིན་པ་གསོ་བ་བསྟན་ཏོ། །རིམས་ལ་ཞེས་རྐང་གཉིས་ཀྱིས་རིམས་ཞི་བར་བྱ་བ་བསྟན ༑ རྒྱན་པོ་ཞེས་བྱ་བས་ནི་ཙཀྲ་ལ་བརྟེན་ཏེ་ཤོ་མི་འཕམ་པར་བྱ་བ་བསྟན། དགྲ་བོ་ཞེས་བྱ་བ་ལ་སོགས་པས་དགྲ་ལས་རྒྱལ་བར་བྱ་བ་བསྟན། འབྲས་བུ་ཞེས་བྱ་བ་ལ་སོགས་པས་ནི་ལྷའི་རྣལ་འབྱོར་དང་ཡི་གེ་བདུན་ ( ¡¶ ཞ ¡· བརྒྱ་ ) པའི་གསང་སྔགས་ལ་སོགས་པ་རྣམས་ཀྱིས་ཟས་ལ་སོགས་པ་ཅི་དགར་བསྒྲུབ་པ་བསྟན་ཏོ། །ས་གཞི་ཞེས་བྱ་བ་ལ་སོགས་པས་ནི་ཡི་གེ་བདུན་ ( ¡¶ ཞ ¡· བརྒྱ་ ) པའི་སྔགས་ཀྱིས། རྡུལ་ཕྲ་རབ་ཀྱི་ས་གཞི་ལ་སོགས་པ་བཏེག་ན་འཕར་ཞིང༌། མནན་ན་ནེམ་པ་ལ་སོགས་པར་བྱ་བ་སྟེ་ཡིད་དུ་འོང་བར་བྱ་བ་དང༌། རྒྱ་མཚོ་ལ་ཅི་བདེར་ལམ་བྱ་བ་ལ་སོགས་པ་བསྟན་ཏོ།། །།ལེའུ་ཞེ་བཞི་པས་ཀྱང་འབྲས་བུའི་གསང་བ་ལ་འཇུག་པ་དཔའ་བོ་དྲུག་གི་གོ་ཆའི་ལས་བསྟན་ཏེ། དེ་ཡང་འཁོར་ལོ་ལ་བརྟེན་ཏེ་དགུག་པའི་ལས་དང༌། ཙནྡ་དམར་པོའི་སྦྱིན་སྲེག་གིས་སའི་བདག་པོ་ནོར་དང་བཅས་པ་དགུག་པའི་ལས་དང༌། ཡང་ཡི་གེ་བདུན་ ( ¡¶ ཞ ¡· བརྒྱ་ ) པ་དེ་ཉིད་ཀྱིས་ལྕགས་ཕྱེ་ལ་སོགས་པའི་རྫས་ལ་མནན་པ་བྱ་སྟེ

༑ གསད་པའི་ལས་བསྒྲུབ་པ་དང༌། ཡང་སླར་རྗེས་སུ་གཟུང་བ་དང༌། མྱོས་བྱེད་ལྔ་ལ་སོགས་པས་སྨྱོ་བར་བྱ་བའི་ལས་རྗེས་སུ་གཟུང་བ་དང་བཅས་པ་གཅིག་བསྟན་ཏོ།། །།ལེའུ་ཞེ་ལྔ་པས་ནི་འབྲས་བུའི་གསང་བ་བསྟན་ཏེ། འདི་ཡང་རྣལ་འབྱོར་མ་དྲུག་གི་སྔགས་འབའ་ཞིག་གི་ལས་བསྟན་ཏོ། །དེ་ཡང་དང་པོར་གོ་ཆའི་སྔགས་ཀྱི་བསྙེན་པ་སྔོན་དུ་སོང་བས། ཡང་ཀ་ར་བཱི་རའི་མེ་ཏོག་དམར་པོ་ལ་སྔགས་བཟླས་ཏེ། དེ་འོ་མས་བཀྲུས་ནས། དེ་ནས་ཞག་བདུན་དུ་དབང་ཕྱུག་ཆེན་པོའི་ལིངྒ་ལ་གཏོར་ཞིང་སྔགས་བཟླས། དེ་ནས་ནུབ་གསུམ་ཚོགས་ཀྱི་མཆོད་པ་བྱ། དེ་ནས་ཉི་མ་དང་པོ་ལ་སྒྲུབ་གཡོག་ལྷའི་ང་རྒྱལ་དང་ལྡན་པས་སུམྦྷ་ནི་བརྗོད་ཅིང་རལ་གྲི་ཐོགས་པས་བར་ཆད་བསྲུང༌། རང་གིས་མེ་ཏོག་རེ་ལ་སྔགས་རེ་བརྗོད་ཅིང་ཆུ་ལ་བསྐུར་བས། མེ་ཏོག་ཡར་ལོག་པ་ན། ཆུ་དང་བཅས་པས་སོ་ལ་མ་རེག་པར་འཐུངས་ན། ངག་གི་དབང་ཕྱུག་ཏུ་འགྱུར་རོ ༑ ༑ དེ་ནས་སྐྱེ་གནས་ཞེས་བྱ་བ་ལ་སོགས་པས་རྣལ་འབྱོར་མ་དྲུག་གི་ཙཀྲ་བྲིས་ཏེ། དགུག་པའི་ལས་བསྒྲུབ་པ་བསྟན། ཙཀྲ་ནི་རྩིབས་དྲུག་པ་བྱ་ལ་གོ་ཆའི་སྔགས་དྲུག་དགོད་དོ། །ལྟེ་བ་ལ་བསྒྲུབ་བྱའི་སྔགས་སྤེལ་ཚིག་དང་བཅས་པ་བྲི། ལྟེ་བའི་ཕྱིའི་མུ་ཁྱུད་ལ་ཨཱ་ལི་གཡོན་སྐོར། ཕྱི་རོལ་གྱི་མུ་ཁྱུད་ལ་ཀཱ་ལི་གཡས་སྐོར་དུ་བཀོད་ལ། དེ་ནས་རབ་ཏུ་གནས་པ་བྱ་སྟེ། དེ་རྐང་རྗེས་ལ་སོགས་པ་ལས་གཟུགས་བརྙན་ ( ¡¶ ལུ ¡· ཅན་ ) བྱས་པའི་སྙིང་གར་བཅུག་སྟེ། ལིངྒའི་སྐྱེ་གནས་ནས་སྐུད་པ་དམར་པོ་བཏོན་ཏེ། སྒྲུབ་པ་པོའི་མཐེའུ་ཆུང་ལ་བཏགས་ནས། དེ་བསྐུལ་ཞིང་སྔགས་བཟླས་ན་དགུག་པའི་ལས་འགྲུབ་བོ།། ཙནྡན་དམར་པོ་ཞེས་བྱ་བ་ལ་སོགས་པས། ལིངྒ་མེས་གདུང་བར་བྱས་པས་དབང་དུ་བྱ་བའི་ལས་བསྟན་ཏོ། །མི་ཡི་བདག་པོ་ཞེས་བྱ་བ་ལ་སོགས་པས་ལིངྒ་མེ་ལ་གདུངས་བའི་སྦྱོར་བས། རྒྱལ་པོ་འཁོར་དང་བཅས་པ་དབང་དུ་བྱ་བའི་ལས་བསྟན་ཏོ། །བསྒྲུབ་པར་བྱ་བའི་རྐང་རྗེས་ལ་ཞེས་བྱ་བ་ལ་སོགས་པས་སྦྱིན་སྲེག་གི་ལས་ཐམས་ཅད་འགྲུབ་པར་བསྟན་ཏོ།

།རྡུལ་དེ་ཞེས་བྱ་བ་ལ་སོགས་པས་མནན་པའི་སྒོ་ནས་སེམས་རྨོངས་པར་བྱས་ནས་དགུག་པའི་ལས་བསྒྲུབ་པ་བསྟན་ཏོ། །ཀ་ར་བཱི་རའི་ལྕུག་མ་ཞེས་བྱ་བ་ལ་སོགས་པའི་རྫས་ཀྱི་སྦྱོར་བས། སྐྱེས་པ་དང་བུད་མེད་དབང་དུ་གྱུར་ཏེ་དགུག་པ་བསྟན་ཏོ། །ཀ་ར་བཱི་ར་ཞེས་བྱ་བས་ངག་གི་དབང་ཕྱུག་སྒྲུབ་པའི་ཐབས་གཞན་བསྟན་ཏོ། །དེ་ནས་གཞན་གྱི་ལས་ཞེས་བྱ་བའི་ཙནྡན་གྱི་ལིངྒ་ལ་བྱ་བའི་སྦྱོར་བས་དབང་དུ་བྱ་བའི་དོན་ཏོ།། །།ལེའུ་ཞེ་དྲུག་པས་ཧ་ལྔའི་འཁོར་ལོ་ལ་བརྟེན་ནས་ཁྲག་དགུག་པ་ལ་སོགས་པའི་ལས་བསྟན་ཏོ། །འདི་ཡང་ཧེ་རུ་ཀ་ཡབ་ཡུམ་གང་ཡང་རུང་བའི་རྣལ་འབྱོར་ལ་གནས་ཏེ། རང་གི་སྙིང་གར་ཧ་ལྔའི་ཙཀྲ་བསྒོམས་ལ། ཧ་ལྔ་ལ་སྔོན་དུ་བསྙེན་པ་འབུམ་ཕྲག་ལ་སོགས་པ་སོང་བས་ལྷ་རྣམས་བསྒྲུབ་པར་བྱའོ།། །།ལེའུ་ཞེ་བདུན་པས་ནི་རྗེ་བཙུན་མའི་སྙིང་པོའི་ཡི་གེ་ཉེར་གཅིག་པ་ལ་བརྟེན་པའི་ལས་ཀྱི་ཚོགས་བསྟན་ཏེ། དེ་ཡང་དབང་པོ་རབ་ཀྱིས་བཀླགས་པ་དང་དྲན་པ་ཙམ་གྱིས་འགྲུབ་པར་བསྟན། མིང་རུས་ཞེས་བྱ་བ་ལ་སོགས་པས་སྔོན་དུ་བསྙེན་པ་སོང་བས་ལས་ཀྱི་སྦྱོར་བ་ཇི་ལྟ་བ་བཞིན་དུ་བྱས་ཏེ ༑ དགྲ་རྣམས་གསད་པ་ལ་སོགས་པའི་ལས་འགྲུབ་པ་བསྟན། དེ་ནས་རྒྱལ་པོའམ་ཞེས་བྱ་བ་ལ་སོགས་པས་ནི། རྗེ་བཙུན་མའི་ཙཀྲ་ལ་བརྟེན་ཏེ་དབང་དུ་བྱ་བའི་ལས་བསྟན། ལྷག་མ་དགུག་པ་ལ་སོགས་པའི་ལས་རྣམས་གཞུང་ཉིད་དུ་གསལ་བས་འདིར་མ་བྲིས་སོ།། །།ལེའུ་ཞེ་བརྒྱད་པས་ནི་ལས་དེ་དག་བསྒྲུབ་པའི་སྔགས་དང་པོར་བསྟན། སྔགས་འདི་ཞེས་བྱ་བ་ལ་སོགས་པས་རྗེས་བཙུན་མའི་སྙིང་པོའི་སྔགས་ཀྱི་ཆེ་བ་བསྟན། མེ་ཏོག་འབྲས་བུ་ཞེས་བྱ་བ་ལ་སོགས་པས་གོང་དུ་བསྟན་པའི་དངོས་གྲུབ་རྣམས་ཀྱང་བསྐྱེད་པའི་རིམ་པ་ལ་བརྟེན་ནས་འགྲུབ་པ་དེ་ཡང༌། ལྷ་སོ་བདུན་གྱི་དཀྱིལ་འཁོར་རྫོགས་པ་ལ་བརྟེན་ཏེ་བསྒྲུབས་ན་ལས་ཐམས་ཅད་འགྲུབ་པར་བསྟན་ཏོ།། །།ལེའུ་ཞེ་དགུ་པས་ནི་འབྲས་བུའི་གསང་བ་ལ་འཇུག་པ་སྐྱེ་བ་བདུན་པའི་བསྒྲུབ་པ་བསྟན་ཏེ།

དེ་ཡང་ལེའུ་བཅུ་གཅིག་པར་སྐྱེ་བ་བདུན་པའི་མཚན་ཉིད་ལ་སོགས་པ་མདོར་བསྟན་ནས་བཅུ་བཞི་པས་རྒྱས་པར་བཤད། འདིར་བརྟག་པ་དང་སྒྲུབ་པའི་ཐབས་ལ་སོགས་པས་སོ་སོར་བཤད་དེ། དེ་ལྟར་ན་ལེའུ་གསུམ་གྱིས་སྐྱེ་བ་བདུན་པ་བསྒྲུབ་པ་བསྟན་ཏོ། །དེ་ཡང་དང་པོར་ཚིགས་བཅད་གཅིག་གིས་མཚམས་སྦྱར་བ་བསྟན། སཱ་ལུ་ཞེས་བྱ་བ་ལ་སོགས་པས་བརྟག་པའི་ཐབས་བསྟན། དེ་ཡང་སྔོན་དུ་བསྙེན་པ་སོང་བ་གཅིག་གིས་རྗེ་བཙུན་མའི་ཉེ་བའི་སྙིང་པོའི་འཁོར་ལོ་བཅས་ལ་དེའི་སྙིང་གར། ཕྱུགས་ལ་སོགས་པ་གང་ཡང་རུང་བའི་སྐྱེ་བ་བདུན་པ་སྟོན་པའི་སྤེལ་ཚིག་དང་བཅས་པ་བྲིས་ཏེ། དེ་ལ་རབ་གནས་བྱས་ལ། དེ་ནས་འབྲས་སཱ་ལུའི་ཟན་ལ། བོང་བུའི་གཟུགས་བརྙན་གཅིག་བྱས་ལ། དེའི་སྙིང་གར་འཁོར་ལོ་བཅུག་ལ། དེའི་མདུན་དུ་མེ་ལོང་ཚིལ་ཆེན་ལ་སོགས་པས་བྱུགས་པ་བཞག་ལ། བོང་བུའི་གཟུགས་ཀྱི་ནང་ནས་བལྟས་ན་གང་གི་སྐྱེ་བ་བདུན་པ་ཡིན་པ་དེའི་གཟུགས་བརྙན་བདུན་འབྱུང་ངོ༌།། དེ་ནི་བརྟག་པའོ། །གཞན་ཁ་དྲི་ཞིམ་པོ་ལ་སོགས་པའི་རྟགས་ཀྱང་ཤེས་པར་བྱའོ ༑ ༑ དེ་ལྟར་ངོ་ཤེས་ནས་རྣམ་ཤེས་འཕོ་བའི་ཐབས་ནི། བླ་མ་བགྲེས་པོ་རྣམས་བསྐྲལ་མི་དགོས་ཏེ། དེ་ནི་བྱང་ཆུབ་སེམས་དཔའ་མངོན་ཤེས་ཅན་ཡིན་པས་སྐྱེ་བ་ལ་དབང་ཐོབ་པ་ཡིན་པས། དེ་ལ་མཎྜལ་ལ་སོགས་པ་བྱས་ཏེ། བསྙེན་བཀུར་བྱས་པས་རང་གི་ལུས་འདོར་གསུང༌། བརྡ་བསྟན་པ་མི་སྐྱེ་བ་བསྟན་པ་ཡིན་ན། སཾ་པུ་ཊ་ལ་སོགས་པ་ནས་འབྱུང་བ་ལྟར། ཧི་ཀ་ཟློས་སུ་བཅུག་པས་འཕོ་བར་བཞེད་དོ། །དབུགས་ནང་དུ་འཇུག་པ་ན་ཧི་ཀ་ཞེས་མྱུར་བར་བཟླས་པས་དབུགས་ཐུང་དུ་འགྲོ་སྟེ། དབུགས་ཕྱིར་འབྱུང་ཙམ་ན། ཧིའི་གི་གུ་ཀ་ལ་སྤོས་ལ། ཧ་ཀི་ཞེས་པ་བཟླས་པས་འཕོ་གསུང༌། དུད་འགྲོ་ཡིན་ན་ལིངྒ་བྱས་ལ། སྔར་གྱི་འཁོར་ལོ་དང་འདྲ་བ་གཅིག་བྱས་ལ། ལྟེ་བར་རྣམ་ཤེས་འཕོ་བར་གྱིས་ཤིག་པའི་སྤེལ་ཚིག་དང་བཅས་པ་བྲིས་ཏེ། རབ་གནས་བྱས་ཏེ། གཟུགས་བརྙན་གྱི་སྙིང་ག་ནས་སྔར་སྔགས་བཟླས་པས་ཡི་གེ་ཉེར་གཉིས་པ་ལྟ་བུ་བཏོན་ལ་སྙིང་གའི་ཙཀྲ་ལ་བསྟིམ།

དེ་ནས་སྔགས་ཀྱི་འོད་ཀྱིས་བསྒྲུབ་བྱ་སྐྱེ་བ་བདུན་པ་དེའི་ཤེས་པ་ཧཱུྃ་གི་རྣམ་པས་བཀུག་ལ། འཁོར་ལོ་ལ་བསྟིམས་པས་འཕོ་གསུང༌། ཇོ་བོའི་གཞུང་འདི་བླ་ ¡Ã མའི་ལས་ལོགས་ ( ¡¶ ལུ ¡· མས་ལེགས་ ) པར་མཛད ༑ དེ་ནི་ཟོས་པ་ཞེས་བྱ་བ་ལ་སོགས་པས་ཁ་ཟས་སུ་བྱ་བ་བསྟན་ཏེ། དེའི་ཤ་ལ་སོགས་པ་དང་རང་ལས་གང་འདོད་ཀྱི་རྫས་བསྲེས་ལ་བྱ་སྟེ། ཞི་བ་ལ་ཙནྡན་དཀར་པོ་དང༌། དབང་ལ་གུར་གུམ་ལ་སོགས་པ་དང༌། རྒྱས་པ་ལ་གི་ཝང་ལ་སོགས་པ་དང༌། དྲག་པོ་ལ་སྐེ་ཚེ་ལ་སོགས་པ་དང་བསྲེས་ལ། སྦྱང་བ་དང་ ༑ སྦར་བ་དང༌། རྟོགས་པ་ལ་སོགས་པས་ཐ་མལ་པའི་དུག་སྦྱངས་ཏེ། དེ་ནས་ཁ་ཟས་སུ་བྱས་པས་མཁའ་སྤྱོད་ལ་སོགས་པ་འགྲུབ་པར་འགྱུར་རོ། །ཡང་ཐིག་ལེའི་སྦྱོར་བས་ཀྱང་དངོས་གྲུབ་ཏུ་འགྱུར་རོ། །ཡང་ན་ཤིང་ཡོངས་འདུ་ཞེས་བྱ་བའི་མེ་ཏོག་དམར་པོ་ཅན་དེའི་ཁུ་བ་དང༌། གོ་རོ་ཙ་ན་བསྲེས་པའི་ཐིག་ལེས་ཀྱང་གང་འདོད་པའི་གཟུགས་སུ་འགྱུར་རོ། །དེ་ཡང་དེའི་གོ་རོ་ཙ་ན་དང་གོང་དུ་འདོད་པ་དེའི་སྤུ་བསྲེས་ལ། སྐུད་པ་བྱས་ཏེ་བཏགས་པས་ཀྱང་གང་འདོད་པའི་གཟུགས་འཛིན་པར་འགྱུར་རོ། །དེ་ཡང་སྐྱེ་བ་བདུན་པའི་ཤ་དང༌། རུས་པ་དང༌། པགས་པ་དང༌། སོ་དང༌། སེན་མོ་དང་ལྔའི་ཕྱེ་མ་དང༌། གང་འདོད་པའི་སྤུ་བསྲེས་ཏེ་སྐུད་པ་བྱས་ལ། སྤྱི་བོར་བཅིངས་པ་ལ་སོགས་པས་ཀྱང་དེ་བཞིན་དུ་འགྱུར་རོ། །རྣལ་འབྱོར་ཞེས་བྱ་བ་ལ་སོགས་པས་སྐྱེ་བ་བདུན་པ་བསྒྲུབ་པ་འདི་གང་རུང་དུ་མི་བྱ་བར་བསྟན། གླང་པོ་ཞེས་བྱ་བ་ལ་སོགས་པས་སྐྱེ་བ་བདུན་པའི་ཤ་ལ་སོགས་པ། དུད་འགྲོ་གང་གིས་ཟོས་པ་དེ་ཡང་མཁའ་སྤྱོད་ལ་སོགས་པར་འགྱུར་བའི་ཕན་ཡོན་བསྟན་ཏོ།། །།ལེའུ་ལྔ་བཅུ་པས་ནི་རྗེ་བཙུན་མའི་ལས་བསྒྲུབ་པ་བསྟན་ཏོ། །དེ་ལ་བསྐྱེད་པའི་རིམ་པ་ནི་དཀྱིལ་འཁོར་རྫོགས་པའི་བསྙེན་པ་བསྐྱལ་བ་ཡང་རུང༌། ཨ་བྷི་ངྷ་ན་ནས་འབྱུང་བ་བཞིན་ཡི་གེ་བཅུ་གསུམ་པ་འདི་ཉིད་ཀྱིས་སྒྲུབ་པའི་ཐབས་བསྒོམ་པས ༑ སྔོན་དུ་བསྙེན་པ་འབུམ་ཕྲག་བཞི་བསྐྱལ་ཏེ།

དེ་ནས་འདི་སྔར་དབང་དང་རྒྱས་པ་གཙོ་བོར་བསྟན་པ་དང༌། ལས་གང་བསྟན་པའི་འཁོར་ལོ་བཞི་དང་ ༑ སྣ་ཚོགས་པ་ཤེས་པར་བྱས་ལ། བཛྲ་བཻ་རོཙ་ནའི་འཁོར་ལོ་ཇི་ལྟ་བ་བཞིན་བྲིས་ཏེ། རབ་གནས་ལ་སོགས་པ་བྱས་ལ། དེ་བཏགས་པ་དང་ལྡན་པའི་ལག་པས་ཤ་ལྔ་བདུད་རྩི་ལྔ་ཟོས་ནས་སྐྱུགས་པ་ལ་སོགས་པའི་སྦྱིན་སྲེག་གིས་དབང་དུ་བྱ་བ་དང༌། དབྱེ་བ་དང་བསྐྲད་པ་ལ་སོགས་པ་རྣམས་འགྲུབ་པའང་གཞུང་ཉིད་དུ་གསལ་ལོ། །དེ་ནས་གཞན་ཡང་བཤད་བྱ་བ། །ཞེས་བྱ་བ་ལ་སོགས་པས་ཕྱིའི་དཀྱིལ་འཁོར་བསྐྱེད་པའི་རིམ་པ་ལུས་ཀྱི་དཀྱིལ་འཁོར་དང་བཅས་པ་དང༌། ཡང་ནང་རྩ་སོ་བདུན་ལ་བརྟེན་ནས་རྫོགས་པར་བསྒོམ་པས ༑ ཚོགས་བསགས་པ་དང་སྒྲིབ་པ་སྦྱངས་བས། ཕ་རོལ་ཕྱིན་པ་ལ་གྲགས་པའི་ཀ་ ( ¡¶ ལུ ¡· ས་ ) བཅུ་ཐོབ་པར་བཤད་དོ།། །།ལེའུ་ང་གཅིག་པས་གོང་དུ་བསྟན་པའི་ལེའུ་ཞེ་དགུས་རྒྱས་པར་བཤད་པའི་ལམ་དང་འབྲས་བུའི་གསང་བ་རྣམས་འདིས་མཇུག་བསྡུས་ཏེ་བསྟན་ཏོ། །དེ་ཡང་དེ་ཁོ་ན་ཉིད་བཅུ་བཞི་ནི་ལམ་གྱི་གསང་བའི་གཙོ་བོར་མཇུག་བསྡུས་ཏེ་བསྟན་ལ། ཡང་དེས་འབྲས་བུའི་གསང་བ་ཡང་མཇུག་བསྡུས་གསུང༌། མི་མཆོག་ཅེས་བྱ་བས་ནི་ལམ་གྱི་གསང་བའི་ཕན་ཡོན་གྱི་ཚུལ་གྱིས་འབྲས་བུའི་གསང་བ་མཇུག་བསྡུས་ཏེ་བསྟན་ཏོ། །དེ་ཉིད་བཅུ་བཞི་ཡང་མངལ་སྐྱེས་ཀྱི་དབང་དུ་བྱས་ནས་བསྐྱེད་རིམ་འབའ་ཞིག་ཏུ་བཤད་ལ། སྒོང་སྐྱེས་ཀྱི་དབང་དུ་བྱས་ཏེ་བསྐྱེད་རྫོགས་གཉིས་ཀ་བསྟན་ཏེ། དེ་ལྟར་ན་ལམ་གྱི་གསང་བ་ཐམས་ཅད་བསྡུས་ནས་བསྟན་ཏོ། །དེ་ལ་མངལ་སྐྱེས་ལྟར་ན་གོས་ནི་འབྱུང་བ་ཆེན་པོ་སྔོན་དུ་སོང་བའི་གཞལ་ཡས་ཁང་བསྐྱེད་པའོ། །རྗེས་སྤྱོད་དང་སྦྱར་ན་སྟག་གི་ལྤགས་པའི་ཤམ་ཐབས་ལ་སོགས་པའོ། །ཕྱག་རྒྱ་ལྔ་ནི་རྗེས་སྤྱོད་ཁོ་ན་སྟེ། རྗེས་ལ་རུས་པའི་རྒྱན་འདོགས་པའོ། །ཤེས་རབ་ཡན་ལག་ནི་མཉམ་གཞག་ལ་ཡབ་ཡུམ་བསྟན་ཏོ ༑ ༑ རྗེས་སྤྱོད་ལ་རྣམ་ཐར་སྒོ་གསུམ་གྱི་ངོ་བོ་ཁ་ཊྭཱཾ་གའོ། །ཕུར་བུ་མཉམ་གཞག་སྲུང་བའི་འཁོར་ལོ།

རྗེས་སྤྱོད་ལ་ཡང་ཚོམ་སྔགས་ཀྱིས་བསྲུང་བ་བྱའོ།། ཨཱ་ལི་ཀཱ་ལི་ལ་སོགས་པ་མཉམ་གཞག་ཁོ་ན་ལ་སྦྱར་ཏེ། ཉི་མ་དང་ཟླ་བ་དང༌། མེ་ལོང་ལྟ་བུ་དང་མཉམ་པ་ཉིད་དོ། །རྒྱུ་སོགས་སྟོང་པ་ནི་བསོད་ནམས་དང་ཡེ་ཤེས་ཀྱི་ཚོགས་སོ། །འདི་ནི་མངོན་བྱང་ལྔ་ལས་ལྷ་བསྐྱེད་པའི་སྔོན་དུ་བྱ་བ་སྟེ། ཨཱ་ལི་ལས་ཟླ་བ་ལ་སོགས་པ་དང་འདྲ་བར་ལྷ་བསྐྱེད་པའི་རྒྱུར་བྱས་ལ་ཁོངས་སུ་སྡུད་དེ། མངལ་སྐྱེས་ཀྱི་ལུགས་ལ་སོགས་པར་མི་བགྲངས་ན། སྔོན་དུ་འགྲོ་བ་སོ་སོར་རྟོག་པ་སྟེ། བར་གྱི་ས་བོན་ཧཱུྃ་འཇུག་པས། སྒྲ་ལ་སོགས་ཞེས་བྱ་བ་ཧཱུྃ་ལས་འོད་འཕྲོས་པས་བྱ་བ་གྲུབ་པའོ། །འོད་སླར་བསྡུས་ཏེ་གཅིག་ཏུ་གྱུར་པ་ལས་རྒྱུ་རྡོ་རྗེ་འཛིན་པ་བསྐྱེད་ལ་ཆོས་ཀྱི་དབྱིངས་སོ། །བདུད་རྩི་ནི་རྒྱུ་རྡོ་རྗེ་འཛིན་པ་འོད་དུ་ཞུ་བའོ། །མྱང་འདས་འབྲས་བུ་རྡོ་རྗེ་འཛིན་པར་བཞེངས་པའོ ༑ ༑ ལག་པའི་མཆོད་པ་ལག་མཆོད། དབང་བསྐུར་ནི་མཉམ་གཞག་གི་དབང་བསྐུར་དང་ཁྲུས་གཉིས་ཀ་བསྟན། གོ་ཆ་ཆེན་པོ་ནི་མཉམ་རྗེས་གཉིས་ཀ་ལ་སྦྱར། སྔགས་ཀྱི་མཆོད་པ་ནི་རྩ་བའི་སྔགས་ལ་སོགས་པ་ཐམས་ཅད་དོ། ། སྒོང་སྐྱེས་དང་སྦྱར་ན་གོས་ཞེས་བྱ་བས་སྔགས་གཉིས་ཚངས་པའི་གནས་བཞི་བསྒོམ་པ་ལ་སོགས་པ་བསྟན་ཏེ། ཕྱག་རྒྱ་ལྔས་མཉམ་གཞག་གི་ལྷའི་རྒྱན་ནོ། ། ཤེས་རབ་ཡན་ལག་ནི་ཕུང་པོ་སོགས་ཀྱི་དག་པ་དྲན་པའོ། །ཕུར་བུ་དྲ་བ་ནི་སྲུང་བའི་འཁོར་ལོའོ། །ཨཱ་ལི་ཀཱ་ལི་ནི་འོག་མ་གཉིས་ལུས་ངག་ཡིད་གསུམ་དག་པར་ཤེས་པར་བྱའོ། །རྒྱུ་སོགས་སྟོང་པ་སྔོན་འགྲོ་ནི་ཚོགས་གཉིས་བསགས་པའོ། །འཇུག་པ་སྒྲ་ལ་སོགས་པ་བྱ་བ་ནི། ལཱུ་ཧི་པ་ལྟར་ཨཱ་ལི་ ( ¡¶ ལུ ¡· + ཀཱ་ལི་ ) ལས་ཟླ་བ་བསྐྱེད་པ་ལ་སོགས་པ་རིམ་གྱིས་རྟེན་དང་བརྟེན་པ་དུས་གཅིག་བསྐྱེད་པའོ། །ཅི་ ( ¡¶ ལུ ¡· ཇི་ ) སྲིད་བསྡུས་པ་ཞེས་བྱ་བ་ནི། ལུས་ཀྱི་དཀྱིལ་འཁོར་བྱང་ཆུབ་ཀྱི་ཕྱོགས་ཀྱི་ཆོས་ལྷའི་རྣམ་པར་དྲན་པར་བྱས་པ་ལ་སོགས་པའོ ༑ ༑ བདུད་རྩི་ནི་བདུད་རྩི་མྱང་བའོ།

།མྱང་འདས་ནི་ཕྱག་རྒྱ་བཞི་དང་རྡོ་རྗེ་སེམས་དཔའ་བསྡུས་པ་ལ་སོགས་པ་རྫོགས་པའི་རིམ་པ་རྣམས་སོ། །དེ་དག་གིས་ནི་དོན་བསྡུས་མཉམ་གཞག་ཁོ་ན་བསྟན་ཏོ། །ལག་པའི་མཆོད་པ་ལ་སོགས་པ་བཞི་ནི་རྗེས་སྤྱོད་ཁོ་ན་ལ་སྦྱར་བར་ལེགས་སོ། །ལྷག་མ་རྣམས་ནི་གཞུང་ཉིད་དུ་ཤེས་པར་བྱའོ། །རྗེ་བླ་མ་དམ་པའི་གདམས་ངག་གསུངས་པ་ལས་དཀའ་བ་རྣམས་རང་དོན་དུ་ཡི་གེར་བཀོད་པའོ། །ས་མཱཔྟ་མི་ཐི། ། དགེའོ ༑༑ ༑༑ ༄༅། །ནག་པོ་ ( ¡¶ ཞ ¡· + པའི་ ) དཀྱིལ་འཁོར་གི་ཐོ་ཡིག་བཞུགས་སོ། ། ༄༅། །བླ་མ་དམ་པའི་ཞབས་ལ་སྤྱི་བོར་གུས་པར་ཕྱག་འཚལ་ལོ། ། དཀྱིལ་འཁོར་གྱི་ཆོ་ག་འདི་རྩ་བའི་རྒྱུད་དང་བཤད་པའི་རྒྱུད་གཉིས་ལས། བཤད་པའི་རྒྱུད་ལ་བརྟེན། བཤད་རྒྱུད་ལྔ་ལས། སློབ་དཔོན་གྱི་ལས་ཚོགས་གཙོ་བོར་སྟོན་པ་ཧེ་རུ་ཀ་མངོན་པར་འབྱུང་བའི་རྒྱུད་ལ་བརྟེན་ཏེ། དེའི་ལེའུ་བཞི་པ་ནས་སའི་ཆོ་ག་བསྟན། བཅོ་ལྔ་པ་ནས་ཚོགས་དང་དབང༌། བཅུ་དྲུག་པ་ནས་དམ་ཚིག་དང་སྡོམ་པ་བསྟན་ཏེ། དེ་དག་ལ་བརྟེན་ནས་མཛད། སློབ་དཔོན་ཀརྞ་པ་བྱ་བ་ཀརྞ་ཌའི་རྒྱལ་ཁམས་སུ་སྐྱེས་པས་ན་ཀརྞ་པ་ཟེར། ཀརྞ་པ་བྱ་བ་རྣ་བ་ལ་ཟེར་བས་སྙན་རིང་བ་ལ་ཡང་ཟེར། འདིའི་སློབ་དཔོན་ཤྲཱི་

¡Ã ཛཱ་ལནྡྷ་ར་བ་ཞེས་བྱ་བ་ཛཱ་ལནྡྷ་ ( ¡¶ ཞ ¡· ཛ་ལན་དྷ་ར་པ་ཛ་ལན་དྷ་ ) རར་གྲུབ་པ་བརྙེས་པ་དེ་ཡིན་ཏེ། དེའི་གདམས་ངག་ལ་བརྟེན་ནས་སྒྲུབ་ ( ¡¶ ལུ ¡· བསྒོམ་ ) པ་བྱས་པས་ནུས་པ་ཅུང་ཟད་སྐྱེས་ནས་འཁོར་རྣལ་འབྱོར་ཕོ་མོ་མང་པོས་བསྐོར་ནས་བཞུགས་པ་ལ། སྤྱོད་པ་བྱ་སྙམ་ནས་སློབ་དཔོན་ལ་ཞུས་པ་ལ། སློབ་དཔོན་ན་རེ་སྤྱོད་པ་ལ་ནུས་པ་ཐོབ་དགོས་ན་ཁྱོད་ལ་ནུས་པ་ཡོད་དམ་གསུངས་པ་ལ། དང་པོར་རི་དྭགས་ཁྲི་སྙན་སཱ་ལེ་ལ་དགུག་པའི་ལྟ་སྟངས་ཀྱི་མདུན་དུ་ཁུགས། སྟག་ལ་རེངས་པའི་ལྟ་སྟངས་བྱས་པས་རེངས་སོ། །མ་ཧེ་ལ་དབང་གི་ལྟ་སྟངས་ཀྱིས་བལྟས་པས་དབང་དུ་འདུས་ཏེ་མདུན་དུ་འོངས། ཤིང་རློན་པའི་ཡལ་གའི་ཤིང་ཐོག་ལ་ལྟུང་བའི་ལྟ་སྟངས་ཀྱིས་བལྟས་པས་ལྟུང་བར་ནུས་པ་ལ། སློབ་དཔོན་ན་རེ་ཚར་བཅད་པ་བྱ་བ་རྣམ་པར་ཤེས་པའི་ལས་ཡིན་གྱིས། ཡེ་ཤེས་ཀྱི་ལས་རྗེས་སུ་གཟུང་བ་དགོས་པས་དེ་རྣམས་ཁྲོལ་གསུངས་པ་ལ་དགྲོལ་མ་ནུས་སོ། །དེར་སློབ་དཔོན་གྱིས་སྤྱོད་པ་བྱེད་པའི་དུས་ནི་ཡེ་ཤེས་དང་རྣམ་ཤེས་ ¡Ã ཀྱི་རླུང་ཆ་མཉམ་པས་ཚར་བཅད་པ་དང་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· ཕན་གདགས་པ་དང་ ) ། ཕན་གདགས་པ་མཉམ་པར་སྦྱོར་ནུས་པའི་དུས་སུ་ཕྱེད་པ་ཡིན་པ་ལ། ཁྱོད་ཀྱིས་མཉམ་པར་སྦྱོར་མི་ནུས་པས་སྤྱོད་པའི་དུས་ལ་མ་བབ་བོ་གསུངས་ནས་དེ་རྣམས་སློབ་དཔོན་གྱིས་བཀྲོལ་ལོ།། དེ་ལྟར་སློབ་དཔོན་གྱིས་མ་གནང་ཡང༌། ཅིས་ཀྱང་སྤྱོད་པ་བྱེད་དོ་ཞེས་ང་རྒྱལ་པོ་བྱེད་པ་ལ་སློབ་དཔོན་ན་རེ། འོ་ན་ཁྱོད་དེ་ཝཱི་ ( ¡¶ ཞ ¡· བི་ ) ཀོ་ཊ་སྟེ་ལྷ་མོ་རྫོང་དུ་མ་འགྲོ་ཞིག མཁའ་འགྲོ་མས་བར་ཆད་བྱེད་དོ་གསུང༌། དེ་ནས་འཁོར་དང་བཅས་ཏེ་ཆུ་ཁ་ཅིག་ཏུ་ཕྱིན་པ་ལ། མཛེ་མོ་ཞིག་ང་སྒྲོལ་ཟེར་བ་ལ། འདི་ཁུར་ན་འདི་ཡི་མཛེས་གོས་ན་དངོས་གྲུབ་ཚེ་གཅིག་གིས་བསྒྲུབ་པའི་བར་ཆད་དུ་འགྲོ་སྙམ་ནས་མ་བསྒྲལ་བ་ལ། འཁོར་ཀུ་ས་ལི་ཞིག་གིས་བསྒྲལ་བས། རྡོ་རྗེ་ཕག་མོར་གྱུར་ནས་ཀུ་ས་ལི་དང་གཉིས་མཁའ་སྤྱོད་དུ་སོང༌།

དེ་ནས་ཕར་ཕྱིན་པས་རྨོད་པ་གཅིག་དང་ཕྲད་པ་ལ། རྨོད་པ་དེ་ཐུག་ཆང་ས་འོག་ཏུ་སྦས་ནས་འདུག་པ་ལ་བསླངས་པས་མ་སྟེར་ཏེ། ཕ་ཟད་ན་རྡོ་བ་ཞིག་ལས་ཆུ་བག་ཙམ་འཛག་པ་ཅིག་འཐུང་ཟེར། དེས་མི་ངོམས་བྱས་པ་ལ་འཐུང་ནུས་པར་གྱིས་ཤིག་ངོམས་པར་བྱ་ཡིས་ཟེར་ཏེ། ཆུ་འཐུང་པ་ལ་ཆུ་ཟད་པར་མ་གྱུར་ཏོ། །དེས་ཐུག་ཆང་མ་སྟེར་བ་ལ་གླང་ལ་སྡིགས་མཛུབ་བྱས་པས་གླང་ཤི་བ་ལ་ད་འདི་རྗེས་སུ་གཟུང་དགོས་པས་གསོས་ཤིག་ཟེར་བ་ལ་མ་ནུས་སོ། །ཡང་དེ་ནས་ཕར་བྱོན་པ་ལ་སྐྱེད་མོས་ཚལ་ཞིག་ན། ན་ཆུང་ཞིག་འདུག་པ་ལ་སྐོམ་ནས་ཤིང་ཐོག་བསླངས་པས། ཁྱོད་རང་ལ་ནུས་པ་མེད་དམ་ཟེར་བ་ལ་ཤིང་ཐོག་ལ་ལྟུང་བའི་ལྟ་སྟངས་ཀྱི་བལྟས་པས་མར་ལྟུང་ཞིང་བྱུང་བ་ལ། མོས་མིག་གི་རྫི་མ་བཏེག་པ་ཙམ་གྱིས་ཡར་འབྱར་གྱིན་སོང༌། ཡང་ཕར་ཕྱིན་པ་ལ་ལྷ་ཁང་ཞིག་ན་མི་ཞིག་ ( ¡¶ ལུ ¡· གཉིས་ ) བརྒྱད་སྟོང་པ་ཀློག་གིན་འདུག་པ་ལ། ཁྱེད་པཎྜི་ཏའམ། རྣལ་འབྱོར་པའམ ༑ དགེ་བསྙེན་ལ་སོགས་པ་ཅིས་གནས་བྱས་པས། ཁོ་ན་རེ་རྣལ་འབྱོར་གྱི་ཆ་ལུགས་ཀྱིས་གནས་སཽ་བྱས་པས། སློབ་དཔོན་ན་རེ་རྣལ་འབྱོར་པ་བྱ་བ་ཀློག་བྱེད་དམ་བྱས་པ་ལ། འོ་ན་ཁྱོད་ཀྱིས་ཀློག་ཟེར་བ་ལ། སློབ་དཔོན་གྱིས་ཀློག་མ་ཤེས་ཏེ་ཕམ་མོ། །དེ་ནས་སློབ་དཔོན་གྱིས་ཤར་ཕྱོགས་བྷཾ་ ( ¡¶ ཞ ¡· བྷ་ ) ག་ལར། རཱ་ཛཱ་ཤྲཱི་སུ་ཙནྡྲ་དང༌། བློན་པོ་ཀུ་ ¡Ã ས་ལི་ནཱ་ཐའི་ ( ¡¶ ཞ ¡· སུ་ལ་ན་དའི་ ) གྲོང་ན་རྒྱལ་པོ་མུ་སྟེགས་པ་ཡིན། བློན་པོས་སློབ་དཔོན་ཀརྞ་པ་གྲོང་དུ་བྱོན་བྱ་བ་ཐོས་ནས། བསུར་ཕྱིན་པ་ལ་ང་ལ་བསྙེན་བཀུར་བྱེད་ཅིང་བསུ་བ་ཁྱེད་ཅི་དོན་དུ་གཉེར་བྱས་པ་ལ་བློན་པོ་ན་རེ། ང་ལ་དོན་དུ་གཉེར་རྒྱུ་གཞན་མེད། ལྷ་ཁང་ཞིག་ན་ལོ་ ¡Ã ཀེ་ཤཱ་ ( ¡¶ ཞ ¡· ཀི་ཀྭ་ ) རའི་སྐུ་ཏིར་ཏི་ཀ་དང་མཐུན་པ་ཞིག་བྱས་ནས་ཡོད་པ་འདིའི་རབ་གནས་ཞུའོ་བྱས་པ་ལ། བྱ་གསུང་ནའང་ཡུན་རིང་པོར་བྱེད་དུ་མ་འདོད་པ་ལན་ཅིག་ད་ལྟ་རང་བྱ་གསུང༌། འོ་ན་རབ་གནས་ཀྱི་འཚོག་ཆས་མི་དགོས་སམ་བྱས་པ་ལ་ཅི་ཡང་མི་དགོས་གསུངས་ནས་མེ་ཏོག་གཅིག་སྐུའི་སྤྱི་བོར་བཞག་སྟེ་སུ་པྲ་ཏིཥྛ་གསུངས་ནས་རབ་གནས་ཟིན་ནོ་བྱས་པས།

བློན་པོ་དེ་ལ་ཡིད་མ་ཆེས་པ་ལ་སློབ་དཔོན་གྱིས་མེ་ཏོག་ཕོག་པས། སྐུ་དེ་སློབ་དཔོན་གར་འགྲོ་བའི་རྟིང་ལ་སོང་བ་ལ་བློན་པོས་ཞུ་བ་ཕུལ་ཏེ་ད་ཡིད་ཆེས་པས་རབ་གནས་མཛོད་ཅིག་ཞུས་པས། མེ་ཏོག་གཅིག་མགོར་བཞག་སྟེ་ཡང་སུ་པྲ་ཏིཥྛ་ཞེས་བརྗོད་པས་སྐུ་དེར་འདུག་སྟེ། མི་མང་པོས་ནང་དུ་སྤྱན་དྲངས་པས་མ་དྲོངས་ནས་དེ་ཉིད་དུ་ལྷ་ཁང་བཞེངས། ད་ཅིའི་ལྷ་ཁང་ལ་མཆོད་རྟེན་ཞིག་ཁོང་པར་བཅུག་ནས་བཞག དེ་ནས་བློན་པོས་སློབ་དཔོན་ལ་ནང་པའི་རས་བྲིས་ཤིག་མཛད་པས་ཞུས་པ་ལ་གནང་ནས། འདྲི་མཁན་དང་གསུམ་ལས་གཞན་ལ་གསང་སྟེ། རྒྱལ་པོའི་ལྷ་འཇིགས་བྱེད་དང་དུས་མཚན་གདན་དུ་བཏིང་བ་བྲིས་པ་ལ། གཅིག་ནས་གཉིས་སུ་འཛགས་ཏེ་རྒྱལ་པོས་ཐོས་པ་ལས། རྒྱལ་པོ་ན་རེ་བློན་པོ་འདི་སྙིང་དང་འདྲ་བ་ཡིན་པས་འདི་མི་བདེན་མི་སྲིད་བྱས་པ་ལ། གཏམས་མང་དུ་གྲགས་པ་ལ་ད་ནི་བདེན་ནམ་སྙམ་ནས་འདུག་པ་ལ་བློན་པོ་གྲོགས་པོ་ཞིག་གིས་ཐང་ཀ་འདི་སྤོངས། ཁྱོད་ལ་རྒྱལ་པོ་འཁོར་དང་བཅས་པས་ཆད་པ་ཆེན་པོ་གཅོད་པར་འདུག་གོ་བྱས་པས། དེ་བཞིན་དུ་བྱའོ་ཟེར་ནས་རས་མང་པོས་དྲིལ་ནས་རྒྱས་བཏབ་སྟེ་ཉི་མ་ལྷོ་ཕྱོགས་ནས་ཕར་བསྐུར་ཏེ་བཏང་བ་ལ ༑ རྒྱལ་པོ་གྲོང་ཁྱེར་ཕྱོགས་བཞི་ན་སྒོ་བཞི་ཡོད་པའི་སྒོ་བ་རྣམས་ཀྱིས། འདི་ཙམ་གྱི་རྒྱས་འདེབས་པ་འདི་ན་ནོར་མང་པོ་ཡོད་ཟེར་ནས་གཡེངས་པས་བྲིས་སྐུ་དེ་བྱུང་བ་ལ། རྒྱལ་མཚན་གྱི་རྩེ་མོ་ལ་བཏགས་ནས་རྒྱལ་པོའི་གན་དུ་ཁྱེར་བ་ལ། རྒྱལ་པོའི་འཁོར་ཐམས་ཅད་འདི་སྲོག་དང་ཕྲལ་ལ། འདིའི་ནོར་ཐམས་ཅད་དང་དབྲལ་ལོ་ཟེར་བ་ལ། རྒྱལ་པོ་ན་རེ་ལར་ ( ¡¶ ཞ ¡· - ལར་ ) བྱ་བ་ཐམས་ཅད་བརྟགས་ནས་བྱ་དགོས་པས་ད་དུང་ཆད་པས་མི་བཅད་ཀྱིས། འུ་ཅག་ཀྱང་བདེ་མཆོག་གདན་དུ་བྱས་པའི་འཇིགས་བྱེད་དང་དུས་མཚན་སྟེང་དུ་བཏོན་པ་བྲིས་ལ་བདེན་པའི་བྱིན་རླབས་འགྲན་ཟེར་ན། མུ་སྟེགས་ཀྱི་དེའུ་ལའི་ནང་དུ་བཅུག་སྟེ། ཁྱེད་ཀྱི་ལྷ་ཡང་མཆོད་དོ་ཟེར་ནས་ཞག་བདུན་ན་ཆོ་འབྱེད་པ་ལ།

བློན་པོ་ཀུ་ས་ལི་ནཱ་ཐས་སློབ་དཔོན་ལ་ཞག་བདུན་ནས་ཆོ་འབྱེད་པས་ནུས་པ་བསྟན་དགོས་ཞུས་པ་ལ ༑ ནུས་པ་བསྟན་ཏོ་གསུང༌། དེ་ནས་ནམ་ཕྱེད་ན་ཏིར་ཏི་ཀའི་མཚན་མ་འཛིན་པ་ཐོད་པ་འཛིན་པ་གཅིག་བྱུང་བས། ངས་ཚོགས་དཔོན་བྱའམ་ཟེར་བ་ལ། ཁྱོད་ཀྱིས་ཚོགས་དཔོན་མི་བྱེད་དོ་བྱས་པ་ལ། ནམ་མཁའ་ལ་འཕུར་ནས་ཧེ་རུ་ཀར་བྱས་ནས་སོང་བ་ལ། བདག་རྗེས་སུ་གཟུང་དུ་གསོལ་བྱས་པ་ལ། ཁྱོད་ལ་ཚེ་འདི་ལ་མ་ཧཱ་མུ་དྲཱ་ ( ¡¶ ཞ ¡· དྲ་ ) ཐོབ་པའི་སྐལ་བ་མེད་ཀྱིས་གཞུང་དྲུག་རྩོམས་ཤིག་པར་གནང་བ་བྱིན་ཏེ། བཞི་ཐུན་མོང་མ་ཡིན་པ། གཉིས་ཐུན་མོང་བའོ། །འོ་སྐོལ་གྱི་ལྷ་ཁོའི་གདན་ན་བདོག་པ་བྱས་པ་ལ། འཇིག་རྟེན་གྱི་ལྷ་འདྲ་བས་གདན་དུ་གནོན་ནུས་སམ། གསུངས་ནས་དེ་ནས་ཞག་བདུན་ནས་བལྟས་ཙམ་ན ༑ འཇིགས་བྱེད་དང་དུས་མཚན་ཞབས་ཀྱིས་མནན་ནས་འོད་དང་བཅས་པར་བཞུགས་པ་མཐོང་ནས་ཐམས་ཅད་མོས་པར་གྱུར་ཏེ། སློབ་དཔོན་རྒྱལ་པོའི་གདན་སར་སྤྱན་དྲངས་ཏེ་རྫས་ལ་སོགས་པ་ཕུལ་ཏེ། དབང་བསྐུར་ལ་སོགས་པ་མནོས་ནས་བྷཾ་ ( ¡¶ ཞ ¡· བྷ་ ) ག་ལ་པ་ཙམ་སངས་རྒྱས་པར་བསྒྱུར་གསུང༌། དེ་ནས་སློབ་དཔོན་གྱིས་གཞུང་དྲུག་བརྩམས་སོ། །དེ་ནས་མཛེ་མོ་དེ་ཡང་མཛེ་མོ་མིན་ཏེ་རྡོ་རྗེ་ཕག་མོའི་རྣལ་འཕྲུལ་ལོ། །ཞིང་རྨོད་དེ་ཡང་ཞིང་རྨོད་མིན་ཏེ་ཧེ་རུ་ཀ་ཡི་རྣམ་འཕྲུལ་ལོ། །མཛེ་ལྡན་བུ་མོ་དེ་ཡང་བུ་མོ་མིན་ཏེ་རྡོ་རྗེ་ཕག་མོའི་རྣལ་འཕྲུལ་ཡིན ༑ ༑ ཐོད་པ་ཅན་ཡང་ཐོད་པ་ཅན་མ་ཡིན་ཏེ་ཧེ་རུ་ཀ་ཡི་རྣམ་འཕྲུལ་ཡིན། ཁྱོད་ཀྱི་ཞབས་ཀྱི་ལྗིད་ཀྱིས་མནན་ན་འོག་གི་རླུང་ཡང་འཇིག་འགྱུར་ཞེས་ཏེ་ཞབས་ཀྱི་རྩེ་མོས་མནན། །ཕྱག་རྣམས་དྲང་པོར་བརྐྱང་ན་ཕྱོགས་སྐྱོང་རྣམས་ཀྱང་སྐྲག་འགྱུར་ཞེས་ཏེ་ཕྱག་རྣམས་བསྐུམ་པས་གནས། །དབུ་ཡི་གཙུག་ཏོར་དྲང་པོར་བསྲངས་ན་ཚངས་པའི་འཇིག་རྟེན་འཇིག་འགྱུར་ཞེས་ཏེ་དབུ་ནི་ཡོ་བར་གནས ༑ རྗེ་བཙུན་ཁྱོད་ཀྱི་གར་ནི་དགོས་པ་དང་ལྡན་མཐོང་བས་བསྟོད། །ཅེས་གསུངས་ནས་ཛཱ་ལནྡྷ་རར་དགོན་པ་བྱེད་གསུང་པ་ལ།

ནད་ཀྱིས་ཐེབས་ནས་གཡོག་པོ་རྣམས་གནས་ཉི་ཤུ་རྩ་བཞིར་བཏང་སྟེ། བླ་མའི་དཀའ་བཅག་པ་ལ་བཤགས་པ་བྱས། དེ་ནས་དེ་མེ་ཏོག་དང་སྨན་བླངས་ནས་འོངས་པ་ལ། ལམ་ན་ཁྲུས་ཀྱི་རྫིང་ཡོད་པ་ལ་ཁྲུས་བྱ་ལོང་མེད་དེ་ལ་མོར་འགྲོ་བ་ལ། བུད་མེད་མང་པོ་དེ་ན་ཁྲུས་བྱེད་ཅིང་ཁྱེད་ཅག་ཀྱང་ཁྲུས་གྱིས་ཟེར་བ་ལ་ལོང་མེད་སློབ་དཔོན་ན་བས་མྱུར་དུ་འགྲོ་བྱས་པ་ལ། སློབ་དཔོན་སོས་ཟེར་ནས་ཁྲུས་བྱེད་ཅིང་བསྡད་པ་ལ ༑ སྨན་རྣམས་བུད་མེད་དེ་རྣམས་ཀྱིས་ཁྱེར་ནས་བུད་མེད་རྣམས་གར་སོང་ཆ་མེད ༑ དེར་ཁྱིམ་དུ་ཕྱིན་ཙམ་ན་སློབ་དཔོན་གྲོངས། ང་ལ་མཁའ་འགྲོ་མའི་ཆད་པ་བྱུང་བས་ཚེ་འདི་ལ་མ་ཧཱ་མུ་དྲཱ་ ( ¡¶ ཞ ¡· དྲ་ ) མེད་ཀྱང༌། བར་དོར་རྟེན་འདི་ལ་ཞུགས་ནས་ཕྱག་རྒྱ་ཆེན་པོ་ཐོབ་པར་བྱེད་ཀྱིས་ཞག་བདུན་དུ་མ་སྲེགས་ཤིག་གསུང་བ་ལ། ཞག་ལྔར་གསང་ཐུབ་པ་ལ། རྒྱལ་པོའི་རྩིས་པས་རྩིས་བྱས་པས་ཡུལ་ཕྱོགས་ན་རོ་ཡོད་པས་ཡུལ་ཐམས་ཅད་འཚོགས་ཤིག་ཟེར་ཏེ་བསྲེག་ཏུ་བཏང་ངོ༌། །བསྲེགས་པས་དང་པོའི་གནས་དང༌། དེ་ནས་བསྲེག་པ་ན་ཌ་མ་རུ་ལ་སོགས་པའི་སྒྲ་དུ་མ་གྲག་གོ །དེ་ནས་གཞན་དུ་སྐུ་འཁྲུངས་ནས་ང་སྤྱོད་པ་བྱེད་པའི་དུས་ལ་བབ་ན། རྒྱན་རང་ཆས་སུ་ཤོག་ཅིག་ཅེས་སྨོན་ལམ་བཏབ་པས་བྱུང་ངོ༌། །དེ་ལྟ་བུའི་སློབ་དཔོན་ཁྱད་པར་ཅན་དེས་ལྷ་ལས་གནང་བ་ཐོབ་ནས་བརྩམས་སོ། །འདི་ལ་དོན་གསུམ་སྟེ། དང་པོ་ཀླད་དོན་བསྟན་ཏེ། དེ་ཡང་དང་པོ་ལ་ཕྱག་འཚལ་ཞིང་བཤད་པར་དམ་བཅའ་བ་ཤློ་ཀ་གཅིག་གིས་བསྟན། དེ་ནས་བརྩམ་པའི་རྒྱུ་ཡོད་པ་ཤློ་ཀ་གཅིག་གིས་བསྟན། དེ་ནས་ཁེངས་བསྐྱུང་གཅིག་གིས་བསྟན། དེ་ནས་བརྗོད་བྱ་ཁྱད་པར་ཅན་དང་ལྡན་པ་གཅིག་གིས་བསྟན་ཏོ། །དེ་ཡང་ཕྱག་དང་བཤད་པར་དམ་བཅའ་བ་གཉིས་ལས། དང་པོ་ལ་ཕྱག་འཚལ་བ་བསྟན་ཏེ། དཔལ་ལྡན་ཞེས་པ་དཔལ་ནི་ཡེ་ཤེས་སམ་རྡོ་རྗེ་ཕག་མོ་ལ་བྱ་ལ། དེ་དང་ལྡན་པས་དཔལ་ལྡན་ནོ། །འགྲོ་བའི་མགོན་གྱིས་གཙོ་བོ་ཡབ་ཡུམ་དངོས་སུ་སྨོས་ལ།

ཤུགས་ལས་མཁའ་འགྲོ་མ་ལ་སོགས་པ་བཞི་དང༌། དུར་ཁྲོད་མ་བརྒྱད་ཤུགས་ལས་བསྟན། ཉི་ཤུ་རྩ་བཞི་གཉིས་མེད་ཅེས་པས་དཔའ་མོ་བསྟན་ཏེ། དེ་ལྟར་དཀྱིལ་འཁོར་གྱྀ་འཁོར་ལོ་ཚང་བ་ལ་ཕྱག་འཚལ་བར་བསྟན། དེ་ནས་བཤད་ཅེས་སྦྱར། གང་བཤད་ན་དང་པོ་བསྙེན་པ་བཤད་ཅེས་སྦྱར་ཏེ། རབ་འབྲིང་གསུམ་སྟེ། རབ་ལྷས་གནང་བ་བྱིན་པ་དང༌། གཉིས་པ་རྨི་ ¡Ã ལྟས་སུ་ ( ¡¶ ཞ ¡· ལམ་ལྟ་བུས་ ) མཐོང་བ་དང༌། གསུམ་པ་སྔགས་ཀྱི་གྲངས་ཚད་དུ་སྐྱེལ་བའོ། །དེ་ནས་དཀྱིལ་འཁོར་གྱི་སྤྱོད་པ་སྤྱད་པ་ཞེས་བྱ་སྟེ། སའི་ཆོ་ག་དང༌། ཐིག་དང༌། སྟ་གོན་དང༌། བྲི་བ་དང༌། ཚོན་བྱིན་གྱིས་བརླབ་པ་དང༌། དཀྱིལ་འཁོར་བྱིན་གྱིས་བརླབ་པ་དང༌། འཇུག་པ་དང་བརྒྱད་དོ། །དེ་ནས་བརྟུལ་ཞུགས་ཀྱི་རིམ་པ་སྟེ། མེ་ཏོག་ཆུའི་དབང་བསྐུར་ལ་སོགས་བཅུ་བཞིའོ། ། འོ་ན་ཁྱོད་ལ་གཞུང་རྩོམ་པའི་རྒྱུ་ཡོད་དམ་ཞེ་ན། ལན་དུ་ལུགས་སུ་འབྱུང་དང་ལ་སོགས་པ་སྟེ། ལུགས་སུ་འབྱུང་བ་སྟེ། ལྷ་དང་སྔགས་དང་རྒྱུད་ཀྱི་རིམ་པ་མ་ནོར་བ་ཡང་ཤེས་ཞེས་སྦྱར། བཟློག་པ་དང་ལྷ་གོ་རིམ་བཟློག་པས་བཀོད་པ་དང་ ༑ སྔགས་དང་རྒྱུད་ཀྱི་གོ་རིམ་བཟློག་ནས་བསྟན་པ་ཡང་ཤེས་ཞེས་སྦྱར་རོ།། ཡང་སྦྱིན་བྱ་ཐ་དད་དེ། བུམ་པ་དེའི་དབང་དང༌། སྤྱི་སྟེ་གསང་དབང་དང༌། སྦས་པ་སྟེ་ཤེས་རབ་ཡེ་ཤེས་དང༌། ¡Ã བཞི་པ་ ( ¡¶ ལུ ¡· ཞི་བ་ ) ཡང་ཤེས་ ( ¡¶ ལུ ¡· + ཡེ་ཤེས་ ) ནས་བརྩམས་པ་ཡིན་ནོ། །ད་ནི་གསུམ་པ་ཁེངས་བསྐྱུང་བསྟན་ཏེ། སྒྲ་སྟེ་ལེགས་པར་སྦྱར་བའི་སྐད་ཀྱང་སྤངས་ལ། སྒྲ་ཉམས་ཏེ་ཟུར་ཆག་ཀྱང་སྤངས་ལ། ¡Ã བདེ་བས་སྦྱར་ ( ¡¶ ལུ ¡· བསྡེབས་སྦྱོར་ ) ཏེ་རྟགས་དང་བྱེད་ཚིག་དང༌། ཉེ་བར་བསྒྱུར་བ་སྐྱོན་མེད་ཀྱང་སྤངས་པའི་དཀྱིལ་འཁོར་གྱི་ཆོ་ག་བཤད་དོ། །འོ་ན་རྩོམ་པ་ལ་གཞུང་མཛེས་པ་དང༌། དོན་ལྷག་པ་གཉིས་ཡོད་དགོས་ཏེ། དེ་ཡོད་དམ་བྱས་པ་ལ། དེ་ཀུན་མེད་ཀྱང་རང་ཉིད་དྲན་པར་བྱ་བའི་ཕྱིར་བརྩམས་པ་སྟེ། འདིས་གཞན་དོན་དུ་མ་བརྩམས་ཞེས་ཁེངས་བསྐྱུང་བའོ།

།གར་གྱི་གནས་བྱ་བ་ལ་གར་མཁན་དང༌། དེའི་གནས་ཏེ་སྦུབས་སོ། །དཔེར་ན་གར་གྱི་གནས་གཅིག་ལ་ཚངས་པ་དང་ཁྱབ་འཇུག་ལ་སོགས་པ་དུ་མའི་འདྲ་འབག་ཏུ་སྟོན་པ་བཞིན། ཕ་རོལ་ཏུ་ཕྱིན་པའི་ཐེག་པ་གཞན་ཡང་ཐོས་པ་ཙམ་ལམ་དུ་ཁྱེར་ཏེ། ངེས་པ་མ་བསྐྱེད་པས་དེ་ཁོ་ན་ཉིད་གཅིག་གྲུབ་མཐའ་དུ་མར་གདགས་སོ། །དེ་ལྟ་བུ་དེ་མནྟྲ་ཏྭ་མཐོང་བ་ལ་ལམ་དུ་ཁྱེར་བ་ཡིད་ཆེས་པ་བཞི་ལ་བརྟེན་ནས་སྐྱེས་པས་ངེས་པའི་སྒོ་ནས་འཇུག་པ་མེད་དོ་ཞེས་སུན་འདོན་ནོ། །དེ་ནས་ཕ་རོལ་ནི་ཚིག་ཙམ་ལ་སྟོན་པ་ཡིན་པར་སྔགས་ནི་མཚོན་བྱེད་ཁྱད་པར་ཅན་གྱིས་དེ་དངོས་སུ་རྟོགས་པས། དོན་ལ་སྟོན་པས་དེས་དཔྱད་པ་སྟེ། ལོ་ཀའི་སིངྔྷི་དང༌། ཐར་པ་སྟེ། མཆོག་གི་དངོས་གྲུབ་དང༌། རང་དགའ་སྟེ། དུས་ཀྱི་ངེས་པ་ལ་མི་ལྟོས་ཏེ་ཚིག་གཅིག་གིས་ཅི་བདེ་བར་ཐོབ་པར་བྱེད་པས་སོ། །དེ་ལྟ་བུ་དེ་ཡང་དབང་ཐོབ་པ་ལ་སྟོན་པ་ལ་དབང་གི་དཀྱིལ་འཁོར་དགོས་ན་དཀྱིལ་འཁོར་དེ་སུའི་དོན་དུ་བྲི་ཞེ་ན། ལན་དུ། ལེགས་པར་བརྟགས་པའི་སློབ་མ་ལ་ཞེས་པ་སྟེ། བརྟག་པ་ནི་གཉིས་ཏེ། རང་གི་སྡེ་པའི་གཞུང་གིས་བརྟག་པ་དང༌། གཞན་གྱི་སྡེ་པའི་གཞུང་གིས་བརྟག་པའོ། །དེ་ལ་གཞན་གྱི་སྡེ་པའི་གཞུང་གིས་བརྟག་པ་ནི། རྣམ་པར་སྣང་མཛད་བྱང་ཆུབ་པ་ལས། རྒྱལ་རིགས་དང༌། རྗེའུ་རིགས་དང༌། བྲམ་ཟེ་དང་ ༑ ཚོང་དཔོན་ལ་སོགས་པ། རིགས་དང་ཡོ་བྱད་སྙེམས་པ་ལ་དབང་མི་བསྐུར་བར་བཤད་ལྟ་བུའོ། །རང་གི་རྒྱུད་ཀྱིས་བརྟག་པ་ནི་ཧེ་རུ་ཀ་འབྱུང་བ་ལས ༑ སྤུན་དང་བུ་དང་ལྷན་ཅིག་ཏུ། །བདེ་དང་སྡུག་བསྔལ་ཀུན་འབྱུང་བྱའི། ། གསང་སྔགས་དེ་ལ་སྦྱིན་མི་བྱ། །ཞེས་གསུངས་པ་དང༌། ཡང་དེ་ཉིད་ལས། སྔགས་དང་དེ་བཞིན་ཕྱག་རྒྱ་དང༌། །ཡེ་ཤེས་དེ་ལ་སྦྱིན་མི་བྱ། །ཞེས་གསུངས་པ་ལྟ་བུ་སྟེ། མདོར་ན་རིགས་སྙེམས་པ་དང༌། ལོངས་སྤྱོད་སྙེམས་པ་དང༌། བུ་ལ་སོགས་པ་ཉེ་བའི་སྙེམས་པ་དང༌། རྒན་པོ་དང་བྱིས་པ་ལ་སོགས་པ་ནུས་པ་མེད་པ་དེ་རྣམས་མ་ཡིན་པ་དེ་ནི་ལེགས་པར་བརྟག་པའོ།

།དེ་ལྟར་བརྟགས་པའི་རྗེས་ལ། སློབ་མས་བླ་མ་ལ་ནུས་པ་དང་སྦྱར་བའི་བསྙེན་བཀུར་དང་མཎྜལ་བྱ། བླ་མས་ཅི་དོན་དུ་གཉེར་དྲི། སློབ་མས་བདག་གསང་སྔགས་ཀྱི་སྒོ་ནས་རྗེས་སུ་བཟུང་དུ་གསོལ་ཞེས་བརྗོད། དེ་ནས་བླ་མས་དེ་ལ་སྐྱབས་འགྲོ་སྔོན་དུ་བཏང་ལ་བླ་མ་ལྔ་བཅུ་པ་བཤད་དགོས་པས། ཡོན་ཏན་ཐམས་ཅད་ཀྱི་གཞིར་གྱུར་པ་སོ་སོར་ཐར་པའི་སྡོམ་པ་ལ། ¡Ã གསུམ་པ་ ( ¡¶ ལུ ¡· - གསུམ་པ་ ) སྐྱབས་སུ་འགྲོ་བའི་ཚུལ་གྱིས་བསྙེན་ནམ་ཞེས་དྲི། བསྙེན་ན་བཤད། མ་བསྙེན་སྐྱབས་འགྲོ་ཁྱད་པར་ཅན་སྔོན་དུ་འགྲོ་བའི་ཚུལ་གྱིས་སེམས་བསྐྱེད་དང་སྡོམ་པ་སྦྱིན་ནོ། །དེ་ནས་སློབ་དཔོན་དང་སློབ་མའི་རྒྱུད་བརྟག སློབ་དཔོན་སློབ་མ་ཕན་ཚུན་གཅིག་ལ་གཅིག་མགུ་ན་སློབ་མའི་འབྲེལ་བ་བྱ། མ་མགུ་ན་བདེ་བྱཻེས་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· འབྱེས་ ) བྱའོ། །མགུ་ན་སློབ་དཔོན་གྱིས་དཀྱིལ་འཁོར་གསུམ་ལས་རྡུལ་ཚོན་གྱི་དཀྱིལ་འཁོར་བྲིའོ། །འོ་ན་སློབ་དཔོན་མཚན་ཉིད་ཇི་ལྟ་བུ་དང་ལྡན་པས་བྲི་ཞེ་ན

༑ ཡང་དག་པའི་ཡེ་ཤེས་སྟོན་པར་བྱེད་པའི་རྒྱུད་ནི་བདེ་མཆོག་གོ །དེའི་གསང་སྔགས་ཀྱི་དོན་ཤེས་པ་ཡིན། མ་ཤེས་ན་སློབ་མ་ལ་བྱིན་རླབས་དགོད་མི་ནུས་སོ། །གཙང་མ་ནི་དམ་ཚིག་ཉམས་པའི་སྐྱོན་གྱིས་མ་གོས་པའོ། །མཁས་པ་ནི་དེ་ཁོ་ན་ཉིད་བཅུ་ཤེས་པའོ། །སྦྱོར་ཤེས་ནི་དངོས་གྲུབ་བཞི་དང་བརྒྱད་བསྒྲུབ་ཤེས་པའོ། །ཡེ་ཤེས་ཕ་རོལ་ནི་མཆོག་འགྲུབ་ཤེས་པའོ། །དཔལ་ལྡན་ཁྲག་འཐུང་ཚོགས་ཆ་ལུགས་ནི། བདག་ཉིད་ཐ་མལ་པའི་རྟོག་པ་སྤངས་ནས་སྤྱོད་ལམ་ཐམས་ཅད་དུ་བདེ་མཆོག་གི་རྣལ་འབྱོར་ལ་གནས་པའོ། །དེའི་སྟེང་དུ་གསང་བའི་ཡེ་ཤེས་སྤངས་ཤིང་ཐེག་པ་དམན་པ་ལ་མོས་པ་ལ། དེ་འཁོར་བ་ལས་འདོན་པར་འདོད་པའི་སྙིང་རྗེ་དང་ལྡན་པའོ། །འོ་ན་གཉིས་པ་དེས་རྗེས་སུ་གཟུང་བར་བྱ་བའི་སློབ་མའི་མཚན་ཉིད་ཇི་ལྟ་བུ་ཞེ་ན། འཇམ་ཞིང་ལ་སོགས་པའི་ཚིག་རྐང་དྲུག་གིས་བསྟན་ཏོ། །ཡོངས་སུ་བཟུང་ཞེས་པའི་སློབ་མ་གཅིག་ནས་བཟུང་སྟེ། གསུམ་མམ། ལྔའམ། ཉི་ཤུ་རྩ་ལྔའི་བར་ཟུང་དུ་མ་གྱུར་པ་ཡོངས་སུ་བཟུང་སྟེ། དེ་ལྟར་ཡང་སློབ་མ་ཟུང་དུ་གྱུར་པ་ནི་ནམ་ཡང་བཟུང་བར་མི་ནུས་སོ། །དེའི་ནང་ནས་ཀྱང་གཙོ་བོ་གཅིག་སློབ་དཔོན་གྱི་འོས་སུ་བཟུང་ངོ༌།། དེ་ལྟར་སློབ་མ་མཚན་ཉིད་དང་ལྡན་དུ་ཟིན་ཀྱང༌། ལས་དང་པོ་པ་ལ་མཎྜལ་དུ་འཇུག་པ་ལ་བསྙེན་པ་ནས་བཟུང་ནས་མཎྜལ་གྱི་ ¡Ã ཡོ་བྱད་ ( ¡¶ ལུ ¡· ལོངས་སྤྱོད་ ) འཚོགས་དགོས་པས། དེ་སློབ་དཔོན་གྱིས་དྲི་ཞིང་སློབ་མས་ཁས་བླང་བ། འཁོར་ལོའི་ཡོ་བྱད་ཅེས་རྐང་པ་བརྒྱད་ཀྱིས་བསྟན། དེ་ལྟར་ཡོ་བྱད་ཚོགས་ཀྱང་དམ་ཚིག་དང་སྡོམ་པ་འཛིན་མི་ནུས་ན། དབང་བསྐུར་བའི་འོས་མིན་པས། དེ་དྲི་ཞིང་སློབ་མས་ཁས་བླང་བ། དམ་ཚིག་སྐྱོང་ཞིང་ཞེས་པ་ཚིག་རྐང་བརྒྱད་ཀྱིས་བསྟན་ཏེ། དེ་ལ་གཟའ་ལ་བརྟེན་པའི་སྡིག་པ་ནི་ཏིར་ཏི་ཀ་འཚེ་བ་ཆོས་སུ་སྨྲ་བ་རྣམས་ཉི་ཟླ་གཟས་འཛིན་ཙམ་ན། སྡིག་པ་བྱས་ཀྱང་སྡིག་པར་མི་འགྱུར་ལ། སླར་ལ་སུ་བསད་པ་བྲན་དུ་སྐྱེ་བར་འདོད་པའོ།

།སྐུ་ལ་བརྟེན་པ་ ( ¡¶ ཞ ¡· ནས་ ) མ་ཧཱ་དེ་བའི་ལྷ་འགྲམ་དུ་མ་ཧེ་ལ་སོགས་པ་བསད་ན་གྲོལ་བར་འདོད་པ་དང༌། དེར་ཤ་དང་རུས་པ་ལ་སོགས་པས་མཆོད་ན་གྲོལ་བར་འདོད་པ་སྟེ། དེ་ལ་རབ་ནི་ལྷས་གནང་བ་བྱིན་པའོ། །འབྲིང་མཎྜལ་བྲིས་ཤིག་པར་རྨི་ལྟས་སུ་བསྟན་པའོ ༑ ༑ ཐ་མ་གྲངས་ཀྱི་བསྙེན་པ་སྐྱོལ་བ་སྟེ། ¡Ã དེ་ཡང་དཔའ་བོ་དང་ཡུམ་གྱི་སྙིང་པོ་གཉིས་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· ཀུན་ཏུ་བརྟགས་པའི་སྡིག་པའོ། །སྐྱེ་བོའི་མང་པོའི་སྡིག་པ་ཅེས་པ་ལྷན་ཅིག་སྐྱེས་པའོ། །ད་ནི་སློབ་མ་དབང་བསྐུར་བའམ་སིདྡྷི་སྒྲུབ་པའི་ཕྱིར་མཚམས་བཅད་ལ་བསྙེན་པ་བྱ་སྟེ༑ ) སྦྲགས་ལ་ཁྲི་ཕྲག་གསུམ་བཟླས་སོ། །དཔའ་བོ་ཉི་ཤུ་རྩ་བཞི་དང་དཔའ་མོ་ཉི་ཤུ་རྩ་བཞིའི་དཔའ་བོའི་ཧཱུྃ་བོར་ཞིང༌། དཔའ་མོའི་ཨྃ་བོར་ལ། གཉིས་སྦྲེལ་ལ་ཁྲི་ཕྲག་གཉིས་བཟླས། རྣལ་འབྱོར་མ་བཞི་དང་ཧཱུྃ་བཞི་སྦྲེལ་མ་བྱ། ¡Ã ཀཱ་ཀཱ་སྱེ་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· ཀ་ཀ་ཤང་ ) ལ་སོགས་པ་རྐང་པ་བརྒྱད་པར་སྤགས་པ་ལྔ་སྟོང་ལྔ་བརྒྱ་བཟླས་སོ། །དེ་ལྟ་བུའི་བསྙེན་པ་དེ་གནས་གང་དུ་བྱ་ཞེ་ན། དེ་ནས་དཔའ་བོ་ཆེན་པོས་འདིར་ཞེས་བྱ་བ་ལ་སོགས་པས་བསྟན་ཏེ། དང་པོ་གནས་ནི་ཁྱད་པར་དུ་ཞེས་བྱ་བ་ལ་གནས་ནི་གཉིས་ཏེ། སྒྲུབ་པ་པོས་བསྙེན་པ་བྱ་བའི་གནས་དང༌། དཀྱིལ་འཁོར་བྲི་བའི་གནས་སོ། །དེ་ལ་དང་པོ་ལ་ཁྱད་པར་དྲུག་སྟེ། རྒྱལ་པོ་ཆོས་མཐུན་པར་སྤྱོད་པའི་གནས་དང༌། རྒྱལ་པོ་ཡུལ་ན་གནས་པ་དང༌། རྒྱལ་པོ་འཁྲུག་པ་ལ་སོགས་པ་འཚེ་བ་མེད་པའི་གནས་སོ། །དེ་བཞིན་དུ་ཡུལ་ལ་ཡང་གོང་མ་ལ་གསུམ་སྦྱར་ཏེ་དྲུག་གོ །དཀྱིལ་འཁོར་གྱི་གནས་ནི་གྲོང་གི་བྱང་ཤར་གྱི་ཆའོ། །འཇིགས་བྲལ་ཞེས་པ་ལ་སོགས་པ་ཁྱད་པར་ཅན་བསྟན། འདྲེས་མེད་ཅེས་པ་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· + ཐེག་པ་ ) དམན་པའི་སེམས་ཀྱིས་མ་འདྲེས་པའོ། །དཔལ་ལྡན་ཧེ་རུ་ཀ་རང་ ( ¡¶ ཞ ¡· དང་ ) ཞེས་པས་ནི་ལྷའི་རྣལ་འབྱོར་ལ་གནས་ཏེ་བསྙེན་པ་བྱ་བར་བསྟན་ཏོ། །གནས་ནས་ཞེས་པ་བཞིས་ལྷ་དེ་བསྐྱེད་པའི་ཚུལ་བསྟན་ཏོ།

། ད་ནི་དཀྱིལ་འཁོར་གྱི་སྤྱོད་པ་བཤད་དེ། དེ་ལ་བདུན་ལས་དང་པོ་ལ་སའི་ཆོ་ག་བསྟན ༑ དེ་ལ་དྲུག་ལས་དང་པོ་ལ་བརྟག་ཅིང་གཟུང་བ་བསྟན་ཏེ། དེ་ལ་གཉིས། ཡིད་ཀྱིས་གཟུང་བ་དང༌། ཡང་དག་པར་རྫོགས་པའི་སངས་རྒྱས་ཀྱིས་ས་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· - ས་ ) བཟུང་བའོ། །ཡིད་ཀྱིས་གཟུང་བ་ནི་གནས་དེ་ཡིད་ཀྱིས་བདག་གིར་བྱེད་པ་སྟེ། དེ་ཡང་དེ་ནས་ཞེས་པ་ནས། མཛེས་ཤིང་ཡིད་ནི་དགའ་བ་རུ། །ཞེས་པའི་བར་གྱིས་བསྟན། གཟུང་བ་གཉིས་པ་གནས་དེར་མཎྜལ་བྱ། དྲིའི་ཐིག་ལེ་དང་མེ་ཏོག་གི་ཚོམ་བུ་སུམ་ཅུ་རྩ་བདུན་བྱས་པས་ ¡Ã བུངྔྷས་ ( ¡¶ ལུ ¡· ཤུདྡྷས་ ) བྱིན་གྱིས་རློབ་སྟེ། སངས་རྒྱས་ཀྱིས་ས་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· - ས་ ) བཟུང་བའོ། །དེ་ནས་བསླང་བར་བྱ་སྟེ། དེ་ལ་གཉིས་ཏེ། སྣང་བ་ལས་བསླང་བ་དང༌། མི་སྣང་བ་ལས་བསླང་བའོ ༑ ༑ དེ་ལ་གཉིས་ཏེ། ཞི་བས་བསླང་བ་ཞིང་སྐྱོང་ལ་གཏོར་མ་སྦྱིན་ལ་བཏང་ ( ¡¶ ཞ ¡· བླང་ ) ། དྲག་པོས་བསླང་བ་མགུལ་རྒྱན་གདུ་བུ་ཞེས་པ་ལ་སོགས་པས་བསྟན ༑ གསུམ་པ་སྦྱང་བ་ལ་གཉིས་ཏེ། ལས་དང་བྱ་བས་སྦྱང་བ་དང༌། ཡང་དག་པར་རྫོགས་པའི་སངས་རྒྱས་ཀྱིས་སྦྱང་བའོ། །དང་པོ་ནི་རབ་འབྲིང་གསུམ་དུ་བརྐོས་ལ། རྡོ་བ་དང༌། རུས་པ་དང༌། ཕུབ་མ་ལ་སོགས་པའི་སྐྱོན་བསལ་ ¡Ã ཏེ ༑ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· ལ་དེ་ ) འཐས་པར་བཅགས་ལ། ཆུ་དང་ལྕི་བ་ལ་སོགས་པས་བྱུགས་ཏེ། དེ་ཡང་རྡོ་བས་ནི་དམུ་ཆུ་སྐྱེད་ལ། རུས་པས་གཟེར། ཕུབ་མས་རིམས་བསྐྱེད་པར་བཤད་དོ། །གཉིས་པ་ནི་དེར་མེ་ཏོག་གི་ཚོམ་བུ་སུམ་ཅུ་རྩ་བདུན་བྱས་ཏེ། ལྷར་བསྐྱེད་པ་སྟེ། དེ་ཡང་དཔའ་བོའི་དབང་པོ་ལ་སོགས་པའི་མཚན་ཉིད་ཀྱིས་བསྟན་ཏོ། །བཞི་པར་བྱིན་གྱིས་བརླབ་པ་ནི། དབང་ ( ¡¶ ལུ ¡· དཔལ་ ) ལྡན་ཕྱོགས་ཞེས་པ་ནས་བགེགས་ལ་སོགས་པ་བསྐྲད་པར་བྱ་བའི་བར་གྱིས་བསྟན་ཏོ། །ཞལ་བཞི་ལ་སོགས་ཞེས་པ་ན་མར་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· + ས་ ) བསྲུང་བར་བསྟན་ཏེ། བདག་སྣ་ཚོགས་རྡོ་རྗེ་སྟེང་དུ་སྙིང་གའི་ཧཱུྃ་ལས་ལྷ་བཅུ་སྤྲོས་པས་བརྒྱད་ཀྱིས་ཕྱོགས་སྐྱོང་བརྒྱད་དགུག

ཁ་གདོང་མས་སྟེང་གི་དགུག ཁྱི་གདོང་མས་འོག་གི་དགུག ཡང་ན་ཁཎྜ་རོ་ཧཱ་ ( ¡¶ ཞ ¡· ཧ་ ) གཉིས་ཀྱིས་དགུག་སྟེ། ཕུར་བུས་གདབ་པ་དང༌། ཐོ་བས་བརྡུང་པ་ལ་སོགས་པ་དང༌། དེ་ཐམས་ཅད་རོར་སོང་བ་དང༌། རོ་བཅུ་ལས་ར་བ་དང་གུར་ལ་སོགས་པ་བསྐྱེད་པ་ལོ་ཧཱི་པ་དང་མཐུན་ནོ། །དེ་ནས་སའི་ལྷ་མོར་མཁྱེན་པར་གསོལ་བ་བྱ་སྟེ། དེ་ཙམ་གྱིས་སའི་ཆོ་ག་རྫོགས་སོ། །ད་ནི་གཉིས་པ་ལྷག་པར་གནས་པ་བསྟན་ཏེ། དེ་ལ་གསུམ་ལས་ལྷ་ལྷག་པར་གནས་པ་ལ་གཉིས་ཏེ། རང་ཉིད་ལྷར་བསྐྱེད་པ་དང༌། མདུན་དུ་བསྒོམ་པ་དང་གཉིས། དེ་ལ་དང་པོ་བདུན་རྣམ་དག་ནས་བཟུང་ནས་ལྷ་སུམ་ཅུ་རྩ་བདུན་སྒྲུབ་ཐབས་ནས་བྱུང་བ་ལྟར་བསྐྱེད། ཡེ་ཤེས་པ་དགུག དབང་བསྐུར་མཆོད་བསྟོད་ཀྱི་བར་དུ་བྱ། དེ་ནས་བདུན་པ་ལ་སོགས་ཀྱི་ལྷ་མོ་བཅུས་ཕྱོགས་སྐྱོང་བཅུ་དགུག ཀཱི་ལ་ཡས་ཐོ་བས་བརྡུང་པས་རོར་སོང༌། རོ་བཅུ་ལས་ར་བ་དང་གུར་ལ་སོགས་པ་བསྐྱེད། ལྷག་མ་གཞུང་ན་གསལ། གཉིས་པ་སློབ་མ་ལྷག་པར་གནས་པ་ནི་སློབ་མ་ཁྲུས་བྱ། ལོ་ཀའི་བ་ལིཾ་གཏང༌། མཎྜལ་བྱ། སྔགས་ཀྱི་དངོས་གྲུབ་དོན་གཉེར་གང༌། །ཞེས་སོགས་ཀྱིས་མྱ་ངན་ལས་འདས་པའི་བསམ་པ་ཁྱད་པར་ཅན་བསྐྱེད། དེ་ནས་རང་ཧེ་རུ་ཀར་བྱས་ཏེ། ¡Ã བཱ་ར་ཧཱི་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· བ་ར་ཧི་ ) དང་སྦྱོར་བ་ལ། སློབ་མ་འོད་ཀྱི་གོང་བུའི་ཚུལ་དུ་སྣ་བུག་གཡོན་པ་ནས་ཞུགས་ནས་ཁོང་བར་སོང་སྟེ། དཀུ་གཡོན་པ་ནས་བྱུང་ནས་མདུན་དུ་འདུག་སྟེ། དགའ་ཆེན་སོགས་ཀྱིས་གསོལ་བ་འདེབས་པ་ཁ་བསྐང༌། དེ་ནས་བུ་ཚུར་སོགས་ཀྱིས་སྤྲོ་བ་བསྐྱེད། དེ་ནས་མཎྜལ་བྱ། སྐྱབས་འགྲོ་སྔོན་དུ་འགྲོ་བའི་སྒོ་ནས་ནང་གི་སེམས་བསྐྱེད་བྱ། དེ་ནས་དམ་ཚིག་དང་སྡོམ་པ་གཟུང༌། སོ་ཤིང་དོར། སྲུང་སྐུད་གདགས། ཀུ་ཤ་སྔས་དང་སྟན་དུ་སྦྱིན། དེ་ནས་འཇིག་རྟེན་ན་ནི་ཀུན་མཁྱེན་དག །སོགས་ཀྱི་ཆོས་བཤད་དོ། །དེ་ཙམ་གྱིས་སློབ་མ་ལྷག་པར་གནས་པར་བསྟན་ཏོ། །དེ་ནས་བུམ་པ་ལྷག་པར་གནས་པ་བྱ་སྟེ། གྲངས་དང༌།

མཚན་ཉིད་དང༌། ཁ་རྒྱན་དང༌། མགུལ་བཅིང་པ་ལ་སོགས་པ་གཞན་དང་མཐུན། མཚན་མ་བྲི་བ་ནི། རྣམ་རྒྱལ་ལ་རིགས་ལྔའི་མཚན་མ་བྲི། བརྒྱད་ལ་གྲི་གུག་བྲི། ལས་ཐམས་ཅད་པ་ལ་གྲི་གུག་གམ་སྣ་ཚོགས་རྡོ་རྗེ་བྲི། བུམ་པ་བྷྲཱུྃ་ལས་གཞལ་ཡས་ཁང་པ་བསྐྱེད། རྣམ་རྒྱལ་དུ་ལྷ་སུམ་ཅུ་རྩ་བདུན་ཚང་བར་བསྐྱེད། ལས་ཐམས་ཅད་པར་བདེ་བ་ཆེན་ ¡Ã པོ་ ( ¡¶ ཞ ¡· པོའི་ ) ལྷ་ལྔའམ། ཁཎྜ་རོ་ཧཱ་ ( ¡¶ ཞ ¡· ཧ་ ) བསྐྱེད། བརྒྱད་དུ་ ¡Ã ཀཱ་ཀཱ་སྱེ་ ( ¡¶ ཞ ¡· ཀ་ཀ་ ) སོགས་བརྒྱད་བསྐྱེད ༑ ཕཻཾ་གྱིས་ཡེ་ཤེས་པ་བསྟིམ། སོ་སོའི་སྔགས་བརྒྱ་རྩ་བརྒྱད་དམ། ཡང་ན་ ( ¡¶ ཞ ¡· + ལས་ ) ཐམས་ཅད་ལ་སརྦ་བུངྔྷ་ཌཱ་ཀི་ ¡Ã ནཱིའི་ ( ¡¶ ཞ ¡· ནིའི་ ) སྔགས་བརྒྱ་རྩ་བརྒྱད་བཟླས། རྣམ་རྒྱལ་གྱི་སྟེང་དུ་ཀ་པཱ་ ( ¡¶ ཞ ¡· པ་ ) ལ་བཞག་སྟེ། བདུད་རྩི་ཐམས་ཅད་དུ་ ( ¡¶ ཞ ¡· - དུ་ ) བླུག་གོ །བཅུ་པོ་དེའི་སྟེང་དུ་ཁྲུས་ཀྱི་བུམ་པ་བརྒྱད། དྲིའི་བུམ་པ་དང༌། མངར་གསུམ་གྱི་བུམ་པ་དང༌། ཞོ་དང་འབྲས་བུ་ ( ¡¶ ཞ ¡· - བུ་ ) དང་འབྲུ་སྣ་ཚོགས་དང༌། འབྲས་ཡོས་དང༌། རིན་པོ་ཆེའི་བུམ་པ་སྟེ་བརྒྱད་དོ ༑ ༑ དེ་རྣམས་ལྷོ་རུ་བཞག་ལ་རྡུལ་ཚོན་སྦྱོང་བའི་དུས་སུ་ཁྲུས་བྱའོ། །དཀྱིལ་འཁོར་ཆོ་གའི་གོ་རིམ་ལ་བསྙེན་པ་དང༌། དཀྱིལ་འཁོར་གྱི་སྤྱོད་པ་དང༌། བརྟུལ་ཞུགས་ཀྱི་རིམ་པ་དང་གསུམ་མོ། །དེ་ལ་དང་པོ་བསྙེན་པ་ནི། སློབ་དཔོན་དང་སློབ་མའི་འབྲེལ་བ་སྔོན་དུ་བརྟག་པ་གཅིག་གིས་བྱ་སྟེ། དེ་ཡང་གཉིས་པ་སྔོན་དུ་སོང་བས་སློབ་མ་ལ། སློབ་དཔོན་གྱིས་ཁྱོད་ཅི་འདོད་ནས་མཉེས་པར་བྱེད། ཅེས་དབུགས་དབྱུང་བར་བྱ་ཞིང༌། དེས་ཀྱང་རྗེས་སུ་གཟུང་བར་གསོལ་བ་བཏབ་ལ། དེ་ནས་སློབ་དཔོན་དེས་རིགས་ཀྱི་བུ་ཡོན་ཏན་ཐམས་ཅད་ཀྱི་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· + ས་ ) གཞིའི་ཆོས་འདུལ་བ་ལས་བྱུང་བའི་སོ་སོར་ཐར་པའི་སྡོམ་པ་ལ། ¡Ã གསུམ་ལ་ ( ¡¶ ཞ ¡·¡¶ ལུ ¡· - གསུམ་ལ་ ) སྐྱབས་སུ་འགྲོ་བའི་ཚུལ་གྱིས་བསྙེན་གཏམ། ཞེས་སྐབས་ཕྱེ་ལ། ཐོབ་ན་ཁས་ལེན་དུ་འཇུག མ་ཐོབ་ན་བླ་མ་ལྔ་བཅུ་པའི་སྣོད་དུ་རུང་བར་བྱ་བའི་ཕྱིར།

མཎྜལ་ལྔའི་ཚུལ་གྱིས་སྐྱབས་འགྲོ་དང༌། ཕྱིའི་སེམས་བསྐྱེད་བྱས་ལ། བླ་མ་ལྔ་བཅུ་པ་བཤད་ཅིང་སློབ་དཔོན་དང་སློབ་མའི་རྒྱུད་སོ་སོར་བརྟག སོ་སོའི་མཚན་ཉིད་དང་མི་ལྡན་ན་བཏང་སྙོམས་སུ་བཞགསོ་ ¡Ã སོར་ ( ¡¶ ཞ ¡· སོའི་ ) འབྲེལ་བའི་ཡུལ་དུ་གྱུར་ན་བསྙེན་པའི་གསོལ་བ་གདབ་པར་བྱ་སྟེ། བསྙེན་པའི་ཡོ་བྱད་ཐམས་ཅད་བསགས་ནས་སློབ་དཔོན་བདག་ཉིད་དབུ་རྒྱན་གྱི་ལྷ་དྲན་པའི་བར་དུ་ ¡Ã བསྒོམས་ལ། དེ་ལ་ཕྱི་ནང་གི་མཆོད་པས་མཆོད་ལ་ཇི་ལྟ་བ་བཞིན་དུ་བསྡུའོ། །ཡང་མདུན་དུ་མེ་ཏོག་ཚོམ་བུ་ལ་དབུ་རྒྱན་གྱི་ལྷའི་བར་དུ་བསྐྱེད་ལ། ( ¡¶ ཞ ¡· བསྐྱེད་ལ། ) དེ་ལ་ཡང་སྔ་མ་བཞིན་དུ་མཆོད་དོ། ( ¡¶ ཞ ¡· དེ་ ) །འཇིག་རྟེན་དང་འཇིག་རྟེན་ལས་འདས་པའི་གཏོར་མ་གཉིས་ཀྱང་དབུལ ༑ ཚོགས་བྱིན་གྱིས་བརླབ་ཅིང་གཏོར་མ་ལ་ཡང་ཕུད་དབུལ། གྲོགས་སུ་གྱུར་པའི་རྣལ་འབྱོར་པ་རྣམས་ལ་ཡང་སྔ་མ་བཞིན་མཆོད་ ¡Ã ཅིང་དམ་ཚིག་ ( ¡¶ ཞ ¡· དེ་དམ་ཚིག་པ་ལ་གཏོར་མ་ ) གཏང༌། ཚོགས་ཅུང་ཞིག་ལོངས་སྤྱོད། འཇིག་རྟེན་པའི་གཏོར་མ་བཏང་ལ། སློབ་མར་འོས་པ་རྣམས་དཀུག་ལ་མཎྜལ་དབུལ་བ་སྔོན་དུ་འགྲོ་བས། ཡོན་ཏན་ཐམས་ཅད་འབྱུང་བའི་གནས། །ཐབས་དང་ཤེས་རབ་བདག་ཉིད་ཅན། །སྟོན་པའི་བསྟན་པ་ལེགས་གནས་པའི། །བདེ་མཆོག་ཆོ་ག་བཤད་དུ་གསོལ། །ཞེས་ལན་གསུམ་པའི་བར་དུ་གསོལ་བ་བཏབ་སྟེ། བདེ་མཆོག་ཆོ་གའི་སྐབས་སུ་དགྱེས་རྡོར་ཆོ་ག་ཞེས་བྱ་བ་ཅི་རིགས་པར་བསྒྱུར། དེ་ནས་འཁོར་ལོ་མཆོད་པའི་རིམ་ཇི་བཞིན། །ཞེས་བྱ་བ་ལ་སོགས་པས་ཡོ་བྱད་ཚོགས་སམ་དྲི། དེས་ཀྱང་ཚོགས་ན་ཁས་བླང༌། ཧེ་རུ་ཀ་དཔལ་ཚུལ་འདི་ལ། ། ཞེས་བྱ་བ་ལ་སོགས་པས་དམ་ཚིག་བསྲུང་ནུས་སམ་དྲི། ནུས་ན་ཁས་བླངས་ཏེ། དེ་ལྟར་མཚན་ཉིད་དང་ལྡན་པས་ཡོ་བྱད་ཚོགས་ལ། དམ་ཚིག་བསྲུང་ནུས་ལ། སློབ་མ་བསྲུང་བར་བྱ་ཞིང་བསྙེན་པ་བརྩམ་སྟེ། བཟླས་བསྒོམ་རྣམ་པ་གཉིས་ཀྱི་སྒོ་ནས་བསྙེན་པ་རབ་འབྲིང་གསུམ་གྱིས་བསྙེན་ཏེ། རབ་མཁའ་འགྲོ་མས་ལུང་བསྟན་པ།

འབྲིང་རྨི་ལྟས་བསྟན་པ། ཐ་མ་སྔགས་ཀྱིས་གྲངས་ཚད་ཚང་བ་སྟེ་བསྙེན་པའོ། །དཀྱིལ་འཁོར་གྱི་སྤྱོད་པ་ལ་དོན་བདུན་ལས། དང་པོ་སའི་ཆོ་ག་སྟེ ༑ ས་བརྟག་ཅིང་གཟུང་བ་དང༌། བསླང་བ་དང༌། སྦྱང་བ་དང༌། བྱིན་གྱིས་བརླབ་པ་དང༌། བསྲུང་བ་དང༌། ས་ཡི་ལྷ་མོ་མཁྱེན་པར་གསོལ་བའོ། །གཟུང་བ་ལ་གཉིས་ཏེ། སངས་རྒྱས་ཀྱིས་ས་ ( ¡¶ ཞ ¡· - ས་ ) བཟུང་བ་དང༌། རང་གི་སེམས་ཀྱིས་གཟུང་བའོ། །དང་པོ་ནི་རང་བྱུང་གི་ཐིག་ལེ་ལ། ལྷ་སོ་སོའི་སྔགས་བཟླས་ལ་བཞག གཉིས་པ་རང་གི་སེམས་ཀྱིས་འདིར་བྱའོ། ། ¡Ã ཞེས་གཟུང་བའོ། ། ( ¡¶ ཞ ¡· - ཞེས་གཟུང་བའོ།། ) བསླང་བ་ལ་གཉིས་ཏེ། སྣང་བ་ལས་བསླང་བ་དང༌། མི་སྣང་བ་ལས་བསླང་བའོ། །སྦྱང་བ་ལ་གཉིས་ཏེ། ཏིང་ངེ་འཛིན་དང་ལས་དང་བྱ་བས་སྦྱང་བའོ། །བྱིན་གྱིས་བརླབ་པ་ནི་སློབ་དཔོན་གྱིས་རྐང་མཐིལ་དུ་སྣ་ཚོགས་རྡོ་རྗེའི་ལྟེ་བར་ཧཱུྃ་ལྗང་གུ་ལས་འོས་འཕྲོས་པས་བགེགས་སྐྲོད་པར་བསམ་ཞིང༌། གཡས་བརྐྱང་བ་ལ་སོགས་པའི་སྟང་སྟབས་ཀྱིས་བྱིན་གྱིས་བརླབ་པའོ ༑ ༑ བསྲུང་བ་ནི་སློབ་དཔོན་གྱིས་མཎྜལ་གྱི་དབུས་སུ་སྣ་ཚོགས་རྡོ་རྗེའི་སྟེང་ན་འདུག་པར་བསམ་ཞིང༌། སྙིང་གའི་ཧཱུྃ་ལས་ཁྲོ་བཅུ་འམ། མཁའ་འགྲོ་མ་བཅུ་ སྟེ ༑ སྒོ་མཚམས་མ་བརྒྱད། ཁཎྜ་རོ་ཧཱ་ ( ¡¶ ཞ ¡· ཧ་ ) གཉིས་སྟེ་བཅུ་སྤྲོ་བའོ། །དེ་རྣམས་ཀྱིས་ཕྱོགས་སྐྱོང་རྣམས་བཀུག་ནས། ཕུར་བུའི་སྟོད་ཁཎྜ་རོ་ཧཱ་ ( ¡¶ ཞ ¡· ཧ་ ) གཉིས་ཀྱི་གཟུགས་ཅན། སྨད་བསད་ཤིང་གི་རྩེ་མོ་ཅན་ཟུར་གསུམ་པ་དབལ་དང་བཅས་པ་སྙིང་གའི་ས་བོན་ལས་སྤྲོ་ ( ¡¶ ཞ ¡· གཏོང་ ) ཞིང༌། གྷཱ་ཏ་ཡས་གདབ ༑ མུཏྐ་རས་བརྡུང་ཞིང་བསད་པ་དང༌། ཡང་རང་གི་ ( ¡¶ ལུ ¡· + སྙིང་གའི་ ) ས་བོན་ལས་འོད་འཕྲོས་པ་དང༌། གནོད་མཛེས་རྣམས་བསྡུ། ཡང་ས་བོན་ལས་འོད་འཕྲོས་པས་དེའི་རྐྱེན་གྱིས་ཕྱོགས་མཚམས་ཀྱི་རོ་ཞུ་བ་ལས། ཕྱིར་རྡོ་རྗེའི་ར་བ། ནང་དུ་དྲ་བའོ། །སྟེང་དུ་གུར་དང་མདའི་དྲ་བའོ། །འོག་ཏུ་འོད་ཟེར་དང༌། ཨཿཚེག་དྲག་ཅན་ ¡Ã གཉིས་ཀྱིས་

( ¡¶ ཞ ¡· གྱི་ ) རྡོ་རྗེའི་ས་གཞིའོ། །སའི་ལྷ་མོ་ལ་མཁྱེན་པར་གསོལ་བ་ནི། བསྐྱེད་རིམ་སྔོན་དུ་སོང་བས་མདུན་གྱི་སྟེགས་བུ་ལ་བའི་རྣམ་ལྔས་བྱུག་ཅིང༌། དྲིའི་ཐིག་ལེ་ ( ¡¶ ཞ ¡· + གཅིག་ ) དང་མེ་ཏོག་ཚོམ་བུ་གཅིག་བཀོད་ལ། རང་གི་ལག་པ་གཡོན་པའི་སྲིན་ལག་ལ་ཧཾ་བསམས་ཏེ། ཧྃ་ཞེས་བརྗོད་པས་མཎྜལ་ལ་བསྣུན་ལ་སེ་གོལ་བརྡབས་ཏེ། དེའི་རྗེས་ལ་ལྷ་མོ་སྐུ་མདོག་སེར་མོ་རྒྱན་ཐམས་ཅད་ཀྱིས་བརྒྱན་པ། ཕྱག་ན་བདུད་རྩིས་གང་བའི་བུམ་པ་ཐོགས་པ། སྐེད་པ་ཡན་ཆད་ཐོན་ནས་བཀའ་ནོད་པར་བསམས་ལ། དེ་ལ་མཆོད་པར་བསམས་ལ། ཨོཾ་པྲྀ་ཐི་བཱི་དེ་ཝཱི་གྲིཧྣེ་དཾ་ཨརྒྷཾ། དེ་བཞིན་དུ་ངྷཱུ་པཾ། དཱི་པཾ། པུཥྤཾ། གནྡྷཾ། དེ་ནས་གཏོར་མ་དཀར་པོ་གཅིག་དབུལ་བར་བྱ་སྟེ། ཨོཾ་པྲྀ་ཐི་བཱི་དེ་བཱི་ན་མཿསརྦ་ཏ་ཐཱ་ག་ཏེ་བྷྱོ། བི་ཤྭ་མུ་ཁེ་བྷྱཿསརྦ་ཐཱ་ཁཾ། ཨུངྒ་ཏེ་སྥ་ར་ ¡Ã ཎ་ཨི་ ( ¡¶ ཞ ¡· ཧི་
